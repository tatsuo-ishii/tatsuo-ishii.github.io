PostgreSQL とはなんですか？
PostgreSQL は Unix で動く Relational Database Management System (RDBMS) です。学術用、教育用、個人用、商用など利用目的に関係無 しに無償で利用できます。 桑村さんのページにもっと良い紹介や、 FAQ の和訳があるので、是非そちらも御覧下さい。

pgsql-jp MLは現在日本PostgreSQLユーザー会 で管理していますので，そちらに行ってください．

PostgreSQL の最新ソースはどこから入手できますか？
本MLトップページの「関連URL」を御覧下さい。

PostgreSQL の日本語対応はどうなってますか？
基本的には、EUC コードが使えます。6.3.1 以降では、日本語 以外のコードも一部サポートしています。詳しくは ソースの中の、src/doc/README.mb.jp を御覧下さい。

SQL に関する良い解説書はありませんか？
「PostgreSQL関連の情報リンク-参考文献」にいくつか参考文献や リンクが掲載されています。

PostgreSQL のドキュメントはどこにありますか？ また日本語訳はありますか？
ドキュメントはソースの中の /usr/local/pgsql/doc/ にあります。 オンラインマニュアルは標準では /usr/local/pgsql/man/ にあります。 ややバージョンが古いものもありますが、日本語訳のあるものは 「PostgreSQL 関係の日本語ドキュメント」に掲載されています。
