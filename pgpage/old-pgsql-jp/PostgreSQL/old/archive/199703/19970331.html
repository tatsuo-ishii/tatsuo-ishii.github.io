「試した環境」
 SPARC/Linux と x86/Linux

・postgreSQL	v6.0	（/usr/local/pgsqlへ）
・kaffe		v0.83	（/usr/localへ）
があらかじめインストールされており、

setenv KAFFEHOME /usr/local/share/kaffe		
setenv CLASSPATH .:/usr/local/share/kaffe/classes.zip:/usr/local/share/kaffe
setenv LD_LIBRARY_PATH /lib:/usr/lib:/usr/local/lib:/usr/X11R6.1/lib

のような環境変数が設定されているとします。
		
次に、

http://www.tara-lu.com/~jimb/jsql

から

java-libpq.tar.gz

を入手します。そして、

% cd /usr/local/share/kaffe
% tar zxvf <hoge>/java-libpq.tar.gz

とします。

kaffeが、完全にJDK1.1(.1)をサポートしてないので
JDBC(SQL関係)のクラスファイルが含まれていません。そこで、

http://splash.javasoft.com/jdbc/index.html

から、
jdbc-0122_tar.Z

を入手します。そして、

% cd /usr/local/share/kaffe
% tar zxvf <hoge>/jdbc-0122_tar.Z
% ln -fs jdbc/classes/java .

します。すると、

> % kaffe jSQL -h localhost -d test
> Welcome to the PostgreSQL Java based sql monitor:
>   Please read the file COPYRIGHT for copyright terms of PostgreSQL
> 
>    type \? for help on slash commands
>    type \q to quit
>    type \g or terminate with a semicolon to execute query
>  You are currently connected to the database: test
> 
> test=>  \l
> datname   |datdba|datpath   
> ----------+------+----------
> template1 |   503|template1 
> test      |   503|test      
> bench     |   503|bench     
> regression|   503|regression
> (4 rows)
> 
> test=>  
>

となります。これでまとめられたでしょうか。
