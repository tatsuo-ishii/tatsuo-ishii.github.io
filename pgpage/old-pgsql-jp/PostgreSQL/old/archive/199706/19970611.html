やや古い文献ですが，奈良先端の吉川先生が ADBS'95 で Tutorial を
なさっています [1]．そちらを手がかりにされるのが良いのではと思います．

# なお，ADBS は情報処理学会が主催です．

INRIA の O^2 を使った研究[2]では，SGML element に対応する class を
作り，これによって SGML DTD を class の is-a/is-part-of 関係の
中にマップします．一度それをやってしまうと SGML 文書を格納する作業は，
element に対応する instance を作成し，SGML intance の
element 構造に合わせてそれらの instance をつなぎ合わせる作業と
なります．問合せは，path expression を拡張して自由度を持たせることに
よって対応していたように思います．
