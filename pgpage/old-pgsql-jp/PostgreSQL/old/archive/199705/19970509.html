私も知りません。で、「2.99」ってことは、ASCII 日本語 TeX 2.99 -
p1.0.9F か何かなんでしょうか？

p1.0.x は使ったことがないのでわかりませんが、日本語 TeX 2.99 - j1.x 系
列（多分 p1.x.x も同じ）は、default の状態のまま build すると、SJIS 版
が出来上がります。EUC 版を作る場合は、compile の前に EUCsetup スクリプ
ト（original の配布に同梱）を実行して patch を当てることになります。

処理形が想定している漢字 code は、ASCII pTeX 3.x p2.x.x では banner に
そのものズバリが表示されます。日本語 TeX 2.99 - j1.x では、「2.99 -
j1.7e」のように `e' がついていれば EUC 版、そうでなければ SJIS 版です。
