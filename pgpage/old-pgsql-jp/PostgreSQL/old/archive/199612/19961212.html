Hi...

	Just a few announcements.

	First off, in order to continue centralizing things under the
postgreSQL.org domain, there have been 6 new mailing lists setup to 
better server the community.  

	All the lists are @postgreSQL.org, and are as follows:

	questions
		- replaces postgres95@postgres95.vnet.com
		- general discussions from an end-user standpoint

	hackers
		- replaces pg95-dev@ki.net
		- developers discussion list (everyone welcome, of course!)

	bugs
		- bug reports, patches, etc
		- keep these off of the questions/hackers list as much as
		  possible

	announce
		- product/release announcements

	mirrors
		- central address for mirror admins

	committers
		- CVS repository commit messages/discussions

	All of the lists are subscribable by sending a "subscribe" request
to '<list>-request@postgreSQL.org', and all of the lists have a digest
associated with them.

	Also, all of the above lists have their digest files stored and
downloadable at:

	ftp.postgreSQL.org/pub/majordomo/<list>-digest

	Any problems with any of these lists should be reported to me
directly, at scrappy@postgreSQL.org.

------


		Upcoming Release of PostgreSQL v6.0
		===================================

	Well, the developers have implemented some pretty dramatic changes
in the server over the past several months.

	As a result, we have just frozen what we now consider to be a 'stable'
branch of the source tree in order that we can now test the server for Release.

	Current release date is set for the week of Jan 2nd.

	This *will* be a dump/reload release, and it is highly recommended 
that, once released, everyone upgrade to it, as it cleans up alot of the
bugs still existing in the v1.x releases, as well as dramatic improvements
to the vacuum *and* indexing areas of the server.
