G++ は GCC の distribution に含まれています。

  ./configure
  gmake bootstrap
  gmake install

とすると、default では gcc のほか g++ や Objective-C も install されま
す。ただし、g++ を正常に使えるようにするには libg++ や libstdc++ を 
install する必要があるかもしれません（いずれも GCC とは独立した 
distribution で、gcc と同じ場所から FTP できるはずです）。私の machine 
では、

  gcc-2.8.1
  libg++-2.8.1.1a
  libstdc++-2.8.1.1

が install してあります。

＃ lib〜 は、使用する gcc に対応した version のものを install する必要
＃ があります。上記のように、version 番号が同じ（ような）数字になって
＃ いるものがそうです。


>  upgrade kit は2.2.6がありませんでした(^^;;。まっさらにして
>  2.2.7を入れた方が早いかもしれませんね。

今からだと 2.2.8-RELEASE のほうがいいでしょう。

＃ CD-ROM が出るまでちょっと時間がかかるでしょうが‥‥。

とりあえず 現行のまま upgrade-2.2.5.tgz の中の bsd.port*.mk だけを取り
出して install してみてもいいかもしれませんが、これをしなくても error 
には（2.2.6-RELEASE の場合）なりにくいと思いますので、こちらはあまり気
にしなくていいかもしれません。
