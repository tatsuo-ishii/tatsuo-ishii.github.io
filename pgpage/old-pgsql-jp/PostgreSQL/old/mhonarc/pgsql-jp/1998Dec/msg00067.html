Mail を file に落とし、

uudecode filename

とすると postgresql.tar.gz という file ができます。これを適当な 
directory に展開し、そこで make します。

もし compile が始まらないようなら、

  ftp://ftp.freebsd.org/pub/FreeBSD/packages-stable/Latest/

にある upgrade kit を install するといいでしょう。


Compile error 自体は  PostgreSQL の distribution に含まれている header
file を定常に参照できていないことが原因でしょうから、configure が 
include path を正しく設定してくれていれば問題ないはずなんですが‥‥。
