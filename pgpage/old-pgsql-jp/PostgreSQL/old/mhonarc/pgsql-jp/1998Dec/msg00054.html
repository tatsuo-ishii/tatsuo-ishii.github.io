ようするに、WWWサーバマシン上に PostgreSQL をインストールせずに perl 
インターフェイス(pgsql_perl5)をインストールしたいということでしょうか。
「インストール」の意味にもよりますが、少なくとも /usr/local/pgsql/ 以
下があれば、pgsql_perl5 をインストールすることは可能です。つまり、

configure
make
make install

で /usr/local/pgsql/ 以下を作成し、ついで pgsql_perl5 をインストールす
れば良いと思います。

initdb や postmaster の起動は必要ありません。

>Pgだけのインストールで良い場合，WWWサーバにはpostgresのアカウントは必要で
>しょうか？Pgのインストールはpostgresでなければ動作しないでしょうか？よろ
>しくお願いいたします．
>
>環境は
>FreeBSD2.2.6Rです．お願いいたします．

postgres アカウントはいりません。どんなアカウント(root以外)でも結構で
すから、/usr/local/pgsql/ が作れればそれで良いのです。
