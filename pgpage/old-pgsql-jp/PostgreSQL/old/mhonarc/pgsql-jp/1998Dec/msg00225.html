エラーメッセージ

とエラーメッセージが出て接続できません。
この場合、どのような原因が考えられるのでしょうか？。タコ

下に今までやってきた過程を書きます

1. FreeBSDでの操作
1.1 postgres6.3.2(pached)をインストール

1.2 initdbでデータベース作成
 pg_hba.confの編集
 host all 0.0.0.0  0.0.0.0 trust

1.3 pgsql.sh でpostmaster起動
 pgsql.sh の内容
 #!/bin/sh
 [ -x /usr/local/pgsql/bin/postmaster ] && {
 su -l ppostgres -c 'exec /usr/local/pgsql/bin/postmaster
  -D/usr/local/pgsql/data
  -S -o -F -i > /usr/local/pgsql/errlog' &
 echo -n ' pgsql'
 }

1.4 ユーザー作成 (ユーザ名:hoge hogeはFreeBSDの側の既存のユーザ名)
 create user hoge

1.5 hogeでログインし、データベース testdb を作成
 create database testdb

1.6 テーブルの作成
 psql　testdb
 testdb=> create table test (a int,b char(20));
 testdb=> \q


2. Windows98での操作

2.1 ログイン(hoge でログイン パスワードはFreeBSDと同一)

2.2 ODBCドライバのインストール
 postdrv で 英語版ODBCドライバをインストール
 日本語版ODBCドライバをコピー

2.3 データソースの追加(ODBCデータソースアドミニストレータにて)
 ユーザーDSN画面で追加
 (詳細)
 Data Source:PostgreSQL
 Database:testdb
 Server:hoge01
 UserName:hoge
 Description:(無し)
 Port:5432
 Password:(無し)
