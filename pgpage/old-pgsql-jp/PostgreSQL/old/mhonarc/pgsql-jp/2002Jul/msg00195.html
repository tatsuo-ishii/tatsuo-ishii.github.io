森岡と申します。
環境が違いますが、私もうまくPerlの環境が構築できませんでした。
稲本さんと同じように過去の記載を参考にして下記の方法でするとできました。
ご参考になれば幸いです。

./configure --enable- .............
make
cd src/interfaces/perl5
make clean
export SRCDIR=./
export POSTGRES_INCLUDE=/usr/local/src/postgresql-7.2/src/interfaces/libpq/
export POSTGRES_LIB=/usr/local/src/postgresql-7.2/src/interfaces/libpq/
perl Makefile.PL
make
make test ← ちゃんとできているか確認をしました。
cd /usr/local/src/postgresql-7.2/src
make install
