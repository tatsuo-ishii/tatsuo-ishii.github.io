PostgreSQL のデータを Excel に取り込む際、カラム名を日本語に
変えようとして、はまってしまいました。

drop table test;
create table test(id int, name text);
insert into test values(1, 'aaa');

なるテーブルに対して MS Query から select * from test を行い、
結果が表示された後、フィールド名 id をダブルクリックして列見
出しに id1 と入力して OK を押すと、

  Error while executing the query;
  ERROR: parser: parse error at or near "'"

というダイアログが表示され、カラムタイトルの変更ができません。

直接 SQL を編集して

  select id as id1, name as name1 from test

などを実行しようとしても同様のエラーとなるので、日本語化云々
の話ではないのかも。どなたかご教授いただきたく。

        PostgreSQL ODBC Driver 日本語版（2001/07/13版）

        MS Excel 2002(10.2614.2625), MS Query 2002(10.2607.2625)
