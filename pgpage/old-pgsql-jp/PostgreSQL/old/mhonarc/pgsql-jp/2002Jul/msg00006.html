PostgreSQL 7.2において
以下のような場合、udpate での left join はできないのでしょうか。
m_adr
 name
 zip
 todofuken

m_zip
 zip
 todofuken

update m_adr set todofuken = m_zip.todofuken
  where m_adr.zip = m_zip.zip

MS Access 等では
update m_adr left join m_adr on m_adr.zip = m_zip.zip 
  set m_adr.zip = m_zip.zip
とできるのですが。
