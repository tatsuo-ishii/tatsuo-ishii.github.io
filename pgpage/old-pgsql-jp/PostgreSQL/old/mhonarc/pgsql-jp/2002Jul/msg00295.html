僕も先日、Apache,PHPをRPMでインストールしましたが、同じような現象に陥り
ました。
httpd.conf,やphp.iniを色々いじったのですが、だめでした。
違っているのかもしれませんが、結局RPMでインストールすると素のApache+PHPの
作業しかできないのではないでしょうか。
僕は、Apache+PHPはソースからインストールし直しました。

では、頑張って下さい。

===<Apache Configure>===
./configure \
"--with-layout=Apache" \
"--prefix=/usr/local/apache" \
"--enable-shared=ssl" \
"--enable-module=so" \
"--enable-rule=SHARED_CORE" \
"--enable-shared=rewrite" \
"--enable-module=ssl" \

===<Apache htttp.conf>===
LoadFile /usr/local/pgsql/lib/libpq.so　<-★ libphp3.soより前に書くのがみそ
らしい
LoadModule php3_module        libexec/libphp3.so

===<PHP Configure>===
./configure  \
 --with-pgsql=/usr/local/pgsql \
 --with-apxs=/usr/local/apache/bin/apxs
 --enable-versioning --enable-i18n \
 --enable-track-vars
