私の環境では、例えば

SET PGCLIENTENCODING=SJIS

の部分を、

SET PGCLIENTENCODING=hoge

などと書きかえた pgaccess.bat を実行すると、
'hoge'というClient encodingはサポートしていないというエラー
メッセージが表示されてDBへの接続に失敗します。
なので、この環境変数の設定が効いてるのは確かです。
