PHP/FI の出力でゴミが出てしまいます。
以下のような単純な、ソースでもおかしいです。

-------- ソース開始 ----------
<html><head></head><body>
<? echo "abcde">
</body>
</html>
-------- ソース終り ----------

-------- 出力結果開始 ----------
<html><head></head><body>
abcde
</body>
</html>
le>
-------- 出力結果終り ----------
最後の le> の部分がゴミです。 ただし、出力されてしまうゴミは、
一定ではなく、また全く出ない場合もあります。


PHP/FI のバージョンは、2.0b12  + 日本語パッチと GD 1.2 日本語パッチ
なし。
OS は、Linux 2.0.30, 因に Apache は、1.24 を使用しています。

どなたか、原因がわかる方、ご教示ください。
