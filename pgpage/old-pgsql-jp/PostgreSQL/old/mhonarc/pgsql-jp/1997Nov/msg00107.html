>>SQL92 の構文には CAST というのがあって、定数なら型変換できます。
>> select cast '0123' as int4
>　なるほど。でも定数だけなんですね。ううむ。

SQL92では
         <cast specification> ::=
              CAST ( <cast operand> AS <cast target> )
         <cast operand> ::=
                <value expression>
              | NULL
         <cast target> ::=
                <domain name>
              | <data type>
ですので、定数以外でもOKみたいですね。
