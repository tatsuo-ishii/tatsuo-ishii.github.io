Toshio Uchiyama <uchiyama@wakeup.it.matsue-ct.ac.jp> さんは書きました。

->	PostgreSQL のあるマシンで JAVA applet のデータベースアプリ
->ケーションを動かそうとすると
->
->	jdbc:postgresql://machine/database?user=username
->
->だとコネクトできないようです。その代わり、
->
->	jdbc:postgres:database?user=username
->
->でコネクトできます。

	以上、私が書きましたが、この症状がでる理由は、

	appletviewer JDBC_Test.html

のように httpd を通さずに applet を呼び出しているからです。
ローカルのマシンにデータベースがあっても、ちょっと面倒ですが、

	appletviewr httpd://localmachine/JDBC_Test.html

のように、httpd 経由で呼び出せば、

	jdbc:posgresql://localmachine/database?user=username

の URL が使えることが分かりました。
						以上です。
