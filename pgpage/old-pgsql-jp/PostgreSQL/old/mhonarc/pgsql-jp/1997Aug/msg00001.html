テーブルの継承について御教授下さい。

以下の様な流れで継承について確認しました。

１．create table test_a ( code varchar(5), name varchar(10));
２．データをいくつかINSERT。
３．create table test_aa ( id varchar(2) inherits( test_a);
４．\d test_aaで内容確認。

\dでtest_aaでテーブルの中身を見ると継承しているtest_aのカラムのLengthが全
て0になっているんです。     実際にtest_aaにINSERTしてみるとtest_aで定義し
ているLengthは無視？されているようで、指定Length以上INSERT出来てしまいま
す。     test_aa上で定義したカラムは問題無いようですが。
varcharをchar(n)に置き換えてやってみましたが、同じ状態でした。
