Tatsuo Ishii wrote:
> CREATE FUNCTION EN2CODE( TEXT ) RETURNS INT2
>   AS 'SELECT EN_CODE FROM AAA
>       WHERE EN = $1'
>   LANGUAGE 'SQL' ;
> 
> として、
> 
> select en2code( 'A123A' ), en_code from aaa;
> 
> で動きませんか？

　動きました。(^^;
en2code|en_code
-------+-------
     10|     10
     20|     20
(2 rows)

（引用順変えました。） 
> やりたいことが今一つ分かりませんが、

　先のテーブル"AAA"はマスターテーブルになってまして、
別なテーブルではEN(TEXT)やEN_CODE(INT2)で格納されて
いるのです。
　表結合用に副問合せをしたかったんです。あまり良くな
い使い方かもしれませんが。

　ありがとうございました。今後もビギナーな質問をする
と思います。よろしくお願いいたします。
