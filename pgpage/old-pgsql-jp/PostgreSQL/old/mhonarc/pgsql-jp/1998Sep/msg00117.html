中嶋です。

From: Naoe Ichiishi <ichiishi@mlg.co.jp>
> プログラム中で、あるテーブルの
> フィールド名とフィールドの型のみを取得したいのですが、
> 何か方法があるのでしょうか？
> ちなみに、使用するデータベースとテーブルは、
> プログラムのオプションとして指定されるため、
> 事前に知ることができません。
> SELECT文を発行して、
> PQfnameとかPQftypeとかで取得するしかないのでしょうか？

の通りです。他には、以下の関数がありますが、既におさらい済みで
しょうか。

results = PQexec(コネクション, SQL文);
タプル数 = PQntuples(result);
フィールド数 = PQnfields(result);
カラムの値 = PQgetvalue(result, ...);

また、PQftypeの結果としては、以下のスレッドを御覧下さい。

[pgsql-jp 4050] return value of PQftype function
