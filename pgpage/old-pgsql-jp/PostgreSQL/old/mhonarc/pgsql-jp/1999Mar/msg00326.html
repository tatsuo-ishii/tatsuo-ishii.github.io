Web サイトから入手できる、

    - Administrator's Guide for Informix Dynamic Server
    - INFORMIX-Universal Server Administrator's Guide

などでは、簡単な内部アーキテクチャを説明し、
それをもとにサーバの設定、チューニング方法を解説しているようです。

Version 7.x のサーバ (Informix Dynamic Server、IDS) は、
Illustora 買収前からの製品ラインなので、
直接の血縁関係はないとおもいます。

Version 9.x のサーバ (Informix Universal Server、IUS) が
Informix、Illustora の子供、PostgreSQL の親戚に当たるものでしょう。
「DataBlade API Programmer's Manual」というマニュアルも
ありますし。

上記 2 種類のサーバは、row デバイスにデータを格納していますが、
Linux 用の無償試供版が提供されている INFORMIX-SE という製品ラインは、
1 つの表につき 2 つの OS ファイルを作成し、データを格納しています。

データの格納法や動いているプロセスを観察してみると、
SE と PostgreSQL が一番近いようですが、
オブジェクトの機能と言う切り口では、IUS と PostgreSQL が一番近いと
言えるでしょう。
