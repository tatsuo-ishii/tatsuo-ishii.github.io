>>>>> On Wed, 10 Mar 1999 17:16:57 +0900, Tatsuo Ishii <t-ishii@sra.co.jp> said:

> 軽い話題です。LinuxWorld 99 での話のネタに使わせて頂きたいのですが、
> 「あなたはなぜ PostgreSQL を選んだのか？」と聞かれたらどう答えますか？
 
 ・ FreeBSD で動かそうとしたから
 ・ タダでソースがあるから
 ・ 日本語化パッチがあったから
 ・ SQL のサポートレベルがまともそうだったから
 ・ 日本語で読み書きできる ML があったから

ですかね。昔の Postgres は SQL じゃなくて独自の問い合わせ言語
だったので、あまり興味はなかったのですが…

いつだったかの日経バイトの記事を読んで、石井さんの Web ページを
見にいったら、その印象が良かったので。というのもあります。
実際に触れるようになってからは、

 ・ アクセスインタフェースが色々ある

というのも気に入ってます。実際にはそんな何種類も使うわけでは
ありませんが、用途に応じたインタフェースを選べるのがいいです。

最近は ODBC ドライバへの関心は個人的に薄くなってるんですが、
どこで必要が生じるかわかりませんから、あるというのはいいと思ってます。
Java からも扱えるようになったのは買ってます。

 ・ 取り扱いが簡単

というのもいいですね。どなたかもおっしゃってましたが、Oracle
なんかだと必ずしもそうではないですから。
まぁ、Oracle がああなっているのがマズいとは思いませんけど。
商用本格 DB としてはある意味当然かな？ と考えていますし。
