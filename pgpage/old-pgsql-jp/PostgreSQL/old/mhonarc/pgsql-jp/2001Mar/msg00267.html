    数値が９８０であるとして、
    select SERIAL from TABLE where VALUE BETWEEN 900 and 1000;  もダメです。

    select SERIAL from TABLE where VALUE > 900;     はOKです。

    〜以下の”＜”はダメみたいです。
    どのマニュアルや、ネットでの記事にも何も見つからないし。。。。
