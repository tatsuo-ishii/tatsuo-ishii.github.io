[pgsql-jp 20250] AUTO COMMIT を外す方法

  • From: Keiichi Takami <takami@mikigw2.paris-miki.co.jp>
  • Date: Wed, 07 Mar 2001 11:15:12 +0900 (JST)
  記事 [12686] Re: AUTO COMMIT の ON/OFF について
の中で、「ただし、JDBC を使うと、ドライバレベルで autocmmit on/off が
見掛け上出来るようになります。」

  というのがありますが、具体的にどのようにすればいいのでしょうか？

  サーバで Postgresql を使う予定にしておりまして、RMIServer から JDBC
を使って Postgresql にアクセスします。この時、RMIServer になんらかのエ
ラーが生じた時に、rollback 出来るように、auto commit を off にして、
明示的に commit, rollback を使いたいのです。
