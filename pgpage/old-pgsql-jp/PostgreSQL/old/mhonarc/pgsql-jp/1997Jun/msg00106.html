> /usr/sbin/install -c -m 555 postgres /export/home/postgres/pgsql/bin/postgres
> install: The -c, -f, -n options each require a directory following!
> make[1]: Leaving directory `/export/home/postgres/postgresql-v6.1/src/backend

Solaris の /usr/sbin/install は BSD 系統とはオプションが違うので
その関係でこのエラーが出ます．かわりに /usr/ucb/install を使えば
問題ありませんので，Makefile.global の中で，

INSTALL=/usr/ucb/install

と入れてしまうか，make INSTALL=/usr/ucb/install install として
実行するのが簡単かと思います．
