‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―
事例［１］ 学術情報処理センター利用者管理システムの紹介
‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐―‐―‐―
氏名： 江藤博文

所属・肩書：佐賀大学学術情報処理センター 助手

講演内容：学術情報処理センター利用者の情報を管理するために
          PostgreSQL を使用して作成したシステムを紹介します。

システム稼働時期または開発時期等: 平成11年度 

PostgreSQL Version: 7.0.3
OS: Solaris 2.6
主要開発言語: PHP

‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―‐―
事例［２］ 地図情報提供サービスDo-mapにおけるPostgreSQLへの移行の現状
‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐―‐―‐―
氏名：中西　一登、藤森　禎雄

所属・肩書：中西　一登　(株)ゼンリンデータコム　技術開発部マネージャ
　　　　　　　　・拠点案内技術グループ　チームリーダー
　　　　　　藤森　禎雄　(株)ゼンリンデータコム　技術開発部アシスタント
　　　　　　　　マネージャ・データベース技術グループ　チームリーダー

講演内容：
　本年4月にカットオーバしました地図情報提供システムDo-mapにおいては、
　従来はDBとしてOracleのみを用いていましたが、コスト削減等を目的として
　大部分をPostgreSQLに切りかえることにしました。
　その現状についてお話させていただきます。
 
システム稼働時期または開発時期等:
　改良システムのカットオーバは本年中を予定

PostgreSQL Version: 7.1.1
 
OS:　Linux Redhat6.2
 
主要開発言語:  非公開とさせていただいております。

‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐―‐―‐―
事例［３］ 社内連絡ＷＥＢシステム
‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐―‐―‐―
氏名：  伊原大輔

所属・肩書： システム開発部　( 株式会社システム総研 )

講演内容：　社内、社外に分散する従業員の相互連絡用に 
　　　　　　構築したＷＥＢシステムで　社内日報、WEBMAIL、
　　　　　　名刺箱の３つのサブシステムで構成した。
　　　　　　データベースの暗号化セキュリティをほどこし、外 
　　　　　　部からデータベースへアクセスされ、データを盗ま
　　　　　　れても、解読できないようにした。
  
システム稼働時期または開発時期等: ２００１．０５．０１

PostgreSQL Version: PostgreSQL　６．５．３
OS: 　　　　　　　RedHat　LINUX　6.2j
主要開発言語:　　PERL、DBI,　DBD

‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐―‐―‐―
事例［４］ AS/400からオープンソースの世界へ
           〜PHPソースコードジェネレータ：WebPG（仮称）〜
‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐−‐―‐―‐―
氏名：大井智幸、近藤直文

所属・肩書：大井智幸  プラットアイズ株式会社 本社開発部
            近藤直文  ソフト工房

講演内容：
IBM AS/400上での基幹業務システム及び BtoC、BtoBなどの Webシステムで
実績を上げてきた弊社が、オープンソース特にPostgreSQLとPHPを利用した
業務システムの開発を行うにあたって見えてきた問題点と課題。
そして、それを解決するために開発を始めた PHPソースコードジェネレータ
（WebPG）の目的と概要を紹介致します。

システム稼働時期または開発時期等: 2001年4月〜開発中

PostgreSQL Version: 7.0.3（7.1に対応予定）
OS: RedHat Linux 6.2J
主要開発言語: PHP4
