石井です．

> Anonymous CVSを利用してPostgreSQLのソースを取得したいのですが、
> ドキュメントを参照して以下の手順を実行しても認証で蹴られてしまい
> ます。いくら試しても同じ結果になるのですが、何故でしょうか？
> 
> 
> (参照したドキュメント)
>  http://osb.sra.co.jp/PostgreSQL/Manual/PostgreSQL-7.1-ja/cvs.html
> 
> (手順)
>  $ cvs -d :pserver:anoncvs@postgresql.org:/home/projects/pgsql/cvsroot login
>  Logging in to :pserver:anoncvs@postgresql.org:2401/home/projects/pgsql/cvsroot
>  CVS password: (実際には表示されないが postgresql と入力)
>  cvs login: authorization failed: server postgresql.org rejected access to
>  /home/projects/pgsql/cvsroot for user anoncvs

最近CVSのリポジトリが移動したせいでしょう．

cvs -d :pserver:anoncvs@anoncvs.postgresql.org:/projects/cvsroot login
cvs -z3 -d :pserver:anoncvs@anoncvs.postgresql.org:/projects/cvsroot co -P pgsql

でやってみて下さい．
