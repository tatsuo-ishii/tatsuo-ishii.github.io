川崎　勝 wrote:

> はじめまして、最近メーリングリストに参加した川崎と申します。
> 早速ではありますが、primary keyの設定が分かりません。やりたい事は次の
> 通りです。
>
> すでに作成してあるテーブルのカラムをprimary keyとして宣言する
>
> と、言うないようです。テーブル作成時にprimary keyに設定すれば問題ないの
> ですが、すでに膨大な量のデータが入っています。
> 初歩的なことですが、どうかご教授よろしくお願いします。

残念ながら正式な手段はまだありません。　
ですが実際的には次のようにすれば十分だと思います。

　テーブル名_pkeyという名のunique indexを該当カラム
　に作成する。

たとえば
# create table tbl (id int4 primary key);
を実行すると
NOTICE:  CREATE TABLE/PRIMARY KEY will create implicit index 'tbl_pkey' for
table 'tbl'

というメッセージを出力して自動的にtbl_pkeyというunique indexが作成
されます。　自動でなく手動でこれをすることによりprimary keyの機能
はほぼ満たされることになります。
