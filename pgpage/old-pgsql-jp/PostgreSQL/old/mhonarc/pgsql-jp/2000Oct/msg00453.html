以下ソース。
ほとんどしまさんのそのままです。
「test1」はデータベース名。
bash-2.02$ cat testdb.c
#include <stdio.h>
#include "postgres.h"
#include "libpq-fe.h"

main(){
    PGconn *con;
    PGresult *res;

    con = PQsetdb("localhost", "5432", NULL, NULL, "test1");
    if(PQstatus(con)!=CONNECTION_BAD){
        fprintf(stdout,"OK!\n");
        exit(1);
    }else{
        fprintf(stdout,"error!\n");
        exit(1);
    }
    PQfinish(con);
}

以下コンパイルのコマンドライン
bash-2.02$ cat README.testdb
gcc -o testdb.cgi -I/usr/local/pgsql/include -L/usr/local/pgsql/lib -lpq testdb.
c

コンパイルはとくに何事もなく終了。
bash-2.02$ sh README.testdb
bash-2.02$

結果もとくに問題なし。
bash-2.02$ ./testdb.cgi
OK!
bash-2.02$
