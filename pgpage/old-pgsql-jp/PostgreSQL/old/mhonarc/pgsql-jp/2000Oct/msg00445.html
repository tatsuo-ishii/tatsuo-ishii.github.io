[pgsql-jp 18406] usogres-0.0.6 公開

  • From: Hosokawa Tetsuichi <hosokawa@good-day.co.jp>
  • Date: Sun, 29 Oct 2000 03:20:41 +0900
細川です。

Realtime Backup Utility Usogres for PostgreSQL の 0.0.6 を
http://usogres.good-day.net/ にて公開開始しました。

今回の修正は、新しめの gcc と glibc の組合わせでは、「うそ
ぐれす」がコンパイルできないという問題点に対処しました。

具体的には、Plamo 2.1 で発生していた問題への対処です。

# で、うそぐれすのサイトの方では上記の問題に対処するための
# Plamo 2.1 用の gcc295.tgz 修正版をうそぐれすとあわせて公
# 開しています。
# Plamo で c++ な人にはうれしいパッケージかも。(^^;)

ではでは。
