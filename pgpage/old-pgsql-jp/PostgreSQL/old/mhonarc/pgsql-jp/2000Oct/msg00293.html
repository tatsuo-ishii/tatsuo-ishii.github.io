At 11:50 +0900 2000.10.19, Minoru Iida wrote:
> ERROR:  Bad timestamp external representation '2000-09-25 22:01:60.00+09'

Postgres 7.0.2 + FreeBSD 4.1 の環境ですが、私も同じような現象に出くわしまし
た。

  create table foo (
    key       text,
    time_ins  timestamp default timeofday()
  );

のような定義の表に

  insert into foo (key) values ('aaa');

としてレコードを登録したところ、秒の桁が 60 になったことがあります。
うちでは jdbc は使っていないので、他の部分のバグではないかと思います。
