SELECTして得た結果を、配列(@row等)に入れたいのです。
サンプルを実行しても、列ごと取り出しているので、
ちょっと使い方が異なります。

具体的には、
------
use CGI;
use Pg;
use CGI::Carp qw(fatalsToBrowser);

my $dbname = $query->param('dbname');
$conn = Pg::connectdb("dbname=$dbname");
my $qstr = "SELECT * FROM tkd;  ";
$result = $conn->exec($qstr);
$n = $result->ntuples;              # 件数の取得

for ($i=0;$i<$n;$i++) {                              ┐
    print "$result->getvalue($i,0),";# 列Aのとりだし ｜
    print "$result->getvalue($i,1),";# 列Bのとりだし ｜問題の個所
    print "$result->getvalue($i,2)"; # 列Cのとりだし ｜
    print "<br>\n";                                  ┘
}
------
の問題の個所のところを、
------
for ($i=0;$i<$n;$i++) {
    @row = $result->getvalue; # かんな感じで
    print "@row<br>\n";
}
------
として、$row[0]に列Aを、$row[1]に列Bを入れたいのです。
こんなかんじのことできますでしょうか？
