羽田と申します。

postgreSQL 6.5.1 で以下のようなことをやりたいのですが、
SQL文がわかりません。

以下の二つのテーブルがあります。
table_a
----------------------
id   name
----------------------
1    dog
2    cat
3    ls
4    cd
5    select

table_b
----------------------
attr  attrname
----------------------
100   animal
101   unix
102   sql

table_c
----------------------
id    attr
----------------------
1     100
2     100
2     101
3     101
4     101
5     101
5     102

上記で以下のような SQLを作成して検索をさせています。
select c.id , a.name from table_a a, table_c c
    where c.attr in (100:101) 


--------------------------------
id     name
--------------------------------
1      dog
2      cat
2      cat
3      ls
4      cd
5      select

この表の重複した行を削除したいのですが。

select c.id , a.name from table_a a, table_c c
    where c.attr in (100:101) 
    group by c.id

とやるとエラーになります。
このような場合どういうSQLを書けばよろしいのでしょうか？
