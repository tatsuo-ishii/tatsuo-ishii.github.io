小菅昭一さん、いなむらさん
「select した結果をすぐinsert」ではありがとうございました。

さて、
サーバー　LinuxPPC R5　PostgreSQL6.5.2
クライアント　Win98　VB6.0　ODBC

クライアント側のVB
mySQL = "insert into t1 values ('みかん', '冬');"
cn.Execute mySQL, , adcmdtext

これで問題なくinsertできるのですが、

hi = "みかん"
ki = "冬"
mySQL = "insert into t1 values (hi, ki);"
cn.Execute mySQL, , adcmdtext

とすると、変数が存在しないと言うエラーが出ます。
変数を使ったinsertはどのようにすればよいのでしょうか。
