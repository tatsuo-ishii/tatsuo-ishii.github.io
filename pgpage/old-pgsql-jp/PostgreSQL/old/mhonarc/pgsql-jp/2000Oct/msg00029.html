155ページの例を参考に下記のような
PL/Pgsql文を作成したのですが
select 文を実行すると
NOTICE:  plpgsql: ERROR during compile of pg_test near line 5
ERROR:  parse error at or near "w"
と表示されうまくいってないようです
これは漢字のカラムがいけないのでしょうか？

またdeclare宣言で漢字のテーブルを宣言する場合
はどのように宣言するのでしょうか？
w   家賃台帳%ROWTYPE  と書いてみたのですが
うまくいかないようです

よろしくお願いいたします。
kiyoto@sun-net.ne.jp

------------------記----------------------
create function pg_test() returns int as '
declare
    w    bank%ROWTYPE;
begin
    for w in select * from bank loop
       raise notice ''銀行名称: %'', w.銀行名称;
     end loop;
    return 1;
end;
' language 'plpgsql';

PS　bank　テーブルには銀行番号と銀行名称のカラムがあります。
