初めて投稿します。しまと申します。

自分の環境にPostgreSQLをインストールしてinitDBをして
テーブルをテスト的に作成してみたりしております。
今度はCのプログラムでDBにアクセスし、値を取得及び
変更・削除などをやってみたいと思っておりますが
コンパイル時にエラーが出てしまい、どうしても判りません。

どなたかご教授お願いできませんでしょうか。
大変困っております・・・・

以下にコンパイル時に表示されるエラーを記載します。
＃Cのソースファイル名はtestdb.cです。

-----ここから--------
Undefined                    first referenced
 symbol                          in file
PQstatus                     /var/tmp/ccbrbbfd.o
PQfinish                     /var/tmp/ccbrbbfd.o
PQsetdbLogin                 /var/tmp/ccbrbbfd.o
ld: fatal: Symbol referencing errors. No output written to testdb.cgi
collect2: ld returned 1 exit status
-----ここまで--------

また、自分が書いたCのプログラムを以下に記載します。

----ここから-----
#include <stdio.h>
#include "postgres.h"
#include "libpq-fe.h"

main(){
    PGconn *con;
    PGresult *res;

    con = PQsetdb(NULL, NULL, NULL, NULL, "test");
    if(PQstatus(con)==CONNECTION_BAD){
        fprintf(stdout,"error!\n");
        exit(1);
    }
    PQfinish(con);
}
----ここまで-----

ローカル（自分のホームディレクトリ配下）でCのプログラムソース
を作成し、GCCでコンパイルしようとしています。
PostgreSQLは/opt/local/pgsqlにインストールしております。

インクルードとして使用するpostgres.hとlibpq-fe.hは
/opt/local/pgsql/includeの中にありましたのでコンパイル時には

gcc -o testdb.cgi -I/opt/local/pgsql/include testdb.c

と指定しました。

宜しくお願いします。
