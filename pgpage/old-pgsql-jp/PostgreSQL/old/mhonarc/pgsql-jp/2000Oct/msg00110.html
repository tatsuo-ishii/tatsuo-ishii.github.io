[pgsql-jp 18066] ACCESS で登録したデータ

  • From: "Sekinemasahiro" <masa_s@pb.highway.ne.jp>
  • Date: Wed, 11 Oct 2000 11:48:02 +0900
関根です。　初めて投稿させていただきます。

昨日の夜からpostgresをコンパイルして、ようやく動くように
なりました。最初は7.0.2をインストールしていたのですが
initdbが実行できなく、6.5.3では実行できたのでこちらを使っています。

簡単なテーブルをpsqlで作成し、データを一件insertしました。
create table testtbl ( hinmei text, nedan int, hinban int );
create  UNIQUE  INDEX testtbl_ix  ON  testtbl;
insert into testtbl values ( 'isu',1000,1 );

上記を実行後、windoowsからMS-Accessで接続し、データが1件登録
されているのを確認できました。Accessからデータを二件登録し、
psqlでselectするとAccessで登録したデータが表示されません。
Accessを何度か再起動して、テーブルを開いてみると3件登録
されているのですがpsqlからでは見れません。
同ようにPGACCESSでも見てみましたが、同じでした。

ACCESSで登録したデータは
　　　　'table', 10000, 2
         'table2', 9800, 3です。

環境　　Server :Laser5Linux + postgres6.5.3
           client:   Windows2KPro + MS-Access2000  ODBCは日本語パッチ適用。

なぜLINUXではAccessで登録したデータが見れないのでしょうか？
ご存知の方、教えて下さい。
