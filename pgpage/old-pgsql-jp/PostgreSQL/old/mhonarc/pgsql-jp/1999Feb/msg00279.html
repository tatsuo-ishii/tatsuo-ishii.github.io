kaz> います。

PHP3 が PostgreSQL の header file,library を要求するために起きてしまう
現象なんでしょうね。

ちょっと adhoc な手段ですが、以下のような手順をとるとどうなりすか？

1. PostgreSQL を compile するなり、動作しているところから持ってくるな
りして library と header file を手元に準備する。
2.(もし shared library にするなら) ldconfig が読んでくれる path に 
library を copy する。
#標準的な library path にしておくと後々楽だと思います。
3.header file を /usr/local/include/postgres とかに copy
4.PHP3 の Makefile の include file を規定しているところに 
-I/usr/local/include/postgres を追加する。必要なら library path も同様
に追加しておいてください。
5.make

#もっとも私はまだ PHP3 は make していないんですが X-)
