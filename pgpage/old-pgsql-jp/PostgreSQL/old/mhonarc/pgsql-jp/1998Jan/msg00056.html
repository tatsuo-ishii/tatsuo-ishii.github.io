校は一橋大学経済学部、専門は計量経済学ですが、はっきりいって4年間全く勉
強せずに来たので経済学の事はまったく分かりません。お陰で今卒論を書く段階
になってとても困っています(笑)

2〜3年のころは某研究所でプログラマーとしてバイトしたり、学校でパソコン
ルームの管理員をやったり、Digital UNIXのサポートをやったりしてました。3
年の夏頃に、このデジタル革命になんとかして便乗したいとケチな考えをおこ
し、ネットで募集した仲間と会社を起こしました。

性別は男性です(笑)生年月日は1975/03/31で今は22才、趣味はジャズ演奏です。
今でもジャズ研でドラムとギターをやっております。音楽にはジャンルを問わず
興味があります。

会社の関係で、データベースとHTML、Javaの連携に非常に興味があります。
今は主にPHPと、JDBCによるアプレットからのデータベースの接続の実験をして
います。

会社で、お金がないSOHO・中小企業向けに、今売っているLinuxサーバにイント
ラネット的なアプリケーションを入れて安く売ろう、というケチな計画を立てて
おり、開発にはmysqlを使っていたのですが、円安のあおりをうけてmysqlのライ
センス料が2万円を超えたため、他のデータベースも検討しよう、ということで
PostgreSQLをインストールして使ってみている段階ですが、なにぶんよくわから
ないのでいろいろ教えていただきたいと思い、このMLに入会いたしました。

若輩者なのでいろいろと失礼もあるかと思いますが、今後ともよろしくお願いい

ところで、PHP3.0b3がリリースされたようですね。
If you're still scared off the beta tag, don't be.  The code seems to be
stable enough for prime time
と言う事なので、安定性はかなり高いのでしょうか。
どなたかすでにインストールしてお使いの方はいらっしゃいますか？

それで、社内ML向けに
http://www.php.net/changes.php3
を半分和訳してコメントを加えたものを作成したのですが（卒論書きからの現実
逃避の産物です）、もしかしたら誰かのお役に立てるかなと思い、
http://www.hde.co.jp/~ogura/php/changes.php3.html
にこれを公開しました。

1時間で書いたためかなりいいかげんなところもありますが、3.0の新機能がどん
なものか、2.0とどうちがうのか、ぐらいのことは分かると思います。ただ僕も
実際にまだ3.0を使っていないので、もしかするとまちがいがあるかもしれませ
ん。まちがいがあったら指摘してください。

それから、日本でPHPについて議論されているMLというのはあるんでしょうか？
もしどなたかご存知であればメールで教えていただければ幸いです。
