 OSやコンパイル条件の問題かと思っていましたが、
 どうやらPostgreSQLの問題のようですね。

 テーブルの中に可変長の配列を持つことには、
 まだまだ問題があるのでしょうか。
 
 ちなみに count は view でも正しく動作しています。

 create table test(a int4[2]);  
 insert into test (a) values('{1,2}');  
 insert into test (a) values('{3,4}');
 create view test_v2 as select count(test.a[1]) as count1 , count(test.a[2]) as count2 from test;
 select * from test_v2;
count1|count2
------+------
     2|     2
(1 row)

 ひとまず、配列を使わないで対処します。
