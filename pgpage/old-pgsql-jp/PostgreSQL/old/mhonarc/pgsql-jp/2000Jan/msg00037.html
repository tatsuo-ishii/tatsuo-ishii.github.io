> を作り、rc.M から 起動するように設定したのですが、 PGDATA が見つからない
> とpostmasterが起動できません。

  #!/bin/sh
  POSTGRESDIR=/usr/local/pgsql
  PATH=$PATH:POSTGRESDIR/bin # Add
  PGDATA=POSTGRESDIR/data    # Add
  PGLIB=POSTGRESDIR/lib      # Add
  export PATH PGDATA PGLIB   # Add
  if [  -x $POSTGRESDIR/bin/postmaster -a -d $POSTGRESDIR/data ]; then
       rm -f /tmp/.s.PGSQL.5432
       echo "Starting PostgreSQL postmaster..."
       su postgres -c "postmaster -S -i"
  fi

とすればよいのでは？ su -l とするという方法もあるかも。
