こんにちは、海外＠筑波大です。

例えば下のようなSQL文でテーブルを作ったとき、

Create Table sample(	ID INTEGER PRIMARY KEY ,
			name VARCHAR(64) ,
			tel VARCHAR(16) );

上のテーブルに、

Insert into sample(name,tel) values('警察','110');

のような要領でカラムを追加したときに、
自動的に適切な ID が割り当てられるようにしたいのですが、
これはどうすればよいのでしょうか？


以前使っていた MySQL というシステムでは、テーブル生成の時に

Create Table sample( ID INTEGER AUTO INCREMENT PRIMARY KEY,.......
                                ^^^^^^^^^^^^^^
とすれば、ユニークな番号を振ってくれたようなのですが、
どうやらこれはMySQLローカルなオプションっぽく、
PostgreSQLで試したところエラーをお返しになられました。

こういうことをしたい場合、PostgreSQLではどのようにすれば
よいのでしょうか？
