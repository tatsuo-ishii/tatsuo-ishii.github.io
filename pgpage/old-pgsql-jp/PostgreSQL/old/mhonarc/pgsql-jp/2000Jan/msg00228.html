  山田＠町田市です。

  select 'today' + '1 day'; 

  は出来る事が分かっていたのですが、必要があり

  select '2000-3-1'::datetime - '1 day';

  を試みたのですが、減算の場合には明示的に interval を付けないとダメですね。

  select '2000-3-1'::datetime - '1 day'::interval;

ちなみに、上の答えはちゃんと２月２９日になっていましたので、Y２Kクリアでした。
