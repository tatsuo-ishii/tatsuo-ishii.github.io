t-ishii> せん。Solaris のドキュメントでは、「一度IBM拡張文字に変換してから日本

そのものずばりではありませんが、
XKP(http://www.xkp.or.jp) の Windows NT拡張漢字処理仕様書第2.1版 
5.1.2 重複符合文字の対応関係 より、 SJIS -> UNICODE -> SJIS の変換を
行ったときの対応関係から NEC選定IBM拡張漢字を抜き出すと、

NEC選定     IBM拡張
---------  ---------
ed40-eeec  fa5c-fc4b
eeef-eef8  fa40-fa49
eef9       81ca
eefa-eefc  fa55-fa57

となるようです。

もちろん ed40-eeec, fa5c-fc4b は SJISとして有効な領域だけです。
(2バイト目が 40-7e, 80-fc)
