postgres95-1.01 JP patch README				1996/3/25

このパッチは、フリーな RDBMS(Relational Database Management System)
postgres95 の最新版 1.01 で日本語正規表現を使った検索(ただし、EUC コー
ドのみ)を可能にするためのものです。パッチ本体は、大西雅晴さん
(onishi@nova.co.jp)作成され、石井が README を付け加えました。

なお、EUC での正規表現検索の部分は、谷本孝浩さんが fjで発表された「MB
extension v1.02 for GNU grep-2.0 (1/1)」
(<DOHZONO.93Jul13110121@up904.sdsft.kme.mei.co.jp>)の中からregex.[ch]、
mbc.[ch] を借用して実現しています。もともと、postgres95 は何もしなくて
もデータには EUC が使えたのですが(attribute などは駄目) 、これによって 
~ オペレータを使った text や char16 の日本語正規表現による検索が可能と
なっています。

使い方：

0. postgres95-1.01 を入手します。
   ftp://ftp.jaist.ac.jp/pub/dbms/postgres95/postgres95-1.01.tar.gz を
   ftpします。約 2MB ほどあります。また、bug fix パッチも入手した方が
   良いでしょう。
   ftp://ftp.jaist.ac.jp/pub/dbms/postgres95/www/pglite5.html を参照し
   て下さい。

1. パッチは tar+gzip+uuencode されています。解凍すると、
   pg95-1.01-jp.patch というディレクトリが作られ、その下に README
   (このファイル）と jp.patch というファイルができるはずですので、
   postgres95-1.01 のソースを適当なディレクトリに展開した後、そこに行っ
   て

        % patch -p < jp.patch

   とします。

2. SunOS JLE を使っている場合は、src/Makefile.custom を修正します。
   す。

	# uncomment this if you use SunOS JLE
	#CFLAGS += -D_WCHAR_T

   この２行目 #CFLAGS... の最初の # を取り除きます。

3. 以上です。後は普通にインストールするだけです。
