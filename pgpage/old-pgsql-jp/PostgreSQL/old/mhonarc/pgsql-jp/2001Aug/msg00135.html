ソフト工房の近藤です。

bytea 型の列をソートするにはどのようにすれば
よいかお教えください。

下記のように ORDER BY に指定するとエラーになるし、
text, bit などに CAST もできません。hexa か、octal
に変換してくれるような関数でもあればいいのですが、
見つかりませんでした。

$ select tgargs from pg_trigger order by 1; 
(-403)ERROR:  Unable to identify an operator '<' 
for types 'bytea' and 'bytea'
You will have to retype this query using an explicit cast

$ select tgargs::text from pg_trigger order by 1; 
(-403)ERROR:  Cannot cast type 'bytea' to 'text'

なお、ORDER BY をつけなければ、次のような出力になります。

tgargs                                                                  
------------------------------------------------
<unnamed>\000bbjud\000cmst\000UNSPECIFIED\000juid\000cmst_id\000  
 

Version は、PostgreSQL 7.0.2  です。
