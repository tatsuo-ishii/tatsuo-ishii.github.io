# お互い、名寄せには苦労しますね。

私は、漢字で名寄せすることが多いので、やはり、実際の
入力値(若干の整形はします)と、検索用の列と、シノニム
辞書を持つようにしています。

異字体( '亜' と '亞' )は勿論、独自の類似文字( カタカナ
の'ニ' と 漢数字の '二' 等号の '＝' 等々)の統一変換
や、シノニム辞書( 'NEC' '日本電気' 等々) を使用して
名寄せに使ってます。
