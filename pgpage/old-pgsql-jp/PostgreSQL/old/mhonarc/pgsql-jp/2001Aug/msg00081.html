> 残念ながら oid2name は標準ではインストールされません．
> postgresql-7.1.2/contrib/oid2name/ に
> ソースがありますので，ここで make すると
> 同じディレクトリに oid2name というコマンドができます．
> 使い方は以下のようになっています．
> 
> Usage: pg_oid2name [-d database [-x] ] [-t table | -o oid] 
>         dafault action        display all databases
>         -d database           database to oid2name
>         -x                    display system tables
>         -t table | -o oid     search for table name (-t) or
>                                oid (-o) in -d database
>         -H host               connect to remote host
>         -p port               host port to connect to
>         -U username           username to connect with
>         -P password           password for username
