数理計画 田中靖宏です。

7.0.2で使えるかどうかは検証していませんが、6.xの頃に「関数を
使って外部結合」という方法をとったことがあります。
変更が面倒なのが欠点。

create function get_sex(int4)
returns varchar
as 'select 性別 from サブ１
 where 顧客番号 = $1'
language 'sql';

create function get_occupation(int4)
returns varchar
as 'select 職業 from サブ２
 where 顧客番号 = $1'
language 'sql';

create view 顧客情報 as
select メイン.*
,get_occupation(メイン.顧客番号) as 職業
,get_sex(メイン.顧客番号) as 性別
;

で、select * from 顧客情報;
を実行するわけです。

こんなふうに、関数を自分で定義できるってのはいいですよね。
もっと便利な使い方とかありそう。
目から鱗な関数の使い方があったら、教えてください。
