[pgsql-jp 23891] MSQRY32 のページ違反

  • From: "Tadashi Nakamura" <tn_mls@hotmail.com>
  • Date: Sun, 2 Dec 2001 16:07:45 +0900
こんにちは　PostgreSQL 7.1.3 を Linux 2.2.19 ( Plamo 2.2.1 )
に　Install し、ODBC を介して、Excel からデ−タの入出力を
やろうとしています。
PostgreSQL の Table を Excel や MS Query 上に表示することは
できたのですが、レコードの編集、追加、削除ができません。

MS Query で、Table を表示し、レコードに対して
「編集の許可」を、Check On にして、数値などを
MS Query 上で訂正、編集し、他の行をクリックしたときに
「MSQRY32 のページ違反」と Error Message が出ます。
モジュール：PSQLODBC.DLL となっています。

ODBC Driver は、
http://www.interwiz.koganei.tokyo.jp/software/PsqlODBC/index.html
で、Download したものです。この Site の FAQ にありますように
データソースオプションの［OID Options/Show Column］と
［OID Options/Fake Index］をともにオンにしても状況に変化は
ありません。皆様の方ではいかがでしょうか。

参考
Windows 98 SE + IE 6
MS Office 2000 ( SR1 も SP2 も適用していません）
