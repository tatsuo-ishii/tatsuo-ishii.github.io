こんにちは
私もturboファンです。
6.1はどうだったか定かではないのですが、TuboLinux Server6.5の場合ですと、
suにて

/etc/rc.d/init.d/postgesql start

とするとイニシャライズまで、上記スクリプトが面倒を見てくれまして
めでたく起動します。

私も昨年まで6.1であれこれしていましたが、結局今の結論を申し上げますと、

6.5を使うべきだと言うことです。
いろんな面で設定が楽ちんだからです。

Tubolinux Server 6.5 スターターキット　（株）ラトルズ　1380円
がお勧めです。
