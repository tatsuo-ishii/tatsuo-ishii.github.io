copyコマンドについては、まつうら様と同じ使い方をしていますが、加えて次のような
手順で取り込んでいます。

１）pg_exec ($con,"create temp table 一時テーブル名・・・・");
２）pg_exec ($con,"copy 一時テーブル名 from '/hoge/huga/csvfile名' using
delimiters ','");
３）pg_exec ($con,"insert into 本物テーブル名 select * from 一時テーブル名");

取込データで痛い目を見たことがあって直接本物テーブルに読むのが怖いからです。
こうしておくと、一時テーブル上で整形するのも億劫になりませんし、結果的に安全に
本物を更新できるような気がしてます。
