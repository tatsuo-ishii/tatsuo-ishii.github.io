ヒズミです。

> みねまつ です
> 
> どのようなcreate indexを発行されたか 併記されますと、より多くの回答が
> 得られると思いますが、併記されないとコレと同じような回答が多数続くと
> 思われます。

以下のようなINDEXを発行しました。
create index index名 on table名(column1, column2);
column1:char(5)
column2:char(6)

ERROR:  cannot create index名
と返ってきます。

ここでindex名を変更すると成功します。

drop index index名;
とすると
ERROR:  pg_ownercheck: class "index名" not found
となります。
