「as is」という言葉がわからない方も多いのではないでしょうか。
「as is」は、「見たとおりのままに」ということですが、要するに、
タイプしたコマンドを何らかの方法でメールの文面にコピー＆ペース
トしてくださいうことです。コピー＆ペーストする方法がわからなけ
れば、タイプしたコマンドを一言一句間違えないように写経するわけ
ですが、ここでミスをする初心者も多かったり...

# 本当に上記のコマンドを発行したということもありうるかも。
# create table名 がうまくいくんだったら
# create index名 もうまくいきそうな気もするなあ....

hotta=# select version();
                           version                           
-------------------------------------------------------------
 PostgreSQL 7.2b2 on i686-pc-linux-gnu, compiled by GCC 2.96
(1 行)
hotta=# create table table名(column1 char(5), column2 char(6));
CREATE
hotta=# create index index名 on table名(column1, column2);
CREATE
