みなさん、はじめまして。小松仁と申します。
今日（99/11/4）からお世話になります。

PostgreSQL とのきっかけは、仕事の関係で
今年の５月から自分でInternetサーバの運用を始め、
ＤＢを使ったコンテンツを作りたいと思いいろいろと
リサーチしました。
元々私はＷｉｎｄｏｗｓ系デスクトップアプリの開発を
行っており、開発ツールとしてＤｅｌｐｈｉを主に使って、
ＤＢはＤｅｌｐｈｉとの親和性が高い（同じベンダということで）
ＩｎｔｅｒＢａｓｅを使っていました。
InternetサーバはLinuxで構築しており、フリーのＤＢとして
ＩｎｔｅｒＢａｓｅ４やSYBASE SQL SERVERなどがありますが、
ＩｎｔｅｒＢａｓｅ４は同時セッション数の制限（たしか４セッション）
があり、SYBASE SQL SERVER はＯＤＢＣドライバがない
（フリーではない？）ため、また将来的なサポートのことも
考えてこれらは使わないことにしました。
そこでいろいろと話題になっている PostgreSQL を
使うことにしました。
PostgreSQL は８月の終わりごろからこのＭＬの
オーナーでもある石井達夫さんの本を参考にして使いはじめ、
使ってみると高機能なことに感激しています。

ということで、これから PostgreSQL のファンの１人として
更なる発展を期待しています。
長くなってしまいましたが、みなさんよろしくお願い致します。
