初めまして、大川と申します。

ただいま、データベースの移植と再構築を行っているものです。
「シーラカンス本」には随分お世話になっております。

早速ですが、教えて下さい。
「シーラカンス本」の例にもありました。'current' と 'now' の件で、
思うように 'current' データが登録されません。

create table t1 (d date);
insert into t1 values ('current');

としても、「シーラカンス本」のように 'current' と表示されません。
その状態で、１日経過しても日付は昨日の状態で、あたかも 'now' のような
振る舞いをしています。
そこで、create table t1 (d date); の 'date' を 'datetime' とかの時間を
含めたフィールドタイプにすると 'current' とうまく表示してくれます。

環境変数等の不足でしょうか？、コンパイルのミスでしょうか？
コンパイルは、ワーニングが少し出た程度なのですが、、、

どうかご教授願いますよう、お願いいたします。
