ちなみに 3.2-RELEASE ならば PostgreSQL 6.4.2j が入っていますから

# cd /usr/ports/japanese/postgresql
# make install

で終わりです。initdb も自動でやってくれますからあとは createuser して
createdb すればもうデータを投入できます。6.5系が使いたければ日本語化
されていませんが 6.5.1 が入っていたと思います(6.5かも?。それでも

# cd /usr/ports/databases/postgresql
# make MULTIBYTE=EUC_JP install

だけです(6.5系は使ってないので他にpatchが必要か未確認ですが

ぜひ ports/packages を使いこなせるようになってください
