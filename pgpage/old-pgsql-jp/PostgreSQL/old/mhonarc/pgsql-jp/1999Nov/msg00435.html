'in', 'except' 勉強になりました。

ところで、同じ結果を得るのに何通りかの方法がありますが、
--たとえば、--
A B 両方にあるレコード
select f from A, B where A.f = B.f;
select f from A in (select f from B);
A だけにあるレコード
select f from A except select f from A;
select f from A not in (select f from B);
