渋谷です。

pgsql-jp, PHP-jp 両方読まれている方はご了承ください。

Linux 用 PHP-3.0.12_jp_beta4 の RPM を作成しましたので、メールします。
動作確認は Vine Linux 1.1 で行っています。
RedHat 5.2 系であればそのままインストールできると思います。
http://alpha.or.jp/VinePlus/
からリンクをたどってください。
(一応 VienPlus 用に作っていますが、確認事項があってまだ登録はされてません。)

インストール時の注意点として、Vine に入っている Apache は
--disable-rule=WANTHSREGEX 付きで作られていますので、このオプションを
外したものに入れ替えないと regex 関係でエラーがでます。
ホームページ上に同じく置いてありますので rpm -Uvh してください。
ほとんどの RedHat 系では同じ物が入っていると思います。

この RPMS は RedHat 6.1 のものをベースにしていますので PostgreSQL 用の
モジュールが別になっています。ですので、MySQL 等の環境があれば別途、
それようのモジュールを外付けで作成できるようになっています。

一応の動作確認は取っていますが、使ってみて不具合等ありましたら連絡いただけ
ればと思います。

Cobalt の不具合対策版の RPMS も置いてありますので、必要な方はお使いください。
