福田というものです。今回はODBC についての質問です。

Windows98 クライアントからFreeBSD上にあるPostgreSQL にADOを使ってアクセスし
ようとしています。PostgreSQL ODBC Driver と片岡さんのページから日本語版をダ
ウンロードして使い始めたのですが、ある一台のクライアントから接続しようとする
と

	実行時エラー:3706
	指定したプロバイダが見つかりませんでした。

というエラーメッセージがでてうまくいきません。

ソースとしては
    Set cnMain = CreateObject("ADODB.Connection")
    cnMain.Open ("PostgreSQL")  -> この行で止まっているようです。
    rsMain = cnMain.Execute("SELECT * FROM user")
    以下続く...
のようになっています。

ODBC の設定とこのソースで別の2台のWindowsクライアントからはうまく接続できる
ので、そのクライアントの環境が他のと違うのかとも思いましたが見当がつきませ
ん。(もしそうならこのメーリングリストにきくべきではないのですが)

後、違うのは PostgreSQL ODBC Driver をインストールするときに

    1 Do not install Driver Manager
    2 Install Driver Manager
    3 Install Driver Manager(with version check)

と３つの選択枝がありますが、うまくいっていっているクライアントにインストール
したときはデフォルトのまま 1  を選択したのですが、うまくいかないものは 2 を
選択しました。一度Driver をアンインストールして再インストールして、そのとき
は 1 を選択してみたのですが結果は同じエラーメッセージがでます。

今のところどうしたらよいかわかりません。もしわかる人がいたらよろしくお願いし
ます。
