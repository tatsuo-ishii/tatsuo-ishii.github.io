最初と最後ってなんのことでしょうか？。

SELECT * FROM hoge ORDER BY foo ASC LIMIT 1; で、最初のデーターを
取り出したのなら、ORDER BY の ASC(昇順)を DESC(降順) に変えれば
昇順ソートの最後の1件、即ち、降順ソートの最初の1件が取り出せます。

# ご存知でしょうが、RDB では、データーの格納順序とか、列の並び
# 順自体には意味が有りません。何らかの順番に取り出したいときは、
# ORDER BY で、ソート順を指定する必要があります。
