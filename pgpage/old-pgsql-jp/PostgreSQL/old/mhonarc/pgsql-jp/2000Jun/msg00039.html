こんにちは、中野です。
伊藤さんレスありがとうございました。

> > まず、テーブルの構成ですが
> > create table hoge(
> >  hoge_id       serial  primary key,
> >  hoge_array    int4[],
> > );
> > という構成であるとします。
> >
> > ここで
> > $a = array(0, 1, 2, 4, 7, 15);
> > として
> > 配列$a[]の内容をすべてhoge_arryaの中に入れたいのですが
> > insertのやりかたがいまいちよくわかりません。
> > できれば取り出しかたも教えていただけるとうれしいです。
> >
> implode(), explode() はどうでしょう?
>
> $a = array(0, 1, 2, 4, 7, 15);
> $str = "{" . implode($a, ",") . "}";
>
> で，$strを，hoge_array にinsert．
>
> 取り出しは，hoge_arrayをselectして，
> 両端の{}を取り除いてから，explode()にて配列に分割．
>
> 文字列の場合，「{}"'」などのエスケープをどうするかという
> 問題はありますが，数字だけならこれでいけると思います．
>
> 他にもっとスマートなやり方があるかもしれませんが．．．

教えていただいたやり方でやってみました。
エラーがでました。それは配列に入れる時に'で囲まないといけない
ということでした。
そして取り出しもうまくいったのですが思ったことがありました。
これってデータベースは配列である意味がないのでは？
ということです。
ためしに
create table hoge(
  hoge_id       serial  primary key,
  hoge_array    text
 );
で同じようなことをやったらできました。
結局元のほうはhoge_array[1]だけにstringとして入っていた
ということだと思います。
でも結局やりたいことができたのでよかったと思います。
伊藤さんありがとうございました。
