佐藤と申します。

現在、Vine Liuux2.0 + JSP + Servlet + JDBC + PostgreSQLの環境で
サーバアプリケーションを作成しています。

JSPの文字コード：ISO-2022-JP
PostgreSQL：６．５．３（EUC_JP）

この環境でブラウザからアクセスしDBへの登録を
おこなうとほんの”一部”の漢字が文字化けを起こします。
デバッグでみてみると、DB内でＳＱＬを処理する際に文字化けを起こしている
ような感じです。
（SQL文を確認すると、正しく漢字がセットされている）
例）
１：insert into table values ('本店') → ×「”謀”の漢字に化けます。」
２：insert into table values ('支店') → ○
