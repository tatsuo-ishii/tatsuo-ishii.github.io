FUNCTIONS
       The functions have been divided into three sections:
       Connection, Result, Large Objects. For details please read

ってありますよね。
# PGRES_TUPLES_OKはFUNCTIONじゃないといえばそうなんですが^^;

postgresパッケージに付属のドキュメント
 postgres/libpq-chapter18058.htm
によると


 PGRES_COMMAND_OK,       /* the query was a command returning no data */
 PGRES_TUPLES_OK,        /* the query successfully returned tuples */

で、
 If the result status is PGRES_TUPLES_OK, then the routines
 described below can be used to retrieve the tuples returned
 by the query.
と、行を返すSELECTが成功したら PGRES_TUPLES_OK で、その他の、
行を返さないようなsqlが成功したら PGRES_COMMAND_OK となると。
