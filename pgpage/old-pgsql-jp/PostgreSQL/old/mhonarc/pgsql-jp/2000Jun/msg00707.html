> 以下のコマンドを実行してみて、プロセス postmaster が起動できるか
> 試してみてはいかがでしょうか ?
>
> # /etc/rc.d/init.d/postgres start
>
> このコマンドで起動できれば、デーモンとして起動できると思いますけど...

早速やってみました。
$/etc/rc.d/init.d/postgres start
Starting postgres: postmaster []
このようになり起動してそうな感じなのですが、実際には起動してないようです。

$ps ax
で見つかりません。

$ postmaster -S -i
その後にこうやると起動します。
