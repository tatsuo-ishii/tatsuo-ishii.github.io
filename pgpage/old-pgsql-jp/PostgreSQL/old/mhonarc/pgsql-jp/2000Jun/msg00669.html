こういうのは ML なんかを調べても出てきません。SQL の本でも買って
SQL を勉強しましょう。

pgsql-jp で、SQL 文が話題に上る場合もありますが、それは複雑な例とか、
PostgreSQL での動作が標準 SQL と違うとか、書き方に差があるとかいう
話題が出るだけです。

質問内容について言えば、この程度の事は、SQL の入門書にも書いて
ありますし、pgsql-jp は SQL の基礎の勉強をするための場でもないはずです。

 select name, sum(kosuu) from lunch where day >='05-21-2000' and
    day <= '06-23-2000' group by name;
