佐野です。
皆様、いつもご教示ありがとうございます。
下記のtable(hit)カルク名は、dateでファイルタイプは、datestampです。
既存のデータに対して、 + '1month'の時は、成功しますが、- '1month'
の時には、下記の通りエラーになってしまします。
１ケ月前等の減算をする方法をどなたか、ご教示お願い申しあげます。



abc=> select * from hit;
date
----------------------------
Mon Jun 05 11:46:04 2000 JST
(1 row)

abc=> select date + '1month' from hit;
?column?
----------------------------
Wed Jul 05 11:46:04 2000 JST
(1 row)

abc=> select date - '1month' from hit;
ERROR:  Bad datetime external representation '1month'
