PL/pgSQLのインストール ..createlang:
/usr/local/pgsql/plpgsql.soというファイルが見つかりません。

>The file contains the call handler for PL/pgSQL.By default,only PL/pgSQL is
>built and installed;other languages must be explicitly enabled at configure
>time.

このファイルはPL/pgSQLへの呼び出しハンドルを含んでいます。
何もしなければPL/pgSQLだけが作成され、インストールされます。
他の言語はconfigure時に無効にされているはずです。

>To install PL/tcl,make sure the option --with-tcl is given to configure,then
>recompile and install.failed

PL/tclをインストールするにはオプションに--with-tclを指定して
configureし、再コンパイルしてください。(？install.failedがよくわからない)
