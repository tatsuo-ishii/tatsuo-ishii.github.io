普通の Java の本や JDBC の本を買ってくれば十分でしょう。
と、いうか Web ページなんかより書籍の方がよほど情報量が多いです。

# 個人的には、Web サイトの情報と言うのは最新ではない事や、間違いが
# ある事もあるため、それだけに頼ろうとするのはどうかと思います。
# Web サイトの情報は、しょせん二次情報だというくらいの割り切りが
# 必要であって、情報ソースとしてはマニュアルや書籍などが基本でしょう。

PostgreSQL の JDBC ドライバ特有の話などは、PostgreSQL のドキュメントに
書いてあります。したがって、それを読めばよいわけです。
