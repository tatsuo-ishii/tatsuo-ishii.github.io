Pg は使っていないので推測ですが、上の場合は、$i が '' で囲まれているので
展開されませんね。"" で囲むとどうでしょうか？

>  Use of uninitialized value at XXXX.cgi line 56. 
> と言ったようなエラーがいっぱい出てしまいます。

こちらは Perl で構文チェックを厳しくしている時、たとえば
  #!/usr/local/bin/perl -w
のように -w をつけていたり、
  use strict;
で stirict 指定をつけている場合に良く悩まされますね。

初期値を設定していない変数をいきなり参照するとこうなります。
値が渡されない可能性があるものに関しては代入時にデフォルト値を
セットしておくなどしないと、結構この warning はうるさいです。

でも、cgi に限らず、-wT と use strict はなるべく使ってチェック
することをおすすめします。
