> PostgreSQL に限らず、一般にどの DBMS でもこういう変更はできないのでは？

標準のSQLがどうなのかは知りませんが、Informix なら可能です。
（変換できない型もあるとは思いますが…）

参考までに Informix の ALTER TABLEの構文を書いておきます。

ALTER TABLE <表名>
	{ ADD (<新列名> <新列のデータ型> [NOT NULL]
		[UNIQUE [CONSTRAINT <制約名>]] [BEFORE <旧列名>] [, ...])
	| DROP (<旧列名> [, ...])
	| MODIFY (<旧列名> <新列のデータ型> [NOT NULL] [, ...])
	| ADD CONSTRAINT UNIQUE (<旧列名> [, ...])
		[CONSTRAINT <制約名>]
	| DROP CONSTRAINT (<制約名> [, ...]) } [ ...]

もちろん RENAME TABLE や RENAME COLUMN も可能です。
