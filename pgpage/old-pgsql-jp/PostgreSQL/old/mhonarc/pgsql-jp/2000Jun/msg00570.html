最近、似たような質問があったと思いますが。

> date型で誕生月を検索するところで悩んでおります。
> 
> ++++++++++++++++++++++
>     Table "birthday"    
> ++++++++++++++++++++++
>  Attribute | Type 
> -----------+------
>  name      | text 
>  birth     | date 

6月生まれを検索するのであれば
select * from birthday where date_part('month', birth)='6'
で検索できると思います。
