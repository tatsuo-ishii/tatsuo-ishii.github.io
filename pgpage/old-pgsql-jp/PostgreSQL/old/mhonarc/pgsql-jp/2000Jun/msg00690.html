高橋と申します。

ＰＨＰ３を使って簡単なページを作成してみたのですが、
データの削除が正常にできたかどうかの確認方法がわか
らず困っております。

ＩＤとＰＡＳＳＷＤを入力したら消すという処理なので
すが、ＤＥＬＥＴＥコマンド自体は成功してしまうので
エラーの取り方がわからないのです。
(推測なのですが、psqlでいうところのDELETE　0　と返
ってきてるんだと思います。)
実際にデータの削除が成功したかどうかはどうやって
判別したらよろしいのでしょうか？
(psqlでいうところのDELETE　1　という風に)

　よろしくお願い致します。


--
   /* delete */
   $sql = "delete from info where c_nm = '$c_nm' and c_pass = '$
c_pass'";

   $result = PG_Exec ( $conn ,$sql );
   PG_Exec ( $conn ,"commit" );
   if (!$result)
   {
      echo "insert error \n";
      exit ;
   } else {
      echo "正常に削除されました。<BR>";
      echo "登録ＩＤ：";
      echo "$c_nm<BR>";
      echo "登録パスワード：";
      echo "$c_pass<BR>";
   }
--
