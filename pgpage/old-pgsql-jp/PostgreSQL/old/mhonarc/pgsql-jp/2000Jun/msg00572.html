山本＠ｆｓｃと申します。

> date型で誕生月を検索するところで悩んでおります。
> 
> ++++++++++++++++++++++
>     Table "birthday"    
> ++++++++++++++++++++++
>  Attribute | Type 
> -----------+------
>  name      | text 
>  birth     | date 
> 
> > select * from datest where birth between '06/01' and '06/30';
> ERROR:  Bad date external representation '06/01'
> 
> のようなエラーが出ます。
> どのようにすると6月生の人を抽出できるでしょうか。
> それとも別のコラムに年を持たせないといけないでしょうか。

上記の例だとPstgreSQLでは

select * from datest where date_part('month',datetime(birth))='06';

※PostgreSQLだと'06'のところ'6'でも抽出できました。

ポイントとしてはこれらの日付型関数はdate型ではなくdatetime向けに
作られているので何かしらのキャストが必要になるようです。
