majordomo@sramhe.sra.co.jp に body に

index postgres95

と１行だけ書いてメールを送ると、入手できるファイルのリストが送られてき
ます。次に同じアドレスに、たとえば、

get postgres95 pg95-1-50.tar.gz.uu

と送ると、pg95-1-50.tar.gz.uu が入手できます。（これは、ML のアーカイ
ブの 1 から 50 までのメールを tar+gz+uuencode したものです。）他には、
FAQ-j.html(FAQ の日本語版)、*.txt はユーザマニュアルの日本語版（βテス
ト中）などが入手できます。本当は README を置いておくべきなのですが、今
日のところは時間がないのでこれにて失礼。
