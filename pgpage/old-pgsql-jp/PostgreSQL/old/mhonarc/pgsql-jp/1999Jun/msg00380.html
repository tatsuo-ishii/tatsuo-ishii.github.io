>さっそく、表題の件ですが、
>sql文を発行して、ヒットした結果の1レコード目から20レコード目までとか
>50レコード目から75レコード目まで、等を出力させる方法があるのでしょうか？

バージョン6.5であれば、offset と limit を使うのが簡単です。

select * from hogehoge limit 20;	-- 1から20レコードまで表示
select * from hogehoge limit 25 offset 49;	-- 50番目のレコードから
						-- 25レコード表示

ちなみに、offsetは0スタートです。

ただし、これは標準のSQLではないので、標準SQLだけでやりたいのならcursor
を使うことになります。
