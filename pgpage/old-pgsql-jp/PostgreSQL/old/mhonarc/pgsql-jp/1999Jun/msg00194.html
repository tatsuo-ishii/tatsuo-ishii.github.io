> インストールするときに、インストール内容についていくつか選択肢が
> 出てきたと思いますが、このときに "Do not install Driver Manager" を
> 選択しておかないと ODBC Administrator が英語版のものに置き換えられて
> しまいます。

  "Do not install Driver Manager"を選択した場合、条件（インストール済みのド
ライバマネージャのバージョン）によってはPostgreSQL ODBC Driver自身のインス
トールが却下されます。そのような場合には、"Install Driver Manager (with
version checking)"を選択するしかありません。運が悪ければドライバマネージャが
英語になります。

# 私は"Install Driver Manager (with version checking)"の選択を推奨しておりま
す。
