  postgresql-v6.5beta3 が出ていましたので、こちらで bench してみました。
展開するとディレクトリが、pgsql ではなくて postgresql-v6.5 となっていました
ので、もうほとんど 6.5 だと理解していますが、、、。

  regress は int8 が文句を云って来ませんでした(Happy!!)。float8 と
geometry で fail です。

  今回は、PPro x 2 のマシンを 6.4.2 で使っているので、6.4.2 との比較が出来て
いませんが、6.5 が正式にリリースされたら、そちらを入れ換えて再度計って見ます。(
早くなったような気もしますが、、、)


環境：
  FreeBSD-current (5/17)
  P6@200Mhz x 1
  128MB DRAM
  4GB UW SCSI.

コンパイル：
  ./configure --with-tcl --with-perl --with-odbc --with-mb=EUC_JP
     --enable-locale
  問題無くコンパイルできました。

regress test:
  boolean .. ok
    :    :
  float8 .. failed    <- overflow error
    :    :
  geometry .. failed  <- LSD の値の違い
    :    :
  euc_jp .. ok
