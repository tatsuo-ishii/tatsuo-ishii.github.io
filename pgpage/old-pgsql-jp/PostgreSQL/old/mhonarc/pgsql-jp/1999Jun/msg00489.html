堀田％初心者の味方＠諫早市です。

psqlj-6.5.diff.gz をリリースしました。これは psql コマンドのヘル
プやメッセージの一部を日本語化したものです。こんな感じ。

template1=> \h select
コマンド: select
説明: タプルを取り出す
書式:
        SELECT [DISTINCT [ON <属性N>]] <式1> [AS <属性1>], ...<式N>
        [INTO [TEMP] [TABLE] <クラス名>]
        [FROM <fromリスト>]
        [WHERE <制限句>]
        [GROUP BY <グループリスト>]
        [HAVING <having句>]
        [ { UNION [ALL] | INTERSECT | EXCEPT } SELECT ...]
        [ORDER BY <属性1> [ASC|DESC] [USING <演算子1>], ...<属性N> ]
        [FOR UPDATE [OF <クラス名>...]]
        [LIMIT <カウント> [OFFSET|, <カウント>]];

    （ヘルプの意味がほとんど分かってないというのは内緒 :-）

http://www.remus.dti.ne.jp/~sim/postgres/index.html#PSQLJ65
に置いてます。いちおー regression test も通したので大丈夫でしょう。

以前どなたかから、バックエンドからのメッセージも日本語化して欲し
いというつぶやきがあったような気もします。でもそれをやってしまう
と regression test が悲惨なことになりそうなので、手を出せません。
