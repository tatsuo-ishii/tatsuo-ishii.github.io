ftp.gnu.org でも、ftp.iij.ad.jp でもどこでもいいですけど、
GNU make のソースを持ってきて、それを展開して同梱されている
ドキュメントを読みましょう。
IRIX は触ったことがないので断言はできませんが、ドキュメントに
したがって作業を行えば、GNU make のインストールはできるでしょう。

逆に老婆心から言えば、この程度のこともできないようでは
PostgreSQL をソースから make しようなんて考えない方が幸せです。

PostgreSQL は flex も要求してたかと思いますが、IRIX 標準の lex で
通るかどうかは私は知りません。
bison の方は bison でなくともたいていの UNIX 系 OS の yacc で
通るかとは思いますが。
