postgresql.confの設定は
silent_mode = on
debug_level = 1 #range 0-16
debug_print_query = true
syslog = 2 # range 0-2
syslog_facility = 'LOCAL0'
です。

./configureのオプションは
--enable-locale --with-CXX 
--prefix=/usr --with-perl --enable-multibyte 
--with-tcl --with-odbc --enable-syslog 
--with-python --with-openssl 
--with-krb5=/usr/kerberos --sysconfdir=/etc/pgsql 
--mandir=/usr/share/man
--docdir=/usr/share/doc 
--includedir=/usr/include/pgsql 
--datadir=/usr/share/pgsql
です。

syslog.confの設定は
local0.*          /var/log/pgsql
です。
