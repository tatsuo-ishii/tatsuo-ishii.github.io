あくまで、暫定的なものとして考えました。
2案ありますが、 2番で十分かと。


１．	PHP/FI が出力する HTML のページの中に、
	タブ区切りで 抽出結果を表示する。
				~~~

	具体的には、<table><tr><th><td> を使用する。
	罫線はつけない。
	
	ページを訪れた人が、必要な部分を コピーして
	エクセルに張り付ける。

２．	PHP/FI が出力する HTML のページの中に、
	タブ区切りまたは カンマ区切りで 
	抽出結果を出力する。
		  ~~~
	
	タグを使用するわけではありません。
	ページは データの ラレツに見えます。

	もしあんまりなら、<pre> を使用してみる。

	ページを訪れた人は、
	「表示」-「文書のソース」などとして
	HTML ファイルを表示する。
	必要な部分を コピーして
	エクセルに張り付ける。

	「名前を指定して保存」してもよい。
