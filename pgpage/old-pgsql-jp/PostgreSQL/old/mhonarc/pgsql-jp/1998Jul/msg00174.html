やらせたいことは、著者名から書名を検索したいのです。
下の場合はTomという著者名から該当する本の書名を検索しています。
直接走らせるとuserはuenoと返って思ったとおりの結果も得られます。
ブラウザーからだとnobodyとなり、結果が得られません。

#!/usr/bin/perl -w

use Pg;

print "Content-type: text/html\n\n";
print "<html>\n";
print "<center>**************Literatur DATA BASE******************</center>\n ";

$conn = Pg::connectdb("dbname = ueno");
$result = $conn->exec("SELECT buch_title FROM literatur where autor = 'Tom'");

$user = $conn->user;
print " User = $user \n";
$num=0;
while(@row = $result->fetchrow){
my $value = $result->getvalue($num,0);
print "$value\n";
$num++;
}

print "</html>\n";
