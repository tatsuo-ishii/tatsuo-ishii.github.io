竹元と申します。

主キーの無いテーブルを作成したあとで、
あるカラムを主キーとして、再設定したいのですが、
そのような方法はありますか？

ALTER TABLEではCHECKはできますが、主キー設定は
ダメなようです。（実装されてない？）


RedHat6.2J + postgresql-7.1-1を使用しています。

-----------
testdb=# create TABLE t1 ( c1 int4 );
CREATE
testdb=# \d t1
           Table "t1"
 Attribute |  Type   | Modifier
-----------+---------+----------
 c1        | integer |

testdb=# ALTER TABLE t1 ADD PRIMARY KEY ( c1 );
ERROR:  ALTER TABLE / ADD CONSTRAINT is not implemented for that constraint type.
testdb=# ALTER TABLE t1 ADD CHECK ( c1 > 0 );
ALTER
testdb=# \d t1
           Table "t1"
 Attribute |  Type   | Modifier
-----------+---------+----------
 c1        | integer |
Constraint: (c1 > 0)
------------
