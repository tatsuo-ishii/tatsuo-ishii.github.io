> のほうが、Indexが聞くし全件なめるわけではないので早い、と言われ
> ました。

誤解する人がいるかも知れないのでフォローしておくと，PostgreSQL では

	select count(*) from pg_class where relname = 'テーブル名'

の形でも index は効きます．また，relname には unique index がはってあ
るので，「全件なめる」こともありません．したがって，

select count(*) from pg_class where relname = 'テーブル名'

のままで PostgreSQL では十分速いということになると思います．
