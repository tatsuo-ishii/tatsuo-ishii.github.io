ただ意外なことに、v7.1 も v7.2 も傾向としては CPU 数が増えても
query の結果は速くはならず、逆に遅くなる方向でした。
// CPU 数を 1, 2, 4, 6, 8 と変化させて計測したので、間違いない
// とは思うんですが…

勿論、環境にも依存するとは思うのですが、個人的には SMP で頑張る
よりも、爆速な 1 CPU で回す方が効果が高いのではないかと思ってい
ます。
