...略...

UNIQUE 制約をつけたいだけなら、CREATE UNIQUE INDEX 。

PRIMARY KEY として設定したいならば、データーを退避
してから、DROP TABLE , CREATE TABLE で、表を作りな
おして、退避データーを再ロードするしかないです。

ALTER 文では、PRIMARY KEY の追加/変更はできません。
