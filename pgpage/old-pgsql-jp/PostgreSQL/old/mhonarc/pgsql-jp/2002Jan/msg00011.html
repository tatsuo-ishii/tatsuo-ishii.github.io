> explain select count(*) from pg_class where relname = 'tjm';

>     Aggregate  (cost=2.02..2.02 rows=1 width=4)

Oracleで試す時間がないのですみませんが、これを見ると問題なさそう
ですね。
Oracle講習会では有る無しの判断はCount(*)を使うと遅くなるので使う
べからず、と言われていたのでそのまま信じていました。
実際は、Oracle講習会で推奨していた
cursol w_cur 
 is select relname from pg_class where relname='tjm' ;
open w_cur ;
fetch w_cur ;
if w_cur%NOTFOUND then 無いよ。
else                   有るよ。
close w_cur ;
としています。
