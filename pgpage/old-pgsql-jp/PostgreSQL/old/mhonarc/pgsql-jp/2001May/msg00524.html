徳家です。
> > データベースをOracleから，PostgreSQLに移行しようと考えています．
> > Oracleでシーケンスを利用しており，最大値を「9999999999」で設定
> > していたので，そのままPostgreSQLのシーケンスとして利用しようとした
> > ところ，最大値を越えているため設定できなかったのですが，シーケンスの
> > 最大値を変更する方法はありませんでしょうか?
>
> CREATE SEQUENCE test_seq
> 	start 1
> 	increment 1
> 	maxvalue 9999999999
> 	minvalue 1
> 	cache 1
> 	cycle;
>
> CREATE TABLE test (
> 	cd int8 DEFAULT nextval('test_seq') NOT NULL
> );
>
> のようにシーケンスおよびテーブルを作成するとテーブルtestのcdには
> 9999999999まで使用することができます。

シーケンスは、PostgreSQLの場合4バイト整数なので、
2147483647が数値的最大値です。

よって9999999999だとオーバーしてしまって設定できません。

# 8バイト化してくれると嬉しいのですが・・・・。
