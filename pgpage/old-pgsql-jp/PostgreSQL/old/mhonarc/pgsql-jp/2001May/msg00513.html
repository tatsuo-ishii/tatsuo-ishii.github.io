> 
> 1)指定IPアドレスの検索
> 例　ipad='192.0.168.10'　ではだめなようですが
> 
> 2)192.0.168.*　を検索する場合　等
> 
> ご教授いただけないでしょうか。

こちらではうまく動いているみたいです。

hoge=# select version();
                           version                           
-------------------------------------------------------------
 PostgreSQL 7.1.1 on i686-pc-linux-gnu, compiled by GCC 2.96
(1 row)

hoge=# create table test(ip inet);
CREATE
hoge=# insert into test values ('192.168.0.10');
INSERT 19572 1
hoge=# select * from test where ip = '192.168.0.10';
      ip      
--------------
 192.168.0.10
(1 row)

hoge=# select * from test where ip like '192.168.0.%';
      ip      
--------------
 192.168.0.10
(1 row)
