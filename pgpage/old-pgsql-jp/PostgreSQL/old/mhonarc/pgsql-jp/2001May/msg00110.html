笠松と申します。
Postgresql7.1.1のpostgresql.jarをうまく生成できたのですが、
一番期待していたsetBinaryStream()の実行エラーが出てしまいます。
PostgreSQLプログラマガイドChapter21.JDBCインターフェイスに書いてある
ものを参考にしたソースの一部を示します。(画像は自前です。)

File file=new File("kasa.jpg");
FileInputStream fis=new FileInputStream(file);
PreparedStatement ps=conn.prepareStatement("insert into images 
values (?,?)");
ps.setString(1,file.getName());
ps.setBinaryStream(2,fis,(int)file.length());
以下省略

これでコンパイル(成功)し、実行すると
InputStream as parameter not supportedとでてエラーメッセージが長々と
(長いので省略,Throwable.java,Exception.java,SQLException.javaなどの
エラーです)続きます。
jdbc2ディレクトリのPreparedStatament.javaのsetBinaryStreamを見ると
引数は順にint parameterIndex,InputStream x,int lengthなので
上のソースのFileInputStreamをInputStreamにしましたが、同じ実行エラーが
でました。
どなたかうまく使えている方、いらっしゃいますか？
