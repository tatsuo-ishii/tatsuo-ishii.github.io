私は、早速 7.1.1 でもやってみました。vine 2.1.5 上ですけど。

./configure --with-java では当初 ANT がみつからんと怒られたので 
ANTの指定で make一発で postgresql.jar の作成までできました。

$ cd /usr/local/src
$ tar xvfz /hoge/postgresql-7.1.1.tar.gz
$ cd postgresql-7.1.1

$ export ANT=/usr/local/src/build/ant/bin/ant
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
$ ./configure --enable-multibyte=EUC_JP --with-java
                                        ~~~~~~~~~~~
$ make

$ make check

$ make install
