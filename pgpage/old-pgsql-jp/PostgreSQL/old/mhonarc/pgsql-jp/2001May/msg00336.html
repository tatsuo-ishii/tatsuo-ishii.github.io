  こじまみつひろさんの努力のおかげで、Plamo-2.2 シーラカンス本版
ができました。

  contrib/Database/Extra/ というディレクトリに pgsql711.tgz とい
うのがあります。中身は PostgreSQL-7.1.1+jumbo20010508 + psqlj で
す。構成は、
--
% ./configure  --enable-multibyte --enable-syslog --with-tcl \
--with-tclconfig=/usr/local/lib --with-tkconfig=/usr/local/lib \
--with-krb4=/usr/athena --with-openssl --with-perl --with-python \
--with-odbc
--
としてインストールしたものです。

  当初の予定がずるずると伸びてGW中もなんとなく飼い殺し状態のよう
でした(;;)。でも、結局めでたく7.1.1リリース版を収録できました(^^)。
（contrib/Database/aa_pgsql.tgz は 7.0.3 ですので気を付けて下さい）

  ついでですが、これには間に合わなかった PHP-4.0.5 + jstring.so他
を含むテスト中の mod_ssl 付き Apache などセットの Plamo 版を
"JPUG CDROM Vol.3" に収録させてもらってます。この CDROM は Linux
World Expo/Tokyo 2001 の .org コーナの JPUG ブースで配付予定です。
その他のプラットホーム用のPostgreSQL-7.1.1のバイナリも収められて
います。
