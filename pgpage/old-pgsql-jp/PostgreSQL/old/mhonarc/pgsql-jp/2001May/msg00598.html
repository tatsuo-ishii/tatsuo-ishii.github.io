上記のようにホスト変数に"[]"表記による配列表記にするとecpgの構文チェックで
次のように怒られてしまいうまくいきません。

test.pgc:85: ERROR: parse error, expecting `')'' or `','' at or near "["
"test.pgc", line 85: warning: newline not last character in file
"test.pgc", line 85: syntax error before or at: <EOF>
test.c に対して、acomp が失敗しました。

Oracleでのホスト配列による一括登録は、

EXEC SQL FOR :cnt INSERT INTO test VALUES (:name, :inum, :fnum);

のように「FOR　配列の要素数」という記述によって実現していました。
