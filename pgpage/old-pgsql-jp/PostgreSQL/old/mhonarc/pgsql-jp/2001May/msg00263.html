初めまして、玉腰です。

表題の通りなんですが、

date型の日付に３ヶ月足してupdateするにはどのような
方法がありますでしょうか？

create table test (id varchar(8), out date);

「id」には、「WWW00000」、「out」には「05-15-2001」が入っており、
 $iid = "WWW00000"とし、結果として「out」を「08-15-2001」にしたい。

update test set outday + '3 month'::timespan where id='$iid'

では、phpを介していますが、下記のように表示されてだめなようです。
PostgresSQL query failed: ERROR: parser: parse error at or near "+" in url on line 27

ある程度時間をルーズにしたいので、date型を利用したいと思っています。

環境：postogreSQL 6.5.3 + PHP Version 3.0.18-i18n-ja-2
