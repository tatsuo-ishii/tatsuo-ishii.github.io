佐藤通敏です。

以前PostgreSQL-7.0beta5でmakeしていたときに同じ問題に遭遇しました。
そのときは

  configureが終ったら、include/config.hファイルのコメント・アウトされているSTDC_HEADERを有効にする。

% vi include/config.h
*** include/config.h.orig	Mon Apr 10 17:42:01 2000
--- include/config.h	Mon Apr 10 18:29:11 2000
***************
*** 293,295 ****
  /* Define if you have the ANSI C header files.  */
! /* #undef STDC_HEADERS */
  
--- 293,295 ----
  /* Define if you have the ANSI C header files.  */
! #define STDC_HEADERS 1

で直りました。Solaris+gccでva_list関係のエラーが出たらstdarg.h関係を
変えてみてください。
