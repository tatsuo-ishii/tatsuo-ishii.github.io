[pgsql-jp 21352] select の結果が文字化けする

  • From: Manabu Kosaki <kosaki@mx1.freemail.ne.jp>
  • Date: Mon, 21 May 2001 11:47:17 +0900
みなさま

小崎です。

環境は Postgres7.1 FreeBSD3.4R です。
日本語のテーブルを作成しデータを insert しました。
しかし select の結果で文字化けを起こします。

kosaki=# select * from 冷蔵庫;
    ^J^l^N^`    |        ^I^J^L^~        |   ^y^X^F^~^F^|   |  ^~^^^L^c^t^|^x^B  | ^
^t^N^L | ^H^w^y^M
------------+--------------------+------------+------------+------+------
 ^L^n^z^Z       | ^d^x^d^c^d^l^d^d^d^b         | 2001-05-15 |            |    7 |
 ^L^n^z^Z       | ^B^g^z^l               | 2001-05-15 |            |    0 |

copy や pb_dump で export した場合は
less コマンドにてデータが正常に表示されます。

/tmp/db.out
COPY "冷蔵庫"  FROM stdin;
野菜    じゃがいも      2001-05-15      \N      7       \N
野菜    大根    2001-05-15      \N      0       \N

Postgres の環境変数は以下の通りです。

LANG=ja_JP.EUC
POSTGRES_HOME=/usr/local/psql71
PGLIB=/usr/local/psql71/lib
PGDATA=/usr/local/psql71/data
PATH=/usr/local/psql71/bin
LD_LIBRARY_PATH=/usr/local/psql71/lib
MANPATH=/usr/local/psql71/man

Postgres の configure は以下を指定しました。
./configure --enable-multibyte=EUC_JP --enable-syslog --prefix=/usr/local/psql71

また、shell は bash を使用していますが、
~/.inputrc には以下を設定しています。

set meta-flag on
set convert-meta off
set output-meta on

以上です。
よろしくお願いいたします。
