ちなみに　オプションの意味は

     -S   Specifies that the postmaster process should  start  up
          in  silent mode. That is, it will disassociate from the
          user's (controlling) tty, start its own process  group,
          and  redirect its standard output and standard error to
          /dev/null.

     -s   postmaster will stop all  other  backend  processes  by
          sending  the signal SIGSTOP, but will not cause them to
          terminate. This permits system programmers  to  collect
          core dumps from all backend processes by hand.
