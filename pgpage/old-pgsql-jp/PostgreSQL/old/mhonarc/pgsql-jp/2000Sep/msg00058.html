 PGCLIENTENCODING=SJIS や set client_encoding to 'SJIS' でも
 いいですが、TeraTerm Pro であれば
  Setup -> Terminal -> Kanji(receive) を EUC,
  Setup -> Terminal -> Kanji(transmit) を EUC,
 でもいいと思います。

 LANG を ja_JP.eucJP とかにしているなら TeraTerm の設定を変
 えた方が便利だと思います。
