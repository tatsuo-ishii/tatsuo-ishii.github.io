別メールにも書きましたが，

UPDATE T1 SET ...

と

BEGIN;
UPDATE T1 SET ...
END (ないしはCOMMIT);

は同じなので，たとえば2つのUPDATEを同じレコードに対して同時に行うとど
うなるか，なんてことを知りたい場合は，上のようにあえてBEGIN...ENDで囲
めば良いのですよ．偶然にまかせる必要はありません．
