はじめまして、出口と申します。

現在、postgreSql 7 + gcc + apache を利用しcでcgiを作成してます。
そこで、コンパイルするときにlibpqをリンクするとブラウザで表示できなくなっ
てしまいます。libpqをリンクしなければ正常に表示されます。
ソース中にlibpqに含まれる関数をすべて抜いてコンパイル時にリンクするだけ
で表示できなくなります。コマンドライン上ではlibpqをリンクしてもしなくて
も結果は結果は同じように出力されます。

リンク方法は以下のようにしてます。

よろしくお願いします。

リンクした場合
gcc -o sample.cgi -L /usr/local/pgsql/lib -lpq sample.c
辛苦しない場合
gcc -o sample.cgi -L /usr/local/pgsql/lib sample.c


ソース
main()
{
        sample();

}
sample()
{
        printf("Content-type: text/html\n\n");
        printf("<html>\n");
        printf("<head>\n");
        printf("<meta http-equiv=\"Content-Type\" content=\"text/html; charset=e
uc\">\n");
        printf("<title>サンプル</title>\n");
        printf("</head>\n");
        printf("<body>\n");
        printf("sample\n");
        printf("</body>\n");
        printf("</html>\n");
}
