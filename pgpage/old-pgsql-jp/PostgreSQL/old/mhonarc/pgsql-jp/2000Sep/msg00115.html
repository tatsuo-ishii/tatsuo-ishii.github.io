何回か ML で話題になっていることですが、もう一度。

psql の \dt コマンドで、テーブル名が表示されることは
ご存知ですね。

    → NO の場合は、マニュアルかPostgreSQL完全攻略
       ガイドをみなおす。

下記コマンドを、実行します。

psql -d ＤＢ名 -E -c '\dt;'

そうすれば、\dt コマンドがどのような SQL 文で、
テーブル名を表示しているかがわかります。他の
psql 組込みの  \ コマンドも同様。(私も、この
方法をこの ML で教わりました。多謝)。

参考:
psql -d ＤＢ名 -E -c '\dt;'  2>hoge
とすれば、SQL の部分だけが、 hoge というファイル
名にリダイレクションされます。

実行結果の例:
QUERY: SELECT usename, relname, relkind, relhasrules 
FROM pg_class, pg_user 
WHERE usesysid = relowner and ( relkind = 'r') 
and relname !~ '^pg_'  ORDER BY relname 

これを参考にして、自分で工夫してみてください。

あるいは、単純に、
SELECT * FROM pg_tables WHERE tablename = '探したい表名';
としてもいいでしょう。

表へのアクセス権限は調べなくてもいいのかな。
