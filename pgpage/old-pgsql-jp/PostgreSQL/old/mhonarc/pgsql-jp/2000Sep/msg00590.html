> Perlの DBI , DBD::Pg で同時接続数に制限はあるのでしょうか？

単に perl の script が複数動いて、それぞれからコネクションを張る場合は
それぞれの DBI ／ DBD::Pg はお互いに「今コネクション張ってまっせ」と
いった情報をやりとりしないと、接続制限はできないと思いますが、
そのようなことはしていないようです。

ので、状況としては、1つの perl の script 内から複数本コネクションを張る、
という場合の制限でしょうか…？
おそらくこちらも制限はないと思いますが…

なので、

> それともDBIとは関係なくPostgreSQL側での制限だけでしょうか？

のはずです。
