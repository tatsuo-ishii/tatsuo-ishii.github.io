Java は1ヶ月前から、Servlet とか JSP とか EJB とかにいたっては、
その概念を Oracle iDevelop2000 ではじめて仕入れた始末ですので
Java ではないです
＃FreeBSD つかっていると、どうしても Java の開発・実行環境を
＃どうするか、で一つめの障壁があって食わず嫌いモードでした

で、件のサイトは Apache1.3.12 + Perl5.003 + mod_perl + DBI
+ DBD-Pg + Apache::DBI で構成しています。mod_perl と Apache::DBI
を組み合わせての parsistent connect の威力は絶大でした。

Web server と DB は同じマシンの上で動いていますが、瞬間風速で
load avg. が 50 とか 100 とかいっても耐えてくれる点で FreeBSD
を信頼しています。
某 Linux (kernel 2.2.14 だったかな?) でテストした時にはこけました…
