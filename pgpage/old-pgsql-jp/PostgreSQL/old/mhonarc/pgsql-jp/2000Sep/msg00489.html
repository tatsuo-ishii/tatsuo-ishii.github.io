現在、CobaltQube2700というLinuxサーバにpostgreSQL7.0.2をインストールしよ
うと
していますが、configureが終わってmakeを実行したところ、以下のようなメッ
セージ
が出て先に進めなくなっています。

[root@www3 src]# make
make: *** File `Makefile.port' has modification time in the future
make: Failed to remake makefile `Makefile.port'.
make -C utils all
make[1]: Entering directory `/usr/local/src/postgresql-7.0.2/src/utils'
make[1]: *** File `../Makefile.port' has modification time in the future

make[1]: *** File `Makefile' has modification time in the future
make[1]: Failed to remake makefile `../Makefile.port'.
make[1]: Failed to remake makefile `Makefile'.
make[1]: *** File `version.c' has modification time in the future
make[1]: Leaving directory `/usr/local/src/postgresql-7.0.2/src/utils'
make: *** [all] Error 2
[root@www3 src]#

Makefile.portをremakeできない、と言っているようなのですが。
Makefile.portの所有者はpostgres、グループはusersになっており、
書き込みの権限もあるのですが。（postgresでログインしています）


Kernelとインストールされているソフトは以下の通りです。
　　Kernel 2.0.33 on a mips
　　GNU Make version 3.75, by Richard Stallman and Roland McGrath.
　　gcc version 2.7.2
　　flex version 2.5.4
　　GNU Bison version 1.25

　　configure --enable-multibyte=EUC_JP
