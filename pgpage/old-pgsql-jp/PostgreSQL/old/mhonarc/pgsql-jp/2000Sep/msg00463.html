> int(11)ってのは、11桁の整数ということでしょうか？

MySQL のドキュメントを見てみたら、

| `INT[(M)] [UNSIGNED] [ZEROFILL]'
|      通常の整数。符号ありの範囲は
|      `-2147483648'〜`2147483647'。符号なしの範囲は `0'〜`4294967295'

で、

| `M'
|      最大表示サイズを意味します。最大の表示桁数は 255.

で、結局のところ int(11) の範囲は `-2147483648'〜`2147483647'
だと思うのですが。


mysql> create table hoge (foo int(11) , bar int);
Query OK, 0 rows affected (0.01 sec)

mysql> insert into hoge values(12345678901,12345678901);
Query OK, 1 row affected (0.01 sec)

mysql> select * from hoge;
+------------+------------+
| foo        | bar        |
+------------+------------+
| 2147483647 | 2147483647 |
+------------+------------+
1 row in set (0.00 sec)

mysql> select version();
+-----------+
| version() |
+-----------+
| 3.22.27   |
+-----------+
1 row in set (0.01 sec)
