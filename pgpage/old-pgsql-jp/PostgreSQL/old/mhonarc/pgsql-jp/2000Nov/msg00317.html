FreeBSD 4.0 + postgresql-7.0.2
WinNT-Sever 4.0 + Access97（以下「AC97」） + ODBC Driver 日本語版

kanri-# \l
kanri-# ------------+----------+----------
kanri-#  kanri      | postgres | EUC_JP

AC97 からの接続に成功し、データーの追加削除にも成功したのですが．．

psqlにて、
kanri-#　select * from table ;
とすると．．表示の日本語部分が化けてしまいます。
この状況は，正常なのでしょうか？

逆に、psqlにて挿入したデータをAC97で見た場合文字化けします。
