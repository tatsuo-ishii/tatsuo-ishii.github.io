言葉足らずだったようで、すいません。

> >日本語データをソート(order by)すると、結果が期待通りになりません。
> >漢字と半角英字がごちゃごちゃになって表示されます。
> >データ(そうめん、自動車、CCD、CAR)があるとすると、
> >ソートをかけると
> >自動車、そうめん、CAR、CCD
> >となります。
> 
> どうなってほしいのでしょうか？
CAR、CCD、そうめん、自動車 となってほしいのです。

> 一般に漢字を含むものをソートするときは、「よみがな」の様な項目を
> 準備して平仮名でよみを入れておいた方がよろしいのではないでしょうか？
ためしに、
CREATE TABLE T_TEST (T VARCHAR(128));
INSERT INTO T_TEST (T) VALUES ('あ');
INSERT INTO T_TEST (T) VALUES ('い');
INSERT INTO T_TEST (T) VALUES ('う');
INSERT INTO T_TEST (T) VALUES ('え');
とやって、
SELECT * FROM T_TEST ORDER BY T;
とすると
結果は、
え
い
あ
う
となってしまいます。
