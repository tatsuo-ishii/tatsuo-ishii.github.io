佐々木といいます。
初めて質問します。

PostgreSQL7.0.3のパッチ済みソース（postgresql-7.0.3-patched-20001117.tar.gz）を
コンパイルしてインストール後、initdbを実行すると以下のようなメッセージがでます。

------------------------------------------------------------------------------------
We are initializing the database system with username postgres (uid=504).
This user will own all the files and must also own the server process.

Creating template database in /usr/local/pgsql/data/base/template1
ERROR:  Error: unknown type 'oidvector'.

ERROR:  Error: unknown type 'oidvector'.

        syntax error 12 : parse errorinitdb: could not create template database
initdb: cleaning up by wiping out /usr/local/pgsql/data/base/template1
--------------------------------------------------------------------------------------

これは、まだ何かインストールしなければならないものがあるのでしょうか？



環境
	PC:AT互換機
	HDD:残り７Gほど
	OS:RedHat
	コンパイラ：gcc version egcs-2.91.66 19990314/Linux (egcs-1.1.2 release)
	make:GNU Make version 3.77, by Richard Stallman and Roland McGrath.

設定した環境変数
<<.bashrc>>
PATH="$PATH":/usr/local/pgsql/bin
export POSTGRES_HOME=/usr/local/pgsql
export PGLIB=$POSTGRES_HOME/lib
export PGDATA=$POSTGRES_HOME/data
export MANPATH="$MANPATH":$POSTGRES_HOME/man
export LD_LIBRARY_PATH="$LD_LIBRARY_PATH":"$PGLIB"

コンパイル・インストール・dbinitの実行は、postgresユーザで行いました。



以上よろしくお願いします。
