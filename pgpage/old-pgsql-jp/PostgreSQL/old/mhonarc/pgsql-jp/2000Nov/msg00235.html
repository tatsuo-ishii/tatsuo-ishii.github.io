はじめまして。上田といいます。
教えていただきたい事があり投稿しました。

create table test(
    userid int2,	// ユーザID
    startdate date,	// 開始日
    enddate date	// 終了日
);

というテーブルを作ったとします。そして開始日には、
とある作業開始した日の日付を入れますが、作業が完了するまでは
終了日には日付を入れたくありません。作業が完了した時点で
enddateに値を入れたいと思います。それで、

sample=>insert into test values(1, '2000/11/16', '');

とすると、

ERROR: Bad date external representation ''

と怒られてしまいます。date型のフィールドに日付を入れたくない
ときはどうすればいいのでしょうか？知恵を貸してください。
よろしくお願いします。

LASER5 Linux 6.0
PostgreSQL 6.5.2
を使用しています。
