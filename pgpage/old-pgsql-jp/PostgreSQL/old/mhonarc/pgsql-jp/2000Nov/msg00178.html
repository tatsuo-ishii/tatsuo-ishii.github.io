KUBO Takehiro <tkubo@insight-tec.co.jp> さん wrote:
> 久保＠茅ヶ崎市です。
> 
> ところで、直接 SQL でオプションをつけられるようにするとうれしいことあ
> りますか？
> 
> 直接 SQL で、
>     select -d db3 * from test3
> とすると、
>     exec_sql -d db3 "select * from test3"
> を実行したのと等値にするとか。

これは、あまりうれしくないですね。SQL の構文を崩した形式に
なりますので違和感もあります。

それと話しは少し変わりますが、"exec_sql"を省略する方法とし
て、exec_sqlオプションを保持できる機能があれば良いと思って
います。(特に、出力書式のオプションは保持したくなります)

(例)
set EXEC_SQL_OPTION="-d db3";
select * from test3;
