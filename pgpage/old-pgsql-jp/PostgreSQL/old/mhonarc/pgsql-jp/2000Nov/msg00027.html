ども、こんにちは。はじめまして。塚田といいます。

postgresql-7.0.2がコンパイル&ざっと動作確認できました
OSXは日本版を使っています。ADCから開発環境を手に入れる必要があります

http://www.gerga.com/pg_darwin_patch.tar.gz
から、パッチを拾ってきて
＜このパッチはdarwin-MLで出てきた物らしいですが詳細は知りません
解凍し、postgresql-7.0.2が見えるディレクトリ位置から
patch -p0 < pg_darwin_patch

それと、このままだとコンパイル中にエラーが出るので
sem.hをコピーします

cd  postgresql-7.0.2/src
mkdir include/sys
cp backend/port/darwin/sem.h include/sys
./configure
make
make install

php-3.0.16のコンパイルも問題なく終わりました
./configure --with-apxs=/usr/sbin/apxs --enable-track-vars --enable-i18n --enable-mbregex --with-pgsql --with-config-file-path=/etc/httpd

ただ、PHPの動作確認はまだしていません。これから、やってみます

でも、ほんとはOSXでWebObjects & PostgreSQLで動かしてみたいところですね

＃macosx-jp MLにも流しましたが、こちらにも関係あると思うので
＃投稿させていただきました
