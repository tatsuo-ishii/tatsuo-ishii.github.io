菊沢 正明 wrote:
(snip)
> CREATE TABLE T_TEST (T VARCHAR(128));
> INSERT INTO T_TEST (T) VALUES ('あ');
> INSERT INTO T_TEST (T) VALUES ('い');
> INSERT INTO T_TEST (T) VALUES ('う');
> INSERT INTO T_TEST (T) VALUES ('え');
> とやって、
> SELECT * FROM T_TEST ORDER BY T;
> とすると
> 結果は、
> え
> い
> あ
> う
> となってしまいます。

なんかおかしいですね。
上記のとおりのことをやってみると、ちゃんと
EUC コード順に、あいうえ  と出力されます。
SELECT * FROM T_TEST ORDER BY T DESC;
ならば、えういあ の順ですね。

EUC HEX CODE は、次のとおり。
 a4a2 a4a4  a4a6 a4a8
  あ    い   う   え  

環境は、
PostgreSQL 7.0.2  TurboLinux WorkStation 6.0
