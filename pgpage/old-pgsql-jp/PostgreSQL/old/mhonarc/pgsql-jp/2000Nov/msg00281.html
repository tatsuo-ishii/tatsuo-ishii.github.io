松本です。

大場さん、小菅さんのおかげで起動できました。ありがとうございます。

原因は、""でオプション全体をくくっていなかったのと、
postmaster.opts.defaultに一部オプションを書いたのを忘れていた為でした。

/* postmaster.opts.defaultの記述 */
-S -i

/* rc.localの起動] */
su postgres -c "/usr/local/pgsql702/bin/pg_ctl -w -o \"-N 128 -B 256 -o '-F' -p
5432 -D/var/lib/pgsql702\" start"

/* Bシェルからの起動 */
# pg_ctl -w -o "-N 128 -B 256 -o '-F' -p 5432 -D/var/lib/pgsql702" start
postmaster successfully started up.

/* 起動後のpostmaster.opts */
/usr/local/pgsql702/bin/postmaster
-p 5432
-D /var/lib/pgsql702
-B 256
-b /usr/local/pgsql702/bin/postgres
-N 128
-o '-F'

///* Kouichi.Matsumoto */
