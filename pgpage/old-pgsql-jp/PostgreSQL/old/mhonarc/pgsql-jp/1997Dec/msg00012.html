Pg.pm というのがあります．これは他の方も紹介していらっしゃ
いますので改めてかく必要も無いかと思います．

こんな流れになります．

1. Pg.pm で PostgreSQL に接続
2. SQL文の発行（PostgreSQL はその SQL 文を実行）
3. 結果が perl の変数に代入される

例えば，query を行った後に，

$ntups = $result->ntuples;
$nfields = $result->nfields;

とすると，query の結果得られたタプルの数が $ntups に代入
されて，フィールドの数が $nfields に代入されます．

$i 番目のタプルの $j 番目のフィールドの値を取り出したけ
れば，

$value = $result->getvalue($i, $j);

となります．このように，$result に query の結果が入るの
で，それを取り出してあげる形になるわけですね．詳しくは，

http://www.postgresql.org/docs/Pg.pm.shtml

をご覧下さい．いずれにせよ，perl5 は必須です．
