石井です。

郵便番号検索システム pgpost が早くも 1.1 になってしまいました:-)

今回の目玉は、うちの鈴木君(a-suzuki@sra.co.jp)が作った Tcl/Tk インター
フェイス "tkpost" です。お楽しみ下さい。

なお、PHP/FI インターフェイスの方は、郵便番号に数字以外の文字があった
ら無視するくらいしか機能が up されていませんので、1.0 をすでにインストー
ルされてる方は、あえて入れ換える必要性はあまりないと思います。
詳しいことは README の変更履歴を御覧下さい。

また、1.0 の時の postal.data.gz はそのまま使えると言うか、PostgreSQL 
のデータベースは 1.0 のままで大丈夫です。

[取得方法]

1. ftp の場合:

ftp://ftp.sra.co.jp/pub/cmd/postgres/6.2.1/pgpost/pgpost-1.1.tar.gz

2. mail によるアーカイブ取り寄せ機能を利用する場合:

pgsql-jp-request@sra.co.jp 宛てに、

get pgpost-1.1.tar.gz.uu

とだけ本文に書いてメールを送る。サブジェクトは要りません。

以上の方法でも取得できない場合は t-ishii@sra.co.jp まで御連絡下さい。
以上、よろしくお願いします。
