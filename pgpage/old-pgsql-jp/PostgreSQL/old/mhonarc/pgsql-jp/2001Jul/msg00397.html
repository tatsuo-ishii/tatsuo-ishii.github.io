よしむら＠福岡です。

> ---------------------------------
> <?php
> $dbcon = pg_connect ( "dbname=xxxxx user=xxxxx");
> if (!$dbcon) { die( "<hr>$dbcon失敗<hr>"); }
> $max1 = "select max(aute_no) from userdb1";
> if (!$max1) { die( "<hr>$max1失敗<hr>"); }
> echo pg_exec ($dbcon, $max1);
> ?>
> ---------------------------------

マニュアルは読まれていますか？

echo pg_exec ($dbcon, $max1);

の部分を以下のようにするといいでしょう。

$resource_id = pg_exec($dbcon, $max1);
echo pg_result($resource_id, 0, "max");

これでお望みの結果が得られるはずです。

詳しくは、マンモス本等をご覧ください。

> telnetで　psql　=>select max(aute_no) from userdb1";
> を実行した場合は検出されます。

psqlでは結果が表示されるように作られていますが、phpのpg_execの戻り値は結果セット
ではありません。
