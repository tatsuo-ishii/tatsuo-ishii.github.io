            事例セミナー 講師募集のお知らせ

                                    日本 PostgreSQL ユーザー会
                                    業務アプリケーション分科会

昨年、当会主催による事例紹介セミナーを開催し、80名以上の方々が
参加され、成功理に終了いたしました。この経験を含め、さらにひろ
く PostgreSQL を使ったアプリケーションを開発するための参考とす
べく、今年も引き続き「事例紹介セミナー」を開催することにいたし
ました。

この「事例紹介セミナー」で、 PostgreSQL を使用した事例を発表し
ていただける方々を募集いたします。事例発表は、導入を検討されて
いる方々への参考になりますので、 PostgreSQL で、システムを作成
されたり、運用されている方は、ぜひ、そのご経験を発表して頂きた
く、講師募集する次第です。

発表のなかで、発表者や導入先/納入先などの会社/団体の宣伝などを
入れていただいてもかまいません。

自薦、他薦を問いませんので、振るってご応募ください。

応募は末尾の応募フォーマットを参考に、ＪＰＵＧ事務局宛にメール
にてお願いします。 ( Email:   staff@jp.postgresql.org  )
