塩見と申します。

> テーブルをcreateする時、int型フィールドで桁数を指定しなかった
> 場合デフォルトでは何桁になるのでしょうか？

データ型が何になるかということだとすれば、次の通りです。

testdb=# create table int_tbl(int int,int4 int4,int8 int8);
CREATE
testdb=# \d int_tbl
         Table "int_tbl"
 Attribute |   Type   | Modifier
-----------+----------+----------
 int       | integer  |
 int2      | smallint |
 int4      | integer  |
 int8      | bigint   |


各型の範囲はドキュメント(postgres)の3.Data Typeの項に載っています。
