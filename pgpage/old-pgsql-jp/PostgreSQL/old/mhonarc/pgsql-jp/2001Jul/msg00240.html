初めまして。
おおはら＠メガソフト　と申します。

さて、いま、PostgreSQL 7.0　のフロントエンドとして MS-Access2000 を使用して
システムを構築中なのですが、ODBC経由で PostgreSQLに接続しようとすると
MS-Access2000が
　Reserved error (-7709) there is no message for this error
というエラーを出すだけで接続できない

という質問を投げようと思ったのですが、社内の別の環境では動いているものも
あったので、念のために MS-Acess2000のバージョンを確認すると、
　動いていない環境のバージョン  9.0.2812
  動いている環境のバージョン　　　9.0.3821 SR-1
となっていました。
2812の環境を3821 SR-1 に変えるとアクセスできるようになりました。

ということで同じような事で悩んでいる方がおられては、と思い投稿させて
いただきました。

どなたかのご参考になれば、幸いです。
