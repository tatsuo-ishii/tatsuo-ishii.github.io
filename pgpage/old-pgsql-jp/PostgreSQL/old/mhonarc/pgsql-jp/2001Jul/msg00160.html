プログラミングのご経験も十分されているようなので、
DBI や Pg.pm のマニュアルがあれば十分ではないで
しょうか？下記は川合孝典様による Perl モジュール
の日本語訳サイトです。
http://member.nifty.ne.jp/hippo2000/perltips/
# 有名なサイトですので、すでにご存知なら申し訳ありません。

弊社サイトにも多少情報はあります。
http://www.itboost.co.jp/perl/

また、コマンドライン版の PHP でやっても良いかと
おもいます。SELECT してメール数通送信、ぐらいな
らそんなに処理系の速度等が問題になるような感じで
もないようですし、サイトの方で include や require
しているファイル（共通関数とか、共通変数を書いて
おくような）なんかもそのまま使えますし。

コマンドライン版の PHP もインストールしておくと、
ちょっとした PHP プログラムの文法（書き方）のテ
ストなんかに使えて便利ですよ。
