COPY ではできないと思います。psql を使って、

  psql -A -F ',' -t -o '/tmp/hogehoge' -c 'SELECT * from emp where id<10';

という感じでどうでしょうか？

オプションは

-A : 表の要素を出力する際に行揃えをしません
-F : 項目の区切りを指定
-t : データだけを出力
-o : 出力先を指定ファイルに
-c : クエリを実行させて、すぐに終了

という意味です。-A がないと、-F の指定が無視されるようです。

オプション以外での指定方法もありますので、詳しくは man psql 、
もしくは
  http://www.itboost.co.jp/man/postgres_man/man1/psql.1.html
にてご確認ください。
