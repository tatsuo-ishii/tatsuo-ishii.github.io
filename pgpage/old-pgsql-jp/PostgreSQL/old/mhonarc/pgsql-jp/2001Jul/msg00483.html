松嶋＠ITBoost です。

PostgreSQL において、任意の SQL を実行する際にか
かる時間を測定する PGLoad(仮) というアプリを作り
ました。

　★ 任意のテーブルに、任意の量のデータを簡単に挿入。
　★ 任意のテーブルに任意の SQL の発行し、時間の計測。
　★ その際、同時接続状況も簡単につくりだせます。

ということができます。

設計したテーブル構造において、どのくらいの規模であれ
ば、どの程度のパフォーマンスが発揮できるのか、という
ことを運用前に、そして「手軽に」知ることができます。

また、等価な複数の SQL のうち、どの SQL が一番速い
とかを知ることもできます。

設定ファイルを編集することにより、ランダムでテー
ブルに挿入されるデータをある程度コントロールする
ことが可能です。設定ファイルの自動生成もできます。

  http://www.itboost.co.jp/inst/
から辿れる、
  http://www.itboost.co.jp/inst/inst_33.php
に置きました。

興味がありましたら使用していただき、動作報告（OS や各種
バージョン情報などを教えてください）、改善要望、バグ報告、
改良(^^;)、命名(^^;;;)なんかもしていただけると幸いです。
