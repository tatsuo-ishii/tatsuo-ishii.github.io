.bashrcの場所にもよるかと思いますが.bashrcは、個人用設定ファイルだから
rc.localが
起動するときには読み込まれないんじゃないですか？
設定を書くなら、rc.localにも書いたほうがいいかと思いますよ。

こんな具合に

[rc.localにて]
 export PGHOME=/usr/local/pgsql
 export PATH=$PATH:$PGHOME/bin
 export PGDATA=$PGHOME/data
 export PGLIB=$PGHOME/lib
 export MANPATH=$PGHOME/man
 export LD_LIBRARY_PATH=$PGLIB
if [ -x $PGHOME/bin/postmaster -a -d $PGHOME/data ] ; then
     rm -f /tmp/.s.PGSQL.5432
     su - postgres -c "$PGHOME/bin/postmaster -S -i"
     echo -n 'postmaster '
fi
