PostgreSQL 7.1.1 の結果です。

test=# create table tb_test
test-# (
test(# a int
test(# );
CREATE
test=# \d tb_test
        Table "tb_test"
 Attribute |  Type   | Modifier
-----------+---------+----------
 a         | integer |

\dT すると
> integer                  | -2 billion to 2 billion integer,
> 4-byte storage

となっているので int4 と同等です。4 byte ですね。
