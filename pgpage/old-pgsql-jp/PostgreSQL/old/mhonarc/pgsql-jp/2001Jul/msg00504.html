# 少々引用順を元と変えます

| pg_dump やら copy コマンドやらに、「text 型は引用符で囲んで出力」
| みたいなオプションがあればとも思ったのですが、どうやら
| それもなさそうです。

　これは今のところ何方もフォローがないところを見ると、「(現状は)できない」が
正解かもしれません。が、ターゲットのバージョンが不明ですので、答えられないだ
けかもしれません。


| お客さんの要望で、PostgreSQL のデータを Excel で読み込める形式に
| エクスポートする必要が出てきました。

　という事であれば、とにかくフィールドは全部 " で括ることです。


| 最悪、pg_dump やら copy コマンドは使わずにこの例で言うと
|
|   select a, '"' || b || '"', c from t1;
|
| のようにすれば期待通りの出力を得ることはできるのですが、
| 実際のテーブルはこのようなフィールドがいくつかありますし、
| フィールドが増えたときのことなどのメンテナンスを考えると
| あまりこの方法は使いたくありません。

　わたしなら SQL でそのような負荷を RDBMS にかけるより(ご自分で書かれている
通りメンテも面倒)、得られたファイルを Perl でフィルタ書いて整形してしまいま
す。\t(Tab) で split して、" で括って , なりのセパレータを付けて join したも
のを print すればよい筈。数行で書けると思います。
