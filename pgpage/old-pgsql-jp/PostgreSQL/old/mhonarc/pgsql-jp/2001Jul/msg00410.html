　ごもっともなご意見です。どのように実現したかの結果報告を怠っておりまし

で、結果としては、
１．実際に使用する各テーブルのフィールド構成は、すべて同じである。
※使用したテーブル構成
　　CREATE TABLE T1(ID VARCHAR(2),DATA TEXT);
　　CREATE TABLE T2(ID VARCHAR(2),DATA TEXT);
　　CREATE TABLE T3(ID VARCHAR(2),DATA TEXT);
　　CREATE TABLE T4(ID VARCHAR(2),DATA TEXT);
２．同じフィールド構成の複数テーブルを結合する場合は、
　　「natural left outer join」では、すべてのフィールドが結合条件となり、
　　望む結果が得られなかった。
３．以上から「left outer join table_name using(field_name)」にて結合フィー
　　ルドを明示的に条件付けしてみた。
４．作成したSQL文
　　CREATE VIEW TEST_VIEW AS
　　　SELECT T1.ID AS ID,T1.DATA AS D1,T2.DATA AS D2,
             T3.DATA AS D3,T4.DATA AS D4
      FROM T1 LEFT OUTER JOIN T2 USING(ID)
              LEFT OUTER JOIN T3 USING(ID)
              LEFT OUTER JOIN T4 USING(ID);
５．以上で望む結果を得ました。
