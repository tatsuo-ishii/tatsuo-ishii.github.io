>>>>> On Mon, 18 May 1998 16:50:49 +0900 (JST), "大海　治男"<oogai@namos.co.jp> said:

>> postmasterが標準で使用するポートは何番でしょうか。

MANPATH を環境に合わせて設定してあれば、man postmaster すれば
出てきますよ。以下、その一部分。

       -p port
            Specifies the TCP/IP port or local Unix domain socket
            file  extension  on which the postmaster is to listen
            for connections from frontend applications.  Defaults
            to 5432, or the value of the PGPORT environment vari-
            able (if set).  If you specify a port other than  the
            default port then all frontend application users must
            specify the same port (using command-line options  or
            PGPORT)  when starting any libpq application, includ-
            ing psql.

と、いう事で PGPORT 環境変数の指定や -p オプションの指定が
なければ 5432番のはずです。
