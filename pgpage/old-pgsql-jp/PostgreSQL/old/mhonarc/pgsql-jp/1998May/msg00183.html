こんにちは、Koです。

 現在、2台のマシンがありまして、
1台目にpgsql(apache+PHP/FIも一応入ってます。）
2台目にapache+PHP/FI
のような構成です。

それで2台目から1台目のDBにアクセスしよう
とすると
xxx.phtml: Could not connect to database (Invalid packet length) on line xxx
$connect =pg_Connect("xxxxxx", "5432", "", "", "xxxxxxxx");
connect error to database
なるErrorMessageがでてきてしまいます。

ちなみにこのphtmlを1台目内で下記の設定で動かすとちゃんと動きます。
$connect =pg_Connect("localhost", "5432", "", "", "xxxxxxxx");

pg_hda.confは下記の通りです。
またposmasterクンも再起動しました。
local         all                                         trust
host         all         127.0.0.1      255.255.255.255  trust
host         all         2台目IP       255.255.255.255  trust

宜しくお願いします。
