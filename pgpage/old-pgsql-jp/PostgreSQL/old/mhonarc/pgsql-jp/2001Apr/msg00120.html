清水です


> 
> datetime型へのupdateを行なう際に
> 
> update テーブル名 set カラム名 = 'now';
>   
> とすれば現在の日時がupdateできますが、
> 現在の日時の未来・過去（例えば1日後とか5時間後とか）
> は、どのように記述すれば良いのでしょう？
> 
> 'now' + 1day
> 'now' + 5hours
> 
> では、駄目なのですか？
> ----


update テーブル名 set カラム名 = now()+'1day'::interval;
update テーブル名 set カラム名 = now()+'5hours'::interval;

でお望みの結果がでるとおもいます。
