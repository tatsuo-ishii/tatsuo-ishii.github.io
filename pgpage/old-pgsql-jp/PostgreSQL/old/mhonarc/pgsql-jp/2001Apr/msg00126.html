開発に参加されているみなさま、ごくろうさまです。
さっそく Vine-2,1 CR なマシンに入れてみました。

  > uname -a
  Linux rigel.wineroses.org 2.2.17-0vl10 \
  #1 Sat Oct 14 20:20:06 JST 2000 i586 unknown

configure のオプションは以下の通り。

  > ./configure --enable-multibyte=EUC_JP --with-openssl=/usr \
  --enable-syslog --with-perl

make check では無事 "All 76 tests passed." と出ました。
あとは

  make install
  initdb
  pg_ctl start

して、以前のバージョンで pg_dumpall したデータ (db.out) を

  psql template1 -f db.out

してデータ移行。

まだ簡単なチェックしかしてませんが、とりあえず 7.1 に移行できて
いるみたいです。(^^)/~


ところで石井さんのメールの末尾の

> TIP 4: Don't 'kill -9' the postmaster
