  ところで、RC3 で、SYSLOGD 経由でログを出させようとしたのですが、
うまくできません。

	% ./configure  --enable-multibyte --enable-syslog ....

で make install して、syslog.conf に
	--
	local0.*                /var/log/pgsqlog
	--
を追加して
	# kill -HUP `pidof syslogd`
で syslogd を初期化。
出力指定オプションを
	% echo "syslog=2" >> $PGDATA/pg_options
pg_options に追加した後、
	% postmaster -S
で起動しました。
