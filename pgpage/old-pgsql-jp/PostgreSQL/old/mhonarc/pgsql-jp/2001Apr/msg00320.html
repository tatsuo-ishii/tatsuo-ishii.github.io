| /usr/home/postgres/DB_BACK_UP/DB_BACK_UP_TOOL.sh
| をたたくとバックアップファイルが作成されるのですが・・・

　pg_dump: not found が示している通り、その shell script 実行時の PATH では
見つけられないからでしょう。

　帰ってきたメイルの内容：

| X-Cron-Env: <PATH=/usr/bin:/bin>

　の PATH では pg_dump は見つけられないはずです。script の先頭に必要な PATH
を設定するようにすれば良いかと。
