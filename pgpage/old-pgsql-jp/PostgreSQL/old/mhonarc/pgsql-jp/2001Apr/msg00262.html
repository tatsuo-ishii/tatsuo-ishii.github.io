日江ともうします。

PostgreSQL ODBC Driver 日本語版 のページから、PostgreSQL ODBC 
Driver 07.01.0004 日本語版（2001/04/06版）をダウンロードしたの
ですが、ダウンロードした psqlodbc-jp-20010406-bin.zip 中の 
psqlodbc.dll は、

  ファイルバージョン:  07.01.0004.20010402

となっています。同梱されている readme.txt のタイトルも、

  PostgerSQL ODBC Driver 07.01.0004 日本語版 2001/04/02版

となっていますが、これは、バージョンの記述が更新されていないだ
けなのか、それとも 2001/04/02版 そのものなのかどちらでしょう?


このドライバをインストールして、Microsoft Query 経由で 
PostgreSQL(7.0, 7.1 のどちらも) から Excel にデータを取りこも
うとすると、PostgreSQL Connection ウィンドウで、必要な項目を入
力した後、OK をクリックした時点で、Microsoft Query が落ちてし
まいます。

同じくこのドライバを使用して、Access から PostgreSQL(7.0, 7.1 
のどちらも) への接続は、正しく動作しているようです。

また、Japanese PostgreSQL Driver: 6.50.0000.20000617 を使って、
PostgreSQL 7.0 への接続は、Excel, Access とも、問題なく動作し
ていました。


この症状は、2001/04/02版 であることに起因するものでしょうか。
