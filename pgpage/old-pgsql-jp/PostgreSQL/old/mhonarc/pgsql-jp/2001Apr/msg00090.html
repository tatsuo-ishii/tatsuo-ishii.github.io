システムカタログはあります。
PostgreSQL 7.1 Developer's Guide
 + Chapter 3. System Catalogs
を参照してください。

テーブルが占める容量は、
pg_class.relpages
--
Size of the on-disk representation of this table in pages (size BLCKSZ).
This is only an approximate value which is calculated during vacuum. 
--
にあります。vacuumしないと解らないし、近似値ですけれど。
