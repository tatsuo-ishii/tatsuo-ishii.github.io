>> ">" == oku  <oku@shikoku-u.ac.jp> writes:

>> すぐに記事を検索させていただきましたが、目的としている「数の自動的なインクリメント」については残念ながら書かれておりませんでした。
>> 実際には PHP/FIを使って作成しますので、最後の手段は最終行の値を検索し、その数に１を足して直接登録するしかないと考えております。

create sequence id;

として、sequence number generator を生成し、( see man create_seqence )
insert 時に

insert into hogehoge values (nextval(id),boo,foo...);
                             ^^^^^^^^^^^
という形で、increment したい列に記述すれば、insert をするたびに、
値が1づつ加えられて行きます。

ただし、rollback などを行った場合には、値は戻りません。
この辺りは、

[pgsql-jp 3098] Re: rollback 時の sequence      の動作について

に出ているのでしょう。
