  http://www.ipc.chiba-u.ac.jp/~yamaga/gnuplot+/3.6base/index-j.html

から日本語化パッチがたぐれます。

また、最近の readline なら、~/.inputrc に

  set convert-meta off
  set meta-flag on
  set output-meta on

と書いておけば、日本語の入力や表示はできます。ただしコマンドライン編集
に少々難があり、^d で 2 バイト文字の 1 バイトだけ消えたりします。上記
パッチを当てれば大丈夫です。
