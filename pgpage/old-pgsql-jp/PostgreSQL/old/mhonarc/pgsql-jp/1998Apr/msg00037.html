山崎＠道工大さんより -i オプションがない、とのアドバイスを戴きまして

% postmaster -D /usr/local/pgsql/data -S -i

としてpostmasterを起動することにより、TCP/IP connectできるように
なりました。

その後、SoftwareDesign 1998/01号のPHP/FIとの連携例を試してみましたが
今度は

>sample.phtml: Bad response to postgres95 query (ERROR: sukikirai: Permission denied. ) on line 24
>$rtn = pg_exec($con,$query); 
>テーブルsukikiraiの検索に失敗しました。

となりました。ドキュメントを調べると、6.3ではテーブルを明示的に
grantしないといけない、という記述がありましたのでpsqlより

test=> grant select on sukikirai;

としてやることで、なんとかselect * from sukikiraiが実行できるように
なりました。(^^;
