CVSROOT とか、ML のアーカイブ以外は実用的にはあまり意味がないのですが、
コレクションとして大事に取っておきます:-)

# third_party に入っている "dbengine"
# (http://www.cis-computer.com/dbengine/)
# はちょっと面白そうです。

README を引用しておきます。
----------------------------------------------------------------------
Welcome to the first of many PostgreSQL CD Distributions.  

What we have attempted to do is to provide one central, non-bandwidth
method of grabbing ahold of the world as of teh release date, plus 
various improvements as they become available.

Recommendations on things that should be added are always welcome by sending
an email to scrappy@postgresql.org.

The following directories are included, as well as what they contain:

CVSROOT 
	- a snapshot of the CVS repository from the time of release.  For
	  those with cvs installed set your CVSROOT environment variable to
	  <directory to CD>/CVSROOT, and you'll be able to view logs, do
	  diffs and checkout code at will.

pub 
	- a snapshot of our FTP server, based on approximately the time that
	  this CD was burned, which means you can see older releases of 
	  PostgreSQL

bindist 
	- Binary distributions available at the time of burning

patches 
	- patches released for the current source tree

CVSup 
	- various CVSup binaries, as well as config file, for those wishing to
	  follow and work on the newest releases of the code

mailing_lists 
	- mhonarc'd mailing list archive up to the time that the CD image was 
	  created (not necessarily equal to date CD was burned)

third_party 
	- various third party software packages that make use of PostgreSQL, 
	  or provide interfaces into it
