はじめまして。

>Oracleですと
>select to_char(A，YYYYMMDD)・・・
>というように取得できますが、同じ事ができるのでしょうか？

先の日付のインターバルと同様、これも悩みまして、以下の方法
で解決しました。

select (
 date_part('year',HIZUKE) * 10000 +
 date_part('month',HIZUKE) * 100 +
 date_part('day',HIZUKE)
);
でOKです。ただし、int4ですので、これをテキストにしたかったら、
最後の「)」の後に、「::text」をつけて、castします。
これで、YYYYMMDDになります。YYにしたいときは・・・・。
他の方法があるかもしれませんので、そっちに期待しましょう。
でも、これもまた、プログラム側でYYYYMMDDに直したほうが良さげ
な感じでした。

あと、余談ですが、
>現在の時刻を取得するには、aaaテーブルがあったとして
>ex) ．select now() from aaa;
は、テーブルが無くても、
select now();
で、出ますよ。
