SQL や DBMS に関しては市販書籍も色々とでていますが、大抵の本には
インデックスに関する記述もあるでしょう。

PostgreSQL では以下のような文法になります。

  CREATE [ UNIQUE ] INDEX index_name
      ON table [ USING acc_name ]
      ( column [ ops_name] [, ...] )
  CREATE [ UNIQUE ] INDEX index_name
      ON table [ USING acc_name ]
      ( func_name( r">colle> [, ... ]) ops_name )
