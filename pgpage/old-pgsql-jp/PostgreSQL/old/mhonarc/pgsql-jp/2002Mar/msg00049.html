はじめまして。PostgreSQL初級者のminoと申します。

この度、Apache+PHP+PostgreSQLを使いサーバを構築して
勉強をしようと思っております。

まずは簡単な物からやってみようと思い、
次の様なPHPを書いてみました。

---------------------------------------------
<html>
<head><title>test</title></head>
<body>
<?php
$dbcon = pg_connect( "dbname=hoge user=hoge");
if (!$dbcon) { die( "pg_connect 失敗"); }
$sql =  "SELECT COUNT(*) FROM member";
$execid = pg_exec ($dbcon, $sql);
if (!$execid) { die( "pg_exec 失敗<hr>"); }
echo  "".pg_result($execid,0,0). "です。";
pg_close($dbcon);
?>
</body>
</html>
--------------------------------------------

memberテーブルからレコード件数をselectさせて
それを表示するという非常に簡単なモノなのですが
ブラウザでアクセスを試みると次のようなエラーが出ます。


Warning: PostgreSQL query failed:
ERROR: Relation 'member' does not exist in test2.phtml on line 8
pg_exec

クエリーが正しくないというものですがSQL文自体には間違いは無いですし
何が問題なのかが分かりません。
実際にpsqlでの問い合わせはちゃんとselect出来ます。
psql自体は-iオプションで起動はしております。

接続の確認用にif (!$dbcon)を入れてましてここでのエラーは出ませんので
接続自体は出来ていると思われます。
何かおかしい所や確認すべきところはありますでしょうか？
ご教授頂ければと思います。宜しくお願いいたします。
