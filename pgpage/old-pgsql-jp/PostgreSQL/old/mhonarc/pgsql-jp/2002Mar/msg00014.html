arai> >test=# create table テスト (試験中 integer);

テーブル名やカラム名を"(ダブルクオーテーション)で囲むと良いです。
他にも
1) CREATE TABLE TABLE (...);
と
2) CREATE TABLE "Table" (...);
では違う結果になります。
1は
CREATE TABLE table (...);
と等価になり、2は大文字と小文字を区別するようになります。
SELECTする時にも1では
SELECT * FROM table ...
で良いのに対し2では
SELECT * FROM "Table" ...
でないとアクセスできません。
