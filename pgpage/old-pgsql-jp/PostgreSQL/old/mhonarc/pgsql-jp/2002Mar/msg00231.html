例えば、以下の様なテーブルがあったとします。

テーブル名：hogehoge
creat table hogehoge(
id int primary key,
create_date date
);

このテーブルを日付順にソートする場合は、
select * from hogehoge order by create_date desc limit 20;
とすれば、取得できると思います。
