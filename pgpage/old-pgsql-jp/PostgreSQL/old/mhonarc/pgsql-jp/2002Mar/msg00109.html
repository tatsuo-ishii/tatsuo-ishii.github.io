PostgreSQLは7.2だと思います。

すでにWEBからPHP（EUC)で追加処理などを行っているテーブルを
VB6.0SP5でFlexgridに表示させています。
ソースは以下のようなものでVB上から追加した
日本語は正常に表示されます。

ソース
With flex
.Rows = rs.RecordCount + 1
.Cols = rs.Fields.Count + 1
.Row = 0
While rs.EOF = False
.Row = .Row + 1
For i = 0 To rs.Fields.Count - 1
.Col = i + 1
If IsNull(rs(i)) Then
.Text = ""
Else
.text=rs(i)
End If
Next i
rs.MoveNext
Wend
