無事、Pg.pm を使って、ＤＢアクセスが出来るようになりました。

手順を整理させて頂きました。
　./configure --with-perl
　cd src/interfaces/perl5/
　make clean
　export SRCDIR=./
　export POSTGRES_INCLUDE=/usr/local/src/postgres/postgresql-7.2/src/include/
　export POSTGRES_LIB=/usr/local/src/postgres/postgresql-7.2/src/interfaces/libpq/
　perl Makefile.PL
　make
　make test
  (rootに変わって）
　make install

　最終的に接続が完了した Pg.pm の大きさは、18086 バイト
　やはり、かなり大きくなっておりました。
