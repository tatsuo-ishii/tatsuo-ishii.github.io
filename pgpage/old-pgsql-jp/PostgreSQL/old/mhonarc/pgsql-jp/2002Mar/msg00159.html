# この手のって何度も出てきてると思うんだけど...

worms> 先日停電でシステムがダウンしたので、再度postmasterを起動したのですが
worms> 下記のようなエラーが発生して、起動できなくなりました。
worms> 
worms> FATAL: StreamServerPort: bind() failed: Address already in use
worms>         Is another postmaster already running on that port?
worms>         If not, remove socket node (/tmp/.s.PGSQL.5432) and retry.
worms> /usr/local/pgsql/bin/postmaster: cannot create UNIX stream port


メッセージを良く読んでみましょう。
If not, remove socket node (/tmp/.s.PGSQL.5432) and retry.
って書いてありますよね。
postmasterが起動していなければ、/tmp/.s.PGSQL.5432を削除してもう一度やってみて下さい。
