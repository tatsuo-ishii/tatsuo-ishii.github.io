NULL の件は、フォローがあったので、細かいことですが。

datetime も now() も、ANSI SQL99 の規格ではないので、
特別な理由がなければ、ANSI SQL 規格に従った方がよい
と思います。
# 列名 = NULL も ANSI SQL ではないです。

datetime   →   timestamp

now()      →  CURRENT_TIMESTAMP
