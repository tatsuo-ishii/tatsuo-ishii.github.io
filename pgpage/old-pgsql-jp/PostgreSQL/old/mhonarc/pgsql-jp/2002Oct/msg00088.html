始めまして、kojicと申します。

最近ようやくADSL回線（pppoe接続プロバイダ、固定IP無し）を引き、Redhat
v7.3JにてADSL接続環境を悪戦苦闘しながら構築中です。
どうしても外部にpingは出来るのですが、WEBサイトなどのIPアドレスではなくドメ
イン名でpingをすると
     ping:unknown host www.yahoo.co.jp
のようになってしまいます。
また（当然でしょうが）接続後WEBブラウザで「http://www.yahoo.co.jp」を開こう
とすると、
     An error occured while loading http://www.yahoo.co.jp:
     Unknown host www.yahoo.co.jp
となってページが開きません。

これまでの経緯としまして、Redhatv7.3Jに
１）rp-pppoe-3.3-7をインストールしADSLに接続はできました。
２）ifconfigを行って、ADSLプロバイダ側のグローバルアドレスを調べPingを打つ
とOK。
３）知っている他のグローバルIP宛にPingをしてもきちんと帰ってきてOK。
しかしながら、
４）ping www.yahoo.co.jpとすると
     ping:unknown host www.yahoo.co.jp
となってしまいます。

ちなみに、pppoe.confの設定は、
DNSTYPE=SERVER
USEPEERDNS=yes
DNS1=
DNS2=
としております。

いろいろやってみているのですが、どうしてもうまくいきません。
