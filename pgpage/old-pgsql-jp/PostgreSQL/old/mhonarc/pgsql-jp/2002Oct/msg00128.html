Postgresqlにて以下のエラーのため、データベースの作成ができません。


$ createdb
psql: connectDBStart() -- connect() failed: Connection refused

         and accepting connections on Unix socket '/tmp/.s.PGSQL.5432'?
createdb: database creation failed

ネットで調べたところ、postmasterが起動していない可能性があるとのことで
下記のコマンドで起動状態を調べました。

$ pg_ctl status
pg_ctl: postmaster is running (pid: 460)
Command line was:
/usr/local/pgsql/bin/postmaster '-S' '-i'

起動はされているのに、なぜ上記のようなエラーが出るのでしょう？
