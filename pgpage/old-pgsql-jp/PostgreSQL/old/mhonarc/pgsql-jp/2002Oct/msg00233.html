桑原です。無事postmasterを起動することができました。
本当にありがとうございました。

<生じていた問題>

$ net start PostgreSQL
に対し、エラーメッセージが表示される。

--------------------------------------
PostgreSQL サービスを開始します.
PostgreSQL サービスを開始できませんでした。

サービスはエラーを報告しませんでした。

NET HELPMSG 3534 と入力すると、より詳しい説明が得られます。
--------------------------------------

/var/log/PostgreSQL.logへの出力
--------------------------------------
/usr/bin/postmaster does not know where to find the database system data.
You must specify the directory that contains the database system
either by specifying the -D invocation option or by setting the
PGDATA environment variable.
--------------------------------------

<解決手順>
1) $ cd /
2) $ touch .bashrc
3) .bashrcを編集

   PATH=$PATH:/usr/pgsql/bin:/usr/local/bin
   PGDATA=/usr/local/pgsql/data
   PGLIB=/usr/local/pgsql/lib
   LD_LIBRARY_PATH=/usr/local/pgsql/lib:/usr/local/lib
   export LD_LIBRARY_PATH PATH PGDATA PGLIB

4) $ source /.bashrc
5) $ ipc-daemon.exe &
6) $ postmaster -i &
