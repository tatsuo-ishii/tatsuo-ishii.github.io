プログラムで工夫します。
円マークとはバックスラッシュのことですが、
この機能はpostgreSQL独自の機能ということで、
すなおに受け入れて、プログラムで工夫することにします。

参考
サーバー・ソフトははPerlでPgモジュールを使っています。
クライアント・ソフトはDelphiで作成しています。

 ------------------------------------------------------------
> template1=# create table hoge (aaa varchar(128));
> CREATE
> template1=# insert into hoge values ('〓〓');
> INSERT 8621490 1
> template1=# select * from hoge;
>  aaa
> -----
>  〓
> (1 row)
> 
> ------------------------------------------------------------
>
