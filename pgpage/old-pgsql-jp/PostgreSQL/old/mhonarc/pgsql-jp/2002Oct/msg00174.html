> 一時表は、カタログ上では、pg_tmp で始まる表名になるようです。

s/pg_tmp/pg_temp/  です。

PostgreSQL 7.2 で試した時は、pg_temp_NNNN_mm とういう
ような pg_class.relname になりました。
# NNNN   mm   の部分は、数字のようです。
NNNN の部分の命名規則は未調査ですが、同一セッション内
で、複数の一時表を create すると、NNNN は、固定で、
_mm の部分が、0, 1, 2, 3 と連番になるようです。
当然のことながら、セッションが異なれば NNNN の部分も
異なります。

（メタデーター表示例）
 列番 |      表名      |  列名  |  型名   | 長さ | 小数 | NULL | DEF
------+----------------+--------+---------+------+------+------+-----
    1 | pg_temp_2318_0 | juid   | int4    |      |      | f    | f
    2 | pg_temp_2318_0 | shimei | text    |      |      | f    | f
    3 | pg_temp_2318_0 | jugc   | int4    |      |      | f    | f
    4 | pg_temp_2318_0 | juqty  | numeric |   12 | 0    | f    | f
    5 | pg_temp_2318_0 | juamt  | numeric |   17 | 0    | f    | f
