太田ともうします。
;; Windows2000上でpostgreSQLをインストールしましたが、
;; postmasterを起動することができません。
;; エラーメッセージは以下の通りです。
;; 
;; FATAL 1:  could not read configuration file `postgresql.conf': Invalid argument

「could not read configuration file」は
src/backend/utils/misc/guc-file.c: ProcessConfigFile()
    fp = AllocateFile(filename, "r");
    if (!fp)
    {
		free(filename);
        /* File not found is fine */
        if (errno != ENOENT)
            elog(elevel, "could not read configuration file `" CONFIG_FILENAME "': %s", strerror(errno));
		return;
    }
なので、フォーマットの問題では無いはずです。
おそらくサンプルのファイルをそのまま使ってもエラーになると思います。
では何の問題かというとWin2KのpostgreSQLは使ったことが無いのでわかりません。
