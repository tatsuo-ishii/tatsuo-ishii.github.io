  • From: Hiroyuki Hanai <hanai@venusux1.kek.jp>

大阪大学の花井です。

普段は HP-UX 上で postgres95 を動かしているんですが、みんなが解析など
をしているマシンをグチャグチャいじくると文句言われるのと、D論の逃避の
ため自分の BSD/OS 2.0.1 で postgre95 を make してみました。

src/Makefile.global を見ると、

#  The name of the port.  Valid choices are:
#  ................
#       BSD44_derived   -       OSs derived from 4.4-lite BSD
#                               e.g. NetBSD, FreeBSD and BSD/OS

などと書いてあるので、「おぉ！ BSD/OS でも動くんやんけ！！」と気軽にやっ
てみたのがそもそもの間違いでした。
＃ 疲れた．．．．

結論としては、make出来て、src/tutorial/basics.sql なんかもちゃんと \i 
出来るので恐らく動いているんであろうと思われます。

本業があることだし、「取り敢えずmake出来たらええんや！」という考
えのもとにグチャグチャやったら出来たのですが、もし BSD/OS で pg95 を動
かしたいけどmakeでけへんなぁという方がいらっしゃいましたら、僕にメール
をくれたらその時に作ったメモをお送りします。
希望者がそれなりにいらっしゃる場合はこのＭＬに流そうと思います。
＃ けどちょっと恥ずかしい．．．．

あっ、それから、
BSD/OS で簡単に pg95 が作れた方がいらっしゃいましたら、是非教えて下さ
い。
＃ 皆さんの自己紹介なんかを読んでると、linuxを使ってらっしゃる方はたく
＃ さんいるみたいだけど、BSD/OS を使ってる人はいないかな？
