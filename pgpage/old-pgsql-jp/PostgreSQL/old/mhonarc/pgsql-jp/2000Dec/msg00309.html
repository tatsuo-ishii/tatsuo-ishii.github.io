At 13:59 +0900 2000.12.22, 松本豊 wrote:
>松本＠川崎市在住です。
>
>date の null の表現方法について教えてください。
>
>次のようなテーブルで、
>
>mydb=> create table dtest ( num int , dd date );
>CREATE
>mydb=> insert into dtest values ( 1 , '2000/7/15'::date ) ;
>INSERT 71702 1
>mydb=> insert into dtest values ( 2 ) ;
>INSERT 71703 1
>mydb=> select * from dtest ;
> num |     dd
>-----+------------
>   1 | 2000-07-15
>   2 |
>(2 rows)
>
>とすると num = 2　の dd は何が入っているのでしょうか？
>dd に値が入っていないものを select しようとして以下の方法を
>試しましたがだめです。
>
>mydb=> select * from dtest where dd = null::date ;
> num | dd
>-----+----
>(0 rows)

自分の環境では、次のようにしていけました。
[PostgreSQL 6.5.1 on sparc-sun-solaris2.6, compiled by gcc 2.7.2.1]

tmp=> select * from dtest where dd = null;
num|dd
---+--
  2|
(1 row)
