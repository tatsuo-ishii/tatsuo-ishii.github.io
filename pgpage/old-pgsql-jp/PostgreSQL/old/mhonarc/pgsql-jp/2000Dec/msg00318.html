すみません、私は頭堅いのかなぁ。否定の書き方が分かりません。

mydb=> select * from dtest where dd != null ;
 num | dd
-----+----
(0 rows)

mydb=> select * from dtest where dd <> null ;
 num | dd
-----+----
(0 rows)

mydb=> select * from dtest where dd > null ;
 num | dd
-----+----
(0 rows)

mydb=> select * from dtest where dd > 0 ;
ERROR:  Unable to identify an operator '>' for types 'date' and 'int4'
        You will have to retype this query using an explicit cast
mydb=> select * from dtest where dd > 0::date ;
ERROR:  Bad date external representation '0'
