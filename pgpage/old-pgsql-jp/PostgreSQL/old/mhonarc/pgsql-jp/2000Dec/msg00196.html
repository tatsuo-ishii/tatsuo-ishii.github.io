私のところでは、以下のように何ら問題はありません。
ただし、PostgreSQL ver 7.0.2 ですので、別の問題かもしれません。
（7.0.3 は数ヶ月前には無かったので、以前からの懸案と考えました）

senshu=> CREATE TABLE test_tbl (number int4, cname varchar(80));
CREATE
senshu=> INSERT INTO test_tbl values (1, '奉');
INSERT 1273546 1
senshu=> INSERT INTO test_tbl values (1, '薬');
INSERT 1273547 1
senshu=> select * from test_tbl ;
 number | cname
--------+-------
      1 | 薬
      1 | 奉
(2 rows)

おそらく、コンパイルオプションが不適切なのだと思います。
どんな指定でコンパイルしたのかを教えてください。
