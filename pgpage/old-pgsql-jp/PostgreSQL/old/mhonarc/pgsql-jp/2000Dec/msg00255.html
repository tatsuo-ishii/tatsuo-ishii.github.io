　byte[] byte = "あいうえお".getBytes();
として得られたバイト配列をbyteaを使ってＤＢに格納しようと思ってます。

ＤＢは
create table (index varchar, byte bytea);
ＪＤＢＣ2.0を使ってます。

byte[] byte = "あいうえお".getBytes();

(?,?)");

ps.setBytes(2, byte);
ps.close();
っとしてます。
