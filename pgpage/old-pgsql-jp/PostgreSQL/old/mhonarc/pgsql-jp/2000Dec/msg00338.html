  以前、VMware 上の Linux で postgreSQL を動かしたらどうか、
質問をしましたが、冬の休暇中にやっておきたい事があり、Win2K
+ Vine 2.0 + PostgreSQL 7.1beta1 を Vaio Z505GR/K (650Mhz/
256MB) に入れて見ました。

  結論として、マルチユーザで使うのには問題がある(遅い)が、
個人使用或は移動開発環境としては使えそう、と考えられました。
スピードも思ったよりは遅くありませんでした(昔の Pentium 100Mhz
の時よりは早い？ー当り前？)。

  勿論、巨大なDBはだめですが、私の所で使っている物は数百MB
程度なので、使う為に遅いラインを繋ぐ事を考えると、こちらの
方にメリットがあります。  また、入力や出力分析に excel を
使っているので、こちらの方が大変重宝します。

  更に、apache を入れて Win2K 上の IE で繋いで、php3 のデバッグ
に使おうと考えています。

  usogres などで、同期を取るようにすると面白い使い方が考えら
れるのではないかと思います。
