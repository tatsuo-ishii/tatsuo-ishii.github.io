login shell は、/etc/passwd に記述してあります。
但し、/etc/passwd は、編集しそこなうと、全くログイン
できなくなって大変なことになるので、慎重に変更してく
ださい。

私ら古い人は、vipw を使いますが、多分、わかりやす
いツールがあるかもしれません。

ところで、/usr/bin/bash と /usr/local/bin/bash と
は、使い分ける必要があるんでしょうか？。

Linux では、sh と bash は同じもので、これが動か
ないとシステムが動かなくなる危険があるので、
私は、慎重派なんで、いままで使っていた bash は、
そのまま残して、login shell にも使っています。
新しい bash を 旧 bash と同じ DIRECTORY に bash2
という名前(これは、Vine RPM の標準の名前)でおいて
あります。

必要により、bash2, source XXXX で、必要な設定をおこ
なうようにしています。
