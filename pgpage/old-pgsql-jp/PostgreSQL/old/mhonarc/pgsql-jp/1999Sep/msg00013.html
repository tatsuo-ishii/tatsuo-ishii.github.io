Oracle も PostgreSQL も勉強中なので、あまり詳しくありませんが、
方法云々の前に、ちょっと気になった点だけ述べさせていただきます。

PostgreSQL で Null のレコードが返るのは、ごく一般的な動作だと思います。
（集計関数は１つの値を返す）
むしろ、レコードが返ってこないのは、おかしくありませんか？
私なら、逆の質問になります。（Oracle に対して）

Oracle は、私の環境（WGS 8.0.5）と、ODBC 8.00.05.00、Access97 の
組み合わせですが、パススルーで、ちゃんと、Null レコードを返しています。
これは、SQLPlus を使っても同様でした。
ODBC を含めた、お使いの Oracle の動作を、いま一度、確認されたほうが
良いと思います。後々、問題になる気がします。
