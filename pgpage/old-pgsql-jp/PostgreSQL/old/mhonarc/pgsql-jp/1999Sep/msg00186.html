> PHPを つかって TABLEの存在確認する 方法が あるのでしょうか？

ＤＢ名がわかっているなら、そのＤＢに connect してか
ら、次の SQL で、表名の一覧がでます。表名もわかって
いるなら、検索条件に、  WHERE relname = ＄hogetable 
を追加すればいいです。

select relname from pg_class where relkind = 'r';


ＤＢ名の一覧( psql -l )なら、template1 などどれで
もいいからＤＢに connect して、次の SQL 文。

select datname from pg_database;

こういうことでよろしいのでしょうか。


ところで、ＤＢ  template1 というのは、必ずあるも
のなんでしょうか？。 psql -l 相当のＤＢ名一覧表を
SQL 文で、出力するには、どれでもいいから任意のＤＢに
connect しなければいけないと思うのですが、そのときの
ＤＢ名は、どうやって探せばいいのでしょうか？ 
base/ DIRECTORY の下を探すのかな？。
