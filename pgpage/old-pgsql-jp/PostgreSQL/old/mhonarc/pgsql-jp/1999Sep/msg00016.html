はじめまして　中畑と申します．
VBもPostgreSQLも最近始めたばかりの初心者です．よろしくお願いします．

早速ですが，VB6とPostgreSQLとの接続がうまくいかず悩んでいます．
見よう見まねで以下の様なソースを書いたのですが，
*が付いているcn.Openで
”[Microsoft][ODBC Driver Manager]データソース名および指定された規定の
ドライバが見つかりません．”
とのエラーが出て接続できません．

ちなみに，ODBCデータソースはコントロールパネルのODBCデータソースで設定し
ており，ACCESS97からは同様の設定でPostgreSQLに接続が成功しています．

そもそもこういう風には接続することはできないのでしょうか．
お忙しいとは思いますが，よろしくお願いします．

・WindowsNT4.0 SP4
・VB6 SP3
・ADO 2.0
・PostgreSQL 6.5.1
・PostgreSQL ODBC Driver 日本語版（1999/08/26版） 


Private Sub Form_Load()
    Dim cn As New ADODB.Connection
    Dim rs As New ADODB.Recordset
    Dim cString As String
    Dim SQL As String
    
    Set cn = New ADODB.Connection
    cString = "DNS=test;UID=testid;PWD=testpass;DATABASE=test"
    
    With cn
       .ConnectionString = cString
*      .Open
    End With
    
    SQL = "SELECT * From test_tab"
    
    Set rs = New ADODB.Recordset
    rs.Open SQL, cn, adOpenForwardOnly, adLockReadOnly, adCmdText
End Sub
