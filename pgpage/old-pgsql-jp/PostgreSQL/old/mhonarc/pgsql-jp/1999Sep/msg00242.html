石井です。

9/15に大阪で開催された LinuxWest に行って PostgreSQL とユーザ会の宣伝
をしてきました。ユーザ会としてのプレゼンテーションは、ユーザ会結成以来
始めてです。

当日は台風が会場付近を通過する最悪のコンディションでしたが、心配してい
たよりは多くの人が集まりました。プレゼンテーションの場は日本 Linux 協
会のブースの中で、観客の居場所が通路と言う状況もあり、黒山の人たかりに
なりました。

時間の関係(20分)でプレゼンテーションは極簡単なものでしたが、それでも境
田さんの pgbash 紹介と簡単なデモ(本当に簡単！)を動かして見せることがで
きました。プレゼンが終った後は、持参したユーザ会の CD-ROM 約80枚を配付
しました。また「シーラカンス本」1冊をJLAの抽選大会に寄付して来ました。

今回は関西地区のエネルギーを肌で感じる良い機会でした。MLメンバー、ユー
ザ会のいろんな人にも会えました。

# 谷田さん、パッチ、ありがとうございました。頂いたパッチを反映した 
# pgbench を間もなくリリースします。

なお、当日のOHPは 
http://www.sra.co.jp/people/t-ishii/PostgreSQL/LinuxWest/index.htm に
あります。また pgbash のプログラムは付属のような超簡単 CGI プログラム
です。
---
Tatsuo Ishii

---------------------------------------------------------------------
#! /usr/local/bin/bash
echo "Content-type: text/html"
echo
echo "<html>"
echo "<header><title>pgbash demo</title></head>"
echo "<body bgcolor=white>"
enable -f /usr/local/pgsql/lib/libpgbash.so  exec_sql
exec_sql -q "connect to postal"
echo "<h1>pgbash による郵便番号検索</h1>"
exec_sql -H -O 'BORDER' "select oldcode as 旧郵便番号,newcode as 新郵便番号,city as 市町村名, town as 町域名 from postal where oldcode = '530' limit 100"
exec_sql -q "disconnect all"
echo "</body>"
echo "</html>"
