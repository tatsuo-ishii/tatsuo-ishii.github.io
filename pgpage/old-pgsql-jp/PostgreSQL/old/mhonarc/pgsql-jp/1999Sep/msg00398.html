> プロンプトが、「***=>」から「***->」や「***'>」になり、

> こういった初歩の初歩のFAQのページがありましたら、

これは、SQL 文終了の  ; （セミコロン)を入力してい
ないからでしょう。  xxx-> は、継続行のプロンプト
です。

＾D （ CTRL D ）で、psql が終了します。

入力済みの SQL 文を実行してよいのであれば、 ;  だ
け入力すれば、動きます。 

pgbash なら、 ; （セミコロン）省略できます。でも、
引用符が必要になります。
