AppletViewer では OK とのことですが、ブラウザはどういう環境
で動かしているのでしょうか。AppletViewer を操作している環境で
ブラウザを立ち上げても接続できないということでしょうか。
もしそうであるか或いは上記のインスタンスで解決したのであれば
以下は無視して下さい。

アプレットはブラウザによって自分のマシンにダウンロードされ、自分の
マシン上の Java VM を使って実行されます。したがって JDBC ドライバ
も自分のマシンにあるか、アプレットと JDBC ドライバを jar パックして
一緒にダウンロードしないと DB との接続はできません。
