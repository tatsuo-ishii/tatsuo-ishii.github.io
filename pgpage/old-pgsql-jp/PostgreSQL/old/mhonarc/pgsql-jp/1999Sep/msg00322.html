jar を展開する必要はありません。APPLET タグ中の ARCHIVE タグで
jar ファイルを指定して下さい。

> ( 使用しているソース )
> --------------------------------------------------------------  eg1.java
> START
> import java.io.*;
> import java.sql.*;
> import java.text.*;
> 
> public class eg1
> {
>   Connection connection;
>   Statement  statement;
>   ResultSet  resultset;
> 
>   public eg1() throws ClassNotFoundException, FileNotFoundException,
> IOException, SQLException
>   {
>     String url = "jdbc:postgresql:template1";

ホスト名が抜けています。
PostgreSQL の JDBC ドライバでの URL の指定は
「jdbc:postgresql://ホスト名/データベース名」
です。
