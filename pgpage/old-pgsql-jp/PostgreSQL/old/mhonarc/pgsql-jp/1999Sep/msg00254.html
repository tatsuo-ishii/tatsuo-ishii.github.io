PostgreSQL JDBC Driver 6.5.2
   Peter Mount - September 16th 1999, 16:05 EST 

   This package provides a JDBC Driver for the PostgreSQL DBS.

   Changes: This is a bug fix release. The main fix is to how the driver determines
   what version (JDBC1.2 or JDBC2) to run. Originally in 6.5, this was done by testing
   the JVM version, but not all JVM's are equal, so it wouldn't work. This is now fixed.
   Also, transactionIsolation, getPrimaryKeys(), and some speed improvements have
   been made.
