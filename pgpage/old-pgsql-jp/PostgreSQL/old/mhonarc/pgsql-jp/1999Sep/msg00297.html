  東野です．

  環境は Postgresql 6.5.1 + PHP4.0b2 on Vine Linux 1.1 です．

  PHP についての質問です．現在 PHP で class を使っていますが，
クラスのオブジェクトを form で渡すことは可能なのでしょうか？
といいますのも，

---a.php---
...
<form method=POST action="b.php">
<?
  $object new myClass();
  echo "<input type=hidden name=\"class_object\" value=$object>";
?>
  ...
</form>
...
------------

という php ファイルがあって，

--- b.php ---
...
<?
$class_object->member_function();
?>
...
-------------

とすると，

Fatal error: Call to a member function on a non-object in /home/..... line xx

というエラーになります．要は class_object はオブジェクトではないよと
いうことみたいです．ただ，実際に，echo "$class_object"; のように
書いてみると Object というように Web には表示されます．

  オブジェクトでのデータの渡し方について参考になる文書またはアドバイス
などありましたら教えて頂けないでしょうか．
