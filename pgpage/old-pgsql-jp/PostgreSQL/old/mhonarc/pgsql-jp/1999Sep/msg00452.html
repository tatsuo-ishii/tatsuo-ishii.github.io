全パラメーターを書いていただかないとわからないのですが、
ひょっとして  -o と シングルクォートが抜けているとか。
タイプミスでしょうけど、-s 小文字でなく −S 大文字です。

/etc/rc.d/init.d/postgresql での記述例です。

su - postgres -c "$Postmaster -i -S -D/ii/db/pg \
	-B 2000 -o '-S 32000 -F'"

マニュアルからの引用(参考迄)
.BR "-S" " kbytes"
Specifies the amount of memory to be used by internal sorts and hashes
before resorting to temporary disk files.  The value is specified in
kilobytes, and defaults to 512 kilobytes.  Note that for a complex query,
several sorts and/or hashes might be running in parallel, and each one
will be allowed to use as much as -S kilobytes before it starts to put
data into temporary files.
