お世話になります。船木＠秋田県立図書館です。
テーブルの更新についてご教示下さい。
やりたいことは「テーブルを別のテーブルのデータで更新したい」
というものです。
既存のテーブルがあり、その中の、あるタプルについて更新するのは
updateコマンドで１つ１つやれば可能であることは分かりますが
ある程度数がまとまったデータがASCIIファイルになっており、
それを用いてテーブルを更新する場合、更新データ（ASCIIファイル）
がテーブルに入れるべきデータをすべて含む（つまり部分更新でなく
データを完全に入れ替える）とすると、いまのところ

１）ASCIIファイルをテンポラリテーブルにコピー
２）テンポラリテーブルを用いてキーを検索
３）そのキーに該当するタプルをテーブルから削除
４）ASCIIファイルをテーブルにコピー
５）１）でつくったテンポラリテーブルを削除

と言うふうになると思いますが、できれば４）の段階で
テンポラリファイルを直接既存のテーブルにくっつけられれば
５）が必要なくなり、手間がかからないのでいいなあと考えたわけです。
これを実現させる方法としてJOINがあるとおもいますが
JOINはサポートされていなかったと記憶しています。

＃じつはJOINがサポートされていたとしても１）〜３）は同じで
＃４）が
＃ select into temp2 from  table .... join
＃ select into temp2 from temp_table....
＃となって drop table,drop temp_table そして
＃alter table temp2 ...という感じになるのかなあと考えてまして、
＃ほとんど同じかなぁと思ったりもしています。

なにか他に効率がよいと考えられる方法はありますでしょうか？
よろしくご教示下さい。
