こんにちは、齋藤＠寺岡精工です。

NUMERIC型での索引についてなのですが、
石井さんの書かれた 「改訂版 ＰｏｓｔｇｒｅＳＱＬ完全攻略ガイド」に
「numeric/decimalは、今のところ索引が作れないという欠点がある」
とありますが、
私の所の環境
Redhat Linux 6.1
PostgreSQL 7.0.2
で
create table a(a numeric(5,0) primary key);
explain select * from a where a='1';
と行うと、「Index Scan using・・・」となり
索引が出来ているようです。
これは本当に出来ているのでしょうか？
それとも、
「numeric/decimalは、今のところ索引が作れないという欠点がある」
は、なにか別のことを意味するのでしょうか？（それとも7.0.xになって直った？）

皆様のご意見宜しくお願いいたします。
