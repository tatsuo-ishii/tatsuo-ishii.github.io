[pgsql-jp 16447] 環境変数 PAGER に more とか less を入れておけば

psql で何とか more っぽく表示してくれないかな〜
等と考えながら、「地理データ型」のML履歴を
調べていると・・・
[pgsql-jp 7511]に、師匠の 石井氏が件名の
「環境変数 PAGER に more とか less を入れて」云々の
コメントがあり、早速、利用せねばと考えております。
ところで小生は、座っている場所で利用しているPCが
MS-Windowsである関係で、MS標準の telnet で psql
を利用したりしていますが、環境変数の設定で随分と
利用しやすくなりました。
(MS-telnetではスクロールができない)

環境変数 PAGER の設定で無用となりましたが、
それでも psql の「￥」から始めるコマンドの
結果をファイルに保存するための簡単なシェルを
UNIX初心者のためにUpしておきたいと思います

#!/bin/sh
if [ "$1" -a "$2" ]; then
 if [ "$3" ]; then
 cat <<EOF | psql $1 > $3
\\$2
EOF
 else
 cat <<EOF | psql $1 | grep -v "^EOF$" | more
\\$2
EOF
 fi
else
 THIS=`echo $0 | sed 's/\.\///'`
 cat <<EOF

usage:
	${THIS} dbname command [filename]

 example: ${THIS} test "dd date"              : \dd date | more
	  ${THIS} test "dd date" dd_date.dump : \dd date > dd_date.dump
	  ${THIS} test "h *"                  : \h * | more
	  ${THIS} test "h *" help.dump        : \h * > help.dump

EOF
fi
ここまで　^J~;
