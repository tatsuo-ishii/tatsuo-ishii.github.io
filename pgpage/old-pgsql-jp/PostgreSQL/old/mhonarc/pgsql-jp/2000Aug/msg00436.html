はまもと です。
前から、pgsql-jpに参加はしていたのですが、書き込むのは初めてです。

PostgreSQLの管理用ツールを調べようと思い立ちまして、検索エンジンや、
pgsql-jpの過去ログなどを漁って、以下のツールを探してきました。他に、
データベース管理に便利なツール情報をご存知の方、教えていただけな
いでしょうか。

また、実際に使っていらっしゃる方がいらっしゃいましたら、その感想な
ども聞かせていただけないでしょうか。

よろしくお願いします。

●pgMan : PostgreSQL Manager
http://www.lisonal.com/itools/pgMan/index.html
・データベースの起動・停止
・データベースの作成・削除
・テーブルの作成・削除
・テーブルカラムの追加
・レコードの閲覧・追加・変更・削除

●WUI
http://www.hde-wui.com/jp/
http://www.hde-wui.com/jp/download/index.html#wui-mod_postgresql
HDEの組み込みモジュール「wui-mod_postgresql-0.51-1.i386.rpm」
テーブルの編集とかはできない。
・バージョンの表示
・-iオプション（外部からの接続許可）のON/OFF
・pg_hba.confの、接続可能ホストの記述
・PostgreSQLユーザーの一覧、削除、作成（DB作成権限、ユーザー作成権限含む）
・Databaseの一覧、削除、作成

●PgAccess
http://www.flex.ro/pgaccess/
Tcl/Tkで書かれたGUIベースの対話型プログラム。PostgreSQLの多くの操作をマウス
を使ったオペレーションが可能。
○Windows日本語環境でPgAccessを使うには（PostgreSQL関連情報）
http://www.interwiz.koganei.tokyo.jp/software/PostgreSQL/index.html

●pgAdmin
http://www.vale-it.demon.co.uk/freeware/index.html
Windows環境で動作するデータ・ベース設計ツール。

●Data Explorer for SQL
http://tech.millto.net/~maacyan/php3/Exp/dataexpforsql.html
RDBMS(OracleとPostgresSQL)のデータをブラウズし操作（抑止可能）するツール
・データ分析機能 （簡易OLAP的な考え方をもった検索ツール） 
・上記と連動したデータ操作(メンテナンス)機能 
・データベース・ブラウズ機能 
・SQL文バッチ処理機能 

●データベース職人
http://www.cybertech.co.jp/db/index.html
有料 98,000円（税別）
