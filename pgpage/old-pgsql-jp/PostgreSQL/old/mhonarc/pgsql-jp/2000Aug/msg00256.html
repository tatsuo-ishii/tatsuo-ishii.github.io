手持ちの書籍って何を見たのですか？
本の種類やレベルによっては「カーネル再構築」は
知ってることを前提のものもあると思うし
逆に、この本はこんな事も書いてないって意味で

(たいがいの場合　バージョン上がれば使えるオプションが増えている)

cd /usr/src/sys/i386/conf
cp GENERIC hoge

hogeを編集して

./config hoge
cd ../../compile/hoge
make depend
make
make install
