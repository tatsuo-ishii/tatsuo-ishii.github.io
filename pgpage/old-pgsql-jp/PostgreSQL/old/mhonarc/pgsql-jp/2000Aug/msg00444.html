吉川です。

> PostgreSQL-6.5.2 を使っています。
> 
> テーブルの中身（すべて半角英数字）を、大文字、小文字に関係なく
> selectすることはできるのでしょうか？
> 不躾で申し訳ないのですが、よろしくお願いします。

upper関数でテーブルに格納された文字列を全部大文字にしてしまうか、
lower関数で全部小文字にしてから比較すればいいんじゃないでしょうか。

全部大文字に統一して比較するとき
select * from table where upper (str) = 'HOGEHOGE';

全部小文字に統一して比較するとき
select * from table where lower (str) = 'hogehoge';
