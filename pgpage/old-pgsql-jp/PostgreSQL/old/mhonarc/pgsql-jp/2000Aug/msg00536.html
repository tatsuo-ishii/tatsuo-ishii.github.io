こんばんわ、以前手ブル設計について質問をさしていただいた宮尾と申します。
あれから何通かテーブルの事についてメールが行き来していたので
僕への解答かなぁと考えましたが、メールの件名が違っていたり、
内容が違うので「あー違う」と思って読み流してしまいました。
もし、下記のテーブルを配列で作る方法が解る方がいっらしゃいましたら
教えて頂けないでしょうか？
配列のテーブルの考え方が解りません。

例えば下記のようなかんじの場合。

create table test_tab
(		
	n0	char(2),
	n1	char(2),
	n2	char(3),
	n3	char(2),
	n4	char(2),
	ac01	varchar(128),
	ac02	varchar(120),
	ac03	varchar(64),
	ac01	int,
	ac09	int,
	ac04	int,
	ac06	char(128)
);

アドバイスお願いします。

件名をＲＥ：で返していただけると大変嬉しく思います。
