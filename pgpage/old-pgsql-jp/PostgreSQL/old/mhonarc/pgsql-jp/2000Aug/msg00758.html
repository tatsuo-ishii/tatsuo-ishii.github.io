> >> 
> >> 何故このようなおかしな現象が起きるのでしょうか？
(略)
> 基本の話ですが、
> select * from test order by id;
> で実現できます。

MS-SQL(MSDE)や、MS-Access では、プライマリキーでソートされたものが返って
きますので、「order by id」が不要( id がプライマリキーの場合)です。
そのため、MS に慣れてしまっていると、psql でクエリを実行した時にレコード
がソートされていないのを見て、「おかしな現象」と思ってしまったのではない
でしょうか？
私は最初、違和感を覚えましたから・・・。(MS に慣れてしまってて)
