まずは PostgreSQL のマニュアルを見ましょうよ。

PostgreSQL のソースを展開したディレクトリの doc というディレクトリの
下に PostgreSQL の公開されたドキュメントはすべてあります。
ここを見ましょう。

マニュアル翻訳プロジェクトが扱っているのは 6.5 ベースなので、7.0 で
追加された情報は載ってないでしょうね。

to_char と to_number に限って言えば、これらは Oracle 互換関数なので、
Oracle のそれと使い方はほぼ同じです。
# まったく同じかどうかまでは検証してないけど。
