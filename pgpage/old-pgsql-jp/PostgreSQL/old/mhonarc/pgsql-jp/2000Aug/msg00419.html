takeda> という、大ボケな疑問にぶち当たりました。

takeda> $conn = Pg::connectdb("dbname=test");
takeda> $result = $conn->exec("select nedan from shinamono where hinmei = 'りんご'");

シーラカンス本(改訂版？)の search.pl を参照されているのでしたら、

$conn - Pg::connectdb("dbname=$dbname");
if (PGRES_CONNECTION_OK ne $conn->status) {
    printf("...");
    exit;
}

というところ("..."は面倒なので略しただけです)が接続できたかどうかのチェッ
ク、

$result = $conn->exec($qstr);
if ($result->resultStatus ne PGRES_TUPLES_OK) {
   printf("...");
   exit;
}

というところが、クエリが成功したかどうかのチェックになっています。

p. 312, 313 にも書いてありますから良く読まれてください。
