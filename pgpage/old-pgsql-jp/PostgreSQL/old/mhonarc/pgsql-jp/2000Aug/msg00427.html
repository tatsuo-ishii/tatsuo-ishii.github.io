[pgsql-jp 16813] 初めまして参照整合性エラーが返らない、日本語ODBC使用

  • From: 中西　健<nakanisi@mahoroba.ne.jp>
  • Date: Mon, 21 Aug 2000 18:22:34 +0900
　初めまして
　ODBC経由の場合、参照整合性に違反するデータを追加してもでエラーが返りませ
ん。PrimaryKeyの重複は正しくエラーが返ってきます。psqlで同様の操作をするとエ
ラーメッセージを返します。参照整合性が崩れるわけではありませんがアプリケー
ション側でエラートラップできないのが辛いところです。

スクリプト
  create table master(code int2 primary key, name varchar(10));
  create table data(id int4 primary key, code int2 references master(code)
on update cascade,mount int4);
  insert into data (id,code,mount) values(1,1,1); <- エラーが出るはずだが出
ない

環境
　postgreSQL V7.0.2
　日本語ODBCDriver 6.50.0000
　サーバ TurboLinux server v6.0
  クライアント Windows98,Access2000,Access以外でも同様

　以上ご報告いたします。勘違い等があればご指摘ください。
