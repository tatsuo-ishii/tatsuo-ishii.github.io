【環境】
【サーバー側】
    ・RedHat Linux6.2 日本語版

石井達夫氏のPostgreSQLを教科書とし、お蔭様でMSアクセス上から
ODBC経由でPostgreSQL上のDBにアクセスするところまでいきました。

M$アクセス上で日本語もバッチリ表示され、さて、VBAに挑戦しようか
と思ったのですが問題発生しました。

M$アクセス上で新規レコードを追加しようとすると、
”＃Ｄｅｌｅｔｅｄ”と表示され、追加したレコードが表示されません。

一度ウインドウを閉じ、次に開けば正しい内容が日本語表示されるのですが、
美しくありませんし、不便です。この#Delete表示問題は私だけですか。

Delete表示問題について、わずかでも情報あればご教示願います。
あるいは、もしかしてこれは仕様ですか。

例）

 | KONNA    | KANJI    |
 | AAAAAAAA | 111111   |
 | BBBBBBBB | 222222   |
 | #Deleted | #Deleted | ← レコードを追加すると、
*|          |          |    このようにDeleted表示されます。
