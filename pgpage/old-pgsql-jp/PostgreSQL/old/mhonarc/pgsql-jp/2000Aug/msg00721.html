 >      select name, '1' as ID from table_a

 >      order by ID;

 >      
>      エラーメッセージは、
>      「Unable to identify a binary operator '<' for types
>           unknown and unknown」と出ます。

>      代替できるものはありますでしょうか？
     
IDであるところの '1','2'をシングルクォーテーションで囲まずに、
     
select name, 1 as ID from table_a
union all
select name, 2 as ID from table_b
order by ID;
     
…としてみてはいかがでしょう。
