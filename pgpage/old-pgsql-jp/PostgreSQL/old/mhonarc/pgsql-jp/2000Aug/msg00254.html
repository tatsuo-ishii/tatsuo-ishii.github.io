PostgreSQLネタというよりFreeBSDネタなのですが、
FreeBSD 3.2-RELEASE(古いバージョンのFreeBSDは別）はGENERICカーネル
に標準で実装されています。
GENERICカーネルは、
/usr/src/sys/i386/conf/GENERIC        --PC/AT kernel config
/usr/src/sys/pc98/conf/GENERIC98    -- PC98 kernel config
を参照。

また、共有メモリ、セマフォ等のパラメータを増やしたいのであれば、
/usr/src/sys/i386/conf/LINT に設定可能なパラメータが記載されています。
同時接続ユーザー数を増やしたい場合は共有メモリなどこれらののパラメータ
を増やしてカーネルを再構築するとよいでしょう。

コンフィグレーションファイルができたなら再構築の方法は至って簡単なので、
http://www.jp.freebsd.org/www.freebsd.org/ja/handbook/kernelconfig-building.
html
を参照するといいでしょう。

また、FreeBSDのサイトではメーリングリストの検索もできますのでご利用くださ
い。
もし、あなたがFreeBSDについてもっと知りたいならば、メーリングリストに参加
するのもよいでしょう。
http://www.jp.freebsd.org/
