[pgsql-jp 16738] Re: 年と月で範囲指定をするには？

片岡＠インターウィズです。

> 細谷と申します。
>
> PostgreSQL7.0.2 を使っています。
>
> date型のデータを年月日の範囲指定ではなく、年月だけの範囲指定で
> select することはできるのでしょうか？

> select * from table1 where buy_date between '2000-02-01' and '2000-04-30';
> というように、「日」まで指定するのではなく、年と月だけでやりたい
> のですが…。

  「日」を指定したくないということの真意は、きっと各月が何日まであるかを考慮
したくないからだと思うのですが、実際のところどうなのでしょう…。
  もしもそうであるなら、以下の方法があります。

  select * from table1 where buy_date >= '2000-02-01' and buy_date <
'2000-04-01' + '1 month'::interval;

  こうすれば「日」のところは常に“01”固定でOKです。

  テストしていないのでもしかしたらミスしているかもしれません（date型へのキャ
ストを追加する必要があるかもしれません）が、私はtimestamp型の時にこの手をよ
く使っていますし、date型にも十分応用できると思います。

  他の方が紹介されているdate_partなどを利用する方法も良いと思いますが、デー
タ件数が多い場合などにはパフォーマンス上の問題が発生するかもしれません。
