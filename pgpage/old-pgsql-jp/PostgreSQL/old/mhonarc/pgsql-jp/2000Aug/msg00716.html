田中靖宏＠数理計画です。

Tue, 29 Aug 2000 14:54:57 +0900 : chiba-n@honsha.meidensha.co.jpさん wrote:
>      ******************************************
>      select name, '1' as ID from table_a
>      union all
>      select name, '2' as ID from table_b
>      order by ID;
>      ******************************************
>      
>      エラーメッセージは、
>      「Unable to identify a binary operator '<' for types
>           unknown and unknown」と出ます。
>      
>      もともと、PSQLではできないものなのでしょうか？
>      代替できるものはありますでしょうか？

IDであるところの '1','2'をシングルクォーテーションで囲まずに、

select name, 1 as ID from table_a
union all
select name, 2 as ID from table_b
order by ID;

…としてみてはいかがでしょう。
