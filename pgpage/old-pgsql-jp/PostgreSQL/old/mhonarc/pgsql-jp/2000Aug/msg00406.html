稲葉です。深夜に度々恐れ入ります。
福本様の言われたとおり、/usr/local/lib/php3.iniを
見直しました。
何故かよく分かりませんが、そのファイル内の
i18n.http_output = SJIS　を
i18n.http_output = PASS　に
また、
i18n.http_input_default = SJIS　を
i18n.http_input_default = EUC-JP　に変更し、
apache再起動後正常に表示されるようになりました。

どこかの書籍に書いてあった通りに、このファイルには
最終行に
; internationalization
i18n.http_output = PASS
i18n.internal_encoding = EUC
i18n.script_encoding = AUTO
i18n.http_input = AUTO
とだけ追記したのみでした。

皆様色々とありがとうございました。
感謝！感激！です。
それではまた。
