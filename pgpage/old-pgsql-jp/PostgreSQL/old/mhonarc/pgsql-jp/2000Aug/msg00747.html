いつもお世話になっております。

例えば下記のようなテーブルを生成し、
適当なレコードをインサートします。

          Table "test"
 Attribute |  Type   | Modifier
-----------+---------+----------
 id        | integer |
 comment   | text    |

 id | comment
----+---------
  1 | test1
  2 | test2
  3 | test3
  4 | test4
  5 | test5
(5 rows)

その後、適当にアップデートするとレコードの並びが
ばらばらになってしまいます。

例）
test=# update test set comment = 'test6' where comment = 'test3';
UPDATE 1
test=# select * from test;
 id | comment
----+---------
  1 | test1
  2 | test2
  4 | test4
  5 | test5
  3 | test6
(5 rows)

何故このようなおかしな現象が起きるのでしょうか？
どなたかわかる方いらっしゃいましたら、
よろしくご指導お願いします。

動作環境：
OS KONDARA MNU/Linux 1.2
postgreSQL7.0.2
