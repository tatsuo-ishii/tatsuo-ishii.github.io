こんにちわ、松本です。
PostgreSQLは6.5.3です。

   create function test(int2, int2, int2, int2) returns int4 as ‘
   select id from table1 where num1 = $1 and num2 = $2 and num3 = $3 and
num4 = $4
   ‘ language ‘sql’;

   table1のid int4
   table2のid int4
共にindexを張っています。

＜SQLの実行＞
   select * from table2 where table2.id = test(1, 1, 10, 0);．．．．．１

この場合、indexは使われません。
＜SQLの実行＞
   select * from table2 where table2.id = 100;．．．．．２

だと、indexは使われます。

１の場合に、indexを使わせるにはどうすれば
良いのでしょうか。
