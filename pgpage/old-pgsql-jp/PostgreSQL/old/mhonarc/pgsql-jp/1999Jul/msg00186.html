=====================================================================

1. English      オリジナルの文書（英語版）
2. Japanese     和訳文書
3. SRPMS        RedHat SRPM形式のソースパッケージ
4. applications PostgreSQL関連アプリケーション
5. binaries     各プラットホームのバイナリパッケージ
6. commercial   PostgreSQL関連商用アプリケーション
7. examples     サンプルプログラム
8. freeware     バイナリフリーのアプリケーション
9. sources      ソースプログラムとパッチのアーカイブ

=====================================================================

1. English
        オリジナル文書（英語版）

  English/doc:
        配布に附属の文書、HTML書式の文書は展開済み

  English/doc/FAQs:
        FAQおよび種々プラットホーム依存のFAQ

  English/doc/PS:
        Postscript書式の文書
        (admin.ps internals.ps programmer.ps tutorial.ps user.ps)

  English/man:
        配布に附属のオンラインマニュアル(man1 man3 man5 manl)

  English/man/text:
        配布に附属のオンラインマニュアルテキスト版


2. Japanese
        和訳された文書及びFAQ

  Japanese/ja_doc:
        6.4 の和訳文書

  Japanese/ja_doc/html:
        6.4 の和訳文書のHTML版

  Japanese/jman:
        6.4 の和訳オンラインマニュアルとそのHTML版


3. SRPMS
        RedHat SRPM形式のソースパッケージ(by shibuya@alpha.or.jp)
  postgresql-6.5-j3.src.rpm
        --with-mb=EUC_JP を付けたもの
  postgresql-6.5j-2.src.rpm
        --with-mb=EUC_JP, 堀田さん作成のメッセージの日本語パッチをあてたもの


4. applications
        PostgreSQL関連アプリケーション

  applications/mpsql:
        mpsqlはMotifを使ったユーザインターフェース, kpsql はそのKDE版

  applications/pgaccess:
        Tcl/Tk による GUI、レポートジェネレータ

  applications/pgadmin:
        Win32版の管理用インターフェース(ODBC利用)

  applications/psqlodbc:
        PostgreSQL用 Win32 ODBC ドライバ

  applications/psqlodbc_jp:
        日本語版のPostgreSQL用 Win32 ODBCドライバ

  applications/w-agora:
        PHP によるウェブ掲示板フォーラム


5. binaries
        PostgreSQL バイナリパッケージ
        (bsdi casio freebsd linux netbsd sco sun win32)

  binaries/bsdi:
    pgsql-6.5-bsdi4.tar.gz
        x86/BSDI4版(by eiji@papanui.ddt.or.jp)

  binaries/casio:
    postgresql-6.5_ADPS203P1.tgz
        Casio R4000/System_V 4.2MP版(by saito@inetrt.skcapi.co.jp)

  binaries/freebsd/2.2.8:
   pgsql6.5-freebsd2.2.8.tar.gz
        x86/FreeBSD2.2.8版(by kawa@veecity.ne.jp)

  binaries/freebsd/3.2:
    ja-postgresql-6.5.tgz
        x86/FreeBSD3.2版(by saito@mm.sophia.ac.jp)


  binaries/linux/mips:
    pgsql.tar.gz
        mips/Cobalt Linux版(by shibuya@alpha.or.jp)

  binaries/linux/mips/RPMS:
    postgresql-6.5-j2-CU1.mips.rpm
    postgresql-clients-6.5-j2-CU1.mips.rpm
    postgresql-devel-6.5-j2-CU1.mips.rpm
        LinuxMIPS版 RPM(by t-ishii@sra.co.jp)

  binaries/linux/ppc-r4:
    pgsql.tar.gz
        LinuxPPC-R4版(by t-ishii@sra.co.jp)

  binaries/linux/ppc-r4/RPMS:
    postgresql-6.5-j3.ppc.rpm
    postgresql-clients-6.5-j3.ppc.rpm
    postgresql-devel-6.5-j3.ppc.rpm
        LinuxPPC-R4版 RPM(by t-ishii@sra.co.jp)

  binaries/linux/x86:
        x86/Linux版パッケージ

  binaries/linux/x86/RPMS:
    postgresql-6.5j-2.i386.rpm
    postgresql-clients-6.5j-2.i386.rpm
    postgresql-devel-6.5j-2.i386.rpm
        x86/RHL5.2(J),RHL6.0版 RPM(by shibuya@alpha.or.jp)
    postgresql-6.5-j3.i386.rpm
    postgresql-clients-6.5-j3.i386.rpm
    postgresql-devel-6.5-j3.i386.rpm
        x86/RHL5.2(J),RHL6.0版 RPM(by shibuya@alpha.or.jp)

  binaries/linux/x86/libc5:
    perl5.004.tgz
    pgsql6.5-x86-linux-2.0-libc5.tgz
        x86/Linux-libc5版(by juk@rccm.co.jp)

  binaries/linux/x86/libc5/plamo:
        x86/Plamo Linux版 Databaseパッケージ(by juk@rccm.co.jp)

  binaries/netbsd:
    pgsql-netbsd1.4-m68k.b0.tar.gz
        m68k/NetBSD1.4版(by joyhiro@joyful.gr.jp)
   pgsql.m68k.netbsd133.b0.tar.gz
        m68k/NetBSD1.33版(by muc@isr.co.jp)

  binaries/sco:
    postgres-6.5-x86-UnixWare7.1.tar.gz
        SCO x86/UnixWare7.1.0版(by fwif0083@mb.infoweb.ne.jp)

  binaries/sun:
    pgsql.i86pc.Solaris.2.6.tar.gz
        SUN x86/Solaris2.6版(by kei-y@dd.catv.ne.jp)
    pgsql.i86pc.Solaris7.tar.gz
        SUN x86/Solaris7版(by muc@isr.co.jp)
    pgsql.sparc.Solaris7.tar.gz
        SUN Sparc/Solaris7版(by muc@isr.co.jp)


  binaries/win32:
    cygwin32ipc-1.03-bin.tar.bz2
    postgresql-cygwinb20.1-6.5-EUC_JP.tar.bz2
        x86/Win32/cygwinb20版(by yutaka@marin.or.jp)

  binaries/win32/gnuwin32:
        Win32用GNUツール


6. commercial
        PostgreSQL関連商用アプリケーション

  commercial/eliaslx:
        イントラネット構築のためのWeb対応グループウェア

  commercial/realbasic:
        Macのアプリケーションが簡単に作れるビジュアルな開発環境


7. examples
        サンプルプログラム

    geometric-0.1.tar.gz
        PostgreSQL 6.5用 拡張rtreeアクセスメソッド

    pgimage-1.0.tar.gz
        PHP と PostgreSQL で画像データを管理するシステム

    pgpost-1.3.tar.gz
        新旧郵便番号検索システム pgpost version 1.3

    phpgsql.tgz
        データベースにウェブブラウザからアクセスするためのPHP3スクリプト

8. freeware
        バイナリー・フリーソフトウェア

  freeware/openzolar:
        Web-Database連携ミドルウェアソフト


9. sources
        プログラムソースとパッチのアーカイブ

  sources/apache:
        Apache ウェブサーバ

  sources/gd:
        GIF データ取扱ライブラリ

  sources/gnu:
        GNU flex 2.5.4 と readlen 2.2.1

  sources/oldies:
        過去のバージョンのPOSTGRES

  sources/oldies/postgres-v4r2:
        POSTGRES(最終版)バージョン4リリース2

  sources/oldies/postgres95-1.09:
        Postgres95バージョン1リリース9

  sources/perl:
        Perl5 と JPerlパッチ

  sources/perl/modules:
        Perl DBI モジュールとDBD-Pg

  sources/php3:
        サーバサイド埋め込みスクリプト PHP3(3.0.7, 3.0.9, 3.0.11)

  sources/postgresql-6.4.2:
        PostgreSQL バージョン6.4 リリース2

  sources/postgresql-6.5:
        PostgreSQL バージョン6.5 （最新版）

  sources/postgresql-6.5/patches:
        PostgreSQL バージョン6.5 日本語関連パッチ(pgaccess, psql)

  sources/python:
        オブジェクト指向言語 Python

  sources/ruby:
        オブジェクト指向言語 Ruby とその postgres モジュール

  sources/tcltk:
        スクリプト言語Tcl, GUIツールキットTk、およびそれらの
        拡張言語tclX, 拡張ウィジェットTix
