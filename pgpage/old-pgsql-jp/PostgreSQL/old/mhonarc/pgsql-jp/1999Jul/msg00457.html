　シェル変数は、unbind_variable()というのが使用できるのです
ね。参考になります。うっかりしてましたが、SQLのNULL値の代入
はできましたね (^^)。

　update test set tx=NULL where id=2;

とすることができますので、そうすると、シェル変数には、その
まま $ROW='NULL' とし、

  exec_sql "update test set tx=$ROW where id=2"

とすることになりますね。

# そういえば、普通の文字列をセットした場合、例えば $ROW='ABC'
　は、シングルクォートとつけてやらないといけないのですね。

  exec_sql "update test set tx=\'$ROW\' where id=2"
