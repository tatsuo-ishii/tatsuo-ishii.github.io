いわきりです

#6.3.2と6.4の時にはこんなふうにして試してました

Shoichi Kosugeさんの<v04010103b330fadaaebf@str.hitachi.co.jp>から
 >　ひょっとして、こんなんでテストできるのかな。
 >(0) postmaster を止める。
 >(1) /usr/local/pgsql/ を適当な名前にリネーム。
 >(2) 6.5 をmake install する。
 >(3) テストする。
 >(4) テストが完了したら、postmaster を止めて、/usr/local/pgsql/ 
 >　　以下を全部削除して、(1) を/usr/local/pgsql/ にリネーム。

6.3.2のinstall directorをrename
  /usr/local/pgsql -> /usr/local/pgsql-6.3.2

6.4をinstall後rename
  /usr/local/pgsql -> /usr/local/pgsql-6.4

postmasterが動いてないのを確認して
  ln -s /usr/local/pgsql-6.3.2 usr/local/pgsql
または
  ln -s /usr/local/pgsql-6.4 usr/local/pgsql

で好きなほうを選んで動かしてました。
