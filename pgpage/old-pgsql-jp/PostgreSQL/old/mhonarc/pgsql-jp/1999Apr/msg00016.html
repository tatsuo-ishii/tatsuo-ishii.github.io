CREATE RULEに関して教えて頂きたいのですが、３つのテーブル(a_tab,b_tab,c_tab)
でa_tabにデータをinsertした時に、b_tabをupdateするルールとして

「create rule a_tab_ins as on insert to a_tab do update b_tab set b_tab.
b_id=new.a_id where b_tab.bc_id=c_tab.c_id;」

のようなルールを作成しておくと b_tabは、c_tab.c_idとイコールのデータが更
新されます。次にb_tabが更新された後にc_tab.c_idを更新したいのですが、上記
と同じようなルールだとc_idのデータが更新されてからb_tabが更新されてしまい
ます。
要するにイベントが完了した後にルールを実行したいのですがどなたか教えて下
さい。
