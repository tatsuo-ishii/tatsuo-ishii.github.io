On Mon, 19 Apr 1999 15:27:33 +0900, Ishida Akio <iakio@nettaxi.com> wrote:
> SQL関数にchar型の引数を渡す／返すには、どのように書けばよいのでしょう？
> 
> ---------------------------
> create function func1(int2)
>         returns int2
>         as 'select $1'
>         language 'sql';
> ---------------------------
> これができるのはわかるのですが、
> ---------------------------
> create function func1(char)
>         returns char
>         as 'select \'$1\''
>         language 'sql';
> ERROR:  return type mismatch in function: declared to return char,
>  returns unknown
> ---------------------------
> が出来ないのはなぜなのでしょう。

'...' はいろいろな型の値の文字列表現として使われますので，
それで unknown になってしまったのでしょう．これは，明示的に
'$1'::bpchar  としてやれば OK ですが，これですと，'$1' 
という文字列が返されてしまいますよ？

> #postgreSQLのバージョンを確認するにはどうしたら良いのでしょうか。
> #まあ、多分6.4で間違いないとは思うのですが、、、。

これは，

	select version()

ですね．
