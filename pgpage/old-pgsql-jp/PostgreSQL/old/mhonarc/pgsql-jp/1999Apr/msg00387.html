　6.4 では top 相当の機能は無いと思います。


# 6.5 では top と同じ機能の LIMIT が追加されれているようです。

　上記のSQLの場合、
　select * from table where criteria... LIMIT 30 ;
  とします。

  ちなみに"LIMIT n,s"(または"LIMIT n OFFSET s")は 検索結果集
  合の ｓ番目から n件を出力します。
