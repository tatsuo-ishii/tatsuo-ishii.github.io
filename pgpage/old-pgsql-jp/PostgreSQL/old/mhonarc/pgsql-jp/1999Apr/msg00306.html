SQL関数にchar型の引数を渡す／返すには、どのように書けばよいのでしょう？

---------------------------
create function func1(int2)
        returns int2
        as 'select $1'
        language 'sql';
---------------------------

---------------------------
create function func1(char)
        returns char
        as 'select \'$1\''
        language 'sql';

 returns unknown
---------------------------
が出来ないのはなぜなのでしょう。
