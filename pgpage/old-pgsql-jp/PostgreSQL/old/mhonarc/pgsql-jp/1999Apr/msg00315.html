Tsukada> pgenv.h:21: string: No such file or directory

string は libstdc++ に含まれているので、

  1. libstdc++ をインストールしていない場合
	gcc とのバージョンの相性に注意してインストール

  2. libstdc++ をインストールしている場合
	ln -s stringが置かれているディレクトリ /usr/local/lib/g++-include

すると大丈夫だと思います。
