というわけで ftp.sra.co.jp/pub/cmd/postgres/6.4.2/patches/ を更新しま
した。

サイズ	ファイル
---------------------------------------------
5649438	postgresql-6.4.2-patched-990414.tar.gz
43513	jumbo-990414.patch.gz
1092	btree.patch.gz

が追加されたファイルです。今まさに ftp している方がいるので、symlink 
および README などは後で更新します。

すでに postgresql-6.4.2-patched-990308.tar.gz をお持ちの方は 
btree.patch.gz だけ持って行って、

$ cd postgresql-6.4.2/src/backend
$ gunzip -c btree.patch.gz|patch -p1
$ make clean
$ make all
(postmaster を停止)
$ make install

でインストールし直すのがてっとり早いでしょう。今回の修正はバックエンド
だけなので、アプリケーション(フロントエンド)の再リンクなどは必要ありま
せん。initdb も必要ありません。

また、postgresql-6.4.2-patched-990308.tar.gz 以外の 6.4.2 をお使いの場
合でも、btree.patch だけをあてることは可能です。
