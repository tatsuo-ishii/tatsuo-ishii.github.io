  河上です。

  HP-UX 11.0 make 通りました。

  まずは、config.guess の371行あたりでマシンタイプを
  チェックしている所。
  動かしているマシンはK370という奴でuname -m は
  9000/898 を返します。が、case でマッチするエントリが
  ありません。9000/892 の後に | 9000/898 を追加しました。
  これで config.guess は hppa2.0w-hp-hpux11.0 を出力し
  ます。
  次に、config.sub 。
  154行ぐらいと、178行ぐらいの所でCPU TYPE をチェックして
  いますが、わざわざconfig.guess で64bit用にhppa2.0wとして
  いるのにこれを引っかけてくれません。
  hppa2.0 を hppa2.0? 、hppa2.0-* を hppa2.0?-* としました。
  
  これらを見るにPA8000系のチップにHPUX11.0(64/32)を意識して
  作り直しているように見えます。
  # 64bit native のPostgreSQLができるのだろうか、だったらうれしいな。
  
> 4/1 の snap shot で問題なかった mips linux (Cobalt Qube)が configure
