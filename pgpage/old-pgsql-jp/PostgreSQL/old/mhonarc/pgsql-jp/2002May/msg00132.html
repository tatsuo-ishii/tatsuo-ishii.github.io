近藤直文＠JPUG 業務アプリ分科会です。

JPUG(日本PostgreSQLユーザー会)業務アプリケーション分科会では、 
経験者向けの『勉強会』を毎月開催します。

次回は、 ５月１９日(日曜日) 13:00〜17:00 です。

テーマは、分科会 ML で募集中です。
5/29 - 5/31 Linux World Expo(東京ビックサイト)出展の
話などもしたいと思っております。
# Linux World の詳細はこちらをご覧ください．
http://www.idg.co.jp/expo/lw/

---------------------------------------------------
日時 : 5/19(日) 13:00〜17:00 
場所 : ルエラ (レストラン) 東京都世田谷区桜３-１６-７
会場地図:    http://www.leetla.com/map.html
会費:無料または会場費(500円〜1,000円)程度。

参加資格は、JPUG 会員で PostgreSQL 経験者です。

会員以外で参加ご希望の方は、先ず、JPUG に入会して、業務アプリ  
分科会に参加され、同 ML jpug-ap 宛に参加申込みをしてください。
JPUG 個人会員(正会員)は、入会金/会費共に無料です。

・JPUG 日本PostgreSQLユーザー会 入会申込み
http://www.postgresql.jp/registration/index.html

・業務アプリ分科会 参加申込み
http://www.postgresql.jp/subcommittee/ap/index.html

・このメール自体はコピー配布自由です。

・問合わせは、近藤( nkon@postgresql.jp )まで。
