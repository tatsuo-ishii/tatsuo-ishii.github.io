;;; （項目名、データ型、桁数）などをドキュメントにしたいのですが、
;;; このような事ができるソフトは、あるのでしょうか？

  いろいろあると思います。手軽なのは Perl DBD-Pg をインストールすれば使える、

    PostgreSQL AutoDoc v0.50 (http://www.zort.ca/postgresql/)

    A perl script which takes the database structure and exports an XML and /
    or HTML file which Dia can use to make a UML diagram, and your webbrowser
    can use for additional information that cannot be squeezed in. Tested with
    Dia v0.88.1 and Postgres v7.1 or 7.2 (choose correct version).

フィールド名が長い場合に ... で省略する部分で日本語を考えていません。省略しな
いように Perl スクリプトを変更するだけでも使えます。
