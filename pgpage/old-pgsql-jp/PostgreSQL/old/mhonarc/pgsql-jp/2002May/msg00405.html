あるファイルのアクセスログを

名前｜日付時刻
name time
というテーブルで取っています。

このテーブルから各個人が「初めて」そのファイルに
アクセスした日付時刻を得ようとしました。

Select name, min(time) from database group by name;

この月ごとの人数を集計しようと、前のクエリーと組み合わせて

Select count(name) from database where (Select name,min(time) from
database group by name) BETWEEN ’2002/1/1’ AND ’2002/1/31’;

としましたが、min(time)ってこういう用途に使えるものなのでしょうか？
