> ちなみに最初は bison と　flexがないというので bison も flexもいれました。 
> でもまだflexがないと言って先にいけません。 

同じところではまったことがありますが、インストールすればサクット通りま
す。bison/flexともに「正しく」インストールして実行パスを「正しく」指定
すればうごきますよ。PostgreSQLのconfigureを実行する前に

 % which bison
 % which flex
 % bison
 bison: no grammar file given    <-- エラーメッセージがでる!!
 % flex
 ^C                              <-- 標準入力待ち状態なのでControl-C
 %

して、それぞれが『本当に』動いているのを確認するのがよろしいかと思います。
