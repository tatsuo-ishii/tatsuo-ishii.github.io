色々と試してみたのですが、 \pset format unaligned
をセットすると、-F オプションでセパレータを変更でき、
最後に空白行がでないということがわかりました。
たとえば、

#!/bin/sh
cat << EOF | psql db1 -tq -F " " | awk '{ print $2 }'
  \pset format unaligned
  select * from tbl01 where id < 10;
EOF

どちらにしても、プログラムに多少手を加えることに
なりそうですが、修正する方向が見えてきたので、
助かりました。
