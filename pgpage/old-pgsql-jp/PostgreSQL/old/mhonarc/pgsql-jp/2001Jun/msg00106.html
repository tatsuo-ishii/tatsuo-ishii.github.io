On Fri, 8 Jun 2001 01:40:27 +0900
makoto_harita@ma2.justnet.ne.jp wrote:

> フィールド名に
> user
> という項目を作りたいのですがCREATE TABLEで
> 作成しようとすると、エラーが出て作成できませんでした

CREATE TABLE Test ("user" text)
とすれば作成できます。

しかし、将来に渡って、このテーブルを操作するDML全てにおいて、ダブルクォー
テーションをつけなければならず、また、

SELECT * FROM Test WHERE "user" = '1'

とするつもりで

SELECT * FROM Test WHERE user = '1'

などとしてしまってもエラーが発生しないので（もちろん意図した結果は出ない）、
あまりお勧めできません。
なお、userは、current_userと同じ意味です。
