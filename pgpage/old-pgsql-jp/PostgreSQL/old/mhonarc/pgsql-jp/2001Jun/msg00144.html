渡辺です。

"Y.Watanabe" wrote on Sun, 10 Jun 2001 23:09:10 +0900
with Subject: [pgsql-jp 21758] SQL 関数の書き方？ 

> 今まで普通のSQL文呼び出しだけでなんとかしていたのですが、
> とうとうＳＱＬ関数つくらなきゃどうにもならないことになって
> 勉強中なのですが、どうも合点の行かぬ現象が起きてまして悩んでます。
> 
> watanabe@host> cat test_func.sql
> 
> create function add_one (int4) returns int4 as '
>   begin
>     return $1 + 1 ;
>   end;
> ' language 'sql';
> 
> っていう関数をデータベースに入れようとすると
> 
> watanabe@host> psql -d testdatabase -f test_func.sql
> psql:test_func.sql:5: ERROR:  parser: parse error at or near "retu
> rn"

上記のようなことをする前に、

watanabe@host> createlang plpgsql testdatabase

をしなければならなかったんですね。これでうまく動き出しました。
もうちょっと手続き言語のあり方について事前知識として
学んでおくべきだったようです。

みなさんありがとうございました。
