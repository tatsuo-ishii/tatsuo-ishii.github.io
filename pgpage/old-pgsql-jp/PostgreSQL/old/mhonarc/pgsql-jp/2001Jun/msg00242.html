-F は \pset format unaligned した時だけ有効なようです。
でもそれだと目的と違うようなので、
"|" を取りたいということであれば、

\pset border 0

でできます。

$(HOME)/.psqlrc 
に書いておいて

$ echo "select * from tbl01 where id < 10;" | psql db1 -t

とかするのがいいかもしれません。
