[pgsql-jp 21748] v7.1.2 でのreadlineの組み込み

  • From: "Masashi Sasagawa" <sasagawa@dream.com>
  • Date: Sat, 9 Jun 2001 09:56:11 +0900
はじめまして、笹川と申します。
ここ数日PostgreSQLのインストールと格闘していましたが、
v7.1.2にreadlineがどうしても組み込めませんでした。
しかたなくv7.1.1を使用しています。

readlineは4.2に日本語パッチを当てたものを使いました。

これに下記のコンフィグを行うと、v7.1.2とv7.1.1で結果が違ってきます。
$ ./configure --enable-multibyte=EUC_JP --enable-syslog \
--with-java --with-perl --with-openssl=/usr/local/ssl \
--with-libs=/usr/local/lib --with-includes=/usr/local/include/readline

下記がgrep readline でreadlineを含む行を拾った結果です。
using CPPFLAGS=  -I/usr/local/include/readline -I/usr/local/ssl/include
checking for readline... (cached) no
checking for readline/readline.h... (cached) no
checking for readline.h... (cached) no
checking for readline/history.h... (cached) no

一応、このままインストールしてみましたが、当然コマンドヒストリ、
行編集ともにNGでした。

まったく同じオプションを指定してv7.1.1で実行した結果は次のようになります。
using CPPFLAGS=  -I/usr/local/include/readline -I/usr/local/ssl/include
checking for readline... (cached) yes (-lreadline -lncurses)
checking for readline/readline.h... (cached) yes
checking for readline/history.h... (cached) yes

v7.1.2でreadlineを組み込むには、他にも何か設定が必要なのでしょうか？
よろしくお願いいたします。
