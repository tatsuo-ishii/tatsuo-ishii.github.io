OS：LINUX(SlackWare)

　　（レンタルサーバー）です

度々お世話になります。
PostgreSQLの部屋のプログラミングガイド「3. データベースとの連係」にある
スクリプトの要所を変えて実行してみたところ

エラーメッセージ：Parse error: parse error, expecting `','' or `';'' in
/home/dataweb/public_html/db_test2.php on line 11

が出ました。line11は「echo "<table border="1">\n";」を指していると思いますが
修正点がわかりません。おわかりになる方がいらっしゃいましたらご教授ください。
実行スクリプトは以下です。

<?php
$conn = pg_pconnect("dbname=dataweb user=dataweb");
if (!$conn) {
    echo "接続できませんでした\n";
  exit;
}
$sql = "select name, area, db_add from aigo";
      $rs = pg_exec( $conn, $sql );

      /*  検索結果をテーブルにする  */
      echo "<table border="1">\n";
      for( $i = 0; $i < pg_numrows( $rs ); $i ++ ) {
        echo "<tr>";
        echo "<td>" . pg_result( $rs, $i, 0 ) . "</td>";
        echo "<td>" . pg_result( $rs, $i, 1 ) . "</td>";
        echo "</tr>";
      }
      echo "</table>\n";
      pg_close( $conn );
?>
※余談になるかもしれませんが…何をやりたいのかと言いますと、
最終的にはユーザー登録者がwebから入力したデータを
一般ユーザーが検索＆閲覧できるシステムを作りたいのです。
（ユーザー登録者は自分の入力したデータを、管理者は閲覧に
関わる全てのデータをwebから操作（削除・修正など）できる）

先ずはSQLサーバーにあるデータベースにアクセスして、
検索結果を閲覧できるようにしたいのですが、
解らないことが多いので試行錯誤が続くと思います。
宜しくお願い致します。
