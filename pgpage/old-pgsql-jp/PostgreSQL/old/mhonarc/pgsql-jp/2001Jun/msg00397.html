> これで抽出されるということは、そのように登録しているからですね。
> varcharといえども'ABC      'で入れたら空白を除くことはしませんのでそのまま入
> ります。

VARCHAR, TEXT は、入力したとおりにDBに格納されます。
空白も含めて。

文字列を PRIMAY KEY や、重要な KEY にする場合は、
固定長の CHAR 型にします。そうすれば、指定サイズ
より短ければ後に空白を補い、長過ぎれば後をカット
して必ず固定長になるので、プログラムが簡潔になり
ます。

境 さんの例も、VARCHAR(10) でなく、CHAR(10) にす
れば、下記どちらの SQL 文でも表示されます。

select * from 表名 where 列名 ='ABC       '; -- 後に空白
select * from 表名 where 列名 ='ABC';  -- 後に空白なし。
