２つの同じカラムをもつテーブルから特定のカラムが重複しないデータ
を取り出したいのですが。

例えば、

table A
　user_id  mailaddr

table B
　user_id  mailaddr

と２つのテーブルからメールアドレスが重複しない行を取り出すとき、

　　select * from A where not exists ( select * from B 
	where mailaddr = A.mailadddr );

　◆select A.* from A, B where A.mailaddr <> B.mailaddr;

table A, B ともに１万件近くあるのですが、実行したまま結果が返ってきません。
limit 100 程度なら１０秒程度で結果がでます。
