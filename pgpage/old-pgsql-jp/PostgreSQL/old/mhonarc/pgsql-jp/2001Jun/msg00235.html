テーブル food に食品名 food_name というカラムがあるとします。
そこで
=> SELECT * FROM food WHERE food_name = '白玉粉';
とやると、food_nameカラムが「白玉粉」である行だけでなく、
まったく関係のない余計な行までヒットしてしまいます。

そこで手法を変えて
=> SELECT * FROM food WHERE food_name LIKE '白玉粉';
とするとうまく、白玉粉のみがヒットします。

当面は LIKE で動くので問題は無いのですが、普通の一致検索に
LIKEを使うのも不便で気持ち悪いので(笑)、解決策があったら教えて
もらえないでしょうか？
