ゴミ記事になっちゃいますが^^; これ流で行くと，pg_dump -uの場合は

(echo username ; echo password) | pg_dump -u ...

とか

[sh系]
echo 'username
password' | pg_dump -u ...

[csh系]
echo 'username\
password' | pg_dump -u ...

とか

pg_dump -u ... <<EOT
username
password
EOT

ですかね......
