す。
輸血の際にはあまり重要ではなく、親子鑑定などに使用されるもののようです。

> この 3種類しかないのであれば、SQL ファンクションで可能では？

具体例を提示いただきまして、ありがとうございます。

Developer Studio で SQL 文の勉強をし、PHP3 でプログラミングをしております
が、前者で例示いただきました関数は確かに構文として正しいようですが、恥ずかし
ながら、使い方がさっぱりわかりません。

どうも、1 行しか実行できないのか、「;」 よりも前は、なぜか忘れてしまうよう
で、create function で定義した関数を呼び出せないエラーとなります。(ERROR:
function 関数名 does not exist ...)

現在の知識のレベルでは問題がすぐには解決できそうにないので、今後勉強すべき課
題とし、まずは、position や substring などの文字列操作系の関数を使い select
文の中に無理やり埋め込み期待の動作をさせることができました。

これら関数は、手持ちの書籍でもしっかりと解説されていました。m(__)m

今後とも、ご指導をよろしくお願いいたします。

// blood は char(20) です。

$sql =  "select *,
    substr(blood,
        1,
        position(':' in blood)-1
    ) as abo,
    substr(blood,
        position(':' in blood)+1,
        position(':' in blood)+
        position(':' in
            substr(blood, position(';' in blood)+1,20))
    ) as rh,
    substr(blood,
        position(':' in blood)+
        position(':' in
            substr(blood, position(';' in blood)+1,20))+1,
        20
    ) as mn
    from person;";
