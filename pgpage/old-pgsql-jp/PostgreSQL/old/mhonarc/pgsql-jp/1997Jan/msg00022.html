石井＠SRAです。

路川 徹也さん(michkawa@kt.rim.or.jp)という方から postgres95 ユーザマニュ
アルの日本語版を TeX にしたものを頂いたので、。postgres95(J)ホームペー
ジに置きました。いつものように、
http://www.sra.co.jp/people/t-ishii/postgres95/ から「postgres95 関係
の日本語ドキュメント 」メニューで辿って下さい。なお、石井の方で dvi2ps 
を使って ps 版も作成しましたので、すぐに印刷できます。

# なお、TeX 版は、すでにおみつ@trampさんも作成されてます。
# see http://www.tramp.co.jp/Postgres95/

以下、附属の README を引用させて頂きます。

------------------------------------------------------------------------------

  The POSTGRES95 User Manual
                             TeX Version    1997.01.16

  このファイルは、manual-jp.txtとpostgres95-1.09.tar.gzの中のdoc/userguide.psを
を元にして、userguide.psをTeX化したものです。
  このファイル群には以下のようなファイルが含まれています。

  README:  このファイルです。

  postgres95.tex: マニュアルのTeXファイルです。

  postgres95.dvi: postgres95.texをdviにしたものです。

  pg95-1.eps: tgifで作成した絵（その１）
  pg95-1.ps

  pg95-2.eps: tgifで作成した絵（その２）
  pg95-2.ps

  pg95-3.eps: tgifで作成した絵（その３）
  pg95-3.ps

  epsbox.sty: psファイルを貼り付けるためのstyleファイル

  できるだけuserguide.psにそっくりになるように心がけて作ったつもりですが、
結構いい加減に書いてしまったので、TeXファイルを見ると結構汚いコードかも
しれません。(^_^;) 見辛いようでしたら書き直して下さって構いませんし、
さらにそれを公開して下さるともっと嬉しいです。

  また、このファイル群はdviファイルをPostscript化を行なうに当たってjdvi2kpsを
使用したのでstyleファイルとしてepsbox.styを使っていますが、dvi2psを使う場合
にはdocumentstyleと\epsfileの部分を書き換える必要があるかも知れません。

# 自分の環境に合わせて書き直して下さい。

  このファイルが少しでも皆さんの役に立てば幸いです。

  ではでは。
