boardcastclient の指定を /etc/ntp.conf でやってます。

------------------------------ ここから ------------------------------
# Configureation file for broadcast client.
broadcastclient yes             # declare to be broadcast client
# drift file
driftfile       /etc/ntp.drift
------------------------------ ここまで ------------------------------
