指定したところapacheのエラーログはでなくなりましたが
接続は出来ません。
以下のようなソースなのですが

    try {
      Class.forName("org.postgresql.Driver");
      Connection con = DriverManager.getConnection("jdbc:postgresql://IPAdress:
5432/master","username","passwd");
      Statement stmt = con.createStatement();
      ResultSet rs = stmt.executeQuery("select name from test_table");
      rs.next();
      textField1.setText(rs.getString("name"));
      stmt.close();
      con.close();
    }catch(Exception ex) {
      textArea1.setText(ex.getMessage());
    }

と表示されました。
