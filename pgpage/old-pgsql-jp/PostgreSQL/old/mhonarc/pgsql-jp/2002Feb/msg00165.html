| 現行SQL文
| SELECT * FROM tbl WHERE oidrand(tbl.oid,700000);

(snip)

| これを早くする為の方法はございますでしょうか？

テーブルの行数が一定なら、アプリ側で $offset に
ランダムな値を入れておいて、

  select * from tbl limit 1 offset $offset

という形で平均的に速くなるようにおもいます。

$offset が大きくなるほど、

| SELECT * FROM tbl WHERE oidrand(tbl.oid,700000);

の方法に速度が近づくような気がしますが。
