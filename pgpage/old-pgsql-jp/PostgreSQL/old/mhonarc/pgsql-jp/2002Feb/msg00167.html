MLの皆さまこんにちは。
吉田と申します。
今、
create table t1(a text, skey int)
のようなテーブルに対して
「skey昇順で、aが'hoge'でありskeyの最小な行の前後１行ずつ（計３行）」の
結果をSELECTするにはSQL文はどうしたらよいのでしょうか。
select l1.* from t1 l1, (select * from t1 where a='hoge' order by skey
limit 1) as l2 where l1.skey >= l2.skey order by skey desc limit 2

select l1.* from t1 l1, (select * from t1 where a='hoge' order by skey
limit 1) as l2 where l1.skey < l2.skey order by skey limit 1

のように２回SQL文を実行しなければいけないのでしょうか。
