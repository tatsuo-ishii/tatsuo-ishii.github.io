ダウンロードぐらいはしてみようとしたら、上記の Web ページを
ブラウザで表示させてクリックすると Not Found になってダウンロード
できませんでした。

しかし、ここのリンク先の ftp サーバに ftp クライアントから接続して
試すとダウンロードできました。 はて？ まぁいいや。深くは追求しない。

README.jpath には、

-----------------------------------------------------------------------
  本パッチは、

  ・栄田健一(*)氏の作成された、bash-1.12 に対する漢字パッチを起源
    とし、
  ・それを西島が bash-1.14.1 に適応させ、
  ・それを山賀正人(*)氏が bash-2.01 / readline-2.01 に適応されて、
  ・それを横田義徳(*)氏が bash-2.02 / readline-2.02 および
    bash-2.03 / readline-4.0 に適応されたのち、
  ・それを西島が bash-2.04 / readline-4.1, bash-2.05 / readline-4.2
    および bash-2.05a / readline-4.2a に適応させた

  ものです。
-----------------------------------------------------------------------

と、あります。ファイル名からもわかるように 2002年 2月 8日に公開された
もののようです。まさにホカホカ。(^_^)

私みたいにブラウザからパッチがダウンロードできなかった人は、ftp
クライアントから試してみればいいでしょう。

bash は現時点では 2.05a が最新でしょう。 ftp.gnu.org を見ても
それ以上新しいのは無いし。
