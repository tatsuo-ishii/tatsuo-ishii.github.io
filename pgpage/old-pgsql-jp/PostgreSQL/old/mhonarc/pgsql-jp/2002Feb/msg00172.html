# pgbashとは

  pgbashは、bashシェルの中に PostgreSQL用のデータベースアク
セス機能を組み込んだシェルです。 シェルスクリプトを作成して
バッチ処理で SQLを実行したり、端末から対話的に SQLを直接実行
することができます。

# バージョンアップ内容

 1. PostgreSQL-7.0/7.1/7.2に対応 
 2. キャンセル(Ctrl+C)処理のバグを修正 
 3. Pgbash独自のcopyコマンドにおいてデリミター処理のバグを修正 
 4. 左括弧と右括弧の間のセミコロンの構文解析のバグを修正(Dias 
    Badekasの報告) 
 5. set OPTION文において TRUE/FALSE の値を ON/OFF に変更 
 6. set OPTION_HEADERTR/HEADERTH/BODTTAG/INPUTTAG/INPUTSIZE 
    機能を追加 
 7. EXEC_SQL_PREPARE シェル変数の機能を追加 
 8.「SELECT INTO :ホスト変数」句の機能を追加 
 9. ~/.pgbashrc が存在してなければ、/etc/pgbashrc を読む機能を
    追加 
10. backendから受信した改行コードを"\n"で表示する機能を追加
  （ \r, \t も同様 ) 
11. ラージオブジェクト処理用に'pgbash_description'テーブルを追加 
12. 表形式のテキスト出力フォーマットを変更(罫線の左右にスペース追加) 
13. CONNECT文のパスワード指定において、IDENTIFIED BY、USING、/ 
    を追加 
14. 接続状態の表示に client_encoding を追加('?m' コマンド)
