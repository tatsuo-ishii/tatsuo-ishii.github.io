Tadashi Kanbayashi wrote:

> ・テーブル名、
> ・テーブル作成の際にCOMMENT文でつけたコメント、
> 
> を取得する方法を教えていただきたく。

psql -E で起動して、\d+ とやってみるとヒントが。

SELECT relname AS tablename, obj_description(oid) as comment
FROM pg_class
WHERE relkind = 'r'
AND relname !~ '^pg_'

WHEREはお好みにより。

ところで、この関数は
select description from pg_description where objoid = $1
だけなので使わないで直に書いた方がいいのかな？
ドキュメントの「その他の関数」のとこ、少なすぎ？
