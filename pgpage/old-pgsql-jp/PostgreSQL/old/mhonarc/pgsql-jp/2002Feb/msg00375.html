こういうことですね？

[root@note_linux /root]# createuser choduka
Shall the new user be allowed to create databases? (y/n) y

psql: FATAL 1:  user "root" does not exist
createuser: creation of user "choduka" failed

http://osb.sra.co.jp/PostgreSQL/Manual/PostgreSQL-7.1-ja/app-createuser.html
http://osb.sra.co.jp/PostgreSQL/Manual/PostgreSQL-7.1-ja/sql-createuser.html
説明
CREATE USER は、Postgres のインスタンス に、新しいユーザを加えます。ユーザの
管理と認証に関する情報は管理者 用ガイドを参照して下さい。このコマンドを使う
ためには、データベース スーパーユーザでなければなりません。

CREATE USERのページの説明（上記）にあるように、createuserコマンドを使用して
いるユーザーがpostgresのデータベース スーパーユーザーではないからではないで
しょうか？
データベース スーパーユーザーはインストールした時のユーザー名だったと思いま
す。

なので、postgresqlをインストールした時のユーザーへ変更し、同じコマンドをたた
いてみてください。
