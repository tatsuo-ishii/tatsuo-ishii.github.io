# ついでにpgbash-2.0 の宣伝です(^^)

  pgbash-2.0は、12月25日ぐらいまでに公開できるのではないかと
思います。

　pgbash-2.0は、bash-2.03本体を修正して SQL文を直接実行でき
るようにしました。(exec_sqlコマンドの使い方は従来と同じです)

  pgbashはログインシェル（もしくはサブシェル)としても使用で
きますし、シェルコマンドとSQLを同等に実行できますので、使い
安さという点では psqlよりずっと良くなったと思っています(^^)

login: xxx
password: 

Welcome to the PGBASH version 2.0 (bash-2.03 + exec_sql-2.0)

  Type '[time] SQL; [pipeline][redirection][&]' to execute SQL.
  Type 'exec_sql [option] ["SQL"]' to execute SQL with options.
  Type '?' to HELP.

pgbash> connect to db2@www.psn.co.jp user postgres [password];
pgbash> select * from test; 
pgbash> select * from test; &> /tmp/oo & ... redirectとbackground_Job
pgbash> ls                        .......... ファイル一覧
pgbash> vi /tmp/oo                .......... エディタによる編集
pgbash> time select * from test;  .......... 処理時間表示
pgbash> begin;
pgbash> declare cur cursor for select * from test;
pgbash> fetch in cur into :AA,:BB; ... 検索結果をシェル変数に代入
pgbash> echo "AA=$AA, BB=$BB"
pgbash> end;
pgbash> fc fetch            .......... 履歴のfetch文の編集と再実行
pgbash> !echo  ....................... 履歴のechoを再実行

pgbash> ?   ........ ヘルプ　(psqlライクなコマンドも用意してます)
  h  [SQL]: help <SQL> syntax or all SQL_commands
  o       : help OPTIONs of 'exec_sql' function
  v       : print PGBASH VERSION
  s       : print STATUS after SQL execution
  l       : list all DATABASEs
  m       : list all CONNECTIONs
  d  [TBL]: list tables,indices,columns in <TaBLe>, or all tables
  da      : list AGGREGATES
  dD [obj]: list DESCRIPTIONs for <TAble,FIeld,TYp,FU,OP>, or all
  df      : list PosgreSQL FUNCTIONs
  di      : list only INDEXes
  dO      : list OPERATORs
  ds      : list only SEQUENCEs
  dS      : list SYSTEM tables and indexes
  dt      : list only TABLEs
  dT      : list TYPEs
  dg      : list current GRANT/REVOKE permissions
