> そういえば、WWW用の無制限ライセンスも値上げになったのでしょうか?

Oracle8/8i Workgroup Server for UNIX の不特定ユーザインターネット
アクセスライセンスパックは 320万となっています。
前の値段がいくらだったか忘れました。80万だった気もしますが…

また、Oracle8 WGS for Linux には不特定ユーザインターネットアクセス
ライセンスパックがなくなったので、事実上この用途に使用するのは
ライセンス契約的に現実的ではないように思います。

Oracle8i WGS for Linux のそれは for UNIX と同じく 320万となると
あります。
この用途に for Linux を使うのであれば 8i にしなさいという事なのでしょう。
まぁ、オラクルとしては 8i に移行させたいと考えるのは自然な事だとも思えます。

ただし、Oracle8i for NT の場合は何と 80万 (!) のままです。
# 今回は for NT の価格改訂はないですから。

どこかの代理店から「NT 版が売りにくいから、for UNIX や for Linux の
価格やライセンスを変えてくれ」とでも言われたんじゃないかと勘ぐりたく
なってきちゃいますけど、真実や如何に？ (笑)

for NT も同様な価格改訂するのであれば、そんな勘ぐりはしないの
ですけど。:-)
