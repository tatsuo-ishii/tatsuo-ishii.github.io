(solaris2.6ですが)configure時とmake時の環境をわざと変え、
stones.my.domain% more include/config.h
  (SNIP)
/* #undef HAVE_GETTIMEOFDAY_2_ARGS */
#ifndef HAVE_GETTIMEOFDAY_2_ARGS
# define gettimeofday(a,b) gettimeofday(a)
#endif
となるようにしたら、上エラーが再現しました。本来は、
#undef HAVE_GETTIMEOFDAY_2_ARGS 1
#ifndef HAVE_GETTIMEOFDAY_2_ARGS
# define gettimeofday(a,b) gettimeofday(a)
#endif
となる部分です。

原因はよくわかりませんが、もし、include/config.hが上のようになっていた
ら、configureしなおせば直りそうです。make distcleanするのをお忘れなく。
