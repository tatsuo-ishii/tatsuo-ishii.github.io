IIZUKA Satoshi wrote:

> さっそくですが、ソート対象のカラムに NULL 文字列を上位とするか、
> 下位とするかをDBパラメータ等で決めることはできますか？
> （デフォルトでは、NULLは上位になっているようです。）

こっちの7.1.2では逆でした。？？

> SELECT * FROM hoge_table
> ORDER BY sort_key
> 
> で取得すると、
> 
> sort_key name
> ------- ---------------------
> (null)      record1
> (null)      record3
> key1      record2
> key2      record4
> 
> となってしまいます。
> 
> これを、
> 
> sort_key name
> ------- ---------------------
> key1      record2
> key2      record4
> (null)      record1
> (null)      record3
> 
> という形で取得したいのです。
> 
> 上記の方法をご存知の方、いらっしゃいましたらご一報ください。

任意の計算式でソートできます。
SELECT * FROM hoge_table ORDER BY coalesce(sort_key, 'zzzzzzz');
