石川＠池袋本町です。

9月28日に、
Bruce Momjian=著 (PostgreSQL - Introduction and Concept -)
日本PostgreSQLユーザー会=訳の
「はじめての PostgreSQL - データベース問い合わせのコンセプト -」
が出版されました。

出版社=株式会社ピアソン・エデュケーション
ISBN: 4894714612
サイズ(cm): 23 
単行本 - 496 p
価格=4,200円+税金
です。

本書は JPUG 会員の有志によって翻訳された解りやすい解説書と
なっています。読み進めるにあたっては、PostgreSQL がすでに
インストールされている環境を前提として始まっています。
やさしい問い合わせから、込み入った問い合わせまで無理なく読
んで行くことができます。例題にも富んでおり、PostgreSQL の
各種 API も事例付で解説されています。後半の約半分はソース
に付随するリファレンスマニュアルをMomjianさんが再構成した
ものの翻訳です。オンラインで読むのとはまた別の感覚で、解り
易いと思います。ぜひ、ご参照ください。
