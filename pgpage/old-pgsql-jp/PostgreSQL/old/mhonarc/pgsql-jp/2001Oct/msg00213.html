PostgreSQL 7.1.3のソースからJDBCドライバをコンパイルしてみましたが、
結果は同じでした（日本語のnameだけ表示されません）。

松田さんの教えてくれたexample.psqlで、JDBC経由で接続してみました。
結果は、以下の通りです。

[1] select getdatabaseencoding();
getdatabaseencoding
EUC_JP
[2][2] show server_encoding;
Updated 1 rows <---ここのところ、おかしくありませんか？
[3][3] show client_encoding;
Updated 1 rows <---同上

ちなみに、select * from table1;とやると、日本語のnameだけ表示
されません。

普通のpsqlでやると、

testdb=>select getdatabaseencoding();
getdatabaseencoding
----------------------
EUC_JP
(1 row)

testdb=>show server_encoding;
NOTICE: Current server encoding is EUC_JP
SHOW VARIABLE

testdb=>show client_encoding;
NOTICE: Current client encoding is EUC_JP
SHOW VARIABLE

となり、select * from table1;とやると、日本語のnameも、バッチリ
表示されます。

なんでだろうな〜？
