　渡辺と申します。

　PostgreSQL 7.1.3

CREATE TABLE TEST_AR
   (DATA_AR   VARCHAR(2)[5][5][2]);

　というテーブルに対して

UPDATE TEST_AR
   SET DATA_AR [1][1][2] = '1' ;

　というupdate文を発行すると、

ERROR:  Invalid array subscripts

　このようなエラーとなってしまいます。
　ちなみに

UPDATE TEST_AR
   SET DATA_AR [1][1][1] = '1' ;

　これですと、更新は出来るようですが

UPDATE TEST_AR
   SET DATA_AR [2][2][2] = '1' ;

　ですと、エラーとなります。

　配列中の一要素を更新する場合は、どのようにすればよいのでしょうか？
