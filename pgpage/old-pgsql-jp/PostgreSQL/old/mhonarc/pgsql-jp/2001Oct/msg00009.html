テーブルの構造：
create table my (
    no    serial,
    id    int4,
    arr    int4[10],
    t    timestamp,
    flag    bool);
です。
テキストのなかみはこのようなデータです。（TAB区切りです）
１    １    ｛１，２，３｝        true

’copy: line 1, Bad timestamp external representation ''　’とエラーが出ま
す、
そこでデータを
１    １    ｛１，２，３｝    １９９８−０１−１８        true
に変更したら直ります、これは私の意図が違います、’ｔ’になにも設定したくない
です、
どうすればいいですか。
