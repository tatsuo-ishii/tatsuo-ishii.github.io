> 複数の列を合わせて主キーとして利用する方法が分かりません。

面倒くさいので、テーブル毎にＳＱＬ文を１セット作ってます。
（後々を考えてSQL文はテキストで保存してる）

DROP TABLE XXXXX_TBL;
CREATE TABLE XXXXX_TBL (
    KEY_FIELD_1  NOT NULL,
    KEY_FIELD_2  NOT NULL,
    FIELD_3,
    .......... ,         ここの , を忘れずに
    PRIMARY KEY (
        KEY_FIELD_1,
        KEY_FIELD_2
    )
);
GRANT XXX ON XXXXX_TBL TO username;   で権限を与える
(例)GRANT ALL ......は全ての権限を与える
