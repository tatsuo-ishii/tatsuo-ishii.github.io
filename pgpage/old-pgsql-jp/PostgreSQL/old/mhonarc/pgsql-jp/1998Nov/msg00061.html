さくまです。
オラクル7.3 などを使っています。

distinct と distinctrow はよくわからないので、
group by と min,max でなんとかしています。
ピントはずれだったらごめんなさい。

> skk=>  SELECT DISTINCT yomi from minakaji_private_jisyo 
>        WHERE yomi LIKE 'な%' and okuriari = 0 ORDER BY date;

select		yomi, min (date) as mindate
from		minakaji_private_jisyo 
where		yomi LIKE 'な%' and okuriari = 0 
group by      yomi
order by	mindate ;
