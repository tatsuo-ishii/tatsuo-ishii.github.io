  • From: Shoichi Kosuge <s-kosuge@str.hitachi.co.jp>

小菅＠日立です。

　WWW ブラウザ経由でDB にアクセスする方法をいろいろ考えて
ます。
　どこかにあったperl スクリプトをちょっといじくって（どこ
にあったか不明）、任意のSQL 文を発行できるCGI に仕立て上げ
ました。
　create table なんかもそのままSQL 文を書けばテーブルを作成
できるのですが、もっとブラウザらしい方法でできないものかと
考えています。
　テーブル名入力欄と、属性名入力欄と型入力欄をずらっと並べた
ものと、submit ボタンをつけたページ、というような方式を考え
ましたが、これだと属性の個数が決まってしまいます。
　Java を使うと属性が可変個になる問題を解決できそうですが、
既にあるようでしたらそれを利用したいと思います。

　というわけで、WWW ブラウザ経由でcreate table を発行する
いい方法がありますでしょうか。
