On Tue, 25 Jun 2002 11:11:30 +0900
"Kenji Tanaka" <kenji@venture-works.co.jp> wrote:

> フィールド定義の変更したいのですが、下記の方法でやって
> みましたが、うまくいきませんでした。
> 
> alter table 顧客マスター alter 備考 varchar(200);
> 
> 備考のフィールド長さを１００文字から２００文字に変更したいと
> だけです。

現在PostgreSQLでは属性の変更は出来なかったと思います。
ですので、以下の様にフィールドの属性を変更する必要があります。

select * into temp_table from source_table;

drop table source_table;

create table source_table (....);

insert into source_table select .... from temp_table;

drop table temp_table;
