> falseを0に戻さないとデータが不整合になりますので、
> 戻したいのですが
> 
> DB=>UPDATE tblname SET value1=0 WHERE value1=false ;
> 
> ERROR:  Unable to identify an operator '=' for types 'bytea'
> and 'bool'

> 
> DB=>UPDATE tblname SET value1=0 WHERE value1='false' ;

> and 'unknown'

>
