そうですか．
どうして無いのでしょうね？
コンパイル時にエラーが出ていませんでしたか？

ソースコードのディレクトリで，

/bin/ls -l src/bin/psql/psql

を実行して，「そのようなファイルやディレクトリは
ありません」というエラーが出るのであれば，

cd src/bin/psql
/usr/bin/make

と入力してエラーを確認してください．

もし，件のファイルがあるのであれば，postgresユーザ
で，

cd src/bin/psql
/usr/bin/make install

とすれば，psqlがインストールされます．

さんのご指摘にもありましたが，最新の7.2.1をインス

いるのかがよく説明されており，良かったと思います．
