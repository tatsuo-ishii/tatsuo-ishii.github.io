'%' を含む文字の LIKE 検索では、ESCAPE 句を使用します。

ESCAPE に使用する文字は何でもかまいません。
（例）
先頭に % があるものを探す場合；
select * from aaa where content ~ '^%';
と同じことを LIKE 述語で書くと
select * from aaa where content LIKE '#%%' ESCAPE '#';

途中に % があるものを探す場合；
select * from aaa where content ~ '%';
と同じことを LIKE 述語で書くと
select * from aaa where content LIKE '%x%%' ESCAPE 'x';
