[pgsql-jp 5166] PGSQLODBC Driver Install problem

  • From: Kazuhiko NARUMI <narumi@hit.ac.jp>
  • Date: Thu, 27 Aug 1998 13:18:21 +0900
はじめまして、北海道工業大学の鳴海と申します。

さて、今回、PostgreSQL-6.3.2 で作成したデータベースに
Windows 98 にインストールされた MS Access97(SR1) から
アクセス出来るようにするため、PostgresSQL ODBC Driver(
6.30.0249) をインストールしよう postdrv.exe を実行したところ、

ODBC Driver Manager(ODBC32.DLL) is not installed!

と言うメッセージが表示され、インストールすることが出来ません。

ちなみにコントロールパネルの「３２ビット ＯＤＢＣ」を実行し、
「ＯＤＢＣコア コンポーネント バージョン情報」でドライバー
マネージャ 3.0.23.1 が表示されている（インストールされている）
事を確認の後、postdrv.exe を実行し、Select Components 画面に
おいて Do not install Driver Manager を選択してインストールを
開始しました。

以上の問題を解決する方法をご存じの方がいらっしゃいましたら
ご教授願います。

よろしくお願い致します。
