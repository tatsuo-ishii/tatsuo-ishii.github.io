はじめまして、瀧澤と申します。

「PostgreSQLのSQL92サポート状況]
http://www.sra.co.jp/people/t-ishii/PostgreSQL/6.3/sql92.html
で、WHERE句のIS NULLはサポートがyesになっているようなのですが、

SELECT * FROM test_table WHERE test IS NULL;

というSQLを発行すると、testの値がNULLであろうと一致しないようです。

ちなみに

SELECT * FROM test_table WHERE test IS NOT NULL;

の場合は、すべて一致してしまいます。

PostgresSQLではIS NULLは使用できないのでしょうか？
