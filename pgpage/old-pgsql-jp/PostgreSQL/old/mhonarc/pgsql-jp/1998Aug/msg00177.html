相場＠ＩＳＯＬです。

普段業務用のテストマシンとして使用しているマシンに空きが出来たので 6.3
をインストールしてベンチマークを取ってみました。

最初は OS 付属のコンパイラー(xlc) でビルドしたのですがあまり結果がよろし
くなかったので GCC をインストールして再挑戦してみました。結果はほとんど
変わりませんでしたが。
やっぱりユーザーの少ない OS ではチューニングが甘いのでしょうかねぇ。もち
ろん手間がかかったのは bison と flex と GNU Make のインストールぐらい
で、一発でコンパイルできるのは素晴らしいことなのですが。以下記録です。
