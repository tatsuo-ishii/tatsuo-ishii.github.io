ちょっとあげ足取りみたいで、言いにくいのですが、 SQL  では、
NULL と 空文字列 (  ''  )とは、厳密に区別しないと混乱するので、
『 空文字列 』とか、『文字列が空でも』というような表現の方が
いいと思います。

この場合は、意味が十分通じますから問題ないのですが、これから
本格的に SQL を取組まれようという方のための 老婆心 です。

他意はありませんので、どうか気を悪くせずに聞いてください。
