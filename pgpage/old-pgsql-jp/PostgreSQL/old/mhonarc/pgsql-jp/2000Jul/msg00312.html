　こんにちは、わたなべといいます。

> psqlだけ日本語が使えないと類推したのですが、どのようにすれば日本語が使え
> るようになるのでしょうか。

　ホームディレクトリに .inputrc という以下のようなファイルを作成し
て、ログインし直せば日本語が入力できるかと思います。ただし行編集し
ようとすると崩れます。行編集も日本語に対応させたい場合は readline 
ライブラリを作り直す必要があります。

set convert-meta off
set meta-flag on
set output-meta on
