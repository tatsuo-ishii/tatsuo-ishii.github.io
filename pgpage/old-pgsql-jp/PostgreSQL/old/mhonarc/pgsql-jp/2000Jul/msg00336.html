コマンドサーチパス・環境変数の設定ののち、initdbを
実行すると下のようなエラーが出てきます。

[postgres@mmx src]$ initdb

We are initializing the database system with username postgres (uid=40).
This user will own all the files and must also own the server process.

Creating Postgres database system directory /usr/local/pgsql/data

Creating template database in /usr/local/pgsql/data/base/template1

ERROR:  Error: unknown type 'oidvector'.

        syntax error 12 : parse errorinitdb: could not create template
database
initdb: cleaning up by wiping out /usr/local/pgsql/data/base/template1
[postgres@mmx src]$

わかりませんが、おもえば、.configure時に　multibyte   ---disable
のようなメッセージがあり、他にもけっこう　--no　の行が
多かったような気がします。
