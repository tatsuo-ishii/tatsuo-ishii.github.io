>   psqlの中で、(SQL文の“COPY”コマンドではなくて)
> /COPY でテキストファイルをテーブルに取り込もうと
> しています。
> 　man pqsl によりますと、「規定のdelimiterは、
> TAB」となっていますが、これを別の文字（例えば“,”）
> にする書式ってどのように記述するんでしょうか。

# /COPY -> \COPY, pqsl -> psql ですよね。

7.0 より古いバージョンだと TAB 固定じゃないでしょうか。
7.0 以降は SQL の copy 文 と同じように using delimiters 'characters'
で デリミタ文字の指定ができます。

User's Guide には with delimiters 'characters' と書いてあるけど
with だと parse error になります。
