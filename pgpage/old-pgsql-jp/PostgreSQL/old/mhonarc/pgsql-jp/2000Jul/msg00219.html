>ただ、この方法だと、シリアルを取得後、処理が途中で異常終了すると、

PHPだと、
	$sql = "INSERT INTO hogehoge VALUES boohoo " ;
	$ret = pg_Exec( $dbh, $sql ) ;
	$oid = pg_GetLastOid( $ret ) ;
	$sql = "SELECT * FROM hogehoge WHERE oid = " . $oid ;
	$ret = pg_Exec( $dbh, $sql ) ;
と、言う風に oid を持ってきて、serial番号やら他の値やらを取ってきています。

SQLだとやり方がいまいち分からないのですが、INSERTをするとoidが帰ってくるので
その値を使えば、誰かが使っていても大丈夫ですよね。
