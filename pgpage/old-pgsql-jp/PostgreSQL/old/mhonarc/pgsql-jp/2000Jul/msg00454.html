> 姓名の間に必ず半角空白が入っているので、これをうまく利用すれ
> ば検索は可能じゃないのかな？と思って色々と試しています。

問題を単純化して、1つの列に、半角空白で区切られた 姓 と 名 が
格納されていて、姓と名の一部分だけ入力して検索したいというこ
とならば、次の SQL 文でできます。

SELECT * FROM users WHERE namef LIKE '%やま% %たろ%';

注: % %  の間は、ASCII SPACE 1 つです。

これで、'やまだ たろう' にも、'かやま しんたろう' にも、
'やま たろ' にも合致します。ただし、ASCII SPACE でなく、
全角空白が入っている場合は合致しないので注意。

PostgreSQL 独自機能になるけど、正規表現  ~   を使えば、もっと
いろんな検索パターンを作れます。
