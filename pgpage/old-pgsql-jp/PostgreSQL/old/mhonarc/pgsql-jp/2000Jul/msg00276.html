天野です

> PostgreSQL超初心者の川地です。
> 今、ストアドを作成しているのですが、一つ教えていただきたいことがあります。
> create function で戻り値を複数指定することは可能なのでしょうか？
> 
> create function(int4) returns ..... as '
> select field01,field02 from table where field03 = $1 '
> language 'sql';
> 
> というストアドをつくりたいのです。

以前教えていただいた方法に、返したいフィールドを持つ
テーブルを定義し、返り値の指定に、そのテーブル名を指定
する、という方法があります。

が、複数フィールドが詰まったデータを返すことが出来ても、
ひとつずつしか取り出せない為、結局そのフィールド数分
関数を発行しなければならないので、あまり意味がありません。

また、パフォーマンスを期待して、ストアドを使用することも、
あまり意味がないようです。

詳しくは、
 [pgsql-jp 12583] SQL関数で複数の列を返したい
を、参照してください。


いわきりさんのページで検索できます。

http://datula.mio.org/~iwakiri/pgsql_jp/
