Toshiyuki Zaoya wrote;

>create table usr_date
>{
>     usr_id       text;
>     user_ps      text;
>     user_kengen  int2;
>     ps_date      datetime;
>}

>というテーブルを作成しようとして
>
>WARN:type name lookup of datetime failed
>
>とけられて作成できません。
>これはバージョンが低いからでしょうか？
>対処法としてはやはりバージョンをあげて再インストールしかないでしょうか？

文法間違えてるからでしょう。
ミスは
1.　{}じゃなくて()
2.　1.で間違ってるのでしょうがないけど
　　()の最後に「;」がいる
3.　「;」じゃなくて「,」
　　最後のところは「,」いらない(私もペーストするとよくミスる)
　　どっちにしても、この場合「datetime;」は変

なので、上記create table文ではテーブルは絶対にできない。
自分の作ったSQL文が間違ってそうと思いませんでした？

やりたいのはこうじゃないですか？
create table usr_date
(
     usr_id       text,
     user_ps      text,
     user_kengen  int2,
     ps_date      datetime
);

＃シーラカンス本(他の物でもいいけど)見てますか？
