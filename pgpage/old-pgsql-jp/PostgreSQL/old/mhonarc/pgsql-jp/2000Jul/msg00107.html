山本@ｆｓｃと申します。

oracleでいうNVL関数を function として作成しました。
宜しければ、評価をお願いいたします。

nvl(引数１,引数２)
返り値：
　引数１がNullの場合、引数２を返却する。
　引数１がNullでない場合、引数１を返却する。

======================================================
create function nvl(text,text) returns setof text as 
'select case when $1 is null then $2 else $1 end'
LANGUAGE 'sql';
======================================================

わかっていること、
・日付の検索の際、型が違うとはじかれてしまうため、
　select * from hoge 
　where datetime(nvl(modday,creday)) > '2000-07-06';
　としてあげなければならない。
