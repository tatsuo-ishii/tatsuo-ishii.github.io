PHP4のCVSツリーに（4.0.2dev） にも同じ機能ですが、

bool pg_setclientencoding([int connection,] string encoding)
string pg_clientencoding([int connection])

という関数を追加してあります。
PHP4をお使いの方はお試しください。

石井さんが作成されたものと引数と機能は同じですが、混乱を避けるために
関数名も同じにした方が良いかもしれませんね。
