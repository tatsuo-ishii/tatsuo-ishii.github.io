[pgsql-jp 16185] ~*やLIKEに複数のキーワード使いたい

神谷@岡崎市と申します。

"~*" や"LIKE"に複数の検索キーワードを与えることは出来るのでしょう
か？

PostgreSQL+PHP3+Apacheを使用して、Webブラウザ経由でDB検索していま
す。とりあえずキーワードが各項目１つづつの物は検索出来る様になっ
たのですが、今やりたいのは、ブラウザ上の Textフィールドにスペース
区切りでキーワードを入力した場合に、PHP側でスペース毎に単語を区切
って整形後、 PostgreSQLの "~*" 演算子にそれらを一括で与えたいんで
すが、"~*"や"LIKE"に複数のキーワードを与える方法がわかりません。

例えば、
tech=# select * from tr_report where mcname IN ('CP-2', 'CP-4'); 
は問題無く検索できているようですが、
tech=# select * from tr_report where mcname ~* ('CP-2', 'CP-4');
ERROR:  parser: parse error at or near ";" 
tech=# select * from tr_report where mcname LIKE ('CP-2', 'CP-4%');
ERROR:  parser: parse error at or near ";"   
と成ってしまいます。
tech=# select * from tr_report where mcname IN ('CP-2', 'CP-4%');
は、CP-2を含んだ物しか表示されません。

正しい使用法を教えていただけると助かります。
