> select distinct on name * from table;すると、

# DISTINCT ON 属性名 * FROM って、ANSI SQL2 にないよう
# に思ったけど、PostgreSQL の拡張？。

DISTINCT は、重複しないようにするときに指定します。
他の列についても、何らかの指定をしたいときは、それなり
の書方をします。

仮に、各 name の 最大の num を表示したいのであれば、

SELECT name, MAX(num) FROM table GROUP BY name ;

ＳＱＬの参考書で勉強することをおすすめします。
