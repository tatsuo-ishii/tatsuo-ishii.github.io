FreeBSDのversionがわからないのですが，少なくとも3.1-RELEASE
以降であれば，packageにPostgreSQL 6.4.2があるので，こいつを
利用するのが簡単です．
# もちろん，自分でmakeすれば，より一層PostgreSQLについても
# FreeBSDについても理解が深まるのですが．

わたしの手元で動いているやつは，以下のような
ports/japanese/postgresqlです．


  % head /usr/ports/japanese/postgresql/Makefile
  # New ports collection makefile for:    PostgreSQL-j
  # Version required:     6.4.2+Multibyte-patch
  # Date created:         25 Nov 1998
  # Whom:                 Ichiro Fukuhara <ichiro@ichiro.org>
  #
  # $Id: Makefile,v 1.8 1999/04/17 19:43:23 max Exp $
  #

  DISTNAME=       postgresql-6.4.2
  PKGNAME=        ja-postgresql-6.4.2

ports-current(最新版のports)にはPostgreSQL-6.5.1がすでにあり
ます．
# こっちは--with-mb=ほげ が付いていないので，Makefileに手を
# 入れる必要があります．
