桑村です。

  堀田さんはお忙しいようなので、psql のEUCパッチを勝手に6.5.1
に対応させました。添付のパッチを(仮に)/tmpに置いて、

cd postgresql-6.5.1
gzip -dc /tmp/psqlj-6.5.1.diff.gz | patch -p1 

をしてから、configure, make, make install してください。
もとのプログラムは psql、パッチの当たったのは psqlj と言う名前で
インストールされます。
