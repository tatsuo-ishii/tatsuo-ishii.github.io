石井です。ここだけ。

> また、Macintoshからの接続方法って何かありますか?

Java って声はたくさんあるので、井上さんに教えて頂いた pgsql.tcl を御紹
介します。これは libpgtcl と違って全部 tcl で書かれたライブラリです。
Mac 用の Tcl8.0 と一緒に使うと Mac から PostgreSQL にアクセスできます。
ちゃんとselectもできました。PowerBook 2400c/180 + MacOS 8.1 で確認しま
した。日本語については、Tcl 次第です(日本語化された Tcl8.0ってあります
か？)

ついでに pgaccess も動かそうとしたのですが、そもそも Widget が一部正し
く表示されず完動には至っておりません。でも、特に落ちるわけでもないので、
かなり脈はありそうです。
