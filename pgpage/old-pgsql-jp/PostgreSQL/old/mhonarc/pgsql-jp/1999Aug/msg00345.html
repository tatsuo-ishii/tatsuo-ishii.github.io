　この辺りについてですが、シーラカンス本などに載っていますでしょう
か？

>         set client_encoding to 'SJIS';

　を試してみたところ、

≫testbase=> set client_encoding to 'SJIS';

≫SET VARIABLE

≫通知:承認されていない　変化

　と表示されますので、このエンコードは承認されてないという事のよう
です。
　案の定、COPYでテキストを読み込んで、Accessにリンクしてみたら、カ
ナ部分は意味不明の漢字の羅列に変換されてしまっていました。

　半角カナがUNIX環境では嫌われるのは承知ですので、もしクライアント
エンコーディングの変更が上手くいかないときは、半角→全角の変換も考
慮しなければならないでしょうが、perlも始めたばかりの私にそんな大そ
れたプログラムが書けるかどうか‥‥(--;)
