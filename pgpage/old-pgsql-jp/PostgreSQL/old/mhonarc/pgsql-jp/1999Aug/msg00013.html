片岡＠稲城市＠シーラカンス本で勉強始めたばかり  です。

漢字の扱いで困っています。
それも、ML[9841]で話題になっていたような難しい漢字でなく、
ごくありふれた漢字−−例えば、「所」とか「美」とか−−
でつまずいています。おそらく、文字コードの設定がどこかおかしいのだと
思われますが、よく解りません。(文字コードについてよく解っていません)
ご教示よろしくお願いします。

PostgreSQL 6.5 での設定： initdb --pgencoding=EUC_JP
VineLinux 1.1(2.0.36) での設定：インストール時の日本語環境のまま

====== 実例 start =========
[hiro3@localhost hiro3]$ psql test

test=> create table jusho1 (氏名 text, 住所 text);
ERROR:  parser: parse error at or near ")"
test=> create table jusho1 (氏名 text, 住い text);
CREATE
test=> \copy jusho1 from ./jusho1.txt
Successfully copied.
test=> select * from jusho1;
氏名               |住い
-------------------+----
片岡裕j東-    
(1 row)
====== 実例 pause =========

1)テーブル作成で「住所」を「住い」に変えたら CREATE できた。
2)テキストファイル(jusho1.txt、下記)から、データを取り込んだら
  「美」以下が壊れている。
[hiro3@localhost hiro3]$ cat jusho1.txt
片岡裕美        東京都稲城市

====== 実例 restart =========
test=> insert into jusho1 values (
test-> '片岡裕美','東京都稲城市'
test'> );
test'> 
test'> 
[3]+  Stopped                 psql test
[hiro3@localhost hiro3]$
====== 実例 end =========

3)プロンプトから入力しても、うまく行かない。
  プロンプトそのものが「'>」表示になり、抜けれない。
  (C+z で抜けた)
