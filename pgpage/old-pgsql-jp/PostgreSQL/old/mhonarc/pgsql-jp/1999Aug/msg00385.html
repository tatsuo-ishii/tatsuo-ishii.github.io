須貝＠信越半導体（株）と申します。

初めてメール致します。
宜しくお願い致します。

PosgreSQLはTurbo3.0の
「Apache 1.2.6 + PHP 2.0.1 + 日本語+αパッチ 5.3 」
で学習してるレベルです。

一応、
データベースを作成でき、リモートアクセスできる
環境にはなっております。

質問は、
ODBCを使ってAccess97でリンクした場合、
テーブルに新規データを追加できるように
するにはどう設定したらいいのか、アドバイス
頂けますか。

・ODBCのバージョンは「 6.40.00.06」です。

・システムDSNで設定してます。

・Driverの設定は下記のようになってます。
　選択した項目は＊です。

<<PostgreSQL Driver Setup>>
  Data Source:PostgerSQL
  Database:mydb
  Server:sv1
  User Name:postgres
  Description:
  Port:5432
  Password:************

<<Options(Advanced)--Driver-->>
  Disable Genetic Optimizer
  KSQO(Keyset Query Optimization)
  Recognize Unique Indexes
  Use Declare?/Fetch
  CommLog(C:psqlodbc.log)
  ReadOnly(Default)
  Parse Statements
  Cancel as FreeStmt(Exp)
  
  Unknown Sizes
    *Maximum  Don'tKnow  Longest
  
  Data Type Options
    *Text as LongVarChar  Unknowns as LongVarChar  *Bools as Char
  
  Cache Size:100  MaxVarchar:254  MaxLongVarChar:8910
  
  SysTable Prefixes:dd_;
  
  Connect Settings:

<<Options(Advanced)--DataSource-->>
  ReadOnly
  Show System Table
  Row Versioning
  
  Protocol
   6.4  6.3  *6.2
  
  OID Option
   Show Column  Fake Index
  
  Connect Settings:
