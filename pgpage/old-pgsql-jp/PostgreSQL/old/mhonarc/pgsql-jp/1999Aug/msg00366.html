こんにちは。フォローがないようなので、ひとこと。

SELECT COUNT(*) FROM  table ; では、何か支障がある
のでしょうか？。困っていることや、やりたいことを、
具体的に書いた方が、アドバイスを受けやすいです。

私は、大きな表では、SELECT COUNT(*) で、時間がかか
るので困っています。

Informix では、テーブルの件数をどこかに持っているよ
うで、WHERE や GROUP BY がつかない単純な表の件数な
らば、百万件の表も 1 件の表もほぼ瞬間的に件数が表示
されます。

PostgreSQL のシステムカタログを探し回ったのですが、
そのようなデーターはわかりませんでした。もしあるな
らどなたか教えて。
