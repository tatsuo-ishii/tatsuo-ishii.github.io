ありがとう御座います。自分でｇｒｏｕｐ ｂｙについて少し調べてみます。

> ところで、「割り算」に直接対応するSQLってそもそもあるんですか？普通は 
> subselect などの組合せで実現するのでは？

はい、標準のＳＱＬでは出来る事になっています。論理学で真な物は代数学で表せ
ると思うのですけどＳＱＬは代数学と直にリンクしていますので、論理学で真なも
のはＳＱＬでも書けるそうです。
例えばこんな感じです↓

select  SUPP
from    S
where  not exist
          (
	select ......
	from .......
	where not exist (  select ......
			   from .......
                                      where .......
			)
	)

つまりＮＯＴ
ＥＸＩＳＴ（引き算）を２回分をかけると論理学で言う（∀）代数学でも割り算に
相当するものが表されると言うものです。
↑これがＰＳＱＬで出来るのかが知りたかった物で質問致しました。
