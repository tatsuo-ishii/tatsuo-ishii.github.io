Turbo Linux だからどっちか... っていう性質のものではないです。ユーザの
好みでどちらでも使えます(もちろんインストールされてればの話ですが)。

echo $SHELL

とすると、

/bin/bash

などと表示されますので、判断が付きます(この場合は bash)。あとはパスワー
ドファイルを見ても分かりますね。今 postgres でログインしているのだった
ら、

grep postgres /etc/passwd

などとすると、なにやら表示されると思いますが、

...:/bin/bash

と行の最後の方に出てくるのが、そのアカウントで使っている shell です。

ところで、不幸にも

/bin/sh

となっている場合は、相当使いにくいはずですので、Linux 系なら /bin/bash 
に乗り換えることをお勧めします。乗り換えは chsh というコマンドで可能で
す。

ところで、Turbolinux ということでちょっと心配なのですが、すでに 
PostgreSQL のパッケージがインストールされている場合は、それを忘れずに
削除して下さい。

rpm -a -q|grep -i postgres

で何か出て来たら、それを rpm -e というコマンドで削除します。
詳しくは、
http://www.sra.co.jp/people/t-ishii/PostgreSQL/mhonarc/pgsql-jp/1999Jan/msg00359.html
を御覧下さい。
