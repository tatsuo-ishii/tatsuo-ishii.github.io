たぶん、OracleのSavepointのようなことではないですか？
Oracleの場合、
savepoint a
select * from table_ a where ... for update
update table_a set ...
savepoint b
select * from table_b where ... for update
update table_ b set ...
...
commit b
...
rollback a
というような感じでTransactionをネストすることができます。
