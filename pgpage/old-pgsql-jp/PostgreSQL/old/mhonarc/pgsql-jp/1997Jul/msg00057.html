こんなことでしかお役に立てませんが...

kane> BSD4.4 lite 派生の OS は全て持っているのではないでしょうか？

NetBSD 1.2.1/SPARC, gcc version 2.7.2 の場合も,
/usr/include/machine/endian.h, /usr/include/sparc/endian.h がハードリ
ンクされてあります.

内容はFreeBSDと同じくこんな感じです。

#define LITTLE_ENDIAN   1234    /* LSB first: i386, vax */
#define BIG_ENDIAN      4321    /* MSB first: 68000, ibm, net */
#define PDP_ENDIAN      3412    /* LSB first in word, MSW first in long */

#define BYTE_ORDER      BIG_ENDIAN

SPARCなもんで, BYTE_ORDERはBIG_ENDIANにセットされています.
