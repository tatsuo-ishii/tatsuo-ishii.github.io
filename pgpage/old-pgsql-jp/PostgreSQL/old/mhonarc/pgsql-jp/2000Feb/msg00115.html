> 特にnullな必要は無いのですが、某名刺読み取り機を使うと苗字しか

名刺読みとりのアプリケーションがどうなっているのかわか
らないのですが、名前 kana_mei 列の値が省略されるために、
NULL 値になってしまうだけならば、DEFAULT 句をつかって、
列省略時に、NULL でなく空白などをデフォルト値として 
INSERT するようにできます。

create table t_hoge(
    kana_sei text default '' not null,
    kana_mei text default '' not null
);

ただし、insert into t_hoge values ('xxx', NULL); の
ように、NULL 値が明示的に挿入されるような場合には、
この方法はつかえません。
