こんにちは。

> バックエンドの postmaster は PostgreSQL の super user である
> postgres によって起動されますが、ユーザ postgres の環境変数
> LANG に ja_JP.ujis なとの値がセットされていると、データベースの
> unique index の値に日本語を入れた場合、文字列が違っていても
> 一意制約違反で insert できなくなることがあります。

具体例を上げておいた方が良かったですね。

create table test (a varchar(10));
create unique index test_index on test (a);
insert into test values('あああ');
insert into test values('ああい'); <---- ここ
で、
Cannot insert a dupliacte key into a unique index
のエラーとなります。
