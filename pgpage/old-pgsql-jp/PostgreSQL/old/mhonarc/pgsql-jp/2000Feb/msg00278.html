始めまして、菅原@NCDと申します。

count(*) の動きについて疑問を持ったので投稿しました。

下記のように 2 つのテーブル t1, t2 を作成し、 3 件ずつデータを INSERT 
したところ、 select count(*) from t1, t2; すると、 9 ではなくて、 3 が
帰ってきてしまうのが、私が変だなあと思ったところです。

以下にpsqlでの操作の様子をペーストします。
（psqlが返す、細かなお返事は削除しました。）

-----------------------------------------------
sugawara=> create table t1 (id int2);
sugawara=> insert into t1 values (0);
sugawara=> insert into t1 values (1);
sugawara=> insert into t1 values (2);
sugawara=> create table t2 (id int2);
sugawara=> insert into t2 values (0);
sugawara=> insert into t2 values (1);
sugawara=> insert into t2 values (2);
sugawara=> select * from t1,t2;
id|id
--+--
 0| 0
 1| 0
 2| 0
 0| 1
 1| 1
 2| 1
 0| 2
 1| 2
 2| 2
(9 rows)

sugawara=> select count(*) from t1, t2;
count
-----
    3
(1 row)
-----------------------------------------------
この最後のselectが返す数値は、 3 ではなく 9 でなければならないのではな
いかと思うのですが、如何でしょう。

テーブルに含まれるレコード数を変えて試してみたところ、 from で指定して
いるテーブルのうち、最初のテーブルの件数を、そのまま返しているようで
す。

使用しているもの。
