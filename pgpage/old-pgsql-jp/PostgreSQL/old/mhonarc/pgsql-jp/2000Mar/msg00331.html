  • Importance: Normal
PostgreSQL 6.5.3なんですが、ＪＤＢＣ経由で配列オブジェクトを取得しに行くと
その時点で例外が発生してしまい困ってます。
例外の詳細を表示させてみると「サポートされない物が実行された」との意味合いの
英文が表示されました。
ソースは以下で、ResultSet::getArray("column name"); を実行すると例外発生しま
す。
-----------------------------------------------------------
      Statement stmt = con.createStatement();
      String  sql = "SELECT * FROM myarray";
      ResultSet rs = stmt.executeQuery( sql );
      rs.next();

      Array   ar = rs.getArray("ar");　　　　　　　　　　　　<----ここで例外
      Object  obj = ar.getArray();

      BigDecimal  big[] = (BigDecimal[])obj;
      for ( int i = 0; i < big.length; i ++ ) {
        out.println("<BR>Data["+i+"]=" + big[i] );
      }
      rs.close();
-----------------------------------------------------------
もしかしたらＪＤＢＣでは配列アクセスは未実装なのでしょうか？

ご意見よろしくお願いします。
