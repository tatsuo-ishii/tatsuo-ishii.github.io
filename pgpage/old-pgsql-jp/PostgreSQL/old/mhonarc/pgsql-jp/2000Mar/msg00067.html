既存の perl script を、フィルタ−として使いたいなら、
ファイルにおとさなくても、here document を使って、
次のように書けます。 実行可能 perl script ファイルを
hoge.pl  とした場合。

#!/bin/sh
psql -q -t ＤＢ名  <<  \!END_SQL |  hoge.pl
    select .......  ;  -- ＳＱＬ文
!END_SQL
