境田です。

Naofumi Kondoh <nkon@shonan.ne.jp> さん wrote:
> ソフト工房の近藤です。
> 
> ただ、SQL を実行していないスクリプト行で、CTRL+C ( ^C )
> 押下で、スクリプトを中断しようとしても、次のメッセ−ジ
> がでるだけで、スクリプトの実行をとめられません。
> 
> >>> CANCEL request sent >>>

　今、もしよろしければ次のパッチをあててテストしていただけ
ますか。

1."exec_sql_main.c" の 233行に "else"を追加。
------
230   /*------- Interactive defaults(cntl-C=>cancel) -------*/
231   if (!isatty(0) || !isatty(1))
232           pset.notty = 1; /* Noninteractive */
233   else
234           pqsignal(SIGINT, handle_sigint);
------

2. srcディレクトリに移動して make, make install。


# と書いているときに久保さんのパッチがでました。久保さんの
　パッチの方がより適切かもしれません。私の方は、少しサボっ
　ていますが、とりあえず簡単なテストはOKになっています。
