大野です。

>hoge| poo
>----+----
>foo |baa
>foo |boo
>
>というような行が足せてしまうのですが、そうでなくて「キーである項目 
>hoge にはすでに foo という値があるので、もう一個 hoge = foo な行を足す
>ことは出来ません」としたいのですが・・・・


キーというか、そのフィールドに対して重複させたくないというのであれば、
UNIQUE です。

CREATE TABLE hoge
(
 hoge  CHAR(10),
 poo   CHAR(10),
 UNIQUE (hoge)
);

# 構文があっているかどうかは、試してないのでわかりませんが、こんな感じ。
