  a1    INT,
  a2    INT,
  a3    INT,
  a4    INT,
  a5	INT,

３．aaaテーブル内のレコード数

　1,006,636レコード

４．psqlを起動してＳＱＬ文を発行

　select * from di where dd_no=5972;
　　該当レコードは１７件で、瞬時に結果が表示されます。

　select * from di where dd_no=5973;
　　該当レコードは７件で、瞬時に結果が表示されます。

　ところが、
　select * from di where dd_no=5972 or dd_no=5973; 
　　該当レコードは２４件で、結果が表示されるまでに約４２秒かかります。

＊どうして、こんな簡単な検索に４２秒もかかるのか理解できません。
　PostgreSQLのインストールに問題があるのでしょうか？
　それとも、このスピードは仕様とおりなのでしょうか？
