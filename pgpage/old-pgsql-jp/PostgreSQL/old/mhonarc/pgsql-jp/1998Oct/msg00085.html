ken> ところが、ODBCDirectで接続したところ、
ken> 
ken> Set Rs = Con.OpenRecordset(sSQL)
ken> 
ken> でODBC呼び出しに失敗してしまいます。
ken> insert文の実行Executeはうまくいっています。

insert into time_table values(now()); 

このSQL文は、selectなどと違い結果のレコードセットを返すことはありません。
ですから、OpenRecordsetは使えません。Executeメソッドを用いて、
次のようにしてみてください。

Set Rs = Con.Execute(sSQL)
