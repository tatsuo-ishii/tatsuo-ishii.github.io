pg_connect は、正常終了していますか？。

$DB = pg_connect('dbname=hoge_db') or die("ERROR:PG_CONNECT");

などとして確認してみてください。

pg_connect が正常ならば、簡単な SQL 文から実行してみて、
どこでおかしくなるか確めるのも、遠回りながら、一考かと。
ポイントは、pg_numrows が、0 になるか否かです。

例えば、

SELECT version();

SELECT * FROM stock_tb LIMIT 5;

SELECT  relname, relacl FROM pg_class WHERE relname = 'stock_tb';

SELECT * FROM pg_tables WHERE tablename = 'stock_tb';

等々。
