Pg.pmの話題になってしまうのですが、
[pgsql-jp: 25334] Re: 7.2 での Pg.pm のインストールエラーについて 
を見させていただきまして、Pg.pmをインストールしようとしました。
手順は、札幌のかずさんの手順を参考にさせていただきまして、

./configure --with-perl
cd src/interfaces/perl5/
make clean
export SRCDIR=./
export POSTGRES_INCLUDE=/usr/local/src/postgresql-7.2/src/include/
export POSTGRES_LIB=/usr/local/src/postgresql-7.2/src/interfaces/libpq/
perl Makefile.PL
make
make test
(rootに変わって）
make install

としました、
ところうが、

makeを正常に行えませんでした。
