■ 画像の扱い方
[環境]
特になし

[問題]
画像の扱い方(jpg, gif など)

[原因]
特になし

[対策]
・large object で画像データ、一般にはバイナリデータを扱える。
・large object の制約：
　o pg_dumpでバックアップできない
　o 非常にたくさんのlarge objectを登録すると性能が悪くなる
・普通にファイルで画像を管理する手もある。
