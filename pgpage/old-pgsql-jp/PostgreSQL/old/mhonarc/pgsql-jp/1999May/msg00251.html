applet の場合はそうではないでしょう。
前回も書きましたが applet はクライアントにクラスファイルが
ダウンロードされて、クライアントの Java VM で実行されます。

ですので JDBC を applet のクラスと伴に jar でパックして
いない限り、クライアントに JDBC が必要になります。
逆に言えば、applet がダウンロードされる環境が PostgreSQL
のあるサーバと同じでないなら、サーバにJDBC は必要ない
でしょう。

また HTML、CLASS ファイルは Web Server のマシンに置く
必要があるのであって PostgreSQL とは関係ないはずです。
