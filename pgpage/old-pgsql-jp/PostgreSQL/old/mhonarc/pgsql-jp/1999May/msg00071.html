その他思いつくままにお約束を書いてみます．参考になりますで
しょうか．

1) initdb に実行許可がでているかを確認する．
    % ls -l /usr/local/pgsql/bin/initdb
    -r-xr-xr-x   1 postgres   11703  Aug  1   1998 /usr/local/pgsql/bin/initdb*

2) initdb がサーチパスにいるかどうかを確認する (type 版)．
    % type initdb
    initdb is /usr/local/pgsql/bin/initdb

3) shell script が正しく認識されているか確認する．
    % file /usr/local/pgsql/bin/initdb
    /usr/local/pgsql/bin/initdb:      executable shell script
    % head -1 /usr/local/pgsql/bin/initdb
    #!/bin/sh

4) shell を明示的に起動してみる．
    % sh /usr/local/pgsql/bin/initdb
