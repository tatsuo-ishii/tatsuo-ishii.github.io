6.4.2 の doc/postgres.tar.gz の Chapter 8. Data Types の Character
Types の章に書いてあります。同じ章の表にも "Best choice" とあります:-)

SQL92 defines two primary character types: char and varchar. Postgres
supports these types, in addition to the more general text type, which
unlike varchar does not require an upper limit to be declared on the
size of the field.
