毎日、チェックさせて頂いてます。貴重な情報が得られて大変助かっております。
Microsoft Excel97のVBAでlinux上のPostgreSQLにアクセス（レコードの追加）した
いのですが、OpenRecordsetを発行すると、読取専用になってしまい上手くいきませ
ん。どうも、私の切り方が悪いようなのですが、御存知の方が居られましたら教えて
頂けないでしょうか。よろしくお願いします。

１．table_aに対して、Msqueryを使い、接続とレコードの追加、更新は成功しまし
た。
２．Excel97のVBAから、以下のコードを実行。ウォッチ式でOpenConnection発行後の
Updatableプロ
　　パティがTrue（更新可能）である事は確認しました。
　　が、OpenRecordset発行後はUpdatableプロパティはFalseになってしまいます。

※１と２共に同じDSNを使用し、ユーザーもパスワードも同一です。

    Set wrkODBC = CreateWorkspace("NewODBCWorkspace", "username",
"password", dbUseODBC)

    Set conPubs = wrkODBC.OpenConnection("Connection2", dbDriverNoPrompt,
False,

"ODBC;DRIVER={PostgreSQL};DATABASE=AAAAAA;SERVER=111.222.333.444;PORT=5432;
           FAKEOIDINDEX=0;SHOWOIDCOLUMN=0;ROWVERSIONING=0;
           SHOWSYSTEMTABLES=0;DSN=PosrgreSQL;")

    Set rstCnct = conPubs.OpenRecordset("select * from talble_a",
          dbOpenDynaset, dbExecDirect, dbPessimistic)

    With rstCnct
            .AddNew

            ....<<中略>>

            .Update
    End With
