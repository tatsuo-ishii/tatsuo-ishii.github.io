[pgsql-jp 3596] pgsql related archives updated for contrib/Database

桑村です。

  PostgreSQL 関係の Plaiaware 向けパッケージのアップデートができました。
明日(月曜)のお昼くらいに,

	http://www.rccm.co.jp/~juk/outgoing/Database/

の下あたりにアップロードしておくつもりです。宜しくお願いします。

内容は以下のとおりです。
--
pgsql63:  PostgreSQL-6.3 + 6.3mbPL2.patch
pgsql63:  
pgsql63:  PostgreSQL は SQL 準拠のサーバ／クライアント方式のオブジェクト
pgsql63:  リレーショナル・データベースマネージメントシステム(ORDBMS)です。
pgsql63:  端末型クライアントプログラム(psql)、レポートジェネレータ内蔵の
pgsql63:  表形式 GUI クライアントプログラム(pgaccess)、C, C++, tcl 言語の
pgsql63:  インターフェース・ライブラリの他、PERL, JAVA(DJBC), Python の
pgsql63:  インターフェースおよび C 言語の埋め込み SQL プリプロセッサを含
pgsql63:  みます。PHP/FI との連携で WEB ページをインターフェースにもでき
pgsql63:  ます。また、マルチバイトパッチ(EUC)により日本語の正規表現による
pgsql63:  検索も可能となっています。
pgsql63:  
pgsql63:  (注意: psql は readline を、pgaccess は tcl/tk をそれぞれ必要と
pgsql63:  します。)
pgsql63:  

pgdoc63:  PostgreSQL-6.3 Documents
pgdoc63:  
pgdoc63:  PostgreSQL のドキュメント類です。PG95-DOC プロジェクトでの成果
pgdoc63:  等、和訳ドキュメントも含まれます。Apache サーバをインストールす
pgdoc63:  ると、これらのドキュメントのディレクトリへ WEB ブラウザ経由でア
pgdoc63:  クセスできます。
pgdoc63:  

pgexp63:  PostgreSQL-6.3 Examples and Contributions
pgexp63:  
pgexp63:  PostgreSQL のチュートリアル、サンプル、テスト、コントリビュー
pgexp63:  ションなどのプログラムです。C 言語によるファンクションは共有形
pgexp63:  式のオブジェクトにしてあるので、そのまま SQL で登録して利用で
pgexp63:  きます。これらのプログラムは、PostgreSQL を使う上での参考となり
pgexp63:  ます。
pgexp63:  

readln21: GNU readline-2.1 + jpatch.97120
readln21: 
readln21: GNU readline ライブラリです。コマンド行の編集や履歴情報の展開
readln21: などを可能にします。PostgreSQL の文字端末型クライアント・プロ
readln21: グラム(psql)を使うためにはこのライブラリが必要です。
readln21: 

pgpost11:  PgPost-1.1
pgpost11:
pgpost11:  PostgreSQL を使った新旧郵便番号の検索システム。WEB と、Tcl/Tk
pgpost11:  によるインターフェースプログラムがあります。インストールする
pgpost11:  ためには、PostgreSQL のデータベース・ファイルシステムに 40MB 
pgpost11:  近くの空き容量が必要です。
pgpost11:  (注意: PostgreSQL と PHP/FI 組み込み Apache サーバもしくは、
pgpost11:  Tcl/Tk が必要です。)
pgpost11:

apach125: Apache-1.2.5 + PHP/FI module
apach125: 
apach125: PHP/FI(+LIBPQ)モジュールを組み込んだ Apache HTTP サーバです。
apach125: PHP/FI 経由で PostgreSQL へのアクセスが可能です。PHP/FI の記述を
apach125: 含むページの拡張子は、".phtml" となってます。また、URL でユーザ
apach125: ホームを表す '~' を省くことができるパッチを当てています。この機
apach125: 能を使うには srm.confの中で "UserDirWOTilde on" を設定します。
apach125:  
apach125: (注意: PHP/FI と PostgreSQL がインストールされている必要がありま
apach125: す。)
apach125:  

phpfi201: PHP/FI-2.0.1 + kanji2.0.1.patch
phpfi201: 
phpfi201: PHP/FI は Perl に似た記述による HTML ファイル埋め込み型の簡単な
phpfi201: プログラム言語です。当モジュール組み込み Apache WEB サーバを利用
phpfi201: することによって、HTML ファイル内に直接プログラムを書くことが可
phpfi201: 能です。このパッケージでは PostgreSQL にアクセスすることを可能に
phpfi201: してます。
phpfi201: 
phpfi201: (注意: PostgreSQL がインストールされている必要があります。)
phpfi201: 

wagora12: W-AGORA-1.2b2 + Japanese message
wagora12: 
wagora12: W-AGORA はウェブ上で文書を共有することのできるシステムです。
wagora12: 日本語のメッセージを追加してます。PHP/FI で書かれてますので
wagora12: 文書の検索に swish が使えます(日本語での検索未対応)。
wagora12: 
wagora12: (注意: PHP/FI の使える Apache サーバが必要です。)
wagora12:
