石井です。

>本題の char[] がエラーになる件ですが、本家に報告しておきました。

面白い workaround を教えてもらいました。

create foo(c _char);

これで、ちゃんと c は char[]になっています。つまり、配列の場合、内部的
には 型名[] --> _型名 に変換されるようです。同様に、

create bar(i _int4);

も int4 の配列になります。
