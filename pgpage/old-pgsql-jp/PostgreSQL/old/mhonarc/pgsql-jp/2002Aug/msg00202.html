始めまして、油井といいます。

mbディレクトリ下のconvertファンクションですが、
./src/backend/utils/mb/mbutils.c

select convert (test, 'UNICODE') from test1
だとエラーが表示されます。

select convert (test, 'EUC_JP','UNICODE') from test1
では、大丈夫なので、mbutils.cのpg_convert(PG_FUNCTION_ARGS)
がおかしい様です。

 /* free memory allocated by namein */
	pfree((void *)dest_encoding_name);

が
pfree((void *)src_encoding_name);

ですね^^;
cvsの最新版でも同じような記述でしたので、報告しておきます。
