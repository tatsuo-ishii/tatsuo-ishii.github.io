よくよく見てみれば、ALIASだったのですね。
引数は参照のみしかしませんから、別に変数を定義する必要があります。

create function hogehoge ( integer  , integer ) returns integer as '
declare
  a alias for $1;
  b alias for $2;
  a_sum integer;
begin
  a_sum := a + b ;
  return a;
