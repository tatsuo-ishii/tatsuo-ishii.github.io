境田さん、こんにちは。

>   trim(trailing SET from STRING)とすると、STRING文字列の最後
> から先頭に向かって、SET文字列の中のいずれかの文字と一致すれば
> trim します。

あっ、SET文字列そのものを削除すると思い込んでいました(^^;
なるほど、納得です。

「PostgreSQL User's Guide」を調べてみると Description に trim 
characters from string としっかり書いてありますね。


文字列を削除する場合は、substringとpositionの組み合わせで
できました。

select substring('11 a 1' from 1, position(' a 1' in '11 a 1'));
 substr
--------
 11
