それは PHP をどのようにリンケージして作ったかによります。

PostgreSQL の libpq とリンケージを行う際に、スタティックリンクを
行えばお好みのものができるでしょうし、共有ライブラリを使用する
ダイナミックリンクを行えば、ランタイム環境にも PostgreSQL の
共有ライブラリが必要になります。

PostgreSQL のインストール時には共有ライブラリもインストールされる
のが普通ですし、その環境で PHP を普通に make すると共有ライブラリを
使う形になると思います。

したがって、そちらの希望する PHP を make するには、configure 後の
Makefile を自分で書き換える必要があるでしょう。
