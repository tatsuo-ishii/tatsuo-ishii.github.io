> 
> $keyword_m；

> }
> SELECT date_entry,name,keyword,idmagazine FROM $table
>              WHERE   keyword ~* '$keyword'
>                           or idmagazine = $keyword_m";
> 
> PERLのことで申し訳ありませんが数字を判定する方法を教えてください。

Perlで、数字を判定する方法は、以下のパターンマッチ判定で可能です。

if ($keyword =~ /^\d+$/ ){
        print "suuji!!\n";
}else{
	 print "not-suuji!!\n";
}
