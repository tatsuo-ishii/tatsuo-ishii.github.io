> trimの使い方を間違っているのでしょうか？

　PostgreSQLの処理方法は次です。

  trim(trailing SET from STRING)とすると、STRING文字列の最後
から先頭に向かって、SET文字列の中のいずれかの文字と一致すれば
trim します。

  SET="a 0"
  STRING="10 a 0"

　の場合、{ 'a' | ' ' | '0' } が trim の対象になりますので、
STRING は先頭の "1"だけが残ることになります。

　せんのでわかりません。
