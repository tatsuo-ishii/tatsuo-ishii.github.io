yumiko sato wrote:

> test.PGC: In function `main':
> test.PGC:13: warning: return type of `main' is not `int'

main関数はvoidじゃだめです．
  void main() → int main(void)
として，main関数のおしりに
  return 0;
と入れませう．

> test.PGC: In function `error_exit':
> test.PGC:66: warning: passing arg 1 of `printf' from incompatible pointer type
> 
stderrに出力したいなら，fprintfを使わないとダメです．
   printf(stderr,  →  fprintf(stderr,

> この後は私の問題なのでしょうね。

そうです．もうちょっとC言語の勉強が必要なようです．
