シーケンスというのを作ればお望みのことはできます。

create sequence users_id_seq start 1;
create table users (id int, name text, email text);
create unique index users_id_index on users (id);

INSERT INTO users VALUES (NEXTVAL('users_id_seq '),'xxx','yyy');

1からスタートする数字はNEXTVALで呼び出すごとにカウントアップして
行くのでオートインクリメント（自動に増加）機能として使えます。

私もこういう機能があることをここのMLの方に教えてもらいました。
