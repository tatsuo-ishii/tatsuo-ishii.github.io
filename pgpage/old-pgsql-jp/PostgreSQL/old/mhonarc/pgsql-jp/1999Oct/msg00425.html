本田です。度々すいません。

csh/tcshでpgbashを実行すると、
stones.my.domain% pgbash
# PostgreSQL   pgbash   ( bash & exec_sql built-in command )
Unmatched "
stones.my.domain% bash
となってしまいます。

bashでは
bash-2.03$ pgbash
# PostgreSQL   pgbash   ( bash & exec_sql built-in command )
{ USAGE: exec_sql [option] ["SQL"] } { HELP: help exec_sql }
{ alias: type 'alias' (e='exec_sql',etc..  is already set) }
{ Last : type 'exit' or type 'q' }
pgbash> l
と問題ありません。

/usr/local/pgsql/bin/pgbashに、
#!/usr/local/bin/bash
とつけるか、
echo "{ USAGE: exec_sql [option] [\"SQL\"] } { HELP: help exec_sql }"
を
echo '{ USAGE: exec_sql [option] [\"SQL\"] } { HELP: help exec_sql }'
に直す必要がありました。

以上です。
