  FAQ なのかも知れませんが、下のような事をすると、index が使われて
いませんで遅いのですが、早く処理をする方法は無いでしょうか？

(月次の売上データを持たせているのですが、何処まで入っているかを確認
する必要があって、この様な処理をしています。データが多くなってきたら、
いやに時間が掛かっているのでチェックしていて気が付きました。)


create table sales(
    id_article int4, 
    year_month int8,
    value      float8,
    volume     float8,
    unit_price float8,
 primary key(id_article,year_month)
);
create index sales_year_month_key on sales (year_month);

$psql salesdb
salesdb=> explain
salesdb-> select max(year_month) from sales;
