Apache + PHP + Postgre を使って、audio/WAV のファイルのデータベース
を作成中です。pgimage を参考にさせていただいてます。

pgimage では、以下のソースでフレームの1つに画像ファイルを表示させて
いますが、WAV ファイルなのでアンカータグをつかってダウンロードさせ
るようにしたいと思っています。

show_img.php3
<?
  header("Content-type: $type");
  include "init.inc";
  $con = connect();
  pg_exec("begin");
  $fd = pg_loopen($con, $oid, "r");
  pg_loreadall($fd);
  pg_loclose($fd);
  pg_exec("end");
  pg_close($con);
?>

アンカータグから直接データベース内の large object のデータを
