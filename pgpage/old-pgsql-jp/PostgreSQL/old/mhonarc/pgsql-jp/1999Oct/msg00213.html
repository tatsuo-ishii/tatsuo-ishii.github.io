解決にはぜんぜんなっていませんが、ちなみに
create view aaa as select count(val1) from testdb distinct val1;
ではview が作成されず、
create view aaa as select count(val1) from testdb group by val1;
でview を作成したものの、
select count(val1) from aaa;
select count(count) from aaa;
を行なってもだめみたいです。
結果はともに
[pgsql-jp 11168]
で書かれているような内容になります。
