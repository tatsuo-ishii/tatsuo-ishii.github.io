jp.patch ありがとうございました。> 関係者の皆様。

今回は、jp.patch も素直に当たり、make ではちょっと苦労したものの、
regression test も一発で通りました。当方は、6.2 fix 版で JDBC を試して
見ました。石井さんの JDBC in 6.2b8 との違いは、何も削除せずに javac が
通ったという事くらいですか。あと、石井さんの記事には書いてありません
でしたが、当方は、CLASSPATH 

/usr/local/pgsql/postgresql-6.2/src/interfaces/jdbc/

を追加しました。

postgres@agent[63]%javac postgresql/*.java
postgres@agent[67]%javac JDBC_Test.java
postgres@agent[68]%java -Djdbc.drivers=postgresql.pgDriver JDBC_Test "jdbc:postgresql://agent/test?user=uchiyama"
Connecting to Database URL = jdbc:postgresql://agent/test?user=uchiyama
Connected...Now creating a statement
Ok...now we will create a table
Now we will insert some columns
Inserted some data
Now lets try a select
Back from the select...the following are results
row 0   1       1
row 1   2       1
row 2   3       1
Ok...dropping the table
Now closing the connection
postgres@agent[69]%

当方での、6.2 fix 版ウィスコンシンベンチマークは、追って、
学生から報告があると思います。
