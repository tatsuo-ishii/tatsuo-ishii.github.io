　メーリングリストの皆様、はじめまして。
文昭堂印刷の水野さゆりと申します。
ＵＮＩＸは、ど・ど・ど初めてです。現在のレベルは、
やっと、Redhad4.2インストールができたぐらいです。（でも、まだ解りません。）
　ＤＢは、アクセス９５やＤＢＡＳＥ・ｄｂｘｌなどをさわってました。
レベルの低い質問ばかりなってしまうかもしれませんが、
ただ、会社のテスト用のＢＳＤＩ3.0サーバーにPostgre6.2.1を
こっそり入れたくて参加させて頂くことにしました。
もちろん、ＤＢ関係には、とても興味がありPostgre6.2.1を
見つけたときには、「これだ！！」と思っちゃいました。

　ところで、早速低レベルな質問で恐縮なのすが、
実は、まだインストール出来ていません。
過去のＭＬは、見てみたのですが、私のように
インストールで困っていらっしゃる方は、あまりいないみたいで
また、ＢＳＤＩ3.0へのインストール関係は、事例が少ないみたいです。
　ちなみに、ソースを解凍して/usr/local/pgsqlに入れました。
その後、jp patchをあててMakefile.customを作成して
configureをしました。
make allをかけると

You must use GNU make to use Postgres.  It may be installed
on your system with the name 'gmake'.

NOTE:  If you are sure that you are using GNU make and you are
       still getting this message, you may simply need to run
       the configure program.

と怒られました。

　わたしには、この上記のメッセージの意味すら分かりません。
きっと、ＭＬの皆さんにしてみれば、バカバカしいと思うかもしれませんが、
どなたか、インストール方法をご伝授ください。
宜しくお願いいたします。
