now という予約語を使って、

test=> create table t (t timestamp);
CREATE
test=> insert into t values('now');
INSERT 3482258
test=> select * from t;
t                     
----------------------
1997-10-14 14:35:03+09
(1 row)

となります。

ちなみに、6.2 なら SQL92 の予約語 current_timestamp が使えるので、

test=> create table t(t timestamp);
CREATE
test=> insert into t values(current_timestamp);
WARN:unknown type 1083 
test=> insert into t values(current_timestamp);
INSERT 533838 1
test=> select * from t;
t                     
----------------------
1997-10-14 14:33:04+09
(1 row)

となります。
