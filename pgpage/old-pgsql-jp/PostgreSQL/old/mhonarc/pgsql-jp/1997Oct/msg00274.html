->表の中を見ようとするとダンマリになってしまいます。

	PostODBC のセットアップの時に 「 Data Sources 」ダイアログ
ボックスで setup ボタンを押されました? 私は、このボタンを押し、以下
の設定を行いました。

	Data Source Name	テストだけなのでデフォルトの PostODBCTest にした
	Dscription		空白
	Server			PostgreSQL のあるマシンのマシン名
	Database		PostgreSQL に作ってあるデータベース名
	Port			デフォルトの5432
	User Name		データベースサーバーマシンのユーザーアカウント兼
				PostgreSQL のユーザー名

以上の設定がちゃんとなされていればつながると思いますが。
