# バージョンあげねば

> make allをかけると
> 
> You must use GNU make to use Postgres.  It may be installed
> on your system with the name 'gmake'.
> 
> NOTE:  If you are sure that you are using GNU make and you are
>        still getting this message, you may simply need to run
>        the configure program.

要するに GNU make を使いなさい。と言われていますね。 BSD/OS 3.0 を使わ
れているなら、

/usr/contrib/bin/gmake 

が 入っているはずですから、それを使うと良いと思います。

今は GNU make を新たに install して使ってますけど、以前は 

/usr/local/bin/make を /usr/contrib/bin/gmake のリンクにして使っていま
した。

BSD/OS だと、GNU make のinstall は簡単に終わってしまうので、挑戦してみ
るのも良いと思います。
