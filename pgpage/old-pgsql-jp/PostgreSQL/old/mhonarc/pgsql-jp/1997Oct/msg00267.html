At 2:10 PM 97.10.31, Toshio Uchiyama wrote:
>         現在、テーブルのリンクを実行中です。
>         PostgreSQL <-> MS-Access の接続手法の習得なので、Access97 のデータ
> ベースは空で良いと思い、空のデータベースを作成し、それを開いています。
>         次に
> 
> [ファイル]-[外部データの取り込み]-[テーブルのリンク]
> 
> でリンクダイアログボックスを開きました。
>         このダイアログボックスの「ファイルの種類」テキストフィールドで
> 
> ODBC データベース 0
> 
> を選択してみました。すると「データソースの選択」ダイアログボックスが
> 表示されますが、その後がどうして良いか分かりません。

それ以前にコントロールパネルの ODBC の「ユーザDSN」にて PostODBC の
構成を行って正しい値をセットしておく必要があります（必ずしも必須でな
いかもしれませんが、その辺はちょっと不明）。この際 Server には ip
address ではなくて FQDN を書く必要があります。hosts ファイルの中身で
も可。

次に Uchiyama さんの操作に戻り
・「データソースの選択」ダイアログボックス
・「マシンデータソース」タブをクリック
・データソース名「PostODBC ...」をダブルクリック
・コントロールパネルで設定したサーバ・Database・ユーザで接続され、参
　照可能なテーブルのリストが表示されます。
・リンクしたいテーブルを選び「OK」を押します。
・後は適当に答えます。

これでできるはずです。
