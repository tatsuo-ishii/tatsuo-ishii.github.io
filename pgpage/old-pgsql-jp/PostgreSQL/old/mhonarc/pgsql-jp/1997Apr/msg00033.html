そういえば、角川＠広島大学さんが提供する Vflib を利用した立体漢字ビッ
トマップ作成ツール vf3dxpm というのがあるそうです。

実は、年賀状を作る時に gd を使ってみました。vf3dxpm で文字を1文字ずつ
ピックスマップにして、netpbm で ppm に変換してからつないだ後に GIF に
変換して透明化します。それを写真に重ね合わせるというのをしました。

  vf3dxpm というのは直接使わなかったのでコマンドラインの使い方はわかり
ません。とりあえず、ga.xpm と sho.xpm という２つの文字のピックスマップ
データができているとして、

        % xpmtoppm ga.xpm > ga.ppm
        % xpmtoppm shou.xpm > shou.ppm
        % pnmcat -lr ga.ppm shou.ppm | ppmtogif > gashou.gif

で繋がった「賀正」のイメージデータができます。透明化は gd ライブラリ附属の
プログラムを使い、

        % webgif -t 0 gashou.gif

でできます。

  もう1つ、下地用ににイメージを用意します。実際は写真をスキャナで読み込み
640x480 の大きさの GIF データにしたものを使いました。
  PHP/FI で以下の処理をすると写真の中央やや上に文字が入ります。

        <?
        $impic = ImageCreateFromGif( "nenga.gif" );
        $imcha = ImageCreateFromGif( "gashou.gif" );
        ImageColorTransparent( $imcha, 0 );
        ImageCopyResized( $impic, $imcha, 240, 10, 0, 0, 640, 480, 600, 400);
        ImageGif( $impic, "new.gif" );
        >

＃ こんなのは、イメージ作成用のツールを使えば簡単にできると思います。
＃ ただ単に gd を使ってみたさでやってみました。(^^;
