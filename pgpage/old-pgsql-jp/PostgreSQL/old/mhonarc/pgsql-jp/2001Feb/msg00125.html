> Ｏｒａｃｌｅで
> 
> select a.code, b.name
>  from table1 a, table2 b
>  where a.code = b.code(+);

PostgreSQL 7.1 前でしたらOuter Joinができないので、
SQLを書き換えて別の方法で実現するという方法が
「完全攻略ガイド」に載ってます。

select a.code, b.name from table1 a, table2 b
where a.code = b.code
union all
select a.code,null from table1 a
where not exists (
  select 'x' from table2 b
  where a.code = b.code
);

要するに a ∩ b に a から bを引いたものを足した結果が
Outer Joinで実現される結果と同じになりますということです。
