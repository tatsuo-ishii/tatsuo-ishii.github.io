最初のメールのテーブルへのINSERTの際、

INSERT INTO test (f1) VALUES (1);
--対象列１、値の個数１

や、

INSERT INTO test VALUES (1,'abcd','efgh');
--対象列３、値の個数３

なんかは当然通りますが、

INSERT INTO test (f1) VALUES (1,'abcd','efgh');
--対象列１、値の個数３

というのはダメ、とずーっと認識していたのですが。
