早速質問ですが、psql等でのINSERTの際、対象列の個数と値の個数が
一致していなくても追加できる事を偶然知ったのですが、これは周知の
事なのでしょうか？
私は、個数が一致していなければ行の追加はできないものとばかり思っ
ていましたので、ちょっと驚きでした。以下は、psqlのコピーです。

create table test(
f1 int2,
f2 text,
f3 text,
primary key (f1)
);
insert into test(f1,f2,f3) values (1,'abc');
select * from test;
「結果」------------------------>
  f1 | f2  | f3
----+-----+----
  1 | abc |
(1 row)
<------------------------------

以上、何かご存知の方、ご教示下さい。宜しくお願いします。

ちなみに環境は、Red Hat Linux 6.2、PostgreSQL 7.0.3です。
