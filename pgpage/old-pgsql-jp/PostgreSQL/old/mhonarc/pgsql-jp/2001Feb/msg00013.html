細川＠グッデイです。

本日で、ニューヨークで開催の Linux World も幕を閉じました。
Bruce Momjian のセッションにも参加しましてなかなか有意義な時間を過ごす
ことができました。

さてさて、2月の9日午後6時ごろより、大阪駅前第2ビルの(株)グッデイにおき
まして、ニューヨークの Linux World 報告会(実はただの飲み会)を開催する予
定です。

# ニューヨークに行かずとも大阪へ行くだけであの熱い雰囲気を、味わうことが
# できるかも・・・(ほんとか！？)

PostgreSQL な人にはうれしい、Bruce Momjian の「PostgreSQL Introduction
and Concepts」を2名様にプレゼント。
# 2割引とおまけにつられてつい3冊も買ってしまった・・・(^^;)
# PHP 関係の本も多数売られていたけど、Momjian の本でお金が無くなってしま
# ったのでありました・・・(すでに翻訳出てるし良いよね・・・だめ？)

それ以外にもこまかい(?) Linux World グッズも提供の予定です。

ちなみに、Bruce の本は、PostgreSQL 付属のドキュメントを読む気になったこ
とのある人には、はっきり言って、まったく必要の無い本です。
英語の勉強と PostgreSQL のさわりを知りたくって、かつリファレンスは電子
的なものより座右に置いておきたいという人にうれしい内容となっています。

こういう人で、グッデイに遊びに来れそうな方は、どうぞふるってご参加くだ
さいませ。
あ、こういう人でなくとも、Linux World 報告を聞きたいとか、ただ単に飲み
に行きたいとか言う人もぜひとも参加ください。

と、言うわけで参加者は、「参加希望」ということで、このスレッドにつなげ
るか、私宛(hosokawa@good-day.co.jp)にメールください。
よろしくです。

では、みなさん、2月9日に大阪で会いましょう！！
