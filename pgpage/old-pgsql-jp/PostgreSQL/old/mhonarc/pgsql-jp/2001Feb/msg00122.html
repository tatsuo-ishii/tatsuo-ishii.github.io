man pg_passwdを見るかきり、

>・1つのファイルで複数のユーザおよびパスワードが作成される（できる？）

で、pg_passwd実行時は、＄PGDATAにcdするようです。ここに入れるのは、
(マニュアルに従ってインストールした場合は)postgresさんだけです。

1. postgresさんになる。
2. cd $PGDATA
3. pg_passwd <パスワードファイル名>
4. ユーザ名、パスワードを入力

以下、Plamoでの実行例:

db:~/data> ls
PG_VERSION               pg_group_name_index      pg_shadow
base                     pg_group_sysid_index     pg_variable
pg_control               pg_hba.conf              pg_xlog
pg_database              pg_log                   postmaster.opts
pg_geqo.sample           pg_pwd                   postmaster.opts.default
pg_group                 pg_pwd.reload            postmaster.pid

db:~/data> pg_passwd pg_pwd
Username: a
New password:
Re-enter new password:

db:~/data> pg_passwd pg_pwd
Username: b
New password:
Re-enter new password:

db:~/data> more pg_pwd
kamiya:394NJMwt4sYVE
a:jbW9OdwJ3Yu/w
b:kEPE1h6MxF5qI

# とりあえず、manを読んでみてください。
