> あるテーブルにデータをインポートしようとしています。
> ＣＳＶファイルをｐｅｒｌでＩＮＳＥＲＴ文に変換して行
> おうとしていますが、ｄａｔｅ型のカラムの一部を空欄で
> 登録したいのですができません。
> 取りあえず '1000-01-01' などとして、ダミーの日付を入
> 力することで誤魔化しています。
>
> ｄａｔｅ型のカラムに空欄を入力することは出来るのでしょうか？

date 型のカラムに、「空白文字」を登録することはできないようですが、
代わりに null を使用されてみてはどうでしょうか。

update (テーブル名) set (カラム名) = null where (カラム名) = '1000-01-01';

では、失礼します。
