こんにちは、小田切＠ミラクル・リナックスです。

３／１のMiracle Linux for PostgreSQL Version 1.0
http://www.miraclelinux.com/pressroom/details/2001022801.html

と、Miracle Linux for Samba Version 1.0
http://www.miraclelinux.com/products/product_info/samba.html

の発売を記念いたしまして、日本PostgreSQLユーザー会と日本Sambaユーザ会の
皆様に製品パッケージをプレゼントさせて頂きたいと思います。

と、いってもタダのプレゼントではなく、できればモニタ募集という形を
とりたいと思います。
「個人で自宅で使う」という方よりも「会社や学校で業務に使う」という
方に使用してもらって、感想や改良要望点などをあげて頂ければ幸いです。

感想レポートは任意です。量や内容・形式は問いません。
kodagiri@miraclelinux.com へメールでお願いします。
（このメールのReply-Toは、kodagiri@miraclelinux.comになっています）

同時に無料セミナーも開催しますので、ぜひ参加ください。
http://www.miraclelinux.com/event_seminar/event/night/night.html
（今回当選にはずれても、これにでるといいことがあるかも？）
私も３月１５日（木）・４月１２日（木）にSambaセミナーします。

当選した方で、このセミナーで製品を受け取ってもいい、という方は
懇親会を無料で招待致します。

モニタ募集は、日本PostgreSQLユーザー会の方にPostgreSQL版を５個、
日本Sambaユーザ会の方にSamba版を５個です。

さらに希望者には、ユニクロの春の新作帽子もプレゼントします。
帽子の雰囲気は
http://www.plathome.co.jp/item/miraclelinux/with.html
をご覧ください。
後頭部に小さく「MIRACLE」というロゴが入っています。
色は「黒」「むらさき」「オレンジ」「緑」がありますので
希望の色を選んでください。
（しかし、帽子の色は希望に添えないこともあります）


＜Miracle Linux for PostgreSQL Version 1.0モニタ募集申し込み要項＞

以下の事項を記入の上、kodagiri@miraclelinux.comへメールください。
募集締め切りは、３／７とします。

・お名前
・メールアドレス（日本PostgreSQLユーザ会に登録したもの）
  会員登録は、
   http://www.jp.postgresql.org/registration/
  を参照ください。
・所属（会社名、学校名など）
・住所
・電話番号
・当選した場合の、使用目的・使用場所
・当選した場合製品は「郵送して欲しい」「セミナーで受け取る」
・春の新作 ユニクロの帽子「希望する」「希望しない」
  「希望する」場合は、色「黒」「むらさき」「オレンジ」「緑」を
   お選びください。 （帽子の色は希望に添えないこともあります）
