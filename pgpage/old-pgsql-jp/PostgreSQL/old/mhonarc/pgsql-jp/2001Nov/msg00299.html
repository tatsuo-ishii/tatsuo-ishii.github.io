[pgsql-jp 23883] JDBC ドライバの設定がうまくいきません

  • From: hirokazu_inoue@kyuden-is.co.jp
  • Date: Fri, 30 Nov 2001 13:57:10 +0900
　JDBCドライバを使いPostgresSQLにアクセスしたいのですがうまくいきません。
誰か分かる方がいたら教えて下さい。

環境は以下の通りです。
TurboLinux6.1　JDK1.3　PostgreSQL7.1.3

Javaは「/usr/java/jdk1.3.1_01」にインストールし、

.bash_profileには下の2行を追加してあります。
export JAVA_HOME=/usr/java/jdk1.3.1_01
export PATH=$PATH:$JAVA_HOME/bin

JDKの実行確認には下のソースを使いました。

class p15_1 {
  public static void main(String args[]) {
    short num = 1000;
    System.out.println(num + " is the value of num");
  }
}

[postgres@itsrv postgres]$ javac p15_1.java
[postgres@itsrv postgres]$ java p15_1
1000 is the value of num
と実行が確認できました。


次にPostgresSQLは、
「/usr/local/pgsql」以下にインストールし・環境設定を済ませ、
「regression test」で動作確認しました。


それからJDBCドライバの設定についてですが、

JDBCドライバとPostgreSQLにアクセスするサンプルプログラムは、
技術評論社の「PostgreSQL完全攻略ガイド」に添付されていた、
JDBCドライバ：postgresql.jar
サンプルプログラム：postgresql-example.jar
を使いました。

上の二つの.jarファイルを
「/usr/local/pgsql/share/java」以下に配置し、

環境変数「.bashrc」には、
export CLASSPATH=/usr/local/pgsql/share/java/postgresql.jar:/
usr/local/pgsql/share/java/postgresql-examples.jar
を追加してます。

[postgres@itsrv postgres]$ set | grep PATH
でCLASSPATHが設定されたのを確認してます。

そして、下のコマンドでサンプルプログラムを動かしてみるとエラーになります。
[postgres@itsrv postgres]$ java example.basic jdbc:postgresql:test t-ishii
""
Exception in thread "main" java.lang.NoClassDefFoundError: example/basic

その上、上記のソース（p15_1.java）をコンパイル・実行をしてみると、
[postgres@itsrv postgres]$ javac p15_1.java
[postgres@itsrv postgres]$ java p15_1
Exception in thread "main" java.lang.NoClassDefFoundError: p15_1
とコンパイルは成功するのですが、実行はエラーになります。

CLASSPATHを通した途端、javaコマンド自体うまく動作していない様なのですが、
分かる方がいらっしゃったらよろしくお願いします。
