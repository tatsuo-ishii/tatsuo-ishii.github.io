次の方法で、動くようになりました。

NetBSDのパッケージ作成方法を調べ、readlineを使用しない方法でコンパイルすることで動きました。

readlineを使用しないために、パッチを２個当てる必要がありました。
パッチは、NetBSDのftpサイトからpkgsrc.tar.gzをダウンロードしてきて
その中のpkgsrc/databases/postgresql/patches/の下にある
patch-aaとpatch-alを使用しました。
readlineに関係がありそうなのがこれだけだったので、これしか当てませんでした。
これでいいかどうかはわかりませんが。

configure の指定は、次のように指定しました。
./configure \
	--prefix=/usr\
	--enable-multibyte=EUC_JP \
	--enable-unicode-conversion \
	--disable-readline \ ←−−これを追加。
	--enable-odbc \
	--without-java \
	--without-perl \
	--without-python \
	--without-tcl \
	--without-tk \
	--enable-syslog\
	--enable-locale \
	--sysconfdir=/etc/pgsql \
	--mandir=/usr/share/man \
	--docdir=/usr/share/doc/packages \
	--includedir=/usr/include/pgsql \
	--datadir=/usr/share/pgsql
