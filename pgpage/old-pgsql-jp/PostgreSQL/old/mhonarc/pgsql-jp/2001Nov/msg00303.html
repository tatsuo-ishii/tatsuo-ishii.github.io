　postgresql-example.jar の中の、example ディレクトリに basic という
クラスがあります。（まともにインストールできていれば）


　実行ステートメントの

java example.basic jdbc:postgresql:test t-ishii ""

 は、ユーザ t-ishii を所有者とする test というＤＢがある限り正しいです。

　実際、私はこれを動作させています。もちろん、ユーザ名とＤＢ名は異なり
ますが。

　ですので、何故クラスが見つからないのか不思議です。
