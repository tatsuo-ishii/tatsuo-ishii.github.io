MLの皆さま、こんにちは。
吉田と申します。
今、以下のようなテーブルがあります。
create table t(code integer[10],name text)
配列codeの例えば1番目の要素をインデックスにしようとすると
create index ix1 on t(code[1])
ERROR:  parser: parse error at or near "["
上記エラーとなります。

勿論、以下のコマンドでも
create index ix1 on t(code)
ERROR:  DefineIndex: type _int4 has no default operator class
エラーとなります。
配列の要素をインデックスにするにはどうしたらよいのでしょうか。
それとも、そもそもできない（通常のカラムでないと駄目とか）のでしょうか。

環境は
PostgreSQL 7.1.1
OS Debian potato 2.2.17
です。

以上、宜しくお願いします。
吉田
