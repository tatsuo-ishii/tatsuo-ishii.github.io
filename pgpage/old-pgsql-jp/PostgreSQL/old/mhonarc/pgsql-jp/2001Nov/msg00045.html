1. tar コマンドが使える環境(Linux や Cygnus をインストールした Windows など)
でならば、コマンドラインで tar zxf (ファイル名)
2. Cygnus をインストールしていない Windows 環境ならば zip と tar をサポート
しているアーカイバ(WinRAR など)で解凍

　Windows 環境しかご存知でなく、tar とか Cygnus って何？という場合は、2.の方
法でとりあえず何とかなるでしょう。
