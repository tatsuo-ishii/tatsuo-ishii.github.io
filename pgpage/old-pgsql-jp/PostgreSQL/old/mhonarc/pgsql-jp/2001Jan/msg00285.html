ほそいです

「トランザクション」というのを、そういう使い方をした場合、

create table 口座マスタ (
  口座番号 varchar(10) primary key
, 残高 numeric(12,0) default 0
);

create table 入出金トランザクション (
  入出金トランザクション serial
, 口座番号 varchar(10)
, 日時 default now()
, 金額 numeric(12,0)
, 適用 text
);

ってなつくりになってるかと。

地域によったりして、その昔は「マスタ」「トランス」だとか
いわれてたりしました。
