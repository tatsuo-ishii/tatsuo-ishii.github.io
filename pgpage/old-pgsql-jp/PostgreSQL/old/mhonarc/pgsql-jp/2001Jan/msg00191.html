PHPは使っていないので、やりたい事と違うかもしれませんが..

> select ... from ... where ... and ...
> というSQL文を使ってすべての行を検索させる方法はないでしょうか。
> select ... from ...
> で条件をつけなければ検索できるのはわかりますが、その場合、SQLを2個作るか、
> where以降だけ別に作成するなどして、条件で置きかえる必要があると思います。
> たとえば、PHPで、50以上、50未満、全てという項目がある場合、「50以上」と「50
> 未満」は、最初のSQLで検索可能ですが、

whereの所だけ抜き出したとすると、
 50以上 : "where (列名 >= 50)"
 50未満 : "where (列名 <  50)"
 全て   : "where (ワカラン)"

という事でしょうか？

ある程度の範囲がわかるなら between を使うというのはどうでしょう？
最小値が 0 最大値が 100という場合なら
 50以上 : "where (列名 between  0 and  50)"
 50未満 : "where (列名 between 51 and 100)"
 全て   : "where (列名 between  0 and 100)"
これなら、 選択範囲の値を可変にするだけです。
