質問者の意図がわかりづらいので何なのですが、挿入された時点の日時を記録し
たいのであれば・・・
完全攻略ガイド(シーラカンス本)の第二版では、p.107 に 「insert した時点の
時刻が入るようにするには・・・」で説明がされています。

create table t1(i integer, d datetime default text 'now');

ですね。now という text を default 指定します。
(勝手に引用すみません)

current は、 「現在」とマークされるだけなので、select した時点の日時が表
示(select)されます。
