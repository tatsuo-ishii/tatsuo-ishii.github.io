今 Windows の Access97 で Form を作り、ODBC 経由で PostgreSQL
7.0.3 に接続して、改行を含むテキストを入れてみましたが、
エラーにはならず正常に動いているようです。

test=# \dt
  List of relations
 Name | Type  | Owner 
------+-------+-------
 test | table | toshi
(1 row)

test=# select * from test;    
 id |           name            
----+---------------------------
  1 | postgreSQL
Daemon
Penguin
(1 row)

そこで
pg_dump test > test.dump

して中を見てみると

COPY "test" FROM stdin;
1       postgreSQL\
Daemon\
Penguin

と '\' で改行を表わしているようなので、Access の Form から
'\' を入れてみると

COPY "test" FROM stdin;
1       postgreSQL\\\

ときちんと escape されているようです。:-)
という事で、改行を含む text も安心して扱えるという事なの
でしょうか?
