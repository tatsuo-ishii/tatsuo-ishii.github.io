うちでも同じですね。
'today'や'now'という文字列は定数として定義されているけれど、
関数としては、now()だけしか定義されていないということでしょう。


p=# select CURRENT_TIMESTAMP, CURRENT_DATE, CURRENT_TIME ;
       timestamp        |    date    |   time
------------------------+------------+----------
 2001-01-23 11:28:43+09 | 2001-01-23 | 11:28:43
(1 row)

p=# select 'today'::timestamp, 'now'::timestamp ;
        ?column?        |        ?column?
------------------------+------------------------
 2001-01-23 00:00:00+09 | 2001-01-23 11:29:12+09
(1 row)

p=# select now() ;
          now
------------------------
 2001-01-23 11:29:26+09
(1 row)

p=# select today() ;
ERROR:  Function 'today()' does not exist
        Unable to identify a function that satisfies the given argument types
        You may need to add explicit typecasts
p=# select version() ;
                      version
---------------------------------------------------
 PostgreSQL 7.0.2 on i386--netbsd, compiled by cc
(1 row)
