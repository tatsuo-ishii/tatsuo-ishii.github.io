Seedsと申します。

varcharからint4へのキャストで困っています。
6.5.3では問題なく動いていたのですが7.0では
エラーが出ます。

テーブルは

test(
id varchar(5),
num varchar(5)
);

で、データは以下のような感じです。

id    | num
------+-----
a     | 1
b     | 2
c     | 3
d     | 4

# select * from test where cast(num as int)>3;

と入力したら

ERROR:Cannot cast type 'varchar' to 'int4'

と出て、期待通りの結果を得ることができません。
できれば、numはvarcharのままで実装したいと思って
るのですが可能でしょうか？
