| create table B (b_id int4, b_aid int4);

| 得たいデータのイメージは
| 
| c_id | c_bid | a_title
| -----+-------+---------
| 100  | 10    | A0001
| 200  | 20    |
| 300  | 30    | A0002
| 
| となるような select 文を書きたいのですがよろしくお願いします。

select C.c_id, C.c_bid, A.a_title
  from A, B, C
  where A.a_id = B.b_aid and C.c_bid = B.b_id
union 
select C.c_id, C.c_bid, null
  from B, C
  where B.b_aid is null and B.b_id = C.c_bid;

でどうでしょうか？
