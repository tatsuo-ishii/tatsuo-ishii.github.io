Mamadaです。

> ある程度の範囲がわかるなら between を使うというのはどうでしょう？
> 最小値が 0 最大値が 100という場合なら
>  50以上 : "where (列名 between  0 and  50)"
>  50未満 : "where (列名 between 51 and 100)"
>  全て   : "where (列名 between  0 and 100)"
> これなら、 選択範囲の値を可変にするだけです。

例として数を出してしまったのがいけなかったようです。
実際には、数ではなく文字列で切り分けようとしています。
結論としては、部分的にプログラム中でSQL文を変更するというのが妥当な方法とい
うことですね。
