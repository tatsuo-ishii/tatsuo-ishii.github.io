市村＠札幌です。おはようございます。

最初に自己レスですが、使用したい新機能は「外部キー」ではなく
「外部結合」です。混乱した方がいれば、申し訳ありません。

> 再現性がはっきりしているなら是非教えてください．本家に伝えます．

問題を単純にして再現します。次のようなテーブルがあります。

create table table_a (
    id        int,
    data_a    int,
    data_b    int));

ここで、

select * from table_a where data_a > data_b;
-> OK

select * from table_a a
    where (select data_a from id = a.id) > (select data_b from id = a.id);
-> OK

select * from table_a a
    where ((select data_a from id = a.id) > (select data_b from id = a.id));
-> NG

と、カッコで条件を括るとエラーになってしまいます。
環境は、FreeBSD 4.2 Release です。

宜しくお願いします。
