> table B内の FB=[1,2,4,5,7]を抽出します。

table B内の FB01=[1,2,4,5,7]を抽出します。

> それなら GROUP BY と COUNT で対応できそうですが..
ついでに、例として PostgreSQL7.0.3で実行してみたSQLです。
BorlandのBDE上+Paradox形式でもOKなので汎用性は高いと思います。
BDEのローカルSQLは方言が少ないので..

select
 a.fa01,
 a.fa02
from a, b
where (a.fa01 = b.fb02)
  and (b.fb03 in (1,6))
group by
 a.fa01,
 a.fa02
having
 (count(a.fa01)=2);
