| select * from table_1 where name like \'%John%\';

| "\" in ...

| となってうまくいきません。

上記 SQL を見ると、「'」がエスケープされています。
おそらく、php3.ini の magic_quotes_gpc の設定が

magic_quotes_gpc = On   ; magic quotes for incoming GET/POST/Cookie data

となっているのでしょう。

上記を Off にして httpd を再起動するか、北村＠アシェット様のおっしゃるとおり、
stripslashes 関数を使うか、というところが解決方法かと。
