At 13:40 +0900 2001.01.18, Mamada wrote:
>PostgreSQL6.5で使えるデータ型一覧を出す方法を教えてください。

ソースを展開すると、doc/ ディレクトリがあり、その下に
.tar.gz ファイルがいくつかあります。これを展開すると、
postgres/datatype.htm というファイルがあるので、それを
見ればよいかと思います。

そういえば、いつもMac に持ってきてStuffIt Expander で展開して
ますが、ソースを展開すると、postgresql-6.5.1/INSTALL
というファイルの中に、次のように書かれています。

       12.     Install the man and HTML documentation. Type
         $ cd /usr/src/pgsql/doc
         $ gmake install
         The documentation is also available in Postscript
         format. Look for files ending with .ps.gz in the
         same directory.

たぶん普通はこうやるんでしょう。
