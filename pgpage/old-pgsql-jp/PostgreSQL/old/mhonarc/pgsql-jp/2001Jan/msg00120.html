とりあえず、ぱっと見て気がついたことだけ書きます。
perl script 中の

SEQID = '$FILE_SEQ' ;

という記述が間違いではありませんか。
メールにある情報からは、以下の２通りが推測できます。

  (1) SEQID のデータ型が数値型の場合
      → '$FILE_SEQ' という文字列型データを update しようと 
         しているので、 SQL として間違っている。

  (2) SEQID のデータ型が文字列型の場合
      → $FILE_SEQ 変数に格納されたデータを update しようと
         したのではないかと思いますが、意図されたようには動きません。 
         (SQL としては間違ってはいません)
         なぜなら、$FILE_SEQ がシングルクオートで囲われてます。
