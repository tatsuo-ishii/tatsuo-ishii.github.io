postgresを操作するユーザーの概念はわかったのですが、
Linuxユーザーにfooを追加してから
su - postgres
createuser foo
*********  databases?(y/n)　y
*********  new users?(y/n)　n
CREATE USER

と、fooユーザーは出来るんですけど、その後、本にはfooという
アカウントでログインし、以下のコマンドを実行します。
createdb foo
CREATE DATABASE

と、fooというデータベースが作成されるようですが

[foo@server foo]$ createdb foo
bash: createdb: command not foundって怒られちゃうんです(>_<)

ちなみにpostgresユーザーで実施すると出来るんですけど・・・・・
postgresでfooって言うデータベースを作成すれば当然Ownerはpostgresなので

ちょっと頭がこんがらがってしまっています。

くだらない質問でスイマセンがご伝授ください。
