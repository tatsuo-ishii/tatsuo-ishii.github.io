    PostgreSQLやOracleでもできますよ．
    
    PostgrSQL:
    
    SELECT *
      FROM site_info
     ORDER BY siteid
     LIMIT 5;
    
    Oracle:（8i以降じゃないと使えなかったと思います）
    
    SELECT *
      FROM (SELECT *
              FROM user_info
             ORDER BY userid)
     WHERE ROWNUM <= 5;
    
    
    上位ｎ個ではなくて，上位ｎ位だとPostgreSQLでは
    無理だと思いますが(^^;
    （上位ｎ位を簡単にだす関数はOracle8iでもEE版
