> ⊂綉と話がずれてしまいますが、JITやSwingはJDK1.3の

Java2 には JFC が標準で含まれているので、いわゆる Swing も使えます。
Swing を使ったプログラムが動かなかったということですか？

JIT については、これは別に Java2 に含まれているものではないので、
別途インストールが必要なはずです。

> 重ね重ねすみませんが、上記Linax環境まわりについて

このレベルの話では Linux 云々は何も関係ありません。PostgreSQL や Java
自身に関する知識の問題です。

Java の話は置いといて、本件については PostgreSQL のマニュアルや
ドキュメント類を参照することからはじめるべきでしょう。

PostgreSQL の市販書籍で何か… という事であれば、石井さんのいわゆる
シーラカンス本が適当でしょう。
とはいえ、この本は 6.5 について書かれている者なので、最近の 7.0.3
などのバージョンを使う場合にはそれなりの読み替えが必要になります。

市販書籍などはあくまで「参考」にするという事で、あくまでも
オリジナルのマニュアル/ドキュメントを参照するのが基本でしょう。
