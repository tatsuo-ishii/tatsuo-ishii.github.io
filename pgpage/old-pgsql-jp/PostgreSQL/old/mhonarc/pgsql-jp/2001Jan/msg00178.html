ところで，

> ----------------------------------------------------------
> 
> 問題を単純にして再現します。次のようなテーブルがあります。
> 
> create table table_a (
>     id        int,
>     data_a    int,
>     data_b    int));

かっこが多いですよ．

> ここで、
> 
> select * from table_a where data_a > data_b;
> -> OK
> 
> select * from table_a a
>     where (select data_a from table_a where id = a.id) >
>           (select data_b from table_a where id = a.id);
> -> OK
> 
> select * from table_a a
>     where ((select data_a from table_a where id = a.id) >
>            (select data_b from table_a where id = a.id));
> -> NG
> 
> とカッコで条件を括るとエラーになります。
