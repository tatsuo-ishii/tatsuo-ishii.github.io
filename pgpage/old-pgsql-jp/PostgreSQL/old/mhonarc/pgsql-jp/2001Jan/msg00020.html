はじめまして．松井と申します．
先日より，PostgreSQLを使い始めました．よろしくお願い致します．

早速で恐縮なのですが，お知恵をお借りしたいと思います．

Postgresql 7.0.3 で下記のようなテーブルを作成しました．

# CREATE TABLE test (test_date date,txt text);
CREATE

次にINSERTしてみました．

# INSERT INTO test VALUES ('2001-01-05','hoge');
INSERT 139349 1

日付けの「日」だけを取り出そうと思い，次のようにしてみました．

# SELECT date_part('Day',test_date) from test ;
date_part 
-----------
         4
  (1 row)


普通であれば，5が出力されるはずだと思うのですが，なぜか，4が
出力されていまいました．同様に，

# SELECT date_part('Mon',test_date) from test ;
# SELECT date_part('Year',test_date) from test ;

を試してみたのですが，こちらは正常に，1と2001を出力します．
どのようにすれば，5が出力されるようになるのでしょうか．
御教授頂ければ幸いです．

こちらの環境は，下記の通りです．

・PostgreSQL 7.0.3 は jumbo-20001204.patch.gzを当ててから，
  --enable-multibyte=EUC_JPをつけて，コンパイルしたものです．
  パッケージは使っていません．

・OSは，FreeBSD 4.1.1R と Kondara Linxu 1.2の両方で試してみましたが，
  どちらでも同じ問題が生じました．

その他，必要な情報があれば，お知らせください．

よろしくお願い致します．
