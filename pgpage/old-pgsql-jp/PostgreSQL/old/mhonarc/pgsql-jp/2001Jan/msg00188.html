どんぴしゃりでした。

** プロバイダのphp3.ini **
; internationalization
i18n.http_output = SJIS
i18n.internal_encoding = EUC-JP
i18n.script_encoding = auto
i18n.http_input = auto
i18n.http_input_default = SJIS

** 開発環境のphp3.ini **
; internationalization
i18n.http_output = SJIS
i18n.internal_encoding = EUC-JP
i18n.script_encoding = auto
i18n.http_input = auto
i18n.http_input_default = SJIS
  :
  :
最後に↓が記述してありました
; internationalization
i18n.http_output = PASS
i18n.internal_encoding = EUC
i18n.script_encoding = AUTO
i18n.http_input = AUTO

『PostgreSQL 完全攻略ガイド』に添付のPHP3.0.12 国際版の
iniとプロバイダにインストールされていたPHPのiniに違いが
あったことが原因でした。

>> 結局、画像出力の前に、i18n_http_output("pass") を追加し、

しょうか？
