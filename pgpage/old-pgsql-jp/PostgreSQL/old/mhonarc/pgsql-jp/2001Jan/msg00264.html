北村＠アシェットです。

> 質問者の意図は
> 1.「today」を使って「2001/1/17」に日付をinsertした
> 2.「2001/1/18」にselectしたら、昨日「today」でinsertした
> 　「2001/1/17」と出た(当たり前といえば当たり前なんだけど)
> 3.今日の日付「2001/1/18」が欲しかったので困った
> 
> ってことだと思うんですが。

質問者の

>      create table {
>              logname text primary key,
>              entry_day date default 'today'
>       };

ですが、この場合 entry_day の default 'today' には「create table を実行し
た日付」がデフォルト値として入ることになる（と思います）。
つまり、いつ何をインサートしても、デフォルトが 2000年1月17日 なのですよ。

実際には log にあるように、「insert 文を実行した日付」が入って欲しいのに
・・・って事だと思ったのですが。

ログ記録を引っ張りだすときに、記録した日付でなく、引っ張り出した日付が欲
しい・・って事は無いと。たぶんね。

まぁ、われわれ部外者があーだこーだ言ってても、本人が居なきゃしょうがない
ですね。笑
