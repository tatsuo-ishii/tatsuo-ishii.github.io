    create table t1( key  int4 , data1 varchar  );

    insert into t1 (data1) values ('hogehoge1') ;
    insert into t1 (data1) values ('hogehoge2') ;

つまり

    select * from t1 ;
    key|data1

      1|hogehoge1
      2|hogehoge2

それとも、関数などを作ることになるのでしょうか？こうすれば出来るよ、とい
