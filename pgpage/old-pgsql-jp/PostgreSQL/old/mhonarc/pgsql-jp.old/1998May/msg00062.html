  Linux の日本語パッケージで、 Slackware をベースにいくつかの日本語化プロ
ジェクト、（PJE(日本語拡張キット),JF(日本語FAQ), JG(ゲーム), JMAN(日本語
オンラインマニュアル)）の成果を一体化して組み込んでます。
# インストーラも日本語化してます。

  電中研のこじまみつひろさんがとりまとめをされてましてβテスト中です。
詳細は、

	http://www.st.rim.or.jp/~isle/Plagia/

を御覧下さい。

  僕は、Database 関係で PostgreSQL のパッケージをコントリブしようと思い立
ったのですが、せっかくなので、ついでに Apache+PHP/FI もパッケージにしまし
た。また、この ML にもいらっしゃる馬場さんや樋口さんらから free-Waise や
namazu などのパッケージの提供もされて、結構サーバ系で contrib が充実きてま
す。
# 日本語化 mozilla や MRTG など Network 関係も結構面白いものがあります。

  ほとんど参加者が趣味的にパッケージを提供している感もあります。(^^;
