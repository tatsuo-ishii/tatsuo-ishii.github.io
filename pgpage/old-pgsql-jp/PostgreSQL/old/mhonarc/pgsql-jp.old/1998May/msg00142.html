FreeBSD-users-jp ML に、これの移植をしている人からの投稿が
ありました。「FreeBSD へのポーティングも考えてはいるのだが、
スケジュールの関係で今は間に合わない。とりあえず Linux
エミュレーションで動かそうとしたらエラーが…」というような
内容です。

Linux エミュレーションで動く動かないという事はさておき、
私は FreeBSD 派 :-) なので、FreeBSD への日本語対応移植版が
出ると嬉しいです。
自分のところに導入するしないは別にして、色々なアプリケーションが
出てくるのは PC UNIX のためにはいい事ですし。

商用 DB だと、ORACLE の FreeBSD 版とか Linux 版が出るような事が
あると世間が PC UNIX を見る目が違ってくるかも知れない。
ORACLE はそういう事しないのかな？ NT 版よりもこっちを出して欲しいぞ。
Workgroup Server でいいからさぁ…
