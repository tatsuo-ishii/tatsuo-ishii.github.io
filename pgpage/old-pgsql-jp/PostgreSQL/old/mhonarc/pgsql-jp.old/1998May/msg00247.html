スズキの内田です。

先ほど、PostgreSQL の CD-ROM が届きました。 ケースに、モノクロプリンタで
印刷したラベルが、セロテープで張ってある SONY 製の CD-R です :-)

本家 WWW から申し込んだのですが、結局のところ最終的な方法は、https 経由
などのスマートな方法ではありません。

クレジット番号を書き込んだオーダーフォームを、WWW で公開されている PGP 
公開鍵で暗号化して、sales@hub.org あてに送るというものです。

ところが、この PGP 公開鍵が "Version: PGPfreeware 5.0i" なので、通常
2.6.X + Mew を使っている私としては、おおいに困ってしまいました。

とりあえず、5.0i を Install して、手動で暗号化したファイルを、添付ファイ
ルで送りつけるという方法をとったため、半日がかりとなってしまいました。

発注したのが 4/17 で、そのあと全く何の音沙汰もなかったので、「デコードで
きなかったかなぁ...」などと思っていたのですが、本日ものが届いて一安心で
す。
