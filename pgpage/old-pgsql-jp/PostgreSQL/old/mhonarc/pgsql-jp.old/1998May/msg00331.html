相場＠ＩＳＯＬです。

こちらは最近すっかり Read Only でしたが最近得た情報によると新しい PostgreSQL 
のフロントエンドツールである "KPGsql v0.0.8" がリリースされたようです。

これは Free の PC-Unix 用統合デスクトップツールである KDE 用のモジュール
です。インストールには KDE Beta4 と postgresql-6.3.2 が必要です。

スクリーンショット等の情報は
           http://home.primus.baynet.de/mgeisler/kpgsql/ 
から得られます。

KDE 自体がまだ日本語に対応していませんので英語オンリーになってしまいます
が、見た目はちょっとした psql の GUI 版といった感じのようです。

私自身は KDE Beta2 --> KDE Beta4 及び PostgreSQL 6.1 --> 6.3.2 の移行が
先のためまで試していませんが、インストールしだい使用感等をレポートしよう
と思います。
