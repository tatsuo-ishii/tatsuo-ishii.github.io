乃木様>
> ところで質問ですが、 ODBC を介さずに Java から Postgres に接続する方法
> はあるでしょうか？

Java-Postgres95 というのがあるようですが、JDBC対応では
ないみたいです。私が試したのは、Java-Postgres95 の Ver0.2(？) 
でした。
以下の URL で入手できます。

ftp://substance.blackdown.org/pub/Java/Java-Postgres95

サンプルが付いていたのですが、
FreeBSD-2.1.5 上で稼働している Postgres95 1.08 では
エラー(Unknown character recieved と表示される)、
EWS4800/310LC EWS/UX R10.1 上で稼働している Postgres95 1.07 では
正常にサンプルを実行できました。
バージョンに依存した問題であるのかは切り分け出来てないです。
