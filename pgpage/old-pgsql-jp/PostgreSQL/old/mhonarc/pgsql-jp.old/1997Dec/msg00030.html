気になったので、多重継承のコンフリクトがどうなるかやってみると、

test=> create table inh1(i int4);
CREATE
test=> create table inh2(i text);
CREATE
test=> create table inh3() inherits(inh1,inh2);
WARN:text and int4 conflict for i

と真面目にチェックしているようです。また、子が親の属性を override する
ことは許されないようです。

test=> create table inh3(i float8) inherits(inh1);
