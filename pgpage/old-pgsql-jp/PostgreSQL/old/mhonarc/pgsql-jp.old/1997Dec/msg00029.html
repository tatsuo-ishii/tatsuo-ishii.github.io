多重継承はサポートしているようです．(6.2.1 と 6.1.1 で確認しました）．
以下のコマンドはすべて通ります:

   create table a (x int4);
   create talbe b (y int4) inherits (a);
   create table c () inherits (a, b);
   create table d (z int4);
   create table e () inherits (a, d);.

ごちゃごちゃしたので，図にすると
   
      a(x) <--+----- b(x,y) <--- c(x,y)
              |
              +--+
                 |
      d(z) <--+--+-- e(x,z)

な感じになりますでしょうか. `:-)' で見てください．
