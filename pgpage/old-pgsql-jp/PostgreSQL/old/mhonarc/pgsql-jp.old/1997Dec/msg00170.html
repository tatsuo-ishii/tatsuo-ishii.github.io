皆さん、始めまして。近藤と申します。

Linux 上で、忘備録用にデータベースを使いたくて、Postgres を勉強し出し
ています。
特に Software Design 誌に載った、石井さんの記事は私のような初心者にも
分かりやすく書かれていて、大変参考になっています。

英語が大の苦手な私は、最近、英和辞書(約30万語)を Postgres に取り込もう
と思い立ち、悪戦苦闘の毎日です。現在困っているのは、英単語部分を
 select で検索する場合に、where 句に like を入れてワイルドカード式に
検索させようとすると、驚くほど時間がかかってしまうことです。
何か良い方法があれば、ヒントでも頂けるとありがたいのですが。

趣味の中でしか使っていませんので、何時までたっても初心者の域から
出られませんが、よろしくお願いします。
