 Cobalt Qube は MIPS 系という事で、内容は全く存じておりませんが、
手元の Solaris 7 LP64 プログラムだと

   sizeof (int)  == 32
   sizeof (long) == 64

となります。(WorkShop 5.0 Early Access 2)

 X11R6 のヘッダファイルでは、Alpha と SPARC V9 なら int : 32bit と
定義しているようです。
