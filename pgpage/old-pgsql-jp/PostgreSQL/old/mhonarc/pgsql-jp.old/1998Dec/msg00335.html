At 午後 06:38 98/12/23 +0900, Hiroyoshi　Shigenaga wrote:

> create table で，表は作成したのですが，この表に対し，主キーを定義したい
> のですが，
> どうすればよいのでしょうか？

　ぷらいまりき〜の事なら、
create table hoge (
	hoge1	int	,
	hoge2	int	,
	primary key(hoge1)
) ;
とかできますよ
