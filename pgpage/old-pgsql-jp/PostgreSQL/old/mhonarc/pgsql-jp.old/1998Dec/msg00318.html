> ところが、ＣＲＥＡＴＥ ＴＡＢＬＥ文のなかで、ｄｅｆａｕｌｔ  ’ｎｏｗ’

私のところでもなります。
ですが、これはnowが評価されるのがどの時点かと言うのが問題なのでしょう。
きちんとソースを読んだ訳ではないので間違っている可能性も有りますが、
例えば、

create table test (nowtime time default ' now');

を実行するとこの分が実行される時、つまりテーブルが作成される時に評価され、
default値はこのSQL文が実行された時の時刻となっているのでしょう。
だから、特に値を指定しないでinsert文を実行すると全て同じ時刻、つまり
create table文が実行された時の時刻がデフォルトではいるのでしょう。
