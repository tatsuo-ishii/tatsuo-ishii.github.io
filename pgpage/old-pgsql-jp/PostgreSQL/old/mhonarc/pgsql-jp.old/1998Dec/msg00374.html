いや、そういうことではありません。"A Guide to THE SQL STANDARD, 4th
edition"(「標準SQLガイド第4版」)は優れた著作ですし、翻訳もなかなか良い
と思いますが、この本の目的は SQL 規格の「解説」なんですね。ですから、
すでにSQLに対して十分な知識がある人にとっては、自分の理解を深める意味
で良書だと思います。
# SQLに対する批判のところは読み物として楽しめました(正直言って問題が難
# しすぎて私はちゃんと理解できていない:-)。自ら規格制定に携わったもの
# としての苦悩が出ていると言うか(ちょっとおおげさ)。

それに対して「完全ガイド」の方は、決して初心者向けに不必要に噛み砕きす
ぎていないにも関わらず、根気良く内容を追って行けば、私のような専門家で
ない人間でも十分理解できるように工夫されている点が素晴らしいと思います。
また、「SQL-92の国際化に対応した部分」は特に注目で、類書にない明確な解
説になっています。
# 国産のSQL本でもこのあたりにちゃんと触れている本ってほとんど無いよう
# な気がします。

以上から、値段を考えると「標準SQLガイド」の方は誰にでも薦めるわけには
いかないと書いたわけです。

> ># 最近自分の本棚を見ると 「SQL」or「Java」と付いた本ばかりで嫌に
> ># なっている。
> 
> Windows or VB に埋もれている私に比べれば、(;_;)
> 明るい未来が見えるだけに幸せだと思います。

まあそういわずに。最近 Linux が元気なので、来年あたりは状況が変わって
くるかも知れませんよ。
