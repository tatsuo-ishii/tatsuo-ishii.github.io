 えっとですね、そもそもの元記事の方なので、しゃもとさんとこで
動かないことにはスレッドが終わらないのですが、もうちょっと
情報出してもらえないでしょうか。エラーになったときのログとか、
configure のコマンドライン全文とか。今時点での情報から私の想像力で
言えることは、

  1. configure がちゃんとできていない
  2. コンパイル時に Makefile から呼び出されているコマンド（bison など）が
     おかしい
  3. リソースリミット値が小さい
  4. swap が足りない

くらいです。3 については、

  % limit
  % ulimit -a

4 については

  % swapctl -sk
  % swapctl -lk

の出力をみせて下さい。よろしくです。
