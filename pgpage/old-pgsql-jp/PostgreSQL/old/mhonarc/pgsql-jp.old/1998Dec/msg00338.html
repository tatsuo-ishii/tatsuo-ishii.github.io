add_month() や last_day() のようなものがあるようですが、

sample=> select * from test;

----------+----------
Giants       |1998-12-28

Swallows    |1999-02-01
Bay Stars  |1998-03-03
Carp         |1998-02-10
(6 rows)


sample=> select * from test where 「今日から今月分」 ;

(1 rows)
