石井です。新年まで秒読み体制ですが...

> 橋本です。２度目の投稿になります。
> 
> 現在、Java1.1.5+postgresql-6.3.2+PostgreSQL 6.3.2 付属のJDBCインターフェイス
> でデータベースシステムを構築しています。
> 先日発表されたJAVA2に切替えようと思ったのですが、
> PostgreSQL 6.3.2 付属のJDBCインターフェイスのmakeが
> うまく行きません。
> 
> CLASSPATHをjava1.1.5からjava2に換えて
> .../postgresql-6.3.2/src/interfaces/jdbc
> ディレクトリで
> % make
> すると
> Exception in thread "main" java.lang.NoClassDefFoundError: sun/tools/javac/Main
> make: *** [postgresql/CallableStatement.class] Error 1
> といわれてしまいます。
> 
> どなたかjava2でJDBCインターフェースのmakeに成功した方いますでしょうか？

そりゃ無理です。PostgreSQL の JDBC ドライバは JDBC 1.x ベースで、Java2 
の JDBC ドライバは JDBC2.0 ベースですから。とりあえず Java 1.1.5 で 
JDBC ドライバをコンパイルし、その postgresql.jar ファイルを Java2 で使
うか(これは可能です)、先日本家の ML で公開された JDBC2.0 用の 
PostgreSQL JDBC ドライバをコンパイルするかです。

皆様、良いお年を。
