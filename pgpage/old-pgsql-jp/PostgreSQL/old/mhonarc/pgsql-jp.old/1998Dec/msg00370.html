 NetBSD/sun3 です。NetBSD/m68k のうちの一つです。[pgsql-jp 6145] を
s_lock.c の #if defined(PPC) 〜 #endif の下にはさんで、[pgsql-jp 6333] を
当てたソースで、

  ./configure  --with-mb=EUC_JP --prefix=/usr/local/postgresql-6.4 \
    --with-readline \
    --with-includes="/usr/local/include" \
    --with-libs="/usr/local/lib"

でコンパイルしました。

 Sun3/60 は、unaru という名前です。pg_hba.conf を記述して、

  postgres@unaru% initdb --pgdata=/var/postgres/dbvol0 --pglib=/usr/local/postgresql/lib --pgencoding=EUC_JP

  postgres@unaru% postmaster -i  -D/var/postgres/dbvol0

  postgres@unaru% createuser mochid

  mochid@unaru% env PGDATA=/var/postgres/dbvol0 createdb  -E EUC_JP mochid0
  mochid@unaru% psql mochid0                                           
  Welcome to the POSTGRESQL interactive sql monitor:
