す。

さて、鳴海さんのPostgresSQL ODBC Driverの WIN98へのインストールの件です
が、私が前に2度試した結果です。

postdrv.exe実行

select compornents のダイアログで、

 [*] Do not install Driver Manager
     これを選んだ場合、この先インストールがすすまない。

 [*] Install Driver Manager
     これを選ぶと日本語の32ビットODBCドライバーが英語版で上書きされる。

 [*] Install Driver Manager [with version checking]
     このオプションで無事日本語ドライバーのままインストールされる。

WIN95の場合はどうだったかはさだかではありませんが、書き変わってしまった
場合、SYSTEMディレクトリー内の ODBC*.* 関係を消し(Backupして下さい:-)て
から、Access97などの再インストールで何度も無事に動きました。
