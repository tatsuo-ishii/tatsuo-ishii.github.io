☆ "Tomoaki Kohzu." さん ( pgsql-jp 6698 )：
> postgresql-6.4.2-patched-990114.tar.gz を解凍中に
> CRC チェックサムエラーが発生してしまいます。
> 
> #tar tvz postgresql-6.4.2-patched-990114.tar.gz
> tar: Skipping to next file header
> tar: Skipping to next file header
> gzip: stdin: invalid compressed data--format violated
> tar: Child returned status 1
> tar: Error exit delayed from previous errors

これは f オプションを忘れたために起きたものだと思います。

tar tvzf postgresql-6.4.2-patched-990114.tar.gz

できっとうまくいくはず。
エラーメッセージを読んでみると、tar は標準入力からのデータを
gzip に渡そうとしていて、gzip はそのデータが「圧縮されたもの
ではない」と怒っているようです。
＃「チェックサムエラー」とは書かれていないように思います。
