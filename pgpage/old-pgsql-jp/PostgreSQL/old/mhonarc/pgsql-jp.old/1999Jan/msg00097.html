pgaccess.tcl を眺めてみました。
(ソースの参照を \ で改行してます)

PS 出力対象である canvas の create text の箇所で、

	.rpv.fr.c create text $x [expr $py+$y] \
		-text $rec($field) \
		-font [.rb.c itemcget $objid -font] \		<=== (*1)
		-anchor [.rb.c itemcget $objid -anchor]

としてます。

.rb.c では create text の箇所で

	.rb.c create text $rbvar(xl_auto) [expr $rbvar(y_rpthdr)+5] \
		-text $fldname \
		-tags [subst {t_l mov ro}] \
		-anchor nw \
		-font -Adobe-Helvetica-Medium-R-Normal--*-120-*-*-*-*-*-*

の様に直接 Helvetica フォントを指定していますので、(*1) の箇所では、
font として Helvetica が指定されてしまうことになります。
(*1) で何も指定しなければ Mincho:Helvetica-12 という font が
デフォルトで指定されますので、.rpv.fr.c create text の箇所の -font 
を試しに取って見てはいかがでしょう?
全部で 3箇所程ありました。

ただし、画面と紙で見た目を合わせるためには、もうすこし調整がいると
思います。
