推測による自己レスです。
perlのインストールをちゃんとしたことがないので、間違っている可能性
大ですが。

1.  ./configure --with-perl .....
2.  make
3.  make install
4.  cd interfaces/perl5
5.  make test
6.  既存のPg関連のファイルを何とかして？消す
7.  make install (rootで)

6.4以前もこれでよかったんだろうか ？
間違いがあったらご指摘ください。
