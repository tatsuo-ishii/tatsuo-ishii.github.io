PostgreSQL を勉強するときは、市販の ＳＱＬ の参考書を
一緒に読まれるといいでしょう。  ANSI SQL92 と PostgreSQL
との相違点は、PostgreSQL のマニュアルにも、シーラカンス
本にも載っています。ANSI SQL92 との相違点は、OUTER JOIN 
や、ALTER TABLE などごく一部だけですから、市販のＳＱＬ
参考書で十分です。

MS の SQLSERVER の解説書を間違えて買わないように。 
ANSI SQL92 を中心に、ＲＤＢの解説をしている本がいいで
しょう。好みにより、文法中心か、ＲＤＢの理論中心か等。

(参考)
union は、man select で表示できる SELECT 文の解説のと
ころに、ちょこっと書いてあります。
# $MANPATH を間違えると、UNIX system call の select
# が表示されますのでご注意。
