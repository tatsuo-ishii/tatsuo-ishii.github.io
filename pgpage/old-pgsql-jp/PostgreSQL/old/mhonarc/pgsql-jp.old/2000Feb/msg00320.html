> >psql -f hoge.sql postgre -o psql.log　は結果だけ書かれるし
> >psql -f hoge.sql postgre > psql.log　も同様だし
> 
> csh系ならば
> psql -f hoge.sql postgre >& psql.log
> 
> sh系ならば
> psql -f hoge.sql postgre > psql.log 2>&1
早速試しました。

hoge.sqlの中身
----------------------------------------------------
\connect postgre hoge;
drop table t_kokyaku;
create table t_kokyaku (
    cd_kokyaku int2,
    nm_kokyaku text);
insert into t_kokyaku
     values (1,'you');
select * from t_kokyaku;
\q;

psql.logの中身
----------------------------------------------------
\connect postgre hoge;
drop table t_kokyaku;
create table t_kokyaku (
    cd_kokyaku int2,
    nm_kokyaku text);
insert into t_kokyaku
     values (1,'you');
select * from t_kokyaku;
connecting to new database: postgre as user: coop
DROP
CREATE
INSERT 1015114 1
cd_kokyaku|nm_kokyaku
----------+----------
         1|you
(1 row)

\q;


こんな結果を得たいのですが
----------------------------------------------------
\connect postgre hoge;
connecting to new database: postgre as user: coop
drop table t_kokyaku;
DROP
create table t_kokyaku (
    cd_kokyaku int2,
    nm_kokyaku text);
CREATE
insert into t_kokyaku
     values (1,'you');
INSERT 1015114 1
select * from t_kokyaku;
cd_kokyaku|nm_kokyaku
----------+----------
         1|you
(1 row)

\q;


「問い合わせ」と「結果」をペアで出力したいのです。無理でしょか？
