timespan 型は、如何でしょう。

例えば、

create table ta(aa timespan);
insert into ta values ('00:01:23.45');
select * from ta;   で、
@ 1 min 23.45 secs  と 1/100 秒まで表示されます。

set DateStyle to 'ISO'; などで、表示フォームの変更も可能。
select * from ta;    で、  00:01:23.45  と表示など。

（参考）
datetime や、timestamp は、暦 の日時に使用します。
競技記録など、時間間隔を格納するには、interval や、
timespan などを使用します。
