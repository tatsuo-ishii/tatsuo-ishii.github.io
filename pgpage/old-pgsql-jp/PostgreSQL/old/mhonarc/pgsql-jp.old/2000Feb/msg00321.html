> psql -f hoge.sql postgre > psql.log　も同様だし
> \o　も同様だし

> hoge.sqlの中身も含めて、psql.logに書き出したいのです。

psql を実行する前に script コマンドを叩いておくってのはどう
ですか？

以下は FreeBSD の man script
  script は、端末上に表示したすべてのもののログ (typescript) をとります。

つまるところ、psql を使う前にまず
  % script [logfilename]
してから
  % psql hogehoge
  (必要な処理)
  hogehoge=> \q
  % exit
すると、指定した logfilename に画面に出た文字は全部取れてい
ます。(ちょっと改行コードが ^M 付きだったりしますけど)
