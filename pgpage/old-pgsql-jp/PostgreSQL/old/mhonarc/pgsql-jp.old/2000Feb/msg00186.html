私はPg.pmという、PostgreSQLのアーカイブに付いてくるモジュールを使っ
ています。

use Pg;
$db = Pg::connectdb("dbname = データベース名");
$dataset = $db->exec("sql文");
while (@data = $dataset->fetchrow) {
   # なにか処理
}

というような原型です。
@dataの中身が、各行のフィールド(カラム)をfetchした感じになります。
select文でフィールド名を指定した場合はその順序に@dataの各要素とな
ります。while文ですので、最後まで行ったら自動的にループを抜けてく
れます。また、該当レコードが0件の場合はwhileの中身はなにも実行さ
れません。

Pgについては perldoc Pg をご覧ください。
(Pgがインストールされているのが前提とさせていただいていますが)
