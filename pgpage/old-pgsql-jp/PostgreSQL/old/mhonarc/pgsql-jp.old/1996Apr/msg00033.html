  仕事は、外資系の化学会社の営業企画で、コンピュータそのもの
とは直接関係ありません。道具として使うコンピュータには多いに
関係ありますが、、。

  現在は、家庭で p5@150Mhz, p5@100Mhz のデスクトップ、
p5@75Mhz のラップトップ が、FreeBSD-2.2-960323-SNAP で動いて
おり、また 486@100Mhz のラップトップ、486@100Mhz のデスク
トップが Win3.1 で動いています。出来るだけ早いうちに、全部
lan 接続しようと思っているのですが、家庭内の配線工事の
キッカケがないので、まだ出来ていません。(8 port hub は動いて
いるのですが、、)

  FreeBSD で postgres95 が動き、pg95perl も動くようになったの
ですが、pg95perl の使い方が良く分からなくて、いろいろ探してい
るうちに、ここを見付けました。 仕事は関係ないと言いながら、w3
で営業情報をグラフィック表示して知らせるようなものの prototype 
を会社に提示しようと考えて、手を初めてしまいました。

  C, perl は英語、tcl/tk はドイツ語、その他はほとんどロシア語
(特に SQL は中国語か？、どちらもほとんど出来ませんの意味)
程度です。(母国語は勿論日本語、プログラムの母国語は無しです)。
