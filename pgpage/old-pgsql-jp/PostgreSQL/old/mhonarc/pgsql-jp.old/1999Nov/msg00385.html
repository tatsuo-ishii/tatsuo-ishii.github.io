chown するユーザは Linux のほうのユーザですね。
これは、/etc/passwd の行頭に書かれているのがユーザ名なので、
そこにないユーザだと chown できません。
それを確認してみると良いと思います。

Linux でのユーザ登録は adduser というコマンドで、
PostgreSQL でのユーザ登録は createuser というコマンドです。
Linux user は /etc/passwd で見ることができて、
PostgreSQL のユーザはDBを作成したディレクトリのしたにある
pg_pwd ととうファイルにかいてあります
