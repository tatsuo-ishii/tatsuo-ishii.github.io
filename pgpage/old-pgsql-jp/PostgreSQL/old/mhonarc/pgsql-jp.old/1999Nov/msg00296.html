自己レスです。

textやcharはどうなるか気になったのでテストしてみました。

textではエラーは起こらずにうまく更新できました。

create table test(a int2 primary key, b text[]);
insert into test(a,b) values(1,'{1,3}');
insert into test(a,b) values(2,'{ab,6}');


charでは同様のエラーが起こり更新できませんでした。

create table test(a int2 primary key, b char(8)[]);
insert into test(a,b) values(1,'{1,3}');
insert into test(a,b) values(2,'{ab,6}');
