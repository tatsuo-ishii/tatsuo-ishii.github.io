持田です。

 6.5.2 用の JDBC ドライバ 文字エンコーディングパッチを

    http://www.yo.rim.or.jp/~mochid/postgresql-jdbc/
    http://www.yo.rim.or.jp/~mochid/postgresql-jdbc/pg652jdbc-ce-19991108.patch.gz

に置きました。

 データベースの文字エンコーディングと Java 環境でのデフォルト
文字エンコーディングと区別して扱うようにするパッチです。

 6.4.2 以前と違って、"postgresql.ce.Driver" というクラスをロードして
使うようにしています。"postgresql.Driver" を指定すると、
オリジナルと同じ（文字エンコーディングを区別しない）動作になります。

 文字エンコーディング指定に 6.4.2 以前と同じ方法（Java の
プロパティ、URL のサブプロトコルの部分での指定）を使う場合は、
"postgresql.ce.compat.Driver" をロードするようにして下さい。

 あと、src/interfaces/jdbc/example/Dbview2.java に動作確認用に
使っているサンプルを付けています。

 あんまり動作確認してないので、バグってたら報告よろしくお願いします。
