桑村です。

おおたさんの

> query 32:  38.596 real   2.060 user   0.000 sys

は、bench.out の最後の '>' 以降を取り除いて 

% rm bench.out.perquery; make bench.out.perquery

をすると正確な値となります。



  ところで、石井さん、日本語パッチありがとうございます。

パイルオプションに -m486 を付けて、include/config.h で
#define HAVE_SIGSETJMP 1
と定義してメイクしました。
