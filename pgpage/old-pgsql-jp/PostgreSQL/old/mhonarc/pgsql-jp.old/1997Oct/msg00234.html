PostgreSQL ではなく、bsd-db だったのですね。失礼しました。

というわけで、どっちも知らなかったんで bsd-db と sdbm というのを調べて
みました。

sdbm には文字数制限があります。デフォルトでは 1000 bytes 位。もっと大
きくするには、recompile が必要です。

bsd-db っていうのは、dbm のように、hash など特定の access method だけ
でなく、btree や flat file など、複数の access method への統一されたイ
ンターフェイスだと言うことです(dbm の emulation もできるらしい)。とり
あえずささっと btree のところを眺めた限りでは、文字数制限はなさそうで
す。

>perl5 で利用できる dbm は odbm, ndbm, sdbm, bsd-db, gdbm, だっ
