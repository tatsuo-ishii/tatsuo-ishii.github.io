メッセージ通り、flex のバージョンがふるいためだと思います。
flex -V してみてください。

INSTALL を見ると、バージョンが 2.5.2 か 2.5.4 以上であるといっています。
flex のインストール方法も INSTALL に書いてありますので、最寄りの GNU 
アーカイブからダウンロードしてインストールしてください。

以下、抜粋。

     To install it, type the following:
        cd
        gunzip -c flex-2.5.4.tar.gz | tar xvf -
        cd flex-2.5.4
        configure --prefix=/usr
        make
        make check
        # You must be root when typing the next line.
