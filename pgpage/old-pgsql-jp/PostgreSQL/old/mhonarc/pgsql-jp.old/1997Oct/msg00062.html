Hideki KANEBAKO <kane@gb.sankyoseiki.co.jp> wrote:

> 金箱と申します。
> 
> ユーザーズマニュアルには SELECT * INTO temp from weather;
> で暗黙の create コマンドが実行されるとありますが、
> 
> CREATE TABLE report (
> 	repno varchar(15),
> 	title varchar(80),
> 	issue date,
> 	author int4,
> 	abst  varchar(256)
> );
> 
> のテーブルに対して、
> 
>  select * INTO temp FROM report where report.author = 125849;
> 
> を実行して見たのですが、WARN:parser: parse error at or near "temp" と
> いわれてしまい select の結果を temp に書き出すことができません。
> 
> どこか間違っているのでしょうか？
> 
> postgresql のバージョンは 6.1.1 を使っています。
> 

以下の様にINTOの後にTABLEを入れて見てください。

select * INTO TABLE temp FROM report where report.author = 125849;
