ＤＢ作成当初は正常に動作していたのですが、
突然以下のような現象が発生し困っています。

----+-----------+------------
ID   varchar(13) unique_index
Name varchar(40) 
----+-----------+------------
のようなテーブルを作成し、psqlにて、

SELECT * FROM table_name WHERE ID = '10000000';で抽出すると、
---------+------
ID        Name
---------+------
10000200  Name2
10000000  Name1
---------+------
のように2件のレコードがヒットします。（おかしい）

SELECT * FROM table_name WHERE ID = '10000200';では、
---------+------
ID        Name
---------+------
10000200  Name2
---------+------
のように抽出されます。（正しい）

ちなみに SELECT * FROM table_name WHERE ID Like '10000000';では、
---------+------
ID        Name
---------+------
10000000  Name1
---------+------
と取れました。（正しい）

また、理由は分かりませんが、
UPDATE table_name SET Name = 'Name2' WHERE ID ='10000200';
を行う事によって、現在は正常な動作をするようになりました。


このデータベースは、
通常、perl/CGIからデータをINSERTするのに利用しています。
