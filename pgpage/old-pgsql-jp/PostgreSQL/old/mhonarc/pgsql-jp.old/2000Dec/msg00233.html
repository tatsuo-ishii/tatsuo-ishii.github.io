>私の環境
>・Kondara Linux
>・PostgreSQL 6.5.2
>では、「\copy t1 from 'tmp.data' using delimiters ','」
>を実行すると、「You haveextra tokens after the filename」とのエラーが発生します。
>「copy t1 from 'tmp.data' using delimiters ','」
>でしたら問題無く動作しているのですが。

　\copyのときには、ファイル名を''で囲う必要はないようです。
「'」をはずして試してみてください。

それから、\copyであれば、相対パスでファイルを指定できるようです。（試したのはver6.5.3）
