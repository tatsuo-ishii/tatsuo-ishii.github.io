投票しようかと思ったけど、どこから投票するのかわからん… (^^;

MySQL と PostgreSQL といえば、土曜日の PostgreSQL Technical Seminar 2000
in Tokyo では Brouce Momjian 氏が、「MySQL との比較」という質問に対して、

  「mSQL/MySQL の方が欧米では先に出ていたので、既存のインストール
    ベースで MySQL が多く存在し、これをそのまま引き継がれている。
    日本ではユーザにこれらが普及がし始めた時期にはすでに PostgreSQL が
    存在したので PostgreSQL を選択したユーザが多いのではないか。
    欧米でも今後は PostgreSQL の比率が高まると考えている」

というような趣旨の事を言っていました。
