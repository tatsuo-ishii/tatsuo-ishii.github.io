Takeshi Furuta wrote:
> 
> ふるたです。

(snip)

> update test set kingaku= 200 * honsu;
> 
> とすると、
> ERROR:  Attribute 'kingaku' is of type 'money' but expression is of type
> 'numeric'
>         You will need to rewrite or cast the expression
> 
> と怒られてしまいます。
> 
> どのようにSQL文をなおしたらいいのでしょうか。

ＤＢ列の kingaku が、money 型ですので、
エラーメッセージにあるとおり、cast しましょう。

上の例なら、次のように money 型にキャストします。
詳しくは、PostgreSQL のマニュアルを見てください。

UPDATE test SET kingaku = 200::MONEY * honsu ;

テスト環境:PostgreSQL 7.0.2
