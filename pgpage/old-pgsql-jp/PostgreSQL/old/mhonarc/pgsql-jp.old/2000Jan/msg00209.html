> > 条件付きのデータ挿入方法について質問です。

> > あるテーブルにidというフィールドがあるとします。
> > そこで
> > insert into sysid (id) values( Max(id) );
> > などとINSERTするとします。
> > 
> > ここでレコードが無い場合nullが挿入されてしまいます。
> > このinsert処理内でデータ無いなら１それ以外はMax(id)みたいな
> > 事は可能なのでしょうか。
> > 基本的な質問かもしれませんが、いろいろ調べても、答えが
> > 見つかりません。よろしくお願いします。
> 
> もっといい方法があるのかも知れませんが、
> 
> insert into sysid (id) 
>     select 
> 	case
> 	    when max(id) = null then 1
> 	    else max(id)
> 	end
>     from sysid;

= null は文法的にだめでした。

insert into sysid(id) select coalesce (max(id),1) from sysid;

でできました。ごめんなさい。
