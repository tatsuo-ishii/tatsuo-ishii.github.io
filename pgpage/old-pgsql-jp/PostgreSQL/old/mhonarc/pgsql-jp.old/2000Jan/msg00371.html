gnome-db はソース展開すると、ldap, mysql, odbc, oracle, postgres,
sybase が使えそうです。
DB の選択は、CORBA(orbit)を使っています(どう使っているかはわからんが...(^^;))。
# make はしたけど、make install はしてないもんで、どういうものかはわか
# らない。

他には、gtranscript っていうのもあります。
mysql, postgrasql 用のプラグインがあります。
DB の選択は、libgtrans_iface ってライブラリーが担当しています。
libgtrans_iface のインターフェースに合わせれば、他の DB 用のプラグイン
も作成できます。
SQL Editor としての機能はないけど、テーブルの作成・削除、データの入力・
削除・編集機能はあります。リレーションを表現できないのが難点。

gnome-db, gtranscript ともに、gnome が必須です。
