細かい話ですが、
version 6.5.? のマニュアルで、英文では、
丸括弧で ( actions.... ) となっており、
日本語では、brackets で、 [ actions ... ]
となってました。
# 私のマニュアルが古いのかな？。

ーーーーーーーーーー英文マニュアル抜粋ーーーーーーーーーー
SYNOPSIS
       create rule rule_name
           as on event
             to object [where clause]
           do [instead]
           [nothing | action | (actions...)]

ーーーーーーーーーーーーーーーーーーーーーーーーーーーー

名称
       create rule - 新規のルールを定義します

形式
       create rule rule_name
           as on event
             to object [where clause]
           do [instead]
           [action | nothing | [actions...]]
