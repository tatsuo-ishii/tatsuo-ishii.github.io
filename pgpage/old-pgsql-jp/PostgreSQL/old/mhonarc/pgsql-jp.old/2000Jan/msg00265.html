> PostgreSQL完全攻略ガイド(改訂版)で勉強中です。
> Try!PCのWeb-DB連携システムの構築の記事を見て、

> 早速ですが、今、テーブルの作成でつまづいています。
> 
> create table abc (
> 	a numeric(8,0)[10],
> 	b int8[20]
> );
> 
> の様に、numericやint8で配列を定義しようとすると、
> 
> ERROR: Unable to locate type name '_numeric' in catalog
> 
> の様なエラーが出て、テーブルが作成できません。

numeric のほうはわかりませんが、int8 のほうでしたら、
TODO の中にこういうのがあります。TODO というのは
doc/ の下にあるやつです。6.5.2 のものを見ているので、
6.5.3 で解決しているかどうかはわかりません。

TYPES
(略)
* Allow array on int8[]

ちなみに私の環境でも、numeric の配列は作れません。
(PostgreSQL 6.5.2 + Solaris 2.6)

tmp=> create table tmp01 ( a numeric );
CREATE
tmp=> create table tmp02 ( a numeric[] );
ERROR:  Unable to locate type name '_numeric' in catalog
tmp=>

他の型、例えばint では配列が作れます。

tmp=> create table tmp03 ( a int );
CREATE
tmp=> create table tmp04 ( a int[] );
CREATE
tmp=>
