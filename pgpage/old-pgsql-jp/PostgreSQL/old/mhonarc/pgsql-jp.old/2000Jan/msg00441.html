sytem 関数の引数として、$sql を ' ' (シングルクォート)で囲
んでいるので、その中にlike '03%' と、シングルクォートが２重
になってしまったために、エラーになります。

system や exec, popen 等をつかうには、shell script の
基本を勉強しておきましょう。

SQL 文に シングルクォートが含まれる場合は、-c の後の
SQL 文を エスケープしたダブルクォートで囲むといいでしょう。

system($psql." -c \"".$sql."\" 2>&1 ");

2>&1 で、stderr も stdout すなわち、ブラウザ画面に表示
されます。

お断りしておきますが、PHP の system 関数でpsql を実行
するのは、手抜きというか邪道の類です。デバッグや臨時
処理などで、ちょっと使うには便利ですが、本来は、きちん
と pg_execして、print や echo で表示すべきです。
