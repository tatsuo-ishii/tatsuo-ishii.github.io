石井です。

> 大村です。
> 
> SELECT文で、LIKE '%\\%' をすると検索結果が返らないようです。
> 
> create table test(data varchar(10));
> insert into test values('aaa\\bbb');
> 
> として、
> 
> select * from test where data like '%\\%';
> 
> としても、値が返ってきません。

select * from test where data like '%\\\\%';

ですね。

'%\\%' は '%\%' と解釈されますが、PostgreSQL の like では、\ は特別な
意味を持っており、\% は % という文字自身をあらわすものと解釈されます。
