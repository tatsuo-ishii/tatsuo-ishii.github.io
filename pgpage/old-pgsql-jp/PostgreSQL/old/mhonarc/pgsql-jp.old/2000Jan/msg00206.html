At Fri, 14 Jan 2000 06:03:00 +0900 (JST),
kaigai@jks.is.tsukuba.ac.jp (Kouhei KAIGAI) wrote:
> こんにちは、海外＠筑波大です。
> 例えば下のようなSQL文でテーブルを作ったとき、
> Create Table sample(	ID INTEGER PRIMARY KEY ,
> 			name VARCHAR(64) ,
> 			tel VARCHAR(16) );
> 上のテーブルに、
> Insert into sample(name,tel) values('警察','110');
> のような要領でカラムを追加したときに、
> 自動的に適切な ID が割り当てられるようにしたいのですが、
> これはどうすればよいのでしょうか？

serial を使えば一発ですね。

create table sample(
	id	serial,
	name	varchar(64),
	tel	varchar(16)
);

ただし、id に対応する index と sequence も同時に作られますので、
(特に sequence を) drop しないように気をつけてください。

> 以前使っていた MySQL というシステムでは、テーブル生成の時に
> Create Table sample( ID INTEGER AUTO INCREMENT PRIMARY KEY,.......
> とすれば、ユニークな番号を振ってくれたようなのですが、
> どうやらこれはMySQLローカルなオプションっぽく、
> PostgreSQLで試したところエラーをお返しになられました。

僕は PostgreSQL しか使ったことがないんですが、やはりこういうの
は DBMS によって違うもんなんですかね。
