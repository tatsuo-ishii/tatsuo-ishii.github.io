　あまり、むずかしい方法だと使いにくいかもしれません。pgbash
の範囲としては、

  fmt = "{'----,---,--&.&&', '---&', '#####'}"
  exec_sql -f "$fmt" "select aa,bb,cc from test" 

　この程度の指定方法が手頃かなーと思っています。


　それと、selectで呼んだ変数を加工した後に出力する場合は、近
藤さんの、

>     echo numformat("----,---,--&.&&", $hostvar1);
>     echo numformat("&&&&&&", $hostvar2);
>     echo numformat("#####&", $hostvar2);
>
> # 今は自作ので、v=`csledit.exe '----,---,--&' $var1`;
> # echo $v; なんてやってます。

　が良いのではないかと思います。もちろん、

> # 話は変りますが、この次は、Report Writer が欲しい。
> # 誰か作りませんか。RPG といってもゲームじゃないよ。
> # RPG： Report Program Generator.

  RPGがあればベストです(^^)。echo出力やRPGはお任せします。
