スレッドの順を追ってみてないので外していたらごめん
なさい。

pg_connect()の5番目の引数である データーベース名が
うまくわたってないんでしょう。DB名省略すると、
ユーザー(この場合は、nobody )名と同じ名前のDB名に
なります。

PHP3 の pg_connect は、KEYWORD パラメーターと位置
パラメーターの両方が使えますが、KEYWORD で問題が
あれば、位置パラメーターにしてみては如何でしょう。

標準の設定で、local でだけ使うならば、１〜４の引数
は、空にして、5番目の DB名だけ渡せば動きます。

例えば、DB 名が drink ならば、
pg_connect ("", "", "", "", "drink");

PHP3 の bimanual の pg_connect を読んでみてください。

int pg_connect(string host, string port, 
	string options, string tty, string dbname);
