COBOL の PICUTRE 句 や、NEC PC9801 NBASICの PRINT
USING 見たいのが希望です。
    & : 前ゼロ表示。空白置換しない。 例：  000012.34
    # : 前ゼロは表示しない(空白置換）例：      12.34
    - : 浮動の負符号。
    + : 浮動の正符号。
    , : カンマの挿入位置。
    . : 小数点位置。
bash での拡張表現方法がわからないのですが、勝手に
書くと、
    echo numformat("----,---,--&.&&", $hostvar1);
    echo numformat("&&&&&&", $hostvar2);
    echo numformat("#####&", $hostvar2);
といった感じ。PHP の number_format でもいいですが。
