正確には man initdb によると、 initdb -d directory で指定した
directory に data が構築されます。また環境変数 PGDATA を
設定しておくと default でそのディレクトリに構築されます。
# つまり /usr/local/pgsql/data に作られるとは限らないと
# いうことですね。
