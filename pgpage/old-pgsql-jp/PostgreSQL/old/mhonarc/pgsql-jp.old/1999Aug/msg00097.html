いえ、PostgreSQL の JDBC ドライバは Type 4 です。つまりピュア Java
の JDBC ドライバ。
6.5 以降の JDBC1 だ JDBC2 だというのは、JDK の 1.1.X 対応版、1.2.X
対応版という意味です。つまり、Java1 と Java2 対応。
