小菅さん、後藤さんありがとうございます。
後藤さんのご指摘のとおりMakefileのLIB1に直接書き込んだところ成功しました。

やっとできた；；）

手順
＜環境＞
OS:RedHat5.2-J（Ver1)
Postgresql-6.5.1
Apache-1.3.6
mod_auth_pgsql-0.7.2.tar.gz

＜やっていること＞
Postgresql-6.5.1をインストール(usr/local/pgsql)後、環境変数を.bashrcで変
更
mod_auth_pgsql_0.7.2.tar.gzを展開。
Apache_1.3.6.tar.gzを展開。(usr/local/src/apache_1.3.6)
その後、/auth_pgsql以下のディレクトリーをapache-1.3.6/src/modulesの下にコ
ピー。

＜apacheを下記の方法でコンパイル。＞

# OPTIM="-02" ./configure --enable-module=so
--activate-module=src/modules/auth_pgsql/mod_auth_pgsql.c
--disable-rule=WANTHSGEREX

コンパイル後
# cd src
# vi Makefile
Makefileに追加

Lib1= -L/usr/local/pgsql/bin -lpq
保存して
# cd ../
# make
# make install
# /usr/local/apache/bin/httpd -l
で
mod_auth_pgsql.c
を確認できました。

以上ありがとうございます。
これからもよろしくお願いします。
