PostgreSQL は、単体では(psqlなど)動いているのですか？。

そうだとすれば、Redhat にインストール済みの Apache
( httpd )が残っているのではありませんか？。

/etc/rc.d/init.d/httpd で起動している httpd の PATH
と、御自分でインストールした httpd の PATH があって
いるか確認してみてください。違っていれば、古い方を
消して PATH を張りなおすとか、シンボリックリンクに
するとか。この場合、httpd.conf も、/etc/httpd/conf/の
下と、自分でインストールしたものと新旧両方があること
になりますので、古い方を消すなり、リンクするなりする
ことです。

PHP 自体は動くとのことですから、<?php phpinfo(); ?>
で、Apache の情報も含めて表示されますので、自分が
意図した通りにインストールされているか確認してみて
ください。
