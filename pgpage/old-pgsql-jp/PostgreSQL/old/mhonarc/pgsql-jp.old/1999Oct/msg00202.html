いわきりです

Tomoo Nomuraさんの<38037EC9.107087BC@tmo.co.jp>から
>Debian Linux上に、php3 + Apache でWebを作り、そこから別のDebian上の
>Postgresをアクセスしています。
>フロントエンド上には、postgres本体はいらないと思うし、バックエンドにphp3
>はいらないと思うのですが、正確には、どのライブラリーがいるのでしょうか？

php3のfunctions/pgsql.c functions/php3_pgsql.hを見ると、

functions/php3_pgsql.h
---
#include <libpq-fe.h>
#include <libpq/libpq-fs.h>

functions/pgsql.c
---
(一部抜粋)
static void _close_pgsql_link(PGconn *link)
{
        PQfinish(link);
        php3_pgsql_module.num_links--;
}

となってるので、php3のコンパイルには、
/usr/local/pgsql/include下のヘッダファイルが必須
利用する為には、
PQ?????という関数が呼ばれるので、
/usr/local/pgsql/libpq.so
が必須です。
