この記事で言っているのは COPY でファイルから読み込むときの話です。
通常のテキストファイルでは NULL を表現することができない(空文字列と
NULLは異なる)ので、\N でNULL となるようになっています。



shimizu> psqlで
shimizu> insert into tableA values('\N');として

INSERT で NULL をいれる場合は、単に
    INSERT INTO tableA VALUES (NULL);
です。
