　と思って大喜びでやってみましたらやっぱりエラー。（ちなみに
PHP3でした。）

　結局
$artist1 = addslashes($artist1) ;
$artist2 = addslashes($artist2) ;
$artist3 = addslashes($artist3) ;
で\でエスケープすべきものは全部\が直前につくと言う関数を
「徹底攻略まんもす」で見つけ使ってみました。

で、表示させるまえに
$artist1 = stripslashes($artist1) ;
$artist2 = stripslashes($artist2) ;
$artist3 = stripslashes($artist3) ;
で\を削除してくれ、これでオーケーになっています。
