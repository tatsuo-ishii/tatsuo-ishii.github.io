[snip]

すっかり有名になった感のなる history.h ですが，readline library の
コンパイル元のディレクトリをサクセスするとあるはずです．なのですが，
readline.h は単に，関数のプロトタイプをアクセスするためだけに使って
いるようですから，readline/history.h を作ってしまうという手もあります．

extern void add_history();
extern void using_history();
extern int write_history();

(型が型だけに空のファイルでもこと足れりですが．）
