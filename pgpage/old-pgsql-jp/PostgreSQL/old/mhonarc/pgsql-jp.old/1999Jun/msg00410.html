>
> バージョン6.5であれば、offset と limit を使うのが簡単です。
>
> select * from hogehoge limit 20;	-- 1から20レコードまで表示
> select * from hogehoge limit 25 offset 49;	-- 50番目のレコードから
> 						-- 25レコード表示

> ちなみに、offsetは0スタートです。

ちょっと便利そうな機能ですね。
ところでこの機能は、

   select * from hogehoge limit 20;
                          <- この間に誰かが INSERT か DELETE を実行
   select * from hogehoge limit 20 offset 20

とかやった時に、2回目の問い合わせで間が飛んだりしないで「続き」から
表示されるのでしょうか？
無理っぽいですかね。(^^;
