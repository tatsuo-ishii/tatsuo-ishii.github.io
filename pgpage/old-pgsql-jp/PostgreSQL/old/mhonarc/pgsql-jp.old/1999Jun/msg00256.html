Kentaro Fujinuma <fuji@ask.iiga.co.jp> wrote:

> > sumの合計の値を求めて、その値によるレコードを取得したいのですが。
> 
> SELECT id, SUM(price) as summary FROM HogeTbl 
>  GROUP BY id HAVING summary > 100;
> 
> でどうでしょう。

ありがとうございました。
HAVING句で意図した事が出来ました。

しかし、HAVING句では別名を使えないみたいですね。
結局

> SELECT id, SUM(price) FROM HogeTbl 
>  GROUP BY id HAVING SUM(price) > 100;

とすることによって解決しました。
