しまった。注意するのを忘れていました。
インストールするときに、インストール内容についていくつか選択肢が
出てきたと思いますが、このときに "Do not install Driver Manager" を
選択しておかないと ODBC Administrator が英語版のものに置き換えられて
しまいます。
本当はここを示しておかないといけなかったのですが、うっかりして
いました。ごめんなさい。
http://www.interwiz.koganei.tokyo.jp/software/PsqlODBC/index.html

で、対処法ですが、以下のリンクから ODBC Driver Manager および
各種 ODBC Driver の最新版をダウンロードできます。
"Microsoft Data Access Components（MDAC） 2.1 日本語対応版 提供開始！"
というところです。これをダウンロードして、インストールすれば
最新版の日本語版の ODBC Driver Manager が使えるようになります。

http://www.microsoft.com/japan/developer/data/default.asp

なお、ダウンロードページは各国語対応になっているので、
"Japanese" Version をダウンロードしてください。
