  Linuxについては 2.0.36 libc5と 2.2.5 glibc2 で作ったバイナリを
入れようと思います。
  作成にあたり、

    cd postgresql-6.5/src
    configure  --with-mb=EUC_JP --with-tch --with-perl --with-odbc
    make install
    cd contrib
    make install
    cp vacuumlo/vacuumlo /usr/local/bin/

のように構成してインストールします。
  そうして、アーカイブを以下のように作ります。

    tar cvf pgslq-linux2.0.36-libc5.tar \
	/usr/local/pgsql/bin \
	/usr/local/pgsql/lib \
	/usr/local/pgsql/include \
	/usr/local/pgsql/odbcinst.ini
    gzip pgslq-linux2.0.36-libc5.tar

いかがでしょうか。ところで、perlモジュールはどうしたら良いでしょ
