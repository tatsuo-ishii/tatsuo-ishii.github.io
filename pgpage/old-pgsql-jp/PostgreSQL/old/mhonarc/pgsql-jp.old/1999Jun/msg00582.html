まだ、火曜日の朝でしょうか...

(1) Sparc Linux(RedHat 5.2)

cd postgresql-6.5/src
configure  --with-mb=EUC_JP
make install
cd /usr/local
tar cvf pgsql.sparc.redhat52.tar pgsql
gzip pgsql.sparc.redhat52.tar

(2) NetBSD 1.3.3/m68k

cd postgresql-6.5/src
configure  --with-mb=EUC_JP
make install
cd /usr/local
tar cvf pgsql.m68k.netbsd133.tar pgsql
gzip pgsql.m68k.netbsd133.tar

したものを私のホームページにアップしました。

http://www.isr.co.jp/~muc/pgsql/

(1) 昨日のポストで述べた、

>     cd contrib

は、時間切れのためできませんでした。

(2) 今回のふたつのアーカイブは、PostgreSQLを使用するのに
最低必要な部分だけになっています。

(3) NetBSD/m68k版は、五百田さんと石井さんが昨夜遅くまで
やりとりしていたバグを吸収しておりません。昨晩は、メール
をフォローしていませんでした。ただ、このバグは、psqlの
表示上の問題であって、データベース的には、権限の設定は
ちゃんと出来ているのではないでしょうか。私も、家でしか
確認できないので、どなたか見ていただけませんか。
