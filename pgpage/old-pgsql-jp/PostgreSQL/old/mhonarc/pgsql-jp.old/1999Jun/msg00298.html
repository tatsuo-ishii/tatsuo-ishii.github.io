# 全くPostgreSQL とは関係ない話ですが…

Pentium 133MHz ということですが、マザーのChip Set は何でしょうか？ ちょっ
と古めのマザーだと、ハード上の制限に引っかかる場合があります。

Intel の場合、430HX 以外(TX とかVX とかあったと思いますが)のChip Set 
では64MB 以上のメモリに対するキャッシングができません。つまり64MB を堺
に「遅いメモリ」と「早いメモリ」ができてしまいます。これがコンパイルエ
ラーを引き起こしている可能性はあります。
