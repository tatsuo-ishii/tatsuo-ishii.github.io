# パッチを当てた gd は VFlib をリンクしないと動作しないので、

一番最後の方に、

/* Define if you have the gd library (-lgd).  */
/* #undef HAVE_LIBGD */

という部分がありますが、これを

#define HAVE_LIBGD 1

に直して再コンパイルしていただけますか?
