ご存知とは思いますが、-h オプションを使って
	psql -h hostname database_name
でもOKです。

もちろん、Java だけで書いてあれば、Windows だろうと Mac だろうと、Java 
が使える環境ならば PostgreSQL にアクセスできるようになります。
また、ずばり Java で書かれた psql のイミテーションが存在します。
http://www.tara-lu.com/~jimb/jsql にある jSQL です。

[ext16]t-ishii{48} java jSQL -h srapc451 -d test
Welcome to the PostgreSQL Java based sql monitor:
  Please read the file COPYRIGHT for copyright terms of PostgreSQL

   type \? for help on slash commands
   type \q to quit
   type \g or terminate with a semicolon to execute query
 You are currently connected to the database: test

test=>

らかに Windows です。
