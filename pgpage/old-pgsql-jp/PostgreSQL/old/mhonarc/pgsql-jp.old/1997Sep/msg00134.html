PostgreSQLでの日付型（Ｄａｔｅ）を文字型に変換する方法はないでしょうか？
 厳密に何をやりたいかというと以下の様な感じなのです。（概略）
 
CREATE TABLE T1 (day1 date......);
CREATE TABLE T2 (day2 char8......);
※T2.day2はyyyymmという感じで入っています

T1は日々のデータが入っているテーブルでT2は当月の一日当たりの目標などが入
っているテーブルです。

SELECT T1.*, T2.* FTOM T1, T2 WHERE *****;
の様にT1とT2を結合させたいのですけれど、安易にDate->yyyymm文字型の変換用
の関数（Ｃ言語？）を作ればいいのかとも思ったのですが実現方法が現在わから
ず、他に良い方法や簡単に実現する方法はないかと思いまして質問させて頂きま
した。
