試してみたのですが、6.1/6.1.1 <--> 6.2b8 はつながりませんでした。
#プロトコルが変わったのかな。

単に JDBC をやってみたいのであれば、6.1.1 + 単独のパッケージで試してみ
ては如何でしょう。http://www.demon.co.uk/finder/postgres/ から入手でき
る JavaPostgres95-0.6 は 6.1.1 で動きました。試したのは以下です。

% cd JavaPostgres95-0.6
% java -Djdbc.drivers=postgres95.PGDriver test.test "jdbc:postgres95://srapc451st?user=t-ishii"

これで

11 tests performed, 11 passed, 0 failed.

と最後に出て来るので、それらしく動いているようです。言うまでもありませ
んが、//srapc451/test?user=t-ishii の部分は、適当に置き換えてください。
(srapc451=host名、test=データベース名、t-ishii=ユーザ名)

なお、JDBC ドライバの応用例としては、
http://www.geocities.com/RodeoDrive/1620/jexplorer.htmlという GUI ベー
スのデータベースブラウザがあって、JavaPostgres95-0.4 となら動きます。
(0.6 はダメ)
