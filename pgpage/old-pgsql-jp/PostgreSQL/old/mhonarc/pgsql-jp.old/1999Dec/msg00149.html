寺尾と申します。

  ecpgでconnectの構文は、以下の様になります。

    EXEC SQL CONNECT TO xxx user yyy;

  xxxというDBにyyyというユーザでコネクトします。

  パスワードを付ける場合は、以下の様にすれば良いと思います。
  というのは、プリコンパイルはしたけれど、実際には使わないで、
上で記述した、DB名とユーザ名を指定しただけのものを使っているか
らです。
  参考程度、ということでお願いします。（zzzがパスワード）
  ですから、何の解決にもならないかも・・・

    EXEC SQL CONNECT TO xxx user yyy IDENTIFIED BY :zzz;

  私の使用していた環境は、
    OS      : Soralis 2.6
    Postgres: 6.5.2
  です。

  LinuxJapan1999 8月号＆9月号で笠原さんがecpgを用いたプログラ
ムを作成されているので、それを参考にされては如何でしょうか。

  そこで笠原さんも書かれていますが、ecpgに関する記述は余り見当た
らないので、大変ですが、頑張りましょう。
