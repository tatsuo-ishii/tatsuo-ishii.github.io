[pgsql-jp 12238] PostgreSQL にビジネスチャンスか？ (笑)

  • From: Kazumasa Gotoh <kgotoh@cic-kk.co.jp>
  • Date: Wed, 15 Dec 1999 14:11:16 +0900

日本オラクルが 2000年 1月より Linux 版をのぞく Oralce8 および Oracle8i の
Workgroup Server の値段を改定し、5ユーザ/10クライアント版で 820,000円
となると発表しました。

これまではプラットフォームによって価格差があったようにも思いますが、
これまでの Solaris 版 5U/10C は 22万でした。つまり 4倍近い値上げでして…
一部に強い反発の声があがっています。

PostgreSQL が次期バージョンの 7.0 (?) で、ログ機能や外部キー、外部接合、
そして Decimal,Numeric カラムへのインデックス作成などがサポートされれば
反発を感じた Workgroup Server レベルで業務や提案を行っているユーザが
PostgreSQL に一気になびくかも… ;-)

ビジネスチャンスかも知れませんよ。> 石井さん :-)
