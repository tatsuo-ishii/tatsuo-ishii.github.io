私が使用したバージョンは以下の通りです。

	PostgreSQL6.5 + 付属ODBCドライバ
	iODBC 2.50.3

全てソースからコンパイルしインストールを行ないました。

> 
> まえにも書きましたが、私の場合は、
> 
> PostgreSQL6.5.3+付属ODBCドライバ
> iODBC 2.50.3
> 
> で、すべてソースからコンパイルしてます。また、.odbc.iniの問題もありそ
> うですね。
> --

石井さんが言われた通り問題はodbc.iniにありました。
odbc.iniの各項目と'='の間にタブを入れていたため、odbc.iniの読込に
失敗していたようです※1。(うーん、厳しい・・・）

タブを取り除くことで接続に成功しました。

====
※1
;
; odbc.ini
;
[ODBC Data Sources]
hoge	= PostgreSQL

[hoge]
Servername	= aaa
Port		= 5432
・
・
・
★"Servername"と'='の間にタブを入れてました

====

ひょっとしてPostgreSQL6.5.3のODBCドライバでは大丈夫なのでしょうか？

ご多忙にもかかわらず、貴重な助言ありがとうございました。
