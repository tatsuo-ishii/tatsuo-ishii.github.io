> datetimeとtimespanについて質問させて頂きます。現在の時刻の一日後や一ヵ
> 月後の時刻をDBに入れたいときはどのようにしたらいいのでしょうか？
> 新しいシーラカンス本のP９６、９７を見ると、一旦格納後にselectではもって
> これるのですが、最初に計算した後にinsert文で格納する方法はあれこれやってみて
> もうまくいかず、困っています。
> よろしくお願い致します。

新規にレコードを作るということでよろしければ、

　insert into test (date) values ('now'::datetime+1);

としてやれば1日後のデータが追加されますよ。

ちなみに12時間後なら

　insert into test (date) values ('now'::datetime+'12:00');

ですから、明日の日付ではなく、明日の時刻を入れたいのなら、

　insert into test (date) values ('now'::datetime+'24:00');

としてやれば良いんじゃないかな。
