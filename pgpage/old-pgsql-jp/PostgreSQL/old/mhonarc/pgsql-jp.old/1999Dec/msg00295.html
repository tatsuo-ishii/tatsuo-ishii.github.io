平林と申します。

PostgreSQLを基盤にして、Web上のアノテーション（注釈）システムを
作ってみました。現在試験公開ですが、pgsql-jp mailing list の皆様に
御意見・御感想など伺えると大変嬉しい限りです。

http://www.arc.ritsumei.ac.jp/unchiku/

この成果を卒論にまとめようと思っているので、どうぞよろしくお願いします。

なお、システム構成は
Laser5 Linux 6.0、Apache 1.3.9、PostgreSQL 6.5.3 と、
C言語(libpq)のCGIプログラムを２０個ぐらい組み合わせたものです。

クライアントは以下のものをサポートしています。
  Windows版 Netscape Communicator のバージョン 4.0 以降 
  Windows版 Internet Explorer のバージョン 4.0 以降 
  Macintosh版 Netscape Communicator のバージョン 4.5 以降 
  Macintosh版 Internet Explorer のバージョン 4.5 以降 
  各種UNIXやその互換OS版 Nestcape Communicator のバージョン 4.0 以降
