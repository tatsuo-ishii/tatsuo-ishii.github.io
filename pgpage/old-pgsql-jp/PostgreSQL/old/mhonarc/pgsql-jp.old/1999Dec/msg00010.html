本番用(?)マシンは勉強用には使えなくて… という話でしたら、
その「身の回りにある Windows マシン」のうち 1台に FreeBSD とか
Linux などをインストールして、PostgreSQL + Apache + PHP な環境を
作成してしまうという手もあります。

勉強用/テスト用だったら、さほどのスペックは要求されないでしょうから、
そこそこの性能のマシンとか、Windows95 を動かすと遅いので誰にも使われて
いないマシンがあればそれを活用するのもよいかと…

もっとも、PC のハードウェア自身などに関してあまり得意でないようでしたら、
ISA カード(メーカ製 PC でも古いマシンのネットワークインタフェースは ISA
だったりする事が多い)を使ってるマシンは避けた方がよいかも知れません。

OS に関しては個人的な趣味でいえば FreeBSD の方をお勧めしますけど、(^_^)
身近(物理的にという事)に聞ける人間がいる方を選んだ方がいいでしょう。
