PostgreSQL に直接関係ないんですけど、^^; UNIX 系 OS 上の DBMS の話しと
いう事でご容赦を。_o_

Informix for Linux ですけど、以下の URL からダウンロードできるようです。

        http://www.informix.com/informix/products/freelx.html

ここを見ればわかるのですが、ダウンロードできるのは、

   ・ INFORMIX-SE on Linux
   ・ INFORMIX-ESQL for C
   ・ INFORMIX-Connect

で、すべて Free だという事です。細かい Copyright などは見てないの
ですが、^^; 堂々と 'Free' と書いてあるんだから、普通の意味での Free
だと考えていいんじゃないかと思ってます。(ちょっといい加減 ^^;)

私は Informix の製品体系を知らないのですが、SE というのは簡易版だ
そうです。Oracle で言うと、WorkGroup Server とかそれ以下のものなん
でしょうか？ Oracle の Parsonal 版くらいのもの？
ESQL for C というのは、やはり Oracle で言えば Pro C みたいなもの
ですかね？ Connect は SQL*Net かな？
つまり、C で埋め込み SQL を使えるようにするプリコンパイラと、他の
システム上の Informix と接続するものでしょうか。

SE 自身を使ってもいいだろうし、他のマシン上で動いている Informix DB
を使いに行くために、ネットワーク接続可能であったり、埋め込み SQL を
使ったアプリケーションが Linux 上で開発できるだけでも嬉しい人は多い
んじゃないかとも思います。
後は我々に本陣からすれば、やっぱり日本語サポートが問題ですよね。
これはいつもネックになる… ある意味しょうがないんでしょうが。

'free development licenses' とあるので、Informix としては Linux 上での
Informix アプリケーションの開発をやりやすくして、サーバ DBMS 側は
もちろん Informix の製品を買ってもらおうという戦略なのかも知れません。
