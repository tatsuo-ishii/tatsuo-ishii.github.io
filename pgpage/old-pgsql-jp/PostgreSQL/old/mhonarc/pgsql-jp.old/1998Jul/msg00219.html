はじめまして、金原といいます。宜しくです。

Linux(Debian)でのPOSTGRES活用を目指し、勉強中なのですが、
最初からつまずいてしまっています。
尚、postgresqlバージョンは、6.3.2です。

createdb test3
psql -e test3 < create.sql

を実行すると、以下のメッセージを吐きます。

drop table like;
QUERY: drop table like;
ERROR:  parser: parse error at or near "like"
create table like (foods text,like test);
QUERY: create table like (foods text,like test);
ERROR:  parser: parse error at or near "like"
insert into like values (orange,like);
QUERY: insert into like values (orange,like);
ERROR:  parser: parse error at or near "like"
insert into like values (sukiyaki,hete);
QUERY: insert into like values (sukiyaki,hete);
ERROR:  parser: parse error at or near "like"
insert into like values (cheese,like);
QUERY: insert into like values (cheese,like);
ERROR:  parser: parse error at or near "like"
select * from like;
QUERY: select * from like;
ERROR:  parser: parse error at or near "like"
EOF

create.sqlは、以下の通りです。

drop table like;
create table like (foods text,like test);
insert into like values (orange,like);
insert into like values (sukiyaki,hete);
insert into like values (cheese,like);
select * from like;

何が悪いのか、お教え頂けると嬉しいのですが。
