Apache 1.3.0 で PHP 2.0.1 を動かしています。

以前メモっていた物を付けておきます。v1.3b7 というのは v1.3.0 と
置き換えて読んで下さい。

----- ここから -----
php-2.0.1 の Apache v1.3b7 での利用 メモ

1. php-2.0.1 展開する。

2. php-2.0.1.jp_urat-5.3 パッチを当てる。

3. Web で PHP-FI の ML の archive から見つけたものを利用し以下を変更。
	# diff mod_php.c.org mod_php.c
	27a28,30
	> #if MODULE_MAGIC_NUMBER >= 19980413
	> #include "compat.h"
	> #endif

	# diff php.h.org php.h
	377a378,383
	> #ifdef APACHE_AP_H
	> 	/* symbol name safe ap_ API */
	> #define PUTS(a) ap_rputs((a),php_rqst)
	> #define PUTC(a) ap_rputc((a),php_rqst)
	> #include <compat.h>
	> #else
	379a386
	> #endif

4. ./install

5. (SunOS 4.1.4) の場合以下を変更。
	# diff log.c.org log.c
	120a121,124
	> #ifndef PATH_MAX
	> #define PATH_MAX 255
	> #endif
	> 

	# diff config.h.org config.h
	77c77
	< /* #undef HAVE_REGCOMP */
	---
	> #define HAVE_REGCOMP

6. make

以下は Apache のmake
7. libphp.a,mod_php.[ch] を apache_1.3b7/src/modules/extra/ に移動。
8. Configure
9. make
