>私はそういう使い方をした事はないですけど、PHP/FI は DBMS ごとに
>DB アクセス関数が異なりますから重複はないと思いますけど…
>PHP/FI の Build も複数 DB 対応のものが指定できるようになってますし。
>
>どのあたりがかぶると考えてらっしゃるのですか？

   Apache のConfigureの設定で MySQLの時は

      EXTRA_CFLAGS = -I. -DSERVER_SUBVERSION=\"PHP/FI-2.0.1JP5\"
      EXTRA_LFLAGS=
      EXTRA_LIBS=-L. -lphp -lgdbm -L/usr/local/mysql/lib -lmysqlclient -lregex -lm
      EXTRA_INCLUDES=

   PostgreSQLの時は
   
      EXTRA_CFLAGS= -I./jp.regex -DSERVER_SUBVERSION=\"PHP/FI-2.0\"
      EXTRA_LIBS= -L. -lphp -lgdbm -L/usr/local/pgsql/lib -lpq -L./jp.regex -lregex -lm
      Rule WANTHSREGEX=no

としています。両方組み込む時の環境変数をどう与えたらいいのかがよくわかっていません。

  DBアクセス関数が違うことは承知しています。
