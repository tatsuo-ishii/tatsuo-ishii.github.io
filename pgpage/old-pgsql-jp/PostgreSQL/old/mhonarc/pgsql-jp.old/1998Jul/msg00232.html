この件につきましては、

コンパイル・インストール        ログインUser
など
PostgreSQL                    postgres
PHP/FI                           root
apache                            root

で行いました。

/usr/srcにソースを置いて、apacheコンパイル後、
httpdを /usr/local/etcデレクトリの下におきました。
mkdir cgi-data
chown nobody:nogroup cgi-data
としたいのですが、ｎogroupがないのメッセージが
出ました。
PostgreSQLインストール後に
user      groupe
nobody  nobody
(id=999)

となりました。

nogroupをnobodyと読み替えて
chown nobody:nonobody cgi-dataとして
続けて良いでしょうか。
