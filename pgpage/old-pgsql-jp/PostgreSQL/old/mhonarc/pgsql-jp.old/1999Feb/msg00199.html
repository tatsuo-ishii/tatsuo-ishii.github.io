$PGDATA2を設定して、pgsqlユーザで以下の操作をしました。

queen:~$ initlocation $PGDATA2
We are initializing the database area with username liz (uid=10002).
This user will own all the files and must also own the server process.

queen:~$ createdb -D PGDATA2 test2
ERROR:  Unable to locate path 'PGDATA2/test2'
        This may be due to a missing environment variable in the server
createdb: database creation failed on test2

ということで、データベースの作成に失敗しました。

念のため、このユーザでpostmasterを、別ポートで起動し、
queen:~$ createdb -D PGDATA2 test2 -p 5430
とやりましたが、結果は同じでした。
