内山と申します。

Hiratsuka Noriko さんは書きました:
>plamo linux上でperlのPgモジュールを使ってCGI経由で使う
>検索クライアントを作っているのですが、コマンドライン上では
>検索に成功するのにCGI経由だとなぜかInternal Server Errorに
>なってしまいます。

	直感でお答えしてはずすかもしれません。

	PostgreSQL のユーザに　nobody は作りましたか。

	ユニックスのアカウント　postgres にて　
	createuser nobody をしてみてください。
	検索だけなら権限は　database create も add user は必要でないと思います。
