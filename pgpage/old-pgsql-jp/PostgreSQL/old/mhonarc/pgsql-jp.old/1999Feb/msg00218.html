石井です。

>お世話になっています。船木＠秋田県立図書館です。
>関数translateについてなんですが、”シーラカンス本”の記述に従って
>あるテーブルの２バイト文字を２バイトスペースにするのに
>update test set data=translate(data,'文字１','空白')
>のようなsqlを書いてみたのですが、うまく変更されません。
>もしかして２バイト文字のtranslateはできないのでしょうか？

御推察の通りです。この translate は某商用DBの互換関数なのですが、SQL92
のtranslate と名前が完全にバッティングしているし、私の考えでは、
PostgreSQL が特定の商用DBとの互換性を保つ義理もないと思うので放ってあ
ります。

# とは言え、このことを「シーラカンス本」の中で明確にしておくべきでした。
# 申し訳ありません。
