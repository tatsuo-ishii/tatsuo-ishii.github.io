>こんにちは、佐々木です。

>データベースを利用しようとすると以下のようなメッセージが出て
>使えません。
>
># psql DATABASENAME
>
>Connection to database 'DATABASENAME' failed.
>ERROR:  Index pg_class_relname_index is not a btree
>
>これはどうしたら解消できるのできるのでしょうか?
>いろいろ調べたのですがわかりません。
>このMLの過去ログも調べたのですがわかりませんでした

これはちょっとやっかいなことになりましたね。最悪データが復元できないか
も知れません。まずは tar コマンドなどでバックアップをとってこれ以上事
態を悪化させないようにしましょう。

以下、駄目元で試してみてください。

たとえば /usr/local/pgsql/data にデータベースがあるなら、postgres また
は root で

# cd /usr/local/pgsql/data/base/DATABASENAME
# mkdir aaa
# mv pg_class_relname_index aaa/.

とし、これで DATABASENAME に接続できるかどうか試してみてください。
OKならば pg_dump ですかさずバックアップを取り、destorydb、createdb で
データベースを作り直し、pg_dump のバックアップを戻します。
