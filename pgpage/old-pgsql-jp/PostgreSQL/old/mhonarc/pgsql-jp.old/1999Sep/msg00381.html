> ３．その他
> 
> 　selectで読んだ列名は、次のシェル変数にセットされます。
> 
> 　$SQL_NFIELD　　　   : 列数
> 　${SQL_FILEDNAME[i]} : 列名

  上は間違いです。下の(1)が正しいです。

(1) 列名の取得

　$SQLNFIELD         : 列数
  ${SQLFIELDNAME[i]} : 列名

(2)バージョン 1.0b2との相違点
　
　・$SQLNFIELDS を $SQLNFIELD に変更
  ・$SQLNTUPLES を $SQLNTUPLE に変更

  ・-N :  turn off printing of headings and row count を
  　-T, -B の２つのオプションに変更して -T で heading の
    停止、-Bを row countの停止にしました。

  ・-T :  tern on TABLE type output を
    -L の外枠罫線出力にしました。

　・-F sep : set field separator を
　  -S sep に変更しました。
