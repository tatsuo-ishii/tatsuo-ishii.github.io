> MS-Access97のVBA上だとrollbackが正常に動作しません。
> SQLを１行１行実行する事が問題あるのでしょうか
>     qd.SQL = "SQL文;"
>     qd.Execute

　私は次のようにしています。


''トランザクションの開始を宣言
    Set qdfTemp = Conect.CreateQueryDef("", "begin;")
    qdfTemp.Execute

''使用するテーブルのロック処理
    Set qdfTemp = Conect.CreateQueryDef("", "lock user_tel;")
    qdfTemp.Execute

　この状態でモジュールを試験動作させると、user_telというテーブルは
他クライアントか
らアクセス出来なくなりますので、送り込まれていると思います。
