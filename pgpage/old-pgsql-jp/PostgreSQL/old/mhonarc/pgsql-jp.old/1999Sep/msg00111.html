http://www.sra.co.jp/people/t-ishii/PostgreSQL/6.2/patch1.html

あたりが参考になると思います。こんなことが書いてあります。

>6.2.1jpPL1.patch を置いてある場所を somewhere とすると、 
>               % cd /usr/local/pgsql/postgresql-6.2.1
>               % patch -p1 < somewhere/6.2.1jpPL1.patch

パス名やファイル名は、自分の環境に合わせてください。
オプションについては自分で調べて下さい。

あと、patch コマンドは標準のものではないとか、
SunOS ではshared memory などの設定が標準のままでは不足するという
記述が
http://www.sra.co.jp/people/t-ishii/PostgreSQL/6.4/install-basic.html
のあたりのあったりするので、ML のページはすみずみまで
読んでおくとよいと思います。
