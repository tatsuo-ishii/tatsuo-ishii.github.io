こんにちは。

>   PHP についての質問です．現在 PHP で class を使っていますが，
> クラスのオブジェクトを form で渡すことは可能なのでしょうか？

そりゃ無茶でしょう。

<?
$class_object->member_function();
?>
では、そのまんま「$class_object」というクラスがあると PHP は判断
します。

echo "$class_object";
で「Object」と出るのは、引数が double quotation で囲まれいて、その
先頭に $ がついていると echo というメソッドはそれがリテラル文字
であると判断するからです。

分かりやすく言うと、「$class_object」という入れ物があってその中に
「Object」という文字列が入っています。その入れ物から Object クラス
のメンバ関数を呼ぼうとしているわけですから
Fatal error: Call to a member function on a non-object in /home/..... line
xx
となるのは仕方ないでしょう。
echo はその入れ物から文字列を取り出してプリントするという作業を
するだけです。
