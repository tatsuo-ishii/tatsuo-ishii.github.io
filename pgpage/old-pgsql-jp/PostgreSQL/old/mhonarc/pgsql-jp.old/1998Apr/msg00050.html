>>もしかして、postmaser 起動時に -i オプションを付けてないとか。
>
>はい。それで、サーバーとクライアントを同じマシンにする場合でも
>JDBCを使う場合は -i オプションが必要なのでしょうか。

JDBC の実装に依存するので良く分かりませんが、6.3 からはデフォルトでは 
Unix domain の socket のみ、INET domain の socket が必要な場合は -i を
付けて postmaster を起動することになりました。したがって、同じマシンか
どうかに関係なく、JDBC が 常に INET domain の socket を使うような実装
になっていれば、-i が必要になりますね。
