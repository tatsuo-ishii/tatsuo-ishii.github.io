[pgsql-jp 3853] postgreSQL 6.3.2 on RedHat Linux 4.2

  • From: "Yoshiaki NAKAGAWA" <nakagawa@parasite.co.jp>
  • Date: Tue, 21 Apr 1998 19:19:46 +0900
　postgreSQL 6.3.2 を RedHat Linux 4.2 にインストールし、ベンチマークを
とりましたので送ります。ハードは Pentium/90MHz RAM 64MB。Makefile.custom
は MB=EUC_JP と CFLAGS+=-O2 を指定してあります。コンパイラは gcc version
2.7.2.1 を利用しました。社内のファイルサーバとしても使っているマシンなの
で精度に問題があるかもしれません。

　なお、私の環境だとコンパイル時 bootstrap.c でコンパイルエラーが発生しま
す。これは cpp が PATH の中にないために src/backend/utils/Gen_fmgrtab.sh
が失敗することが原因のようでしたので、とりあえず cd /usr/bin; ln -s
/usr/lib/gcc-lib/i386-linux/2.7.2.1/cpp して回避しました。
