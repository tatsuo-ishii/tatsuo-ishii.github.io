	Jun Kuwamura <juk@rccm.co.jp> wrote:

> 質問１
>   どなたか、GNU readline で日本語(EUC)を通すためのパッチを知りませんか？
> USE_READLINE=true で作った psql は日本語を通してくれません。
>   readline のバージョンは 2.0 です。

EUC を通すだけでよいのなら readline 2.0 では ~/.inputrc に以下の
行をつけ加えれば大丈夫なはずです.
set convert-meta off
set output-meta on

これをデフォルトにしたいということなら, config.h の以下のの行を変
更して下さい.
int _rl_convert_meta_chars_to_ascii = 1;
int _rl_output_meta_chars = 0;
