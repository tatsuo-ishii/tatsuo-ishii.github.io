SQL の書き方について。大したことじゃないんですが、

	select car.name from car;

は、私などはついテーブル名を省略して

	select name from car;

などど書いてしまいがちです。趣味の問題ですが。

ところで、SQL で思い出しましたが、
http://www.itl.nist.gov/div897/ctg/sql_form.htm にて SQL の Test Suite
が入手できます。ちょっとだけ PostgreSQL で試しましたが
(schema/schema1.smi)、やはりぼろぼろ:-) column constraint がないのがつ
らいですね。その他では、

o DECIMAL がない
o FLOAT で精度が指定できない
o テーブル名に "." が含まれているとエラー
o DOUBLE PRECISION がない
o NUMERIC がない

あたりでつまづいています。

# 個人的には、SQL の DECIMAL だの NUMERIC だの、過去の遺物みたいなもの
# はどうでもいいと感じますが。
