6.1.1 でもサポートしてません。SQL エラーにはならないのですが、単純に無
視されるようです。たとえば、

	create table header (hfrom text);

と定義し、pgsql-jp ML のメールから From: を取り出して header テーブル
に格納します。そして、

select hfrom,count(*) from header group by hfrom order by 2;

などとやると、以下のような結果になりますが、

(中略)
t-ishii@sra.co.jp (Tatsuo Ishii)                                |   28
Satoshi Ishikawa <power@sekine.densi.gifu-u.ac.jp>              |   45
Tatsuo Ishii <t-ishii>                                          |   47
Hanai Hiroyuki <hanai@astec.co.jp>                              |   50
kanemasu@dragon.osaka.ish.cae.ntt.jp                            |   51
Mitsuhiro Maeda <mitsu@tramp.co.jp>                             |  133
Jun Kuwamura <juk@rccm.co.jp>                                   |  148
Tatsuo Ishii <t-ishii@sra.co.jp>                                |  310
(400 rows)

having を使って

select hfrom,count(*) from header group by hfrom having count(*) > 50 order by 2;

としても全く同じ結果です。(エラーにもならない)
