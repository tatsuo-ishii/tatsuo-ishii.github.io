値段が高くて購入しづらいです。

　さて、掲記の件で壁にぶつかっているので質問いたします。
0)使用環境
　Solaris2.5.1x86, PostgreSQLv6.1, jp-patch, i386solaris-patch

1)テーブル
CREATE TABLE AAA
(   EN          TEXT,
    EN_CODE     INT2 ) ;

2)データ
INSERT INTO AAA VALUES ( 'A123A', 10 ) ;
INSERT INTO AAA VALUES ( 'B234B', 20 ) ;

3)FUNCTION
CREATE FUNCTION EN2CODE( TEXT ) RETURNS INT2
  AS 'SELECT EN_CODE FROM AAA
      WHERE EN = \'$1\' ;'
  LANGUAGE 'SQL' ;

このような設定で関数を実行してもうまく出力されません。
=> select en2code( en ), en_code from aaa ;
en2code|en_code
-------+-------
       |     10
       |     20
