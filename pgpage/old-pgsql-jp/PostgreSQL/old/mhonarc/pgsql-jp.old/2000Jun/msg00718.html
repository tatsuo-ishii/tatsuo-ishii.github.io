自動起動はまだなのですが、PHPの動作を確認しています。
PHPのプログラム自体は動くのですが、データベース接続ができません。
以下のようなプログラムを動かすと
<?
$conn=pg_Connect("localhost","5432","","","test");
f(!$conn){
        echo "error\n";
        exit;
}
?>
Fatal error: Call to unsupported or undefined function pg_connect() in
/home/top/public_html/testdb.php3 on line 2
このように表示されてしまいます。
「test」というデータベースはできています。
テーブルはまだですが、接続に関係ないと思い、作っていません。
pg_hba.confのコメントははずしました。
host         all        0.0.0.0       0.0.0.0             trust
PostgreSQLのバージョンは、7.0.2で、ソースをコンパイルしました。
起動は以下のようになっています。
/usr/local/pgsql/bin/postmaster -i -D ...

ライブラリがうまく読みこめていないのではないかと疑っています。
httpd.confは
LoadFile /usr/local/pgsql/lib/libpq.so
LoadModule php3_module /usr/libexec/libphp3.so
と書きました。
OSは、TurboLinuxServer6.0で、ApacheとPHPは自動でインストールされるものを使っ
ています。
