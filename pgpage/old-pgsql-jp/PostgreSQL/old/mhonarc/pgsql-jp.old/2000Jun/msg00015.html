> numeric型の項目に1000000という値が入ってるとします
> これを1,000,000という形で取り出したいのですが
> そういった関数等は存在するのでしょうか？
> PostgreSQLは6.5.3ですが（7.0に個人的にはしたいのですが）
> どうでしょうか？

PostgreSQL 7.0 ならば、to_char 関数を使うとできます。

----------------------------------------------------------
  kgotoh=> create table test (kingaku numeric(10,0));
  CREATE
  kgotoh=> insert into test values(1000000);
  INSERT 260114 1
  kgotoh=> insert into test values(123);
  INSERT 260115 1
  kgotoh=> insert into test values(12345678);
  INSERT 260116 1
  kgotoh=> select to_char(kingaku, '999,999,999') from test;
     to_char    
  --------------
      1,000,000
            123
     12,345,678
  (3 rows)
----------------------------------------------------------

PostgreSQL 6.5.3 の場合は、自分の希望する機能を持ったストアード
ファンクションを定義すれば可能だと思います。
