石井さんの本を読んでいてもわかりませんか…
ひょっとして、JDBC どころか Java 自身も初めてとか？

別に備後さんがどうのこうのというわけではないのですが、どうも最近
色々なところ(この ML に限らず)に投稿されている記事を見ると、

  ・ 何も理解しようとせずに、ただ質問を繰り返す人達
  ・ 一度にたくさんの事をやろうとし過ぎて、混乱している人達

というのが目立ってきているように思います。前者は論外として、後者の
方達は、ある最終目的を達成しようとしているのですが、そこに至る
複数のハードルのひとつひとつにビックリしていたり、全体を通して
見る事ができないために混乱なさっているようです。

よくありがちなケースとして、本やサンプルの丸写しで意味がわかって
いないために、動かない時に何も見当がつかないとか。

アドバイスとしては、「落ち着いてよく考えよう」、「詳しく自分でチェック
しよう」、「基本からきちんと勉強しよう」というあたりになるのでは
ないかと思います。

パスワードでエラーになるのであれば、

  ・PostgreSQL 側の設定が不完全でエラーになる
  ・JDBC など、プログラム側の設定が不完全

というのが考えられますが、例えば psql を使用する場合に、そのユーザ名と
パスワードが使えますか？

ひとつひとつ切り分けを行い、問題点を整理していかれるとよいでしょう。
