はじめまして.

初めて投稿します。今後ともよろしくお願いします。

以下のテーブル(test1_tbl, test2_tbl)に対してForeign Keyをはろうと
したのですが、エラーがでます。

test1_tbl
create  table test1_tbl (
        b int primary key ,
        c int ,
        d date );

NOTICE:  CREATE TABLE/PRIMARY KEY will create implicit index 'test1_tbl_pkey' for table 'test1_tbl'
CREATE

test2_tbl
create  table test2_tbl (
        a int ,
        b int ,
        primary key (a,b) ,
        c text ,
        d int ,
        e int ,
        foreign key (b) references test1_tbl(b) );
NOTICE:  CREATE TABLE/FOREIGN KEY clause ignored; not yet implemented
NOTICE:  CREATE TABLE/PRIMARY KEY will create implicit index 'test2_tbl_pkey' for table 'test2_tbl'
CREATE

CREATE TABLE/FOREIGN KEY clause ignored; not yet implemented の意
味がわかりません。

環境
Turbo Linux Server 6.0
PostgreSQL 6.5.3
