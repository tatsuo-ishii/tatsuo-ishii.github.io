Oracle8i Workgroup Server 8.1.5 for Linux の、「不特定ユーザインター
ネットアクセスライセンスパック」は、製品自身は 80万で、年間保守契約料は
BRONZE の場合だと 20万です。SILVER だと 28万ですね。
数億の規模じゃなくても使えるんじゃないかと思いますが…

ただ、Enterprise Edition になると、製品が 640万で、年間保守契約料金は
BRONZE で 128万、SILVER だと 179万 2千円になります。
# Linux 版には Enterprise Edition はないんで、この場合は Solaris とか
# NT などを使う事になるかと思います。

現状では PostgreSQL は Oracle の Enterprise Edition に相当する機能は
ありませんから、PostgreSQL が対応するのは Workgroup Server クラスだと
いう事になると思います。

ちなみに、Enterprise Edition の特定ユーザ無制限ライセンスパックになると、
製品が 2560万で、年間保守契約料金は BRONZE で 512万、SILVER だと 716万
8千円になるようですね。
