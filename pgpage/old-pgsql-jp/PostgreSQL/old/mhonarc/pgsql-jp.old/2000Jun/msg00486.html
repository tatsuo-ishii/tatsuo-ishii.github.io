うちではVine Linux 2.0 ですが、この後環境変数を設定して以下の様に
しています。

　・環境変数設定（~/.bashrc）
----------------------------------------------------------------------
export PATH="$PATH":/usr/local/pgsql/bin
export POSTGRES_HOME=/usr/local/pgsql
export PGLIB=$POSTGRES_HOME/lib
export PGDATA=$POSTGRES_HOME/data
export MAINPATH="$MAINPATH":$POSTGRES_HOME/man
export LD_LIBRARY_PATH="$LD_LIBRARY_PATH":"$PGLIB"
----------------------------------------------------------------------　　

　$ source ~/.bashrc
　の操作で環境変数を反映させる

　・データベースの初期化
　$ initdb

　・postmasterの起動
　$ postmaster -S

　・regression test
　$ cd /usr/local/src/postgresql-7.0.2/src/test/regress/
　$ make all runtest


PostgreSQL 7.0.2 のインストールメモなので、パスは適当に読み替えて
ください。
