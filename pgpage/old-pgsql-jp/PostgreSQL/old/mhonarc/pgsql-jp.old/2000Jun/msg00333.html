# まだ、PostgreSQL-7.0.x のパッケージできてないです。mOm
# が、とりあえず、バイナリを固めるだけはしてあります。

  PostgreSQL-7.0.2 のバイナリ(psqlj付き)です。
http://www.linet.gr.jp/~juk/plamo/parts/2.0/
に起きました。
postgresql-7.0.2-mb-i386-linux-glibc2.1-bin.tar.gz
です。

# ./configure  --prefix=/usr/local/pgsql7 --enable-multibyte
--with-pgport=7654 --with-tcl --with-tclconfig=/usr/local/lib
--with-tkconfig=/usr/local/lib --with-odbc

で構成して構築してあります。インストールの後、 /usr/local ディレ
クトリよりpgsql7以下を tar でまとめて、gzipで圧縮したものです。
regressionテストと mb テストは正常に通過してます。

(注： pgsqlj は postgresql-7.0.2 に堀田さんによる postgresql-7.0
用のパッチを当てて make したもので、 readline も使うようにしてあ
ります。 ) 

  なお、デフォルトポートを 7654 に変えてありますので、これまでの
pgsql65xパッケージと共存できます。ご利用になっているシェルの初期
化については、展開後に pgsql7/_*.pgsql7 を御覧ください。

[注意]
  構築に使った Tcl/Tk はインストール済みの Tcl8.3/Tk8.3 のライブ
ラリを利用するように、 --prefix=/usr/local として構成してありま
す。Plamo Linux 2.0 の Tcl8.3/Tk8.3 は contrib/TclTk/ ディレクト
リにあります。
