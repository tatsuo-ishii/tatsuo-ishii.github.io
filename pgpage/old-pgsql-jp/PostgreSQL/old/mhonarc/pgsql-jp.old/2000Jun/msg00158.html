> PostgreSQL 6.5.3 Pentium II Linux(Vine 1.1)だと、'-2147483648'
> の時だけ、'  ' で囲まないとエラーになります。 面白いですね。
> # int4 じゃないかもしれないけど、これと似た話、前にもあったような。

同じことを Alpha でやってみました。

aaa=> create table tt (aa int4);
CREATE
aaa=> insert into tt values(-2147483647);
INSERT 29193 1
aaa=> insert into tt values(-2147483648);
INSERT 29194 1
aaa=> insert into tt values('-2147483648');
INSERT 29195 1
aaa=>  select * from tt;
         aa
-----------
-2147483647
-2147483648
-2147483648
(3 rows)

Alpha では '' 無しでも -2147483648 は正常に扱われ
るようです。
