この例では
$memo にはいってる複数のデータを取り出そうと
してますが、私は同様の事をしてますが、
@memo = $query->param('memo');
で、各変数 $memo[?] に入ります。

出力するなら、
for ($i=0; $i<25; $i++) {
    print "$memo[$i]\n";
}

これで　出力は出来ます。
