 >[pg::setdb($pghost, $pgport, $pgoptions, $pgtty, $dbname)]を使わないと出
 >来ないのでしょうか？

old styleは今後無くなるかも知れないので使わないほうが無難です。

 >または、[pg::connectdb("dbname=$dbname");]で設定方法があるのでしょうか？
 >実際の記入例を教えていただけると幸いですが、いかがでしょうか？

>>           parameter  environment variable  hard coded default
>>           --------------------------------------------------
>>           host       PGHOST                localhost
>>           port       PGPORT                5432
>>           options    PGOPTIONS             ""
>>           tty        PGTTY                 ""
>>           dbname     PGDATABASE            current userid
>>           user       PGUSER                current userid
>>           password   PGPASSWORD            ""

             ↑ここがoptionに相当
>>
>>           $conn = Pg::connectdb("option1=value option2=value ...")

こんな感じで使えると思います。
--
  #!/usr/local/bin/perl -w
  
  use Pg;
  $conn = Pg::connectdb("host=hogehoge port=5432 dbname=nanigashi user=username password=himitsu");
