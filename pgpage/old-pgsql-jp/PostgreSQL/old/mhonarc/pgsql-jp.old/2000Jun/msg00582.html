市村と申します。

UnixWare 7.1.1 上で Unix ドメインでバックエンドへ接続しようとすると、
エラーになってしまいます。原因等ご存知の方がいらっしゃれば、ご教示頂
きたいと思います。PostgreSQL のバージョンは 7.0.2 です。

1) バックエンドの起動

$ postmaster -S

2) psql の起動

$ psql populadb
psql: No pg_hba.conf entry for host localhost, user postgres, database
populadb

となってしまいます。populadb は データベース名です。

PGHOST に localhost を指定し、postmaster を -i オプション付で起動すれ
ば、問題なく inet ドメインで接続できます。
