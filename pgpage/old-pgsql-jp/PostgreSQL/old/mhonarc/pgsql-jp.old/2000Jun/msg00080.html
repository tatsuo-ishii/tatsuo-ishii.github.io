杉村です。

> abc=> select date - '1month' from hit;
> ERROR:  Bad datetime external representation '1month'

(1) select date - '1 month'::interval from hit;

(2) select date + '-1 month' from hit;


個人的には '1 month' は 時間の量 であって 時刻 ではないので
(1) のように ::interval をつけて型が違うことを明示したほうが
いいような気がしないでもないです。

# なんで + '1 month' が できて - '1 month' がだめなのかはわかりません。
