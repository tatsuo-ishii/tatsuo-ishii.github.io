くわむらです。

  PostgreSQL-7.0.1 がリリースされたので、バイナリにしました。

http://www.linet.gr.jp/~juk/plamo/parts/2.0/postgresql-7.0.1-mb-i386-linux-glibc2.1-bin.tar.gz

  /usr/local/ 下あたりに展開して使ってください。
  デフォルトのポート番号を7654 にしてありますのでご注意ください。
データベースの雛型は作ってないので initdb が必要です。

  ちなみに、わたしの環境変数は：
--
PGLIB=/usr/local/pgsql7/lib
PGDATA=/usr/local/pgsql7/data
PGPORT=7654
MANPATH=/usr/local/pgsql7/man:...
PATH=/usr/local/pgsql7/bin:...
LD_LIBRARY_PATH=/usr/local/pgsql7/lib:...
--

  とりあえず、以下のようには試してはみました。

	% initdb
	% createdb
	% pg_ctl start

	% psqlj
	postgres=# \?
	...
