> $result = $conn->exec($xxx);を実行した時、
> select の場合は $result->resultStatus ne PGRES_TUPLES_OK) で確認出来ますが、
> insert や delete や update を実行した時は
> どうなるのでしょうか？
> 
> $result->????? ne ??????

INSERT のときは
 $result->cmdStatus が INSERT 最後のoid INSERTした行数
 $result->cmdTuples が INSERTした行数
UPDATE のときは
 $result->cmdStatus が UPDATE UPDATEした行数
 $result->cmdTuples が UPDATEした行数
DELETE のときは
 $result->cmdStatus が DELETE DELETEした行数
 $result->cmdTuples が DELETEした行数
って感じになります。行数は0もちゃんと帰ってきます
sqlのエラーなら空っぽです。
# print してみてください

に詳しく書いてます。
