松本様 wrote: 
> manの日本語訳のcreate functionの個所で
> サンプルの意味を教えて頂けないでしょうか？
>
>    create function TP1 (int4, float8) returns int4
>    as ‘ update 銀行口座 set残高 = 銀行口座.残高 - $2
>    where 銀行口座.口座番号 = $1
>    select(x = 1)’
>    language ‘sql’
> 使用例として、
>    select (x = TP1(17, 100.00)
>
> とあるのですが、
> select(x = 1)の意味がよくわかりません。

自分も「どういう意味なんだろう？？」と思っていたのですが、今プログラマ
ガイドの"拡張 SQL:関数"を読んでいた所、松本様が自己レスなさっていた例と
同じような例と解説がのってました。日本語訳を引用させてください。

>SQL 問合せ言語のコマンドの集まりは何でも、一 緒にパッケージになり、関数
>として定義できます。コマンドは更新系（ つまり、 insert 、 update 、
>delete ）も select クエリー も含める事ができます。しかし、最後のコマン
>ドは関数の返す型と指定し たものを返す select でなければいけません。
>
>	CREATE FUNCTION clean_EMP () RETURNS int4
>	 AS 'DELETE FROM EMP WHERE EMP.salary <= 0;
>	SELECT 1 AS ignore_this'
>	 LANGUAGE 'sql';
>
>    SELECT clean_EMP();
>
>	+--+
>	|x |
>	+--+
>	|1 |
>	+--+

select 1 というのはRETURNS int4だからなんすね。
長くなってしまいました。ではでは！
