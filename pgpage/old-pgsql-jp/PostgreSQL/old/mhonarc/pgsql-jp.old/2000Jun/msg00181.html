kuma <aaf35030@pop06.odn.ne.jp> wrote:
> 
> DBD::Pgのtest.plによる全てのテストでOKが出ませんでした。
> 何か問題があるのでしょうか？

date 型で select したばあいに、PostgreSQL の 6.5.3 まではデフォルトで
'MM-DD-YYYY' のようになっていましたが、PostgreSQL 7.x ではこれが
iso 形式 'YYYY-MM-DD' になったようです。

make test する前に環境変数 PGDATESTYLE を Postgres に設定してから
make test してみて下さい。

sh  なら PGDATESTYLE=Postgres; export PGDATESTYLE
csh なら export PGDATESTYLE Postgres

あと、

> $dbh->pg_table_attributes .. not ok: date_

これは自分のところでも出ています。PostgreSQL6.5.3 w/patches
のときも出ていたと思うんですが、まだチェックしていません
