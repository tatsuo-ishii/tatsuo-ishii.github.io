今、２つのテーブルをつなげるouterjoinを使用したいと思っています。
本や過去のＭＬを見ながら、UNIONを使った方法で作成して表示は出来ています。
このSQL文にorder byをつけて番号で昇順にしたいのですがどこへつけたらよいの
でしょうか？ちなみに、番号のカラムはco.co_01になります。

以下のSQLを参考に掲載します。
--------------------> sample sql 
my $qstr = "select 
co.co_01,
mco.mco_03,
mco.mco_06,
mco.mco_09,
co.co_03,
co.co_06,
co.co_13 
>from co,mco 
where co.co_01 = mco.mco_02 
union all 
select 
co.co_01,
null as null_03,
null as null_06,
null as null_09,
co.co_04,
co.co_06,
co.co_13 
>from co 
where not exists 
(select * from mco where co.co_01 = mco.mco_02)
 ";
