Yasuhiro Tanaka wrote:
> 
> 田中靖宏＠数理計画です。
> 
> Informix でいう dbschema や、Oracle でいうところの desc 命令の
> ように、テーブル名から、"create table 文" を逆生成するような
> コマンドはありませんでしょうか？

pg_dump  -s  -t  表名  ＤＢ名 

で、dbschema と同じような事ができます。
SERIAL 型即ち、SEQUENCE を使っている場合は、CREATE SEQUENCE 文
も一緒に出力してくれます。
当然、REVOKE, GRANT, CREATE INDEX 等も一緒に出力してくれます。

-s がないと、データー毎ダンプされるので注意。
