(初期導入の場合)

(1) ApacheをDSOでmake
(2) PostgreSQLを作成

     ・% ./configure --with-mmultibyte=EUC_JP --other-options    ← 7.0.x
     ・% make; make install
　　　 % initdb                  ( 必須です )

(3) phpの作成

    問題は無かったと思います。
