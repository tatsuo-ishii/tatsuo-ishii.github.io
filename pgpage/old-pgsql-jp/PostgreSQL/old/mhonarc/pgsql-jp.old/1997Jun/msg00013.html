    letter		([\241-\376].|[_A-Za-z])+

  この lex 表現は SunOS[3-4]* Solaris2.[0-4]* NEWS-OS[2-6]* HP-UX[7-9]*
EWS* NCR* VAX* Ultrix* AIX* の lex で正 しいスキャナを生成してくれないは
ずです。error を出さずに生成しても、生成された表が役に立たないのもありま
す。flex なら大丈夫です。flex 以外でこれがちゃんと動く lex は見たことが
まだないんです。あったら教えてください。もう、時効だろうから言っちゃうと、
flex がないころに VAX の lex に手を入れて通るようにしたことがあります。

  僕は [\241-\376][\241-\376] というようにして矩形領域を指定して使ってい
たことがあります。SJIS の分割された領域もこの手で表現できて flex で OK 
ですね。EUC と SJIS の flex の表をプログラム起動時に LANG で判定して切り
替える (ここは他の人が作った) ということもやってました。
