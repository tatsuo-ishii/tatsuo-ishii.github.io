[pgsql-jp 1972] MkLinux patch for PostgreSQL 6.1

  • From: Tatsuo Ishii <t-ishii>
  • Date: Tue, 17 Jun 1997 13:38:11 +0900
石井＠SRAです。

JP patch に続き、今度は MKLinux 用のパッチをお届けします。
今回のパッチでは、

o 浮動小数点に関する gcc のバグを回避
o localtime() に関する MkLinux 特有の振舞を回避
o 排他制御の部分を system call から assembler に書き換えることによる
  高速化

を行っております。手元の DR2.1+update2+shared lib/PowerMac 6100/66AV 
ではほぼ問題無く regression を通過することを確認しました。他の環境(PCI
mac や non shared lib など)での動作レポートをお待ちしています。

添付のパッチですが、gzip+uuencode されてますので、展開後 

	cd postgresql-v6.1
	patch -p1 < mklinux.patch

として当てて下さい。インストール手順は INSTALL に書いてあるとおりです
が、http://www.sra.co.jp/people/t-ishii/PostgreSQL/6.1/install.htm に
も解説があります。
