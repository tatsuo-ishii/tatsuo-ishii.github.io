	略。

	pg-sql の話題でないかもしれませんが。データベースの中身を
知るために、postgreSQL を使って、中身を解析してみようと思っていま
す。それと並行して、最先端のデータベース技術も取り入れてみようと
思っています。最先端のデータベース技術とは、自分で新しいデータベー
スの概念を模索することもありますが、オブジェクト指向データベースや
マルチメディアデータベースをおさえておく必要があると思っています。
	この中の、マルチメディアデータベース利用技術習得なのですが、
キーワードとして、Oracle Universal Server と Informix Unisersal 
Server があげられると思います。今後、文部省からお金がつけば、どちら
かのプラットフォームを導入して、学生に最先端データベース利用技術の
習得をさせようと思っています。しかし、どちらが良いのでしょうか？
アーキテクチャーが優れているというような観点からのご意見、市場に
どちらが多く出回るであろうかというような観点からのご意見も頂けた
ら幸いです。
