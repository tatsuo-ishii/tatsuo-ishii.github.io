>PostgreSQL 6.5.3なんですが、ＪＤＢＣ経由で配列オブジェクトを取得しに行くと
>その時点で例外が発生してしまい困ってます。
>例外の詳細を表示させてみると「サポートされない物が実行された」との意味合いの
>英文が表示されました。
>ソースは以下で、ResultSet::getArray("column name"); を実行すると例外発生しま
>す。



オレンジソフト塩田です。

PostgreSQL7.0betaに付属のJDBCドライバでのResultSet.javaでも
以下のようになってるのであきらめましょう(^_^;

    public Array getArray(String colName) throws SQLException
    {
	return getArray(findColumn(colName));
    }
    
    public Array getArray(int i) throws SQLException
    {
	throw postgresql.Driver.notImplemented();
    }
