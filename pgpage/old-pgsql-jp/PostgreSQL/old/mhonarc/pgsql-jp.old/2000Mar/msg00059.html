【質問】
以下のような簡単なテーブルがあるとします。

create table member (
	id		int4,
	name		text
)

実際にはフィールドが非常にたくさんあるのですが、ここでは２つに
してます。このテーブルに対し、insertコマンドでデータを新規登録
するページを作ってますが、

	$qstr = "insert into member values(";
	foreach $i (0..$member_att_fields-1){
			if($i != 0){ $qstr .= ", ";}
			$tmp2 = " \$qstr .= \"\$$member_att[$i]\"; ";←※
			eval $tmp2;
	}
	$qstr .= ")";

てな具合でぐるぐる回して問い合わせコマンド文字列$qstrを作ろうと
してます。ところが、フィールドにより、textとそれ以外のものがあるので、
場合によって、※の部分でシングルクオートでくくる場合と、そうでない
場合を書かないといけません。この判断をさせるのに、それぞれのフィールドが
どういうタイプなのかを知る必要がありますが、どういうコマンドを利用すると
可能でしょうか？（あるいは内部に持たせた方がはやい？）
