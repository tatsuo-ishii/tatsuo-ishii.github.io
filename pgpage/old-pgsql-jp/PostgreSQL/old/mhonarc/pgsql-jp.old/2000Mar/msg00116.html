PostgreSQL の JDBC ドライバに関する解説は、PostgreSQL のパッケージ
自身に含まれているドキュメントに記述されています。
PostgreSQL のマニュアルに目を通せばよろしいのではないかと思います。

VisualCafe 云々については、VisualCafe のドキュメントなりヘルプなりに
他のクラスライブラリを組み込むにはどうするのかの記述があると思われます。
まぁ原則としては、CLASSPATH が通っているところに置いておけばよいのでは
ないかと…

PostgreSQL のマニュアルは英語なのでちょっと… という事でしたら、
石井さんのシーラカンス本にも、ある程度の記述はあります。

また、JDBC ドライバ自身の使い方は PostgreSQL でも ORACLE でも、他の
ものでも共通で、異なる部分は JDBC ドライバのロードや DB 接続を行なう
部分くらいです。

したがって、JDBC の細かい使い方については、JDBC の解説を行なっている
書籍を参照すればよいわけです。これは適当な書籍が何冊かあります。

とは言え、PostgreSQL の JDBC ドライバでは実装されていないメソッドも
ありますから(これは別に PostgreSQL に限らずどれでも同じです)、一応
PostgreSQL 自身のドキュメントも適宜参照した方がよいでしょう。

それから、JAVA Press の記事がどうしても読みたいのであれば、バック
ナンバーを取り寄せる事も可能です。
