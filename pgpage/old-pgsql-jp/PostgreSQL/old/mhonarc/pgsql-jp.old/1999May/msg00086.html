# VCHAR型と書きましたが正確には VARCHAR型でした。

　上例では、DEFAULT句をいじるより、CHAR型をVARCHAR型に宣言する
方が利口(^^)ですね。

  state  VARCHAR(5) NOT NULL DEFAULT 'new'
  とすると、正常に動作します。
