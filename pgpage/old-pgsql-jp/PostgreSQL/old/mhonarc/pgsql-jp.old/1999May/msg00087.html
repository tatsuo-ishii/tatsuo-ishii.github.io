　こんにちは、渡辺 学といいます。

　PostgreSQL に、 Apache + PHP で処理を行うシステムを構築していま
す。このステムで、 postgres プロセスが終了せずに、どんどん増えてし
まうという現象がおきてしまい、原因が分からずに困っています。 
netstat -a で見てみますと、 CLOSE_WAIT で止まってしまっているよう
です。このまま放っておくと、最後にはバックエンドが落ちて core dump 
してしまいます。

　一体何が原因なのかも分からない状態なので、原因の調査方法について、
アドバイスをいただけないでしょうか。よろしくお願いします。


FreeBSD 2.2.8-RELEASE
PostgreSQL 6.4.2(99/4/14パッチ)
