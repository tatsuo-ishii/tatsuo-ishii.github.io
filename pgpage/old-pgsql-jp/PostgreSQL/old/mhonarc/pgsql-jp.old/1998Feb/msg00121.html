『Plagiaware 用 PostgreSQL パッケージ』

						1998-02-28
						桑村 潤(juk@rccm.co.jp)


  この配布には PostgreSQL-6.2.1p7mbPL2(石井達夫さんによるマルチバイト
拡張を EUC_JP オプションにて施してあります) Linux-ELF 版の実行環境が
含まれます。また、オリジナルのドキュメントの他に日本語のドキュメント
も含まれています。

  postgres バックエンドプログラムは、マルチバイト拡張により日本語EUC
によるデータの検索が行なえるとともに、テーブル名とカラム名に日本語が
使えるようになっています。
  psql 文字端末型クライアントプログラムは(EDITOR 環境変数指定)外部コ
マンドでの編集機能をもちます。それと併せて履歴参照やコマンド行編集を
行なえるように GNU readline を使うようリンクしてあります。従って、
GNU readline 共有ライブラリをインストールしておく必要があります。
このため、日本語パッチを当てた GNU readline パッケージ(readline21)を
用意しました。
  pgaccess は Constantin Teodorescu さんによって Tcl/Tk を使って開発
された GUIクライアントプログラムです。pgaccess では、Xウィンドウ上で
表形式のインターフェースからのデータ入力や更新が行なえます。石井達夫
さんによるパッチで、表形式のインターフェースを通して Kinput2 での入力
が可能となってます。PostgreSQL のインターフェースとして便利ですので、
パッケージ(pgaccess)として用意しました。
  アプリケーションの開発用に C, C++, Tcl/Tk のライブラリが含まれてい
ます。また最近、標準的なスクリプト言語として定着した PERL の需要は多
いと思われますので PERL5 のモジュール Pg もパッケージ(pgperl5)として
用意しました。
  PostgreSQL を利用する上で参考となるようなプログラムなどをソース・
ディレクトリから抜き出してパッケージ(pgexample)としてまとめました。

    PostgreSQL は SQL 準拠のサーバ／クライアント方式の ORDBMS
  (オブジェクト・リレーショナル・データベース・マネージメント・
  システム) です。このプログラムの元となった POSTGRES は、
  ストーンブレーカー教授のもとカリフォルニア大学バークレイ校で
  開発されました。詳しくは、配布に附属のドキュメントを御覧下さい。
    PostgreSQL に関する情報は http://PostgreSQL.org/ にて公開
  されています。

  付録として、インストールされた Plagiaware パッケージのリストをデータ
ベースに登録するプログラムなどを ./apps ディレクトリに収めさせて頂きま
した。なにかのお役にたてることがあれば幸いと思っております。


[注] Plagiaware は、こじまみつひろさんが中心となってまとめている日本語
Linux パッケージです。Slackware-3.[34] をベースにして、PJE, JG, JMAN,
JF 等を追加パッケージとしてではなく、パッケージ本体に取り込んでいます。
さらに、インストーラの日本語化がされています。Plagiaware については、
http://www.st.rim.or.jp/~isle/Plagia/ を参照下さい。
  なおこのパッケージは、こじまみつひろさんのご厚意により
ftp://ftp.jaist.ac.jp/pub/os/linux/Plagiaware/contrib/Database/ に収録
して頂いています。
