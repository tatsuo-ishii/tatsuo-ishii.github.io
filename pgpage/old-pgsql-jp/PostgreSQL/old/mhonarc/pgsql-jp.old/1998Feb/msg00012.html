文字コードの区別ってどーやってやっているんでしょうか？思いつく，案とし
て，

1) プロセスに一つのグローバルな環境変数によって決める．
   たとえば， locale のようなもの．
2) 実装に一つ，コンパイル時に，日本語版とか，中国語版とかを決める．
3) 各データベース毎に，文字集合を規定できる．
   たとえば，データベースを生成するときに，日本語の文字とか．
4) 各表ごとに，文字集合を規定できる．
5) 各列ごとに，文字集合を規定できる．
   create table foo(bar char(10) character set euc-jp);
