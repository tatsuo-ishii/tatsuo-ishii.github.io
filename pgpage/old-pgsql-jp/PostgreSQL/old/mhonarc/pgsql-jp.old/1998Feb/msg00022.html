View対象のテーブルを配列で持ってそれを使用してView化しようとしました。

create table table1 (
	item1   int default 0,
        sel	int[],
);

create view view1 as select * from table1 where table1.sel[1] = 0;

insert into table1 values(0,'{0,1}');
insert into table1 values(1,'{1,0}');

select * from view1
を実行するとデータが0けんになってしまいます。

create viewのwhereの項目を配列以外のものにすると問題なく動作します。

配列をViewの選択項目にするのは出来ないんでしょうか？
