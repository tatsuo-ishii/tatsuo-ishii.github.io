suda> PostgreSQLに大量のデータを登録したいのですが、CSV形式にしたも
suda> のを使用してDBにいれることは可能でしょうか。可能でしたら具体
suda> 的な方法を教えていただけないでしょうか。（大量データで、データ
suda> の追加、変更等がある場合、CSV形式が一番簡単かと思ってCSV形式と
suda> してあります）

私は COPY コマンドを使っています. 
delimiter を ',' に指定すればうまく行きます. 標準入力から入
力する場合は, TABLE 名が table であるとして, 

COPY table FROM STDIN USING DELIMITERS ',' 

という命令文が使えます. データが aaa.pg というファイルに入っ
ている場合は, シェルなどから

% cat aaa.pg | psql -c "COPY table FROM STDIN USING DELIMITERS ','" dbname

とやればうまくいきます. ここで dbname とは, データベース名で
あるとします. 

このMLでの過去の議論を見ていただくともう少し詳しい情報が得ら
れると思います.
