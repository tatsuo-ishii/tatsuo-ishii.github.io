佐久間＠パワーソフトです。

Alter Sequence Start ですが、
手元にいる、オラクル７ ではできません。
Drop して Create です。


DML で
	n := NextVal('seq');
	n := JumpValUpTo('seq', 2001);
	
		JumpValUpTo( a,b):	
		　NextVal(a) と　b のうち、小さくない方を
		　CurrVal(a) にセットし、戻り値とする。


みたいなことができたら、
もっと 便利だと思います。
