たかばです。

> ところで、x86/Linux では桑村さんがJDBCを動かしてらっしゃいましたが、
> LinuxPPC や FreeBSD など、他のプラットフォームではどうなんでしょう。
> 使っている方いらっしゃいますか？

クライアント、サーバ共に FreeBSD 2.2.5R で動かしています。
サーバには PostgreSQL 6.3.2 を 'MB=EUC_JP' でコンパイルしたものを、
JDBC には EUCJIS を扱えるようにパッチを当てたものを使っています。
	[pgsql-jp 3617] の jdbc-UTF8.patch を
	    % sed 's/UTF8/EUCJIS/g' jdbc-UTF8.patch > jdbc-EUCJIS.patch
	としてできた jdbc-EUCJIS.patch をあてた。
Java 環境は JDK 1.1.5 を使っています。
	参照: http://www.freebsd.org/java/

しっかり試してはいないので、「ちゃんと動くよ」とはいえないのですが...
[pgsql-jp 3617] の pg_jtest.java はちゃんと動きました。
