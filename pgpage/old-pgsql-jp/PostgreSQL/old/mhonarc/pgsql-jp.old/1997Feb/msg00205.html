あとは、適当にいじってみるのが一番かと思いますが...

# といっても、私の使ってる PostgreSQL 用の PHP 関数など、
# pg_Connect() -> pg_Exec() -> pg_Result() -> pg_FreeResult() -> pg_Close()
# の流れですべてです :-)

必要であろう設定は、

	1. PostgreSQL のユーザーに httpd プロセスの所有者(通常 nobody)
	   を加える。

	2. アップデートなど DB への変更をするなら、GRANT で nobody に
	   許可を与える。

	3. バックエンドサーバーと httpd のサーバーとのホストが異なる
	   時には pg_hba.conf をいじって、httpd のサーバーからもアクセス
	   できるようにする。

くらいではないでしょうか。
もっとも 1 さえ忘れなければ普通の環境なら大丈夫ですね ;-)
