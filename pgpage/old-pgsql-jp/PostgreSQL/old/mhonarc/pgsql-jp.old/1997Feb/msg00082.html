こんにちは。おみつ@Tramp です。

PHP/FI への日本語化パッチの第 3 版ができましたのでお知らせします。
# いまだに MMAP は使ってません m(.)m

== バグフィクス ==
* Mail 関数がデフォルトでは EUC のまま送ってしまう不具合の修正

== 新機能 ==
* 正規表現のマルチバイト拡張 (from gawk)
	石井さんの PostgreSQL への正規表現パッチをみていて、
	PHP/FI にも取り込もうとしたのが始まりでした :-)
	
* マルチバイト文字列用の関数 (MBstrlen と MBsubstr) の追加
	日本語の1文字を"2"ではなく"1"として、数えるものです。

# 詳しくは、配布パッケージ中の README.kanji.euc を御覧下さい。

== 動作確認 ==
*  Linux-1.2.13 + PHP/FI 2.0b9 + (apache 1.1.1 | apache 1.2b6)
	apache モジュールで動かす場合には、この配布パッケージに含まれる 
	libregex.a をリンクさせる必要があります。
