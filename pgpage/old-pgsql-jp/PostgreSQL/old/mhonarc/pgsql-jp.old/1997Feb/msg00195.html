よくわかりませんが, .pic は余計かもしれませんね.

で, .pic をはずしても恐らく上手くいかないと思います.
make install したら, 例えば, 私のところでは,

 /usr/local/PostgreSQL/lib

に libpq.so (私はこういう名前にしてます) がインストールされます.
ld.so がこれを見つけることができるようにするには,
環境変数 LD_LIBRARY_PATH に /usr/local/PostgreSQL/lib を設定するか(あらか
じめ何か設定してある場合は : で区切ってください), もしくは, root になって

 # /sbin/ldconfig /usr/lib /usr/local/lib /usr/local/PostgreSQL/lib

として下さい. /sbin/ldconfig コマンドの引数には, ld.so に見つけて欲しい
シェアードライブラリが入っているディレクトリを全て指定します.

ご自分の環境に合わせて行なってください.
