　データベース名を指定しない場合は、デフォルトのデータベース名
が指定されるみたいです。デフォルトをどこで指定するのかがわからない
ですが。私の所ではpostgres ですが、志田さんのところはpgsql の
ようで。バージョンの違いでしょうか。ログイン名かな？
　で、このデフォルトのデータベースがある場合は、おっしゃるように
問題ないです。

---- ここから ----
postgres@bash-2.02$ psql
Welcome to the POSTGRESQL interactive sql monitor:
  Please read the file COPYRIGHT for copyright terms of POSTGRESQL

   type \? for help on slash commands
   type \q to quit
   type \g or terminate with semicolon to execute query
 You are currently connected to the database: postgres  <- これ
---- ここまで ----

　試しにdestroydb してからやってみるとエラーが出ますが、
上記の"Is the postmaster... " とは違うものでした。

---- ここから ----
postgres@bash-2.02$ destroydb postgres
postgres@bash-2.02$ psql
Connection to database 'postgres' failed.
FATAL 1:  Database postgres does not exist in pg_database
postgres@bash-2.02$
---- ここまで ----
