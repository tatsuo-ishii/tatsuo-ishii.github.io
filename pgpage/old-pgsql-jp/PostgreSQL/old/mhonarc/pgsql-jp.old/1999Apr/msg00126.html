石井さんのおっしゃる通りです

host  all xxx.xxx.xxx.0 255.255.255.255 password password
と変更はしたのですが
host  all 127.0.0.1　   255.255.255.255 password password
は削除してました
＃確かにこれをとったらlocalhostはＮＧですね。

現在の内容は下記の通りです
local all                               password password
host  all 127.0.0.1     255.255.255.255 password password
host  all xxx.xxx.xxx.0 255.255.255.255 password password

この前の自己フォローは修正です
initdbしたときに作成された
local all                               trust
host  all 127.0.0.1     255.255.255.255 trust
を
local all                               password password
host  all 127.0.0.1     255.255.255.255 password password
に修正して動作するようになった
＃127.0.0.1->xxx.xxx.xxx.0と修正し忘れたのがよかったという
＃情けない結果の様です
