はじめまして、松田＠東海大沼津校舎と申します。

東海大では、セメスター制で学年がなくなり、さまざまな学科が入り乱れて科目
を取るので、いろいろな学生の成績を整理しなくてはならなくなり、PostgreSQL
を勉強することにしました。

コンピュータ歴は、２年で皆様には、はるかにおよびません。Windows95の流行
に乗り遅れ、FreeBSDから出発しました。現在、FreeBSD2.2.8-STABLE
 とFreeBSD3.1-RELEASEの上で、/usr/ports/japanese/postgresql を make
install したものと /usr/ports/databases/pgaccess をmake install したもの
を使っています。(2.2.6-PAOノートにものっています。)
pkg_info では、ja-postgresql6.4となっています。Tcl/Tk は、
ja-tcl(resp.tk)-8.0.4となっています。
pgaccessは、/usr/local/bin/pgaccess の一行目

#!  /usr/local/bin/wish8.0

をインストール後、mule で

#! /usr/local/bin/wish8.0jp

にしたら、使えるようになりました。これでいいのかしら？

シーラカンス本では、正規表現が使えるように書いてあったように思いました
が、(今、手元にない。間違ったらごめんなさい。)何故かうまくいきません。そ
のうち改めて質問したいと思います。

＃家を空けていたので自己紹介が遅れました。すみません。
