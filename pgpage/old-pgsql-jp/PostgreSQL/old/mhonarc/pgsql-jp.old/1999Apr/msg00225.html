石井です。表題にありますように、多少コマーシャルっぽい内容なので、その
手の情報に興味のない方は読み飛ばして下さい。

私事で恐縮ですが、4月から「オープンソースビジネス部」という部署に配属
になり、100% 仕事でも PostgreSQL をやっていいことになりました(今までも
仕事中にやってたじゃないか、という突っ込みはなし:-) もちろん SRA も営
利企業ですから、今回の配属の裏には PostgreSQL で利益を上げるように、と
いう意図がちゃんとあるわけです。PostgreSQL に関するコンサルティング、
トレーニング、開発などをお引き受けいたしますので、興味のある方は石井ま
でメール下さい。

表題の「commercial support」ですが、こちらの方は PostgreSQL の有償サポー
トを近々開始する予定で準備中です(主に対象は法人や企業になると思います)。
# メーリングリストがあるのに、お金を払ってまでサポートを受けたい個人ユー
# ザがいるとは思えないので。そんなことないよ、という方は是非御意見下さ
# い。

ところで、こう書くと今までボランティアベースでやってきたメーリングリス
トや、 Web ページでの活動まで有償になってしまうのではないかと懸念され
る方もいらっしゃるかもしれませんが、決してそんなことはありません。石井
としては今までやってきたことはすべて今後も＊まったく変わりなく＊続けて
行くつもりであり、会社ともその線で合意ができていることをお知らせしてお
きます。

むしろ、有償サポートや開発などで得たノウハウを積極的に PostgreSQL コミュ
ニティに還元して行くことにより、今まで以上に PostgreSQL に貢献できるチャ
ンスが増えると思っています。
# そのほか、ずっとやりたかった PostgreSQL の CGI/PHP が使える WWW サー
# バも上げられるかも知れないし、色々計画中ですので御期待下さい。

今後ともよろしくお願いします。
