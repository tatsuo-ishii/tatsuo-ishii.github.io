PostgreSQL ではなくて Oracle ベースでしたが、同じようなシステムを
この間作りました。3人で1ヶ月くらいの仕事でした。
# 某ダムの監視システムです。

そのときは、

RS232C -> VB App -> Oracle -> Oracle Web Application Server ->
Netscape -> Excel

というシステムでした。シーケンサからあがってくるデータをデータベースに
流し込み格納しておき、それをユーザーの条件に合わせて加工して（条件の
指定は WWW 経由で行います）、CSV ファイルとしてダウンロードし、
それを Excel に読ませてマクロで処理するというシステムでした。

すでに要件として Oracle というのがあったので Oracle で作りましたが、
PostgreSQL と PHP でも十分対応できる内容だったと思っています。
# 作った連中はみんなそう思っていた。
# Oracle 独自の機能は全然使っていないので、PostgreSQL でリプレース
  可能。

もし参考になれば幸いです。
