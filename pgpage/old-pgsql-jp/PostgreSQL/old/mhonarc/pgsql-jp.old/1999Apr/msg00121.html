JDBC URL がおかしいという指摘がありましたね。確かにそれもおかしいです
が、

> pg_hba.conf の内容
> local   all   password   password 

これもまずいですね。"local" というエントリは Unix domain のソケットを
使った接続にのみ有効です。JDBC URL に localhost と書くと、これではなくて、

host         all         127.0.0.1     255.255.255.255   trust

こっちをみます。

host         all         127.0.0.1     255.255.255.255   password password

とするとどうでしょう？

> psql -u hogehoge
> Username:hogehoge
> Password:XXXXXXXX
> で接続ＯＫ

psql の場合、-h を指定しないと Unix domain の方で接続するので、当然OK
になります。
