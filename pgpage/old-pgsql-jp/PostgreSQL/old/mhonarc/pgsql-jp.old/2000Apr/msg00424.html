＜試した手順＞
MS-DOSプロンプトを開く
    プロンプトから
        SET PG_CLIENTENCODING=SJIS (Enter)

        psql.exe (Enter)  として、psqlから


        漢字コードEUCのテーブルを参照
            select * from test ;
        とすると漢字が化けます。


また、psql.exeのプロンプトから
        SET CLIENT_ENCODING TO 'SJIS'

        としたら、

    NOTICE: Unrecognized variable client_encoding
    SET VARIABLE

    と怒られてしましました。
