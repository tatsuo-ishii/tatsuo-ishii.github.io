堀田＠諫早市です。

CREATE TABLE    japanese(
    maddr       text        -- メールアドレス
      PRIMARY   KEY,
    q1          int,        -- 1,2,3,4 ....
    q2          int,        -- 1,2,3,4 ...
    q3          int,        -- 1,2,3,4 ...
    q4          int         -- 1,2,3,4 ...
);

んな感じのテーブルを作りまして、q[1-4]の中身はアンケートの選択肢
が入っています。

これを、
q1==1：xx人
q1==2：xx人
...
q2==1：xx人
...
みたいな感じで集計したいんですが、どういった SQL 文を発行すれば
効率よく結果が取り出せるのでしょうか？

SELECT count(*) FROM japanese WHERE q1 = 1;

だと１つしか取ってこれないんですが、こいつを順列組み合わせで発行
するのも何だかダサいような気がしております。
