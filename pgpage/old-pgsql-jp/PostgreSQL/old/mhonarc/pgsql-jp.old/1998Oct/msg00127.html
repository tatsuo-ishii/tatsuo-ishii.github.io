　やりたいこと

　　認証を行わす

　試したこと
　　　(1)gdbmによる認証
　　　(2)mod_mod_auth_pgsqlによる認証
　　表題から想像つくと思いますがいずれもうまく使用でき
　ませんでした。

　(1)FreeBSDにインストールされたdbmanagerにて作成されたdb
　　ファイルはndbmで作成され、gdbmをリンクしたapacheでは
　　認証できない。　ライブラリからgdbmをはずせば可能だが
　　PHP/FIのリンクでgdbmを入れているのではずすと不安

　(2)mod_mod_auth_pgsqlは組み込めたようだが、perlインタ
　　フェースがインストールできない。
　　/usr/local/pgsql/postgresql-6.3.2/src/interfaces/perl5で
　　perl Makefile.PLを実行すると

エラーメッセージ------------------
syntax error in file Makefile.PL at line 9, next 2 tokens "use ExtUtils"

　　が出て先に進めない。

　　前振りが長くなりましたが、解決方法と疑問点を教えてい
　ただけませんでしょうか？

　(1)gdbmを入れずにPHP/FIは作成できましたが、何か不都合は
　　あるでしょうか？

　(2)gdbmを使用したFreeBSDで動くdbmanagerを作成された方は
　　いないでしょうか？
