あと、libpq にトレース機能があるので(PQtrace)、それを使うとフロントエ
ンド <--> バックエンドのやり取りをかいま見ることが出来ます。

たとえば、select * from pg_database を実行するこんなトレースが出て来ま
す。

To backend> Q
To backend> select * from pg_database
From backend> P
From backend> "blank"
From backend> T
From backend (#2)> 4
From backend> "datname"
From backend (#4)> 19
From backend (#2)> 32
From backend (#4)> -1
From backend> "datdba"
From backend (#4)> 23
From backend (#2)> 4
From backend (#4)> -1
From backend> "encoding"
From backend (#4)> 23
From backend (#2)> 4
From backend (#4)> -1
From backend> "datpath"
From backend (#4)> 25
From backend (#2)> 65535
From backend (#4)> -1
From backend> D
From backend (1)> p
From backend (#4)> 13
From backend (9)> template1
From backend (#4)> 8
From backend (4)> 1739
From backend (#4)> 5
