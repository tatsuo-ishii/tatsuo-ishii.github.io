　こちらのほうは、たぶんおかしくないです。
　自分のところで問題なくmake できました。
　おそらく/usr/include のものを指しているのではなく、
/usr/local/lib/g++-include/ の下に有るstring (string.h では
なく) を指していると思います。でも実体は

-- string ここから --
// -*- C++ -*- forwarding header.
// This file is part of the GNU ANSI C++ Library.

#ifndef __STRING__
#include <std/string.h>
#endif
-- ここまで --
