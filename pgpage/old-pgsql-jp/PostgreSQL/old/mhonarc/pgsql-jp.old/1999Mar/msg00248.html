98年の8月からメーリングリストに入って初めての投稿です。
が、最後かもしれません。

ということで、もう人目に触れることがかくなるかもしれなので、
PostgreSQL + PHP3 で私が作った物を公開することにしました。
だいぶ前に作っていた（改良の余地は山ほどあるがめんどくさい）
のですがソースが汚いので恥ずかしい。公開するとセキュリティ上
問題がある、という理由からずっところがしていた物を公開します。
ただし3月15日には消えて無くなります。

http://lax.ddcom.co.jp/postgres.phtml

１.　郵便番号検索
	はじめはやっぱりこれでした。
	どこにでも転がってますね。

２.　Web Editor
	ブラウザから、データベースをいじれる（テーブルの作成、削除等）
	ブラウザから、テキストファイルの編集が出来る（ファイルのパスや
	コメント情報をデータベースで管理）EUC, JIS対応

３.　Web Mail
	PHP3でブラウザからメールを送信。（Mail関数だけで出来るんですけど...）

４.　commerce
	PostgreSQLで商品情報、顧客情報の管理をするPHP3でブラウザから
	商品の編集可。

１と２についてはソースもアップしておきました。興味があれば
持っていって下さい。

一応簡単にセキュリティはかけたのですが、確認するのがめんどうだったので
たぶん、穴がいっぱいあると思います。が、イタズラしないで下さいね。
