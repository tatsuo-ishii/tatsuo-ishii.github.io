内田です。

Linux 上の Apache + PHP + PostgreSQL が、 FileTable Full で落ちてしまう
件に関して、LinuxWorld Expo 会場で :-) 石井さんに教えていただいた方法で
なんとかなったようです。

postmaster 起動時に bash の ulimit -n 20 を行っても、まだ足りないという
状況だったのですが、OS 起動時に以下のようなファイルをつくって起動するこ
とで対応しました。


/etc/rc.d/rc3.d/S00kernel :

     #!/bin/sh
     echo 2048 > /proc/sys/kernel/file-max

# Default では cat /proc/sys/kernel/file-max すると、 1024 でした。
