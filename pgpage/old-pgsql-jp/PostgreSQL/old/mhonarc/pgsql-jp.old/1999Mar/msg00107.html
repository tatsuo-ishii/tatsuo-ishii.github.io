シーラカンス本の例題の場合、基本的に swing 1.1 ですが、ちょこっと 
import 文を替えれば 1.0.3 もOKです。

>Java2の場合、swingは必然的に1.1です。

ご存知とは思いますが、PostgreSQL の JDBC ドライバはまだ Java2 の JDBC 
環境ではコンパイルできませんのでご注意下さい。ただ、JDK 1.1 でコンパイ
ルした JDBC ドライバを持ってきて Java2 で使うことは可能です。

どうしても Java2 でコンパイルしたい人は、PostgreSQL JDBC ドライバ開発
者の Peter T Mount さんのページ(http://www.retep.org.uk だったかな？)
に行けば、開発中の JDBC2 対応バージョンがあると思います。
