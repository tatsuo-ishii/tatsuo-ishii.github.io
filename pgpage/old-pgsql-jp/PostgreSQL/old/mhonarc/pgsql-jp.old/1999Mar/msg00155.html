内田です。

お見せするほど特別なことをしてるわけではありません。squid の RunCache と
同じです。


大まかにはこんなもんです。

#!/usr/local/bin/bash

ulimit -n 20
while : ; do
    su postgres -c 'exec postmaster -d 1 -i -o -F >> /usr/local/pgsql/postmaster .log 2>&1'
    ipcs -m | mail -s "Postmaster DOWN" mailto@hoge.co.jp
    sleep 10
done

この Bash の script を RunPostgres とかいう名前にしておいて、
/etc/rc.d/rc3.d/SXXpostgres の中から RunPostgres & で読んでいます。
