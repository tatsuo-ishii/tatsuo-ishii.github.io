　"トランザクション処理"という言葉では混乱しそうですので、
次に改めます。

--
  ecpgでは、vacuum(exec sql vacuum;)も含めてInsertなどの全て
の更新命令は、end;(exec sql end transaction;) もしくは commit;
(exec sql commit;)が無ければ「無条件」にrollbackされます。
