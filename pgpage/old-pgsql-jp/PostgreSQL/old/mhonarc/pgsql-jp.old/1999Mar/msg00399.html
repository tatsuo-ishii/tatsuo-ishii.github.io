configure は正常に終了していますか？ 

make の Error はともかく、configure の途中の error は案外見落とすもの
で、こうなると正常に Makefile が生成されていないため、この Error の事
例と同じ状況で Make できなくなったと思います。

make distcelan で完全にきれいにしてから、configure の再実行し、そのと
きの出力をファイルに落として再確認してみてはいかがでしょうか。
