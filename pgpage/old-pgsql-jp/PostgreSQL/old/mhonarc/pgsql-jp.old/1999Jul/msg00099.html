堀田＠諫早市です。

標記のようなシステムを構築中です。完成したら全国の宴会好きな方々
に使っていただきたいと思っています。が、相変わらず一人では技術力
不足によりできません(^^;。

で、早速SQLレベルの質問で恐縮です。以下のようなテーブルがあります
（一部省略）。

enkai=> \d xcomment
+-------------+-------------------+-------+
| フィールド  | タイプ            | Length|
+-------------+-------------------+-------+
| enkaiid     | text not null     |   var |
| userid      | text not null     |   var |
| comment     | text              |   var |
+-------------+-------------------+-------+

enkai=> \d xuser
+-------------+-------------------+-------+
| フィールド  |タイプ             | Length|
+-------------+-------------------+-------+
| userid      | text not null     |   var |
| name        | text              |   var |
+-------------+-------------------+-------+

各々以下のようなデータが入っています。

enkai=> select * from xcomment;
enkaiid|userid|comment                 
-------+------+------------------------
luky6  |sim   |これはテストです        
(1 row)

enkai=> select * from xuser;
userid|name
------+------
sim   |堀田１
sim2  |堀田２    
(2 rows)

で、コメントの一覧を表示しようと

enkai=> SELECT comment, xuser.name AS username, FROM xuser, xcomment
WHERE enkaiid = 'luky6';

とやりますと、

comment         |username
----------------+----------
これはテストです|堀田１
これはテストです|堀田２
(2 rows)

となります。本当は

comment         |username
----------------+----------
これはテストです|堀田１

と出て欲しいのですが、どうすればよろしいでしょうか？
