Minami> psqlはreadlineが組みこまれるので入力が楽。
Minami> ただしその場合は日本語の入力をちょくで行えないのがちょっと難。

最近の readline なら、~/.inputrc に

  set convert-meta off
  set meta-flag on
  set output-meta on

と書いておくと、一応は日本語を扱えるようになります。

ただし、カーソル移動や文字削除にやや難があるので、

Hanahara> readlineの日本語化パッチをあててmakeすると、ちゃんと日本語使
Hanahara> えるようになって便利ですよ。

のほうが使い勝手がいいです。

Hanahara> 	readline-2.2.1.jpatch.*
Hanahara> 	readline-4.0.jpatch.*
Hanahara> とかで検索すると引っかかると思います。

  ftp://ftp.ipc.chiba-u.ac.jp/pub/pub.yamaga/readline/
