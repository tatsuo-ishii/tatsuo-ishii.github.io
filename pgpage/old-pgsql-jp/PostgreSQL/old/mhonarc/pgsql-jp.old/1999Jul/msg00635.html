以下のように書いたらどうですか。
「csh」又は「tcsh」の場合、コマンドラインから「env」と
入力してやるとその時に定義されているshellの
環境変数が画面に表示されます。
---------------------------------------
setenv PATH "$PATH":/usr/local/pgsql/bin
setenv POSTGRES_HOME /usr/local/pgsql
setenv PGLIB "$POSTGRES_HOME"/lib
setenv PGDATA "$POSTGRES_HOME"/data
if ($?MANPATH) then
    setenv MANPATH "$MANPATH":"$POSTGRES_HOME"/man
else
    setenv MANPATH /usr/man:/usr/local/man:/usr/local/apache/man:
              /usr/share/man:/usr/X11R6/man:"$POSTGRES_HOME"/man
              （本当は改行、空白を入れずに１行で書きます）
endif
if ($?LD_LIBRARY_PATH) then
    setenv LD_LIBRARY_PATH "$LD_LIBRARY_PATH":"$PGLIB"
else
    setenv LD_LIBRARY_PATH  "$PGLIB"
endif
---------------------------------------
この辺はUnixの基本的な使い方に関する部分なので
お近くの詳しい方に尋ねるか、UnixやFreeBSD等の
入門書を読まれることをおすすめします。
