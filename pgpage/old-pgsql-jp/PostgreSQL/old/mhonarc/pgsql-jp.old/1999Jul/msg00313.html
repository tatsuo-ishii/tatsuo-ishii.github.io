はじめまして。
ALife Gardenという人工生命のゲームサイトのプログラマを
しておりますtesh Nakamuraと申します。

ALife GardenではPostgreSQLを6.3ぐらいからずっと使っています。
が、vacuumが大変で。。。:-)
ボスが、「そろそろPostgreSQLはあきらめて、Oracleぐらいに
変えろ」という指令が出ているのですが、6.5にちょっと期待
しているのですが。
自己紹介代わりに、サイトの宣伝をさせていただきます。
もしよろしければ、ぜひ御覧ください。
http://alifegarden.com

さて、早速質問なんですが、Debianを使っている方で、
PostgreSQL6.5をインストールされた方はいらっしゃいますか？
私のDebian環境では、Geometry関連の型(box等)&関数で必ず落ちて
しまうという現象が起きております。
ALife Gardenでは当たり判定などにどうしてもGeometryが必要に
なってくるので、何とかしたいのですが。

Slackにtar.gzからインストールしたものはうまくいったのですが。。。

環境は以下の通りです。

OS: Debian GNU/Linux 2.2 (potato) Kernel 2.2.10

PGSQL関係パッケージ: dpkg -l|grep postgres
ii  postgresql      6.5-2.1        Object-relational SQL database, descended fr
ii  postgresql-cont 6.5-2.1        Additional facilities for PostgreSQL
ii  postgresql-dev  6.5-2.1        Header files for libpq (postgresql library)
ii  postgresql-doc  6.5-2.1        Documentation for the PostgreSQL database.
ii  postgresql-pl   6.5-2.1        A procedural language for PostgreSQL

libc: libc6.2.1.1

以上、自己紹介のつもりが、宣伝＆質問になってしまいました。
これからもどうぞよろしくお願いします。
