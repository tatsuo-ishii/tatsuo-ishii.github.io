　6.4.3で試しましたが、
 mem_id = $1
で大丈夫のようですね。
ちなみに7.0では、
create function traf_nextvalue( char(4) ) return int2 as
' select traf_index from traffic where mem_id = $1 '
language 'sql';
という設定が可能のようですね。検索も予期した結果を返してくれるようです。
