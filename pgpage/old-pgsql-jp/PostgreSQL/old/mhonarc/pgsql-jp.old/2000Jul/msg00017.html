北村です。
初めまして。

>select date_part('hour',tmptime) from tmptable;
>を実行すると
>ERROR:  No such function 'time_timespan' with the specified attributes
>が返されます．
>
>どのようにSQL文を作成したら、時間(hour)を取り出すことが
>できるのでしょうか？

date_partの引数は、(text,datetime)か、(text,timespan)なので、time型は
使えないみたいですね。
time型を使ってやるとなると、
 select date_part('hour', datetime('today'::datetime, tmptime) )
  from tmptable;
と、datetime関数を使って、一旦datetime型にしてやれば、'hour'が使えます。
datetime関数については、マニュアル見てください。
