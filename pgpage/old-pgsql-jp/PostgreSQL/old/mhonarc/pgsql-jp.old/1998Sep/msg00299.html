小菅です。私も似たようなもんです。

At 19:52 +0900 98.9.29, yasuo imai wrote:
> 草々ですが、POSTGRESQL6.3.2のINSTALLをしていますが、特に問題なくINSTALLが
> 終了するのですが、postmasterを起動しようとすると下記のようなエラーメッセー
> ジが出てしまいます。
> 
> postmaster does not find the database system. Expected to find it in the PGDATA directory "/usr/local/pgsql/data", but unable to open file with pathname "/usr/local/pgsql/data/base/template1/pg_class".
> No data directory -- can't proceed.
> 
> 又、インストール時に/usr/local/pgsql下にdataディレクトリが作成されません
> でした。

　おそらくこれは、initdb をやってないだけだと思います。
　私の環境では、次のようなコマンドを入力しました。

$ initdb --pgdata=/usr/local/pgsql/data --pglib=/usr/local/pgsql/lib --username=postgres

　ただし、これをやる前に、パスが通ってなければならないので、
それを含めた環境変数も設定する必要があると思います。
