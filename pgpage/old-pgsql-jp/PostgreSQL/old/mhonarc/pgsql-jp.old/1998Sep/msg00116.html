石井です。

9/1 に出るはずだった 6.4 のβはバグのためにスケジュールが遅れています
が、昨日 Vadim さんの fix により兇悪なバグが潰れてだいぶ見通しが明るく
なって来ました。そこで待ち切れずに Wisconsin benchmark をやってみたの
ですが、6.3.2 に比べるとかなり速くなっていることが分かりました。

6.3.2: query  9:   0.711 real   0.620 user   0.070 sys

6.4:   query  9:   0.407 real   0.340 user   0.040 sys

これは期待できそうです:-)

#ちなみに、プラットフォームは MkLinux pre-DR3(Mac 7600/120+G3 カード)
#です。
