API Document(JavaDoc)は、postgresql.jarを作成するMakefileを使って作ります

%make doc

実行すると、HTMLファイルを30ぐらい作りますので、それをブラウザで見て下さい。
ちなみに、getTupleCount()はpostgresql.ResultSet.htmlにでています。


   postgresql.ResultSet rs = (postgresql.ResultSet)st.executeQuery(*****);
   System.out.println("getTupleCount:" + rs.getTupleCount());

とすると、確認できると思います。
