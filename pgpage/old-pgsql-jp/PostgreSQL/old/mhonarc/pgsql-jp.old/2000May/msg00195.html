> 参照が簡単になるってことですよね？！（私の理解は、あってますか？）

違いますね。;-)
type はオブジェクトのタイプですから、shun と shinamono が両方とも
表であれば、仮に表を指す type が 1 である場合は両方とも 1 になります。

石井さんが書いたのは、oid2 には、shun と shinamono の表オブジェクトと
しての oid を入れておくというものです。
そうすれば、select 文を作る時に、その oid から表名が参照できますから。

仮に、表オブジェクトを指す type が 1 で、shun と shinamono の oid が
それぞれ、43200 と 43201 だった場合、

   oid|type|oid2
 -----+----+-----
 38519|1   |43200
 38520|1   |43200
 38521|1   |43200
 38522|1   |43200
 38523|1   |43200
 38506|1   |43201
 38507|1   |43201
 38508|1   |43201

と、なるという事でしょう。
