Pg.pm という事は perl を使おうとしているのでしょうか。
私は perl を使わないのでよく知りませんが、DBI と DBD というものが
あるそうです。DBD が DBMS と実際にやりとりを行なうパッケージで、
DBI が perl Script と DBD の間に入り、DBD の切り替えなどを受け持つ
そうです。

で、DBI の方に Proxy 機能があって、どこかに DBI ProxyServer があると、
そこを中継場所にして DB に接続できるらしいです。

これがどう嬉しいかというと、色々な局面はあるでしょうが、ひとつには
DB サーバとクライアントマシンを分けた場合に、Oracle などだと、動作
プラットフォームの制約や契約条件による問題が発生する点への解決策と
して使う事が考えられますね。
PostgreSQL などでも、いちいち make しなければならないなどの手間が
減るので、嬉しい場合があるでしょうね。

DBD には PostgreSQL に対応したものもあります。今だと DBD-Pg-0.93.tar.gz
が最新なのでしょうか？ > 詳しい人
