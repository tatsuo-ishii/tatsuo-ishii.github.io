> PostgreSQLで、現在接続しているユーザ名を知る手段はあるのでしょうか？
> 
> Oracleでは
> 
> SELECT USERNAME FROM USER_USERS
> 
> で取得できます。
> 
> システムカタログをみてみたのですが、同様のものはないようでした。
> 
> 誰かご存じのかたがおられましたら、よろしくお願いします。

psql の中で

=> select USER;

もしくは、

=> select CURRENT_USER;

でみることができます。
PostgreSQLでは上の二つは同じ働き(現在接続しているユーザを返す)をします。
