石井です。

以下のように、PostgreSQL 6.3.1 が正式リリースされました。
ac domain の方は
ftp://ftp.jaist.ac.jp/pub/dbms/PostgreSQL/postgresql-6.3.1.tar.gz
を、co domain の方は
ftp://ftp.sra.co.jp/pub/cmd/postgres/6.3.1/postgresql-6.3.1.tar.gz
を入手されるのが速いかと思います。

また、6.3 からの差分パッチもpostgresql-6.3-6.3.1.gzという名前で同じ場
所にあるので、ご利用下さい(約200KB)。このパッチを当てるときは、
postgresql-6.3 に cd して patch -p1 で当てたほうが良いようです(もしく
は -d 使用)。

changes にある
	LIKE fix for multiple adjacent underscores
は like.patch、
	big-endian fix for backend communications(Kataoka)
は片岡さんの pqcomprim.patch、
	multi-byte character set support, see doc/README.mb(Tatsuo)
は 6.3mbPL2.patch と同じ物です。よって、今回から mb patch は
いらなくなりました(Makeifle.custom に MB=EUC_JP をセットするなどの
使い方は変わっていません)。

Morning...

	Slightly later then we were hoping, we have built a post-release
patch/distribution of PostgreSQL 6.3, labeled as 6.3.1

	Listed below are the many fixes that went into this, based on
problems reported within the first couple of weeks of v6.3 being released,
some of which we felt warranted a very quick "second" release.

	The patch against v6.3's distribution is found at:

	ftp://ftp.postgresql.org/pub/postgresql-6.3-6.3.1.gz

	And the full distribution is available as:

	ftp://ftp.postgresql.org/pub/postgresql-6.3.1.tar.gz

	It is recommended that you upgrade to this latest release at your
earliest convience.

	The WWW site will be updated shortly.

Changes
- -------
ecpg cleanup/fixes, now version 1.1(Michael Meskes)
pg_user cleanup(Bruce)
large object fix for pg_dump and tclsh(alvin@camberlo.demon.co.uk)
LIKE fix for multiple adjacent underscores
LIKE/BETWEEN fix for having function call as target(Thomas)
fix for redefining builtin functions(Thomas)
ultrix4 cleanup
upgrade to pg_access 0.83
updated CLUSTER manual page
multi-byte character set support, see doc/README.mb(Tatsuo)
configure --with-pgport fix
pg_ident fix
big-endian fix for backend communications(Kataoka)
SUBSTR() and substring() fix(Jan)
several jdbc fixes(Peter)
libpgtcl improvements, see libptcl/README(Randy Kunkee)
Fix for "Datasize = 0" error(Vadim)
Prevent \do from wrapping(Bruce)
Remove duplicate Russian character set entries
Sunos4 cleanup
Allow optional TABLE keyword in LOCK and SELECT INTO(Thomas)
CREATE SEQUENCE options to allow a negative integer(Thomas)
Add "PASSWORD" as an allowed column identifier(Thomas)
Add checks for UNION target fields(Bruce)
Fix Alpha port(Dwayne Bailey)
Fix for text arrays containing quotes(Doug Gibson)
Solaris compile fix(Albert Chin-A-Young)
Better identify tcl and tk libs and includes(Bruce)
