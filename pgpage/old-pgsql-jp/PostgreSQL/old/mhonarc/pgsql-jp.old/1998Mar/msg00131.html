平岡です。

私もFreeBSD 2.2.5Rに以前インストールしました。
とりあえずPostgreSQL 6.2.1ですが、参考になれば。
# Webに公開しようとメモをとったのですが、まだｈｔｍｌおに直していません
ので、そのまま書きます。

-----
pgsqlというユーザが必要ですので作成しておきます。
vipwを起動して、最後の行に以下を追加します。
pgsql:*:70:70::0:0:Postgres Maintainer:/usr/local/pgsql:/bin/sh

ports-currentから、postgresqlを取って来て、/usr/ports/pgsql以下に展開します。
/usr/ports/pgsqlに移動します。

ソースを展開してpatchを当てます。
# make patch

patch.mbというディレクトリを作成して移動します。
ftp://ftp.sra.co.jp/pub/cmd/postgres/6.2.1/patches/以下のファイルを全部取って来
ます。

PATCH.mbの最初の行を以下のように修正します。
PATCHDIR=/usr/ports/databases/postgresql/patch.mb

/usr/ports/databases/postgresql/workに移動します。
cd ../work

../patch.mb/PATCH.mbと実行します。

/usr/ports/databases/postgresqlに移動します。
cd ..

インストールします。
make install
-----

これで日本語も通ると思います。
