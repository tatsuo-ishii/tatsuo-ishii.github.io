MS-Access95 Ver7.00aからExecuteメソッド（DATABASE、Querydefオブジェクト）
を使ってinsert、deleteをやっていますが問題無くうまくいっています。
Executeメソッド実行する時ははどのオブジェクトを使用しましたか？

参考までに以下にDATABASEオブジェクトを使用したサンプルです。

 Dim dbs As DATABASE
 
 'データベースオブジェクト取得
 Set dbs = CurrentDb
 dbs.Connect = "ODBC;DSN=PostgreSQL;UID=sasaki;SERVER=smatsv1.smat.co.jp;PORT=5432;DATABASE=testdb"

 dbs.Execute "DELETE FROM aaa;", dbSQLPassThrough

 dbs.Close
