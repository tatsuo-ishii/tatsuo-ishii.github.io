　石井さん、小俣さん、お答えありがとうございます。

>UPDATE r SET b = a * int4(0.9);

　試してみたのですが、これは

>これだと、int4(0.9) が 1 になってしまいますよね。

　ということのようで、b = a になってしまいました。

>UPDATE r SET b = int4(a * float8(0.9));
>
>ではどうでしょうか。


　これだと、

jav=> update itemdata set sprice = int4(price * float8(0.9));
NOTICE:there is no operator * for types int4 and float8
NOTICE:You will either have to retype this query using an
NOTICE:explicit cast, or you will have to define the operator
WARN:* for int4 and float8 using CREATE OPERATOR

(price が a で sprice が b と考えて下さい)

　と出てしまいました。
　そこで以下のようにしたところうまくいきました。

jav=> update itemdata set sprice = int4(float8(price) * 0.9);
UPDATE 4292

　これで一応解決しました。どうもありがとうございました。

　それにしてもこうやってキャストするというのは SQL では一般的なのでしょうか
