->プリプロセスの #ifdef DEBUG などを、
->make の引数で define できたような気がしたのですが、
->どうするんでしたっけ？

 gmake clean して、関係するすべての *.o ファイルを消します。
Makefile の CFLAGS に

 -DDEBUG

を付けたします。そして、 gmake し直します。すると、DEBUG が
有効になっています。ただし、

hogehoge.o: hogehoge.c hogehoge.h
	$(CC) -c hogehoge.c $(CFLAGS)

と、CFLAGS がすべての OBJ 作成に指定されていなければなりませんが。
