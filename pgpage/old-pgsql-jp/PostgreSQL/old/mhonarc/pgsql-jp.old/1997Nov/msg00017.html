なんとか、make の implicit rules (暗黙のルール?) を使えないかなあと思って、
    % make -pf /dev/null | grep CFLAGS
	make: *** No targets.  Stop.
	LINK.c = $(CC) $(CFLAGS) $(CPPFLAGS) $(LDFLAGS) $(TARGET_ARCH)
	COMPILE.c = $(CC) $(CFLAGS) $(CPPFLAGS) $(TARGET_ARCH) -c
をしたら、CPPFLAGS があるのを見つけました。
# 最近 info コマンドが便利だと知りました。
# よく見たら、これにも書いてありました。

PostgreSQL では、普通 Makefile.global を include するようなので、
この場合は、
    % make COPT=-DDEBUG
で良さそう、ということがわかりました。
