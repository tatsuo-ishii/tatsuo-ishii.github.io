石井です。

>桑村です。
>
>  質問です。
>
>  DATE 型に NULL を入れる方法はありますか？
>
>	=> create table test1 ( a date );
>	=> insert into test1 values ( '' );
>	WARN:Bad date external representation

insert into test1 values(NULL);

>  MONEY 型に大きな値を入れると変になります。
>
>	=> create table test2 ( b money );
>	=> insert into test2 values ('325,000,000');
>	=> select  * from test2;
>	b               
>	----------------
>	($18,597,383.68)

MONEY 型の制約でしょう。現在 32bit integer で実装されているので、桁溢
れを起こしていると思われます。

$325,000,000 を表現するためには、セントの部分を含めて10進で 11 桁の整
数が必要ですが、これは 32bit で表現できません。
