|* The query language POSTQUEL has been replaced with SQL (implemented
|in the server). We do not support subqueries (which can be imitated
|with user defined SQL functions) at the moment. Aggregates have been
|re-implemented. We also added support for GROUP BY. The libpq
|interface is still available for C programs.
|
|問い合わせ言語、POSTQUEL は(サーバで実行される)SQLに取って代わられました。
|私達は(ユーザ定義のSQL関数で同様な事ができますから)副問い合わせを
|サポートしません。アグレゲートは再び実装されました。私達はまた、GROUP BY
|のサポートを加えました。libpg インターフェースは C プログラムのために
|現在でも利用できます。
|
|コメント：
|    Aggregates ：集合？ データベースの専門用語だと思う。

Aggregate というのは、多分、SUM とか AVG などによる集計のことを指して
いると思います。「集計機能(関数)」という訳が良いかもしれません。
