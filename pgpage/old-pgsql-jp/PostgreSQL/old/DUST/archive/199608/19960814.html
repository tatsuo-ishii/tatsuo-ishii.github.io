PostODBCのインストール手順を簡単に説明します。

「条件」
postgres95 が稼動しているホスト名を gohan.osaka.ish.cae.ntt.jp
アクセスしたいデータベース名を foo
postgres95のTCPポート番号をデフォルトの 5432
データベースにアクセス可能なユーザ名を kanemasu とします。

1.アーカイブを展開する。

2.展開して出てきた setup.exe を起動する。

3.Install Drivers というタイトルバーのダイアログがでてきますので
 リストの中の Postgres95 を選択します。

4.ファイルのコピーが始まります。

5.Data Sources というタイトルバーのダイアログが出てきますので
 リストの中の PostODBCTest を選択する。

6.Postgres95 Driver Setup というタイトルバーのダイアログが出て
 きますので

Data Source Name : PostODBCTest
     Description : PostODBCのテスト 
          Server : gohan.osaka.ish.cae.ntt.jp
        Database : foo
            Port : 5432
       User Name : kanemasu

のように設定して、OK ボタンをクリックしインストールが終了します。

なお、設定内容の変更や別のData Source Name を追加したい場合は、
コントロールパネルの「32ビットODBC」(Win95の場合)アイコンを
クリックすれば、上記の項番5から始められます。

 >
 >> とりあえず Access95 からpostgres95のテーブルにアクセスして
 >> みました。(ExcelからはMS-Quely経由で...)
 >
 >MS-Quelyというのは標準の Office で使えますか？ 具体的な使い方はどうす
 >るのでしょうか。

MS-Query(この綴りが正解でした)は、Officeの中に含まれているはずですので、
使えるはずです。ただOffice のインストールのしかたによっては、
MS-Queryがインストールされない場合もありますので、もしなければ
再度セットアップで不足分を追加する必要があるかもしれません。

最近、Excel Ver.7 に変えたもんで 以前のバージョンに当てはまるか
どうかわかりませんが簡単に使い方を説明します。
（ちょっと言葉で説明するのが難しので詳しくはマニュアルか市販本を見て
ください）

1.Excel Ver.7 には 「データ」メニューに「外部データの取り出し」という
 項目があります。

2.それを選択するとMS-Queryが起動されます。

3.データソースの選択ダイアログが出てきますので、PostODBCTestを選択
 します。

4.テーブルの追加というダイアログがでてきますので、目的のテーブル
 を追加ボタンを使って選びます。

5.選んだテーブルが画面に張り付きますので、取り出したい カラムをドラッグ
 して、MS-Query上のスプレッドシート上にドロップします。
 (この時点でテーブル内のデータも表示されます。)

6.「ファイル」メニューの「Microsoft Excelにデータを返す」を選択します。

7.Excel に戻ると、「外部データ取り出し」というダイアログがでていますの
 で、取り出し先のワークシートの範囲を指定します。

以上で抽出完了です。

1.2)    What does Postgres95 run on?

        The authors have compiled and tested Postgres95 on the following
        platforms(some of these compiles require gcc 2.7.0):

        - DEC Alpha AXP on OSF/1 2.0

        - i386 Solaris
        - SUN SPARC on Solaris 2.4
        - SUN SPARC on SunOS 4.1.3
        - DEC MIPS on Ultrix 4.4
        - Intel x86 on Linux 1.2 and Linux ELF
        - OSs derived from 4.4-lite BSD (NetBSD, FreeBSD)
        - IBM on AIX 3.2.5
        - BSD/OS 2.0, 2.01 & 2.1
        - SGI MIPS on IRIX 5.3

        The following ports are bundled with the Postgres95 distribution. The
        authors do not have handy access to these platforms but the
        ports have been tested by the others.

        - Motorola MC68K or Intel x86 on NeXTSTEP 3.2
        - Intel x86 on Intel SVR4
