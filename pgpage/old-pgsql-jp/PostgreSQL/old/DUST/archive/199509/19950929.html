近藤＠筑波大と申します。

FreeBSD 2.0.5R で日本語化パッチを当てた postgres95 を作ってみま
した。
	・Makefile.global の PORTNAME を BSD44_derived に修正
	・Makefile.global の SRCDIR をソースのある位置を正しくさ
	　すようにする
	・Makefile.global の USE_READLINE を true にする
	・Makefile.global の READLINE_INCDIR を /usr/includeに、
          READLINE_LIBDIR を /usr/libに修正
	・bin/psql/Makefile の LD_ADD から `-lhistory' を削除
	・mk/postgres.mk の AROPT の書換え
でコンパイルできました。
