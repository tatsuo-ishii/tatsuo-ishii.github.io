名前

pgpool_adm_pcp_pool_status — pgpool.confのパラメータを取得する関数

概要

pcp_pool_status returns record(host,  
 port,  
 username,  
 password,  
 item text,  
 value text,  
 description text); 
text host;
integer port;
text username;
text password;
out item text;
out value text;
out description text;
 
pcp_pool_status returns record(pcp_server,  
 item text,  
 value text,  
 description text); 
text pcp_server;
out item text;
out value text;
out description text;
 

Description

pcp_pool_statusはpgpool.confのパラメータを取得します。

引数

pcp_server

pcpサーバの外部サーバ名

他の引数

pcp_common_optionsを参照ください。

使用例

使用例です。 

 test=# SELECT * FROM pcp_pool_status('localhost',11001,'t-ishii','t-ishii') WHERE item ~ 'backend.*0';

 backend_hostname0       | /tmp                                           | backend #0 hostname

 backend_weight0         | 0.500000                                       | weight of backend #0
 backend_data_directory0 | /home/t-ishii/work/pgpool-II/current/aaa/data0 | data directory for backend #0
 backend_status0         | 2                                              | status of backend #0
 backend_flag0           | ALLOW_TO_FAILOVER                              | backend #0 flag
