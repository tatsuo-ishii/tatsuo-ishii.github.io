名前

pgpool_adm_pcp_node_count — ノード数を取得する関数

概要

pcp_node_count returns integer(host,  
 port,  
 username,  
 password); 
text host;
integer port;
text username;
text password;
 
pcp_node_count returns integer(pcp_server); 
text pcp_server;
 

Description

pcp_node_countはDBノードの数を取得します。

引数

pcp_server

pcpサーバの外部サーバ名

他の引数

pcp_common_optionsを参照ください。

使用例

使用例です。 

test=# SELECT * FROM pcp_node_count('localhost',11001,'t-ishii','t-ishii');
 node_count 
------------
          2
(1 row)
