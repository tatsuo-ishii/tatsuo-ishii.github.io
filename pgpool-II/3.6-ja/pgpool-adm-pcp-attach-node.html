名前

pgpool_adm_pcp_attach_node — 指定ノードをアタッチする関数

概要

pcp_attach_node returns record(node_id,  
 host,  
 port,  
 username,  
 password,  
 node_attached boolean); 
integer node_id;
text host;
integer port;
text username;
text password;
out node_attached boolean;
 
pcp_attach_node returns record(node_id,  
 pcp_server,  
 node_attached boolean); 
integer node_id;
text pcp_server;
out node_attached boolean;
 

説明

pcp_attach_nodeは、ノードをPgpool-IIにアタッチします。

引数

node_id

ノードをアタッチするバックエンドのインデックスです。

pcp_server

pcpサーバの外部サーバ名

他の引数

pcp_common_optionsを参照ください。

使用例

使用例です。 

test=# SELECT * FROM pcp_attach_node(1,'localhost',11001,'t-ishii','t-ishii');
 node_attached 
---------------
 t
(1 row)
