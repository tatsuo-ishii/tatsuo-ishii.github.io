7.1. Resource Usage

Pgpool-II does not consume too much resource. However there are minimum requirements for resource. In this section we are going to explain one by one.

7.1.1. Memory Usage

There are two types of memory usage in Pgpool-II: shared memory and process memory. The former is allocated at the startup of Pgpool-II main server process and will not be freeed until whole Pgpool-II servers shut down. The latter is allocated within each Pgpool-II process and will be freed at the end of the process.
