4.4. Configuring backend information

For Pgpool-II to recognize PostgreSQL backend servers, you need to configure backend* in pgpool.conf. For starters, at least backend_hostname and backend_port paramters are required to be set up to start Pgpool-II server.

4.4.1. Backend Settings

Backend PostgreSQL used by Pgpool-II must be specified in pgpool.conf. See Section 5.4, “Backend Settings”
