3.8. ドキュメントのコンパイルとインストール

3.8.1. ツールセット

Pgpool-IIのドキュメントはSGML(より正確には、SGMLを使って実装された言語であるDocBook)で書かれています。 ユーザが読めるHTMLドキュメントを生成するには、docbookツールを使ってコンパイルする必要があります。 RHEL類似のシステムでツールをインストールするには、以下を実行します。 

	  yum install docbook-dtds docbook-style-dsssl docbook-style-xsl libxslt openjade

3.8.2. ドキュメントのコンパイル

ツールセットがインストールされたら、以下のようにコンパイルします。 

	  $ cd doc
	  $ make
	  $ cd ..
	  $ cd doc.ja
	  $ make

HTMLドキュメントはdoc/src/sgml/htmlに、オンラインマニュアルはsgml/man[1-8]にあります。 日本語のドキュメントはdoc.ja/src/sgml/htmlに、オンラインマニュアルはsgml/man[1-8]にあります。
